    дёҖгҖҒеүҚиЁҖпјҡIIM жҳҜд»Җйәјпјҹ

    зү№зҷјжҖ§зӮҺжҖ§иӮҢз—…пјҲIdiopathic Inflammatory Myopathies, IIMпјүпјҢдҝ—зЁұ myositisпјҢжҳҜдёҖзҫӨд»Ҙж…ўжҖ§иҝ‘з«ҜиӮҢз„ЎеҠӣгҖҒиӮҢй…¶дёҠеҚҮгҖҒиӮҢиӮүзӮҺз—ҮжөёжҪӨзӮәе…ұеҗҢиЎЁзҸҫзҡ„е…Ёиә«жҖ§иҮӘй«”е…Қз–«з–ҫз—…гҖӮдё»иҰҒдәһеһӢеҢ…жӢ¬пјҡDermatomyositisпјҲDMпјүгҖҒPolymyositisпјҲPMпјүгҖҒInclusion Body MyositisпјҲIBMпјүгҖҒImmune-Mediated Necrotizing MyopathyпјҲIMNMпјүгҖҒAnti-synthetase syndromeпјҢд»ҘеҸҠ Clinically Amyopathic DMпјҲCADMпјүгҖӮ

    Stage 3пҪңжү“з ҙе…Қз–«иҖҗеҸ—пјҢз”ўз”ҹиҮӘй«”жҠ—й«”

    1. зө„з№”жҗҚеӮ·йҮӢж”ҫ DAMPsпјҲеҰӮ HMGB1пјү
    2. DAMPs жҙ»еҢ– TLR-3гҖҒTLR-7гҖҒTLR-9 вҶ’ innate immunity е•ҹеӢ•
    3. Plasmacytoid DCs еӨ§йҮҸз”ўз”ҹ Type I IFNпјҲIFNОұ/ОІпјү
    4. Type I IFN вҶ’ дёҠиӘҝ MHC class IгҖҒдҝғйҖІ APC жҙ»еҢ–
    5. APCs е‘ҲзҸҫиҮӘй«”жҠ—еҺҹзөҰ CD4+ T cells
    6. CD4+ T cells жҙ»еҢ– вҶ’ еҚ”еҠ© B cells вҶ’ з”ўз”ҹ MSAs

    MSAs йҖҡеёёеҸӘжңүдёҖзЁ®пјҲmutually exclusiveпјүпјҢжңүжҷӮеңЁиҮЁеәҠз—ҮзӢҖеҮәзҸҫеүҚе°ұе·ІеӯҳеңЁгҖӮ

    Stage 4пҪңдёүжўқе№іиЎҢзҡ„ж•ҲжҮүж©ҹеҲ¶

    и·Ҝеҫ‘ AпјҡиЎҖз®Ўи·Ҝеҫ‘пјҲDM зӮәдё»пјү

    иЈңй«”жҙ»еҢ– вҶ’ MACпјҲC5b-9пјүжІүз©Қж–јеҫ®иЎҖз®Ўе…§зҡ® вҶ’ еҫ®иЎҖз®ЎеЈһжӯ»гҖҒcapillary loss вҶ’ зјәиЎҖпјҲischemiaпјү+ зө„з№”дҪҺж°§пјҲhypoxiaпјү вҶ’ HIF-1Оұ/2Оұ дёҠиӘҝгҖҒд»Је„ҹжҖ§иЎҖз®Ўж–°з”ҹдҪҶдёҚи¶і вҶ’ Perifascicular atrophyпјҲи·қиЎҖз®ЎжңҖйҒ зҡ„ perifascicular еҚҖеҹҹжңҖзјәж°§пјү

    жөёжҪӨзҙ°иғһд»Ҙ CD4+ T cellsгҖҒmacrophagesгҖҒB cells зӮәдё»пјҢдҪҚж–ј perivascular/perimysialгҖӮ

    и·Ҝеҫ‘ BпјҡCTL и·Ҝеҫ‘пјҲPMгҖҒIBM зӮәдё»пјү

    CD8+ CTL иҫЁиӯҳиӮҢзә–з¶ӯиЎЁйқў MHC class I е‘ҲзҸҫзҡ„иҮӘй«”жҠ—еҺҹ вҶ’ Perforin-granzyme B зӣҙжҺҘж®әеӮ·йқһеЈһжӯ»иӮҢзә–з¶ӯ вҶ’ Fas-FasL pathway иӘҳе°ҺеҮӢдәЎ

    PM е’Ң IBM жңү CD8+ T cells зҡ„ clonal proliferationпјҲжҠ—еҺҹй©…еӢ•пјүпјҢжөёжҪӨзҙ°иғһдҪҚж–ј endomysialпјҢзӣҙжҺҘдҫөе…ҘйқһеЈһжӯ»иӮҢзә–з¶ӯгҖӮ

    и·Ҝеҫ‘ CпјҡжҠ—й«”зӣҙжҺҘжҗҚеӮ·пјҲIMNMпјү

    Anti-SRP / Anti-HMGCR жҠ—й«” вҶ’ MACпјҲC5b-9пјүжІүз©Қж–јиӮҢзә–з¶ӯ sarcolemmaпјҲйқһиЎҖз®ЎпјүвҶ’ иӮҢзә–з¶ӯеЈһжӯ»пјҲе№ҫд№Һз„Ўж·Ӣе·ҙзҗғжөёжҪӨпјү

    Anti-HMGCR зҡ„зү№ж®Ҡж©ҹиҪүпјҡStatins дёҠиӘҝеҶҚз”ҹиӮҢиӮүдёӯзҡ„ HMGCR иЎЁзҸҫ вҶ’ жҲҗзӮәиҮӘй«”е…Қз–«йқ¶жҠ—еҺҹ вҶ’ еҚідҪҝеҒңи—ҘеҫҢд»ҚжҢҒзәҢгҖӮ

    Stage 5пҪңMHC Class I йҒҺеәҰиЎЁзҸҫиҲҮ ER StressпјҲйқһе…Қз–«и·Ҝеҫ‘пјү

    жӯЈеёёйӘЁйӘјиӮҢдёҚиЎЁзҸҫ MHC class IпјҢдҪҶ IIM жӮЈиҖ…зҡ„иӮҢзә–з¶ӯе»Јжіӣдё”ж—©жңҹеҮәзҸҫ MHC class I йҒҺеәҰиЎЁзҸҫпјҢз”ҡиҮіеҮәзҸҫеңЁйҒ йӣўзӮҺз—ҮжөёжҪӨзҡ„еҚҖеҹҹгҖӮ

    зӮҺз—Үзҙ°иғһеӣ еӯҗпјҲTNFгҖҒIFNОіпјү+ HMGB1 вҶ’ иӘҳе°Һ MHC class I еӨ§йҮҸиЎЁзҸҫ вҶ’ MHC class I еҲҶеӯҗеңЁ ER е…§ж»Ҝз•ҷе Ҷз©Қ вҶ’ ER stress вҶ’ NFОәB жҙ»еҢ– вҶ’ йҖІдёҖжӯҘдёҠиӘҝ MHC IгҖҒICAM-1гҖҒMCP-1 вҶ’ AutophagyпјҲTRAILпјүгҖҒApoptosisпјҲTWEAKпјүгҖҒNecroptosis вҶ’ иӮҢзә–з¶ӯжҗҚеӮ·пјҲеҚідҪҝжІ’жңү CTLпјү

    йҖҷжўқи·Ҝеҫ‘и§ЈйҮӢдәҶзӮәдҪ•зӮҺз—Үж¶ҲйҖҖеҫҢиӮҢз„ЎеҠӣд»ҚжҢҒзәҢпјҢд№ҹиӘӘжҳҺзӮәдҪ•е–®зҙ”е…Қз–«жҠ‘еҲ¶жІ»зҷӮеҫҖеҫҖдёҚи¶іеӨ гҖӮ

    Stage 6пҪңзҙ°иғһжҝҖзҙ ж”ҫеӨ§иҝҙи·Ҝ

    дё»иҰҒдҝғзӮҺзҙ°иғһжҝҖзҙ пјҡIL-1Оұ/ОІгҖҒTNFгҖҒIFNОұгҖҒIL-6гҖҒIL-15гҖҒIL-18

    • TNF вҶ’ зӣҙжҺҘиӘҳе°ҺиӮҢзҙ°иғһжӯ»дәЎ + жҠ‘еҲ¶ myogenic miRNAsпјҲmiR-1гҖҒmiR-133a/bгҖҒmiR-206пјүвҶ’ жҠ‘еҲ¶иӮҢиӮүеҶҚз”ҹ
    • NFОәB иӘҝжҺ§зҡ„ miRNAsпјҲmiR-146a/bгҖҒmiR-31гҖҒmiR-223пјүвҶ’ йҷҚдҪҺ dystrophin иӣӢзҷҪ вҶ’ еҠ йҮҚиӮҢиӮүжҗҚеӮ·
    • HMGB1 вҶ’ дёҠиӘҝ MHC I + йҷҚдҪҺ sarcoplasmic reticulum зҡ„ CaВІвҒә йҮӢж”ҫ вҶ’ иӮҢиӮүз–ІеӢһ
    • д»Ји¬қжҗҚеӮ·пјҡAMPD1 зјәд№Ҹ вҶ’ ATP/phosphocreatine дёӢйҷҚ вҶ’ иӮҢиӮүз„ЎеҠӣиҲҮз–ІеӢһ

    еӣӣгҖҒеҗ„ MSA зҡ„иҮҙз—…ж©ҹиҪү

    жҠ—й«” йқ¶жҠ—еҺҹеҠҹиғҪ дё»иҰҒи·Ҝеҫ‘ й—ңйҚөж©ҹеҲ¶
    Anti-synthetaseпјҲJo-1зӯүпјү Aminoacyl-tRNA synthetaseпјҲиӣӢзҷҪиіӘеҗҲжҲҗпјү иЎҖз®Ў + иӮә иӮәйғЁзӮәиө·е§Ӣең°пјӣperimysial fragmentationпјӣperifascicular MHC II дёҠиӘҝ
    Anti-Mi-2 ж ёе…§ helicaseпјҲchromatin remodelingпјү иЎҖз®ЎпјҲDMпјү UV вҶ’ complement вҶ’ MAC вҶ’ perifascicular atrophyпјӣType I IFN
    Anti-MDA5 Innate immune receptorпјҲеҒөжё¬ dsRNAпјү Type I IFN innate pathway з—…жҜ’жЁЈи§ёзҷјпјӣType I IFN surgeпјӣrapid progressive ILDпјӣDAD
    Anti-TIF1-Оі Transcriptional regulatorпјҲи…«зҳӨжҠ‘еҲ¶пјү Paraneoplastic и…«зҳӨжҠ—еҺҹиҲҮеҶҚз”ҹиӮҢиӮүдәӨеҸүеҸҚжҮүпјӣfeed-forward loop
    Anti-NXP2 Nuclear matrix proteinпјҲиҪүйҢ„иӘҝжҺ§пјү Paraneoplastic + зјәиЎҖ ж…ўжҖ§зӮҺз—Ү вҶ’ calcinosisпјӣйҲЈиЁҠиҷҹиӘҝжҺ§еӨұеёё
    Anti-SRP Signal recognition particleпјҲиӣӢзҷҪиіӘиҪүдҪҚиҮіERпјү жҠ—й«”зӣҙжҺҘжҗҚеӮ· MAC жІүз©ҚеңЁ sarcolemmaпјӣER stressпјӣз„Ў CTL
    Anti-HMGCR HMG-CoA reductaseпјҲиҶҪеӣәйҶҮеҗҲжҲҗпјү жҠ—й«”зӣҙжҺҘжҗҚеӮ· Statin дёҠиӘҝеҶҚз”ҹиӮҢиӮүдёӯйқ¶жҠ—еҺҹпјӣеҚідҪҝеҒңи—Ҙд»ҚжҢҒзәҢпјӣMAC жІүз©Қ
    Anti-cN1A Cytosolic 5вҖІ-nucleotidaseпјҲеҳҢе‘Өд»Ји¬қпјү CTL + йҖҖеҢ– CD8 endomysial ж”»ж“Ҡпјӣamyloid-ОІ е Ҷз©Қпјӣautophagy зјәйҷ·пјҲIBMпјү

    дә”гҖҒиӮҢиӮүеӨ–иЎЁзҸҫзҡ„иҮҙз—…ж©ҹиҪү

    Heliotrope Rash & Gottron's Papules/Sign

    DM зҡ„иЈңй«”/иЎҖз®Ўи·Ҝеҫ‘дёҚеҸӘеӮ·е®іиӮҢиӮүпјҢеҗҢжЁЈдҪңз”Ёж–јзҡ®иҶҡеҫ®иЎҖз®Ўпјҡ

    MAC жІүз©Қж–јзҡ®иҶҡеҫ®иЎҖз®Ў вҶ’ иЎҖз®ЎйҖҡйҖҸжҖ§еўһеҠ гҖҒиЎҖз®Ўж“ҙејө вҶ’ зҙ…ж–‘пјҲerythemaпјү+ ж°ҙи…«пјҲedemaпјү

    Heliotrope rashпјҡзңјзһјзҡ®дёӢзө„з№”з–Ҹй¬ҶпјҢж°ҙи…«зү№еҲҘжҳҺйЎҜпјӣзҙ«зҙ…иүІ = иЎҖз®Ўж“ҙејөпјҲзҙ…пјү+ иј•еәҰзјәж°§пјҲзҙ«пјүгҖӮ

    Gottron's papules/signпјҡжҢҮй—ңзҜҖдјёеҒҙеҸҚиҰҶж©ҹжў°ж‘©ж“Ұ вҶ’ еҫ®еүөеӮ· вҶ’ DAMPs ж”ҫеӨ§ type I IFN зӮҺз—Ү вҶ’ PapulesпјҲиЎЁзҡ®еўһз”ҹ + зңҹзҡ®зӮҺз—ҮжөёжҪӨпјүvs SignпјҲе–®зҙ”иЎҖз®Ўж“ҙејөжҖ§зҙ…ж–‘пјүгҖӮ

    Mechanic's Hands

    T cell ж”»ж“Ҡи§’иіӘзҙ°иғһзҡ„ tRNA synthetase вҶ’ ж…ўжҖ§зӮҺз—Ү вҶ’ и§’иіӘеұӨйҒҺеәҰеўһз”ҹпјҲhyperkeratosisпјүвҶ’ жүӢжҢҮж©ҲеҒҙ / жҢҮи…№зІ—зіҷгҖҒи„«еұ‘гҖҒзҡёиЈӮ

    Mechanic's hands з„Ўе…ёеһӢиЎҖз®ЎзӮҺжҖ§зҙ…ж–‘пјҢеҪўж…ӢеҸҚжҳ зҡ„жҳҜ T cell ж…ўжҖ§и§’еҢ–жҖ§зӮҺз—ҮпјҲиҖҢйқһ DM зҡ„иЈңй«”/иЎҖз®Ўи·Ҝеҫ‘пјүгҖӮ
