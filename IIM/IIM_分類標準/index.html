    IIM 分類標準完整整理

    特發性發炎性肌病變（Idiopathic Inflammatory Myopathies, IIM）的分類標準演進與各次型診斷準則

    一、演進時間軸總覽

    年代 標準 用途 / 重點
    1975 Bohan & Peter 最早，分 5 群；以肌無力+肌酶+EMG+切片+皮疹為核心
    1995 Griggs / ENMC IBM 初版 首度為 IBM 立病理導向標準
    2004 ENMC Workshop（Hoogendijk） 引入 MSA 概念，提出 IMNM 雛形
    2011/2013 188th ENMC — IBM 現行 IBM 分類（臨床+病理三級）
    2017 EULAR/ACR 第一個 data-driven、計分式總分類（含次群演算法）
    2016/2018 224th ENMC — IMNM 以血清抗體 3 分群（SRP / HMGCR / 陰性）
    2018/2020 239th ENMC — DM 以 MSA 將 DM 分 6 亞型；強制驗抗體
    2024/2025 273rd ENMC + 2025 ACR/EULAR（CLASS）— ASyS 抗合成酶症候群專屬分類，計分式

    核心觀念轉變：從「靠肌肉切片＋肌酶」→「以肌炎特異抗體（MSA）定義臨床-血清-病理一致的同質次群」。現代 IIM 六大臨床表型：DM、CADM、ASS（抗合成酶症候群）、IMNM、IBM、Overlap myositis（PM 已萎縮成「seronegative IIM」）。

    二、Bohan & Peter（1975）

    5 項標準：

    1. 對稱性近端肌無力
    2. 肌肉切片符合肌炎（壞死、再生、發炎浸潤）
    3. 血清肌酶上升（CK、aldolase、AST、ALT、LDH）
    4. 肌電圖肌病變變化（小且多相之運動單位、纖維顫動、正銳波、高頻重複放電）
    5. DM 典型皮疹（heliotrope、Gottron）

    判定：

    • Definite PM：4 項（無皮疹）
    • Probable PM：3 項（無皮疹）
    • Possible PM：2 項（無皮疹）
    • Definite DM：皮疹 + 3 項
    • Probable DM：皮疹 + 2 項
    • Possible DM：皮疹 + 1 項

    限制：無法區分 IBM、IMNM；易把 mimics（dystrophy、代謝性肌病）誤分為 PM。

    三、2017 EULAR/ACR 分類標準

    橫斷面資料建立，分有/無肌肉切片兩模型。先計分達門檻 → 再走演算法分次群。

    計分表

    變項 無切片 有切片
    發病年齡 ≥18 且 <40 歲 1.3 1.5
    發病年齡 ≥40 歲 2.1 2.2
    客觀對稱無力—近端上肢 0.7 0.7
    客觀對稱無力—近端下肢 0.8 0.5
    頸屈肌弱於頸伸肌 1.9 1.6
    腿近端肌弱於遠端肌 0.9 1.2
    Heliotrope rash 3.1 3.2
    Gottron's papules 2.1 2.7
    Gottron's sign 3.3 3.7
    吞嚥困難/食道蠕動異常 0.7 0.6
    Anti-Jo-1 陽性 3.9 3.8
    CK/LDH/AST/ALT 任一上升 1.3 1.4
    肌內膜單核細胞包圍未侵入肌纖維 — 1.7
    肌束膜/血管周圍浸潤 — 1.2
    Perifascicular atrophy（束周萎縮） — 1.9
    Rimmed vacuoles（鑲邊空泡） — 3.1

    判定門檻

    無切片 有切片
    Probable IIM（機率 ≥55%、<90%） ≥5.5、<7.5 ≥6.7、<8.7
    Definite IIM（機率 ≥90%） ≥7.5 ≥8.7

    次群演算法（達門檻後）

    • 有皮膚表現（heliotrope / Gottron papules / Gottron sign）→ DM
    • 發病 <18 歲 → JDM（幼年型）
    • 有皮疹但無客觀肌無力 → Amyopathic DM
    • 無皮膚表現 →
    • 符合 IBM 臨床（指屈肌或膝伸肌無力）+ rimmed vacuoles → IBM
    • 其餘 → PM（涵蓋 IMNM）

    限制：僅含 anti-Jo-1，未納其他 MSA；IMNM、ASS 被併入 PM。

    四、ENMC 系列分類

    188th ENMC — IBM（2011 會議 / 2013 發表，Rose）

    臨床/實驗室特徵

    • 病程 > 12 個月
    • 發病年齡 > 45 歲
    • 膝伸展無力 ≥ 髖屈曲無力，和/或 指屈曲無力 > 肩外展無力
    • 血清 CK ≤ 15× 正常上限

    病理特徵

    • 肌內膜發炎浸潤
    • Rimmed vacuoles
    • 蛋白質堆積（amyloid / p62 / TDP-43）或 15–18 nm 管狀絲狀體
    • MHC class I 上調

    三級分類

    • Clinico-pathologically defined IBM：臨床符合 + 三項標誌性病理（肌內膜發炎 + rimmed vacuoles + 蛋白堆積/絲狀體）全有
    • Clinically defined IBM：臨床符合（兩條件皆具）+ 至少一項病理（但未達全部）
    • Probable IBM：臨床部分符合（兩條件擇一）+ 至少一項病理

    224th ENMC — IMNM（2016 會議 / 2018 發表，Allenbach）

    特徵：肌纖維壞死為主、淋巴球浸潤稀少；MHC-I 上調、肌膜 C5b-9 補體沉積（與 DM/ASS 微血管型沉積不同）。

    血清三分群

    • Anti-SRP IMNM
    • Anti-HMGCR IMNM（再分 statin 暴露 vs statin-naïve；statin-naïve 者較年輕、CK 高、治療反應差）
    • Seronegative IMNM（約 1/3；需切片支持並排除 mimics，注意 paraneoplastic）

    診斷組合：近端對稱肌無力（常急性/亞急性）＋ CK 顯著升高 ＋（anti-SRP 或 anti-HMGCR 陽性）和/或 壞死性肌病切片。

    239th ENMC — DM（2018 會議 / 2020 發表，Mammen/Allenbach/Stenzel/Benveniste）

    核心精神：強制檢測 5 種 DM 特異抗體；典型皮疹 + MSA 可不靠切片即診斷 DM。

    六大亞型

    1. Anti-Mi-2 DM（典型皮疹、預後較好）
    2. Anti-TIF1γ DM（成人腫瘤相關、皮膚重）
    3. Anti-NXP2 DM（成人腫瘤、鈣化；幼年型鈣化明顯）
    4. Anti-MDA5 DM（CADM、RP-ILD 快速進行性間質肺病、血管病變、CK 常正常）
    5. Anti-SAE DM（典型皮疹、吞嚥困難）
    6. Autoantibody-negative DM（抗體陰性）

    臨床型態類別：definite DM、probable DM、amyopathic DM、DM sine dermatitis（無皮疹型 DM）。

    重要排除：帶 anti-synthetase、anti-HMGCR、anti-SRP 抗體者 不歸類為 DM。

    病理：perifascicular atrophy、血管病變、MxA 過度表現（DM 較專一指標）。

    273rd ENMC — ASyS（2024）

    抗合成酶症候群專屬 workshop，與下節 2025 ACR/EULAR（CLASS）標準平行，整合臨床（關節炎、肌炎、ILD、mechanic's hands、Raynaud、發燒）與抗 ARS 抗體。

    五、Anti-synthetase Syndrome — 2025 ACR/EULAR（CLASS）分類標準

    即 CLASS 計畫（EULAR/ACR 資助，分析 948 ASSD vs 1077 對照），2025/01 正式發表。分 Definite / Probable / Possible 三級。

    計分原則：同一域（clinical / serological）只取最高權重；未出現/未評估/未檢 = 0。

    分級切點（cutoff）

    分級 總分 臨床要求 抗 ARS 抗體
    Definite ≥ 5.5 至少一項 clinical domain 必須陽性（mandatory）
    Probable ≥ 5.0 至少一項 clinical domain 非必須（可陰性/未測）
    Possible ≥ 4.0 至少一項 clinical domain 或 other clinical features 非必須（可陰性/未測）

    （Definite 表現：驗證組 sens/spec 86.8% / 99.2%；金標準組 94.3% / 99.7%）

    加權計分（同一域只取最高權重；未評估/未檢 = 0）

    臨床域

    項目 權重
    發炎性關節痛或關節炎 1.0
    亞臨床肌炎（肌酶/EMG/MRI） 1.0
    臨床明確肌炎 1.5
    切片證實肌炎 2.0
    ILD：UIP 或其他/無法分類型 1.5
    ILD：以 NSIP 和/或 OP 為主 2.5

    其他臨床特徵

    項目 權重
    Mechanic's hands / Hiker's feet 1.5
    Raynaud's phenomenon 0.5
    發炎性皮疹（Gottron、heliotrope、V-sign、shawl sign、malar） 0.5
    不明原因發燒 0.5

    血清域

    項目 權重
    Anti-ARS（非 Jo-1，以非 IP 法測得） 3.0
    Jo-1（任何方法）或 非 Jo-1 以 IP 法測得 3.5
    ANA cytoplasmic pattern 1.5
    Anti-Ro52 / anti-SSA 1.0
