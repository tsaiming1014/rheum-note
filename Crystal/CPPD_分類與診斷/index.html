    CPPD 分類與診斷（Kelly Ch98 + 2023 ACR/EULAR）

    核心觀念：分類標準（2023 ACR/EULAR）是研究收案用；臨床診斷黃金標準仍是滑液找到 CPP 結晶。Ryan & McCarty（1960s）為舊版診斷標準。

    一、2023 ACR/EULAR CPPD 分類標準

    套用順序（階梯式）

    1. Entry：曾有 ≥1 次關節疼痛/腫脹/壓痛（周邊關節；CDS 則為軸性 C1/C2）
    2. 絕對排除：所有症狀更可能由 RA、gout、PsA、OA 等解釋 → 停止
    3. 充分條件（任一即確診，免評分）：CDS 或 症狀關節滑液 CPP 結晶（任何量；組織病理證實亦可）
    4. 都沒有 → 評分，總分 >56 即分類為 CPPD

    評分表（同 domain 取最高級；需至少一個症狀周邊關節影像）

    Domain 分級 分數
    A 發病年齡 ≤60 / >60 歲 0 / 4
    B 發炎性關節炎病程 無 / 持續性 / 1 次典型急性 / >1 次典型急性 0 / 9 / 12 / 16
    C 典型發作部位 第一蹠趾 / 無 / 其他周邊 / 腕 / 膝 −6 / 0 / 5 / 8 / 9
    D 相關代謝疾病 無 / 有 0 / 6
    E 滑液結晶分析 2 次陰性 / 1 次陰性 / 未做 −7 / −1 / 0
    F 手/腕 OA 影像（K/L ≥2） 無 / 雙側橈腕 OA / ≥2 項特定型態 0 / 2 / 7
    G 症狀關節 CPPD 影像 US/CT/DECT 皆無 / CR 陰性 / 任一陽性 −4 / 0 / 16
    H 有 CPPD 影像的周邊關節數（不論症狀） 0 / 1 / 2–3 / ≥4 0 / 16 / 23 / 25

    權重洞察：影像（G 16、H 最高 25）與反覆急性發作（B 16）權重最大；第一蹠趾 −6 是把 gout 推開的鑑別設計。

    效能：驗證世代整體 sensitivity 99.2%、specificity 92.5%；約 7 成確定病例直接符合充分條件而免評分，評分只用於抽不到結晶又無 CDS 的少數人。

    二、重要定義（Table 2 腳註——高參考價值）

    Crowned Dens Syndrome（CDS，充分條件之一）

    CPP 結晶沉積於齒突（dens）周圍韌帶，造成軸性發炎。臨床 + 影像兩者皆須具備：

    臨床特徵：

    • 上頸部急性/亞急性劇痛
    • 發炎指標上升、頸部旋轉受限、常合併發燒
    • 須排除 PMR、腦膜炎 等 mimicker

    影像特徵：

    • 傳統 CT：retro-odontoid（寰椎橫韌帶）鈣化沉積，典型為線狀、密度低於皮質骨，軸位常呈兩條平行線
    • 寰樞關節、alar ligament、齒突尖旁 pannus 鈣化亦為特徵
    • DECT：dual-energy index 介於 0.016–0.036

    相關代謝疾病（Domain D，+6 分）

    記憶串「鐵、鈣、鎂、磷、家族」：

    • 遺傳性血鐵沉積症（hereditary hemochromatosis）
    • 原發性副甲狀腺亢進（primary hyperparathyroidism）
    • 低血鎂（hypomagnesemia）
    • Gitelman 症候群
    • 低磷酸酶症（hypophosphatasia）
    • CPPD 家族史

    其他關鍵定義

    • 持續性發炎關節炎：≥1 關節持續腫脹伴疼痛和/或發熱
    • 典型發作（typical episode）：急性發作或急性惡化的關節痛，伴腫脹/發熱，不論治療與否都會緩解
    • 滑液分析：須由受過 compensated 偏振光顯微鏡訓練者執行
    • 影像 CPPD 定義：指纖維軟骨或透明軟骨鈣化；不可計入滑膜、關節囊、肌腱鈣化；僅算周邊關節

    三、Ryan & McCarty 診斷標準（舊版，Kelly 詳述）

    確定度 條件
    Definite 滑液 CPP 結晶 ＋ X 光 chondrocalcinosis（兩者兼具）
    Probable 上述其中一項
    Possible 不尋常關節 OA 或其他特徵

    被淘汰原因：CR 敏感度低、滑液偽陰性率高、進階影像（US/DECT）當年不存在。

    四、CPP 結晶鏡檢特徵

    • 菱形/棒狀（rhomboid/rod-shaped）
    • 正雙折射（positively birefringent） ← 與 gout 負雙折射相反
    • 比 MSU 較暗、較小、較稀疏；最多 20% 不顯雙折射
    • ⚠️ 即使資深醫師辨識準確率僅約 39%——鏡檢是「不完美的黃金標準」

    五、各影像模式的 CPPD 表現（Kelly Ch98）

    模式 典型表現 效能（以滑液結晶為參考標準）
    X 光（CR） chondrocalcinosis——細點狀/線狀軟骨鈣化；另可見肌腱鈣化、軟骨流失型態、大的軟骨下囊腫 敏感度僅 0.59、特異度 0.95、PPV 0.83、NPV 0.76 → 特異高但敏感度差
    超音波（US） 高回音沉積（hyperechoic deposits），大小形狀不一，位於纖維軟骨/透明軟骨內部、肌腱或滑液中；動態檢查時留在軟骨/肌腱內或在滑液中漂動 敏感度 0.85、特異度 0.91、PPV 0.86、NPV 0.90 → 比 X 光好（OMERACT 定義）
    CT / DECT 偵測 CPPD 敏感度高；DECT 是否優於一般 CT 仍有爭議，但 DECT 能藉電子密度與有效原子序（Z）區分 BCP 與 CPP 高敏感度（研究數少）

    ⚠️ 與 gout 超音波鑑別（重要考點）：

    • CPPD：沉積在軟骨「內部」（intra-substance，軟骨中間一條線）
    • Gout：double-contour sign——尿酸鋪在軟骨「表面」（透明軟骨上緣多一條平行強回音線）

    DECT 顏色的迷思

    • DECT 的顏色是後處理軟體指定的顯示色，不是物質本色。痛風套件慣例把 MSU 標綠、鈣標紫；但 CPPD 專用套件（如 Siemens syngo.via CPPD package）會把 CPP 標成綠色。
    • 真正能區分 MSU / CPP / BCP 的是電子密度與有效原子序（Z）：MSU 為有機、低 Z；CPP、BCP 含鈣、高 Z。用痛風演算法跑時 CPP 因 Z 不同不會被判成尿酸。
    • ⚠️ 陷阱：痛風 DECT 中血管鈣化、軟骨鈣化、指甲/皮膚假影可能被誤標為「尿酸綠」造成假陽性 → 須配合臨床判讀。
