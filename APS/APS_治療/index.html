    • CAPS вҶ’ йЎһеӣәйҶҮ + heparin + жҸӣиЎҖ/IVIG

    гҖҮгҖҒaPL profile йўЁйҡӘеҲҶеұӨ (2019 EULAR)

    • Medium-high titreпјҡaCL > 40 GPL/MPLпјҲжҲ– > 99th percentileпјүпјӣanti-ОІ2GPI > 99th percentile
    • High-risk profileпјҲй–“йҡ” вүҘ12 йҖұгҖҒвүҘ2 ж¬ЎпјүпјҡLA йҷҪжҖ§гҖҒdouble жҲ– triple йҷҪжҖ§гҖҒжҲ–жҢҒзәҢй«ҳж•Ҳеғ№
    • Low-risk profileпјҡе–®зҚЁ aCL жҲ– anti-ОІ2GPIгҖҒlow-medium titreпјҢе°Өе…¶жҡ«жҷӮжҖ§йҷҪжҖ§
    • жІ»зҷӮеј·еәҰдҫқ aPL profile + дәӢд»¶йЎһеһӢпјҲvenous/arterialпјү+ еҮәиЎҖ/еҫ©зҷјйўЁйҡӘ жұәе®ҡ

    дёҖгҖҒManagement вҖ” жІ»зҷӮ

    1. Macrovascular Thrombosis

    (1) Primary prevention

    • з„Ўз—ҮзӢҖ aPL йҷҪжҖ§иҖ…дёҚе»әиӯ°еёёиҰҸжҠ—еҮқгҖӮ

    • Low-dose aspirin (LDA)пјҡе”ҜдёҖ RCT йЎҜзӨәе°Қз„Ўз—ҮзӢҖиҖ…й җйҳІйҰ–ж¬ЎиЎҖж “иӯүж“ҡдёҚеј·пјӣдҪҶ (2019 EULAR) д»Қе»әиӯ°д»ҘдёӢж—ҸзҫӨз”Ё LDA 75вҖ“100 mg/dayпјҡ

    • з„Ўз—ҮзӢҖ aPL carrier + high-risk profile

    • SLE + high-risk profileпјҲlow-risk profile еүҮгҖҢеҸҜиҖғж…®гҖҚпјү

    • йқһеӯ•жңҹгҖҒеғ… obstetric APS з—…еҸІпјҲи©•дј°йўЁйҡӘж•ҲзӣҠеҫҢпјү

    • DOACпјҡз„Ўи©Ұй©—пјҢдёҚе»әиӯ°гҖӮ

    • HCQпјҡе°Қ aPL йҷҪжҖ§ SLE е…·дҝқиӯ·дҪңз”Ёпјӣе–®зҙ” aPL йҷҪжҖ§иҖ…и§’иүІжңӘжҳҺгҖӮ
    • StatinпјҡдёӢиӘҝ prothrombotic biomarkerпјҢй«ҳ CVD risk иҖ…еҸҜиғҪжңүи§’иүІгҖӮ
    • йўЁйҡӘеҲҶеұӨе·Ҙе…·пјҡaPL-SгҖҒGAPSSгҖӮ
    • дёҖиҲ¬жҺӘж–Ҫ (2019 EULAR)пјҡеҡҙж јжҺ§еҲ¶ CV riskпјҲжҲ’иҸёгҖҒжҺ§иЎҖеЈ“/иЎҖи„Ӯ/иЎҖзі–гҖҒйҒӢеӢ•пјүпјӣй«ҳйўЁйҡӘжғ…еўғпјҲжүӢиЎ“гҖҒдҪҸйҷўгҖҒй•·жңҹиҮҘеәҠгҖҒз”ўиӨҘжңҹпјүз”Ё prophylactic LMWHгҖӮ
    • йҒҝе…Қ estrogen еҸҠеҗ« estrogen еҸЈжңҚйҒҝеӯ•и—ҘгҖӮ

    (2) Secondary prevention вҖ” Venous

    • жҖҘжҖ§жңҹпјҡheparinпјҲUFH жҲ– LMWHпјүгҖӮ
    • й•·жңҹпјҡVKAпјҢtarget INR 2вҖ“3пјҲINR 3вҖ“4 з„ЎйЎҚеӨ–ж•ҲзӣҠпјүгҖӮ
    • зҷӮзЁӢ (2019 EULAR)пјҡunprovoked йҰ–ж¬Ў вҶ’ й•·жңҹжҠ—еҮқпјӣprovoked йҰ–ж¬Ў вҶ’ дҫқдёҖиҲ¬ж—ҸзҫӨ guideline зҷӮзЁӢпјҲlow-risk + еҸҜйҖҶиӘҳеӣ еҸҜеғ… 3вҖ“6 еҖӢжңҲпјүпјҢhigh-risk profile еҸҜе»¶й•·гҖӮ
    • DOACпјҡRivaroxaban зҰҒз”Ёж–ј triple-positiveпјҲTRAPS еӣ еӢ•и„ҲиЎҖж “йҒҺеӨҡжҸҗж—©дёӯжӯўпјүпјӣASTRO-APSпјҲapixabanпјүstroke ијғеӨҡ вҶ’ DOAC зӣ®еүҚдёҚе»әиӯ°з”Ёж–ј thrombotic APSпјҢеғ… (2019 EULAR) еңЁ VKA й ҶеҫһдёӢд»Қз„Ўжі•йҒ”жЁҷжҲ– VKA зҰҒеҝҢжҷӮиҖғж…®пјҲдёҚеҸҜеӣ й ҶеҫһжҖ§е·®иҖҢжҸӣ DOACпјүгҖӮ
    • еҫ©зҷјпјҲINR 2вҖ“3 дёӢд»Қеҫ©зҷјпјү(2019 EULAR)пјҡе…ҲжҹҘй ҶеҫһжҖ§гҖҒй »з№ҒзӣЈжё¬ INRпјӣйҒ”жЁҷд»Қеҫ©зҷј вҶ’ еҠ  LDAгҖҒINR еҚҮиҮі 3вҖ“4гҖҒжҲ–ж”№ LMWHгҖӮ

    (3) Secondary prevention вҖ” Arterial

    • йҰ–ж¬Ў arterial (2019 EULAR)пјҡVKA е„Әж–је–®з”Ё LDAпјӣVKA INR 2вҖ“3 жҲ– INR 3вҖ“4пјҲдҫқйўЁйҡӘж¬ҠиЎЎпјүпјҢжҲ– INR 2вҖ“3 + LDAгҖӮ
    • Rivaroxaban зҰҒз”Ёж–ј triple-positive + еӢ•и„ҲдәӢд»¶пјӣдёҚе»әиӯ° DOAC з”Ёж–ј APS еӢ•и„ҲдәӢд»¶гҖӮ
    • stroke дҫқ aPL profile иҲҮ CV risk еҲҶеұӨпјӣиҖҒе№ҙгҖҒе–®дёҖ low-titer aCL зҡ„ stroke вҶ’ aspirin е–®з”ЁеҸҜиғҪзӯүж•ҲпјҲAPASSпјҢйҒ©з”ЁжҖ§жңүйҷҗпјүгҖӮ
    • еҫ©зҷј arterialпјҲVKA дёӢд»Қеҫ©зҷјпјү(2019 EULAR)пјҡжҺ’йҷӨе…¶д»–з—…еӣ еҫҢ вҶ’ INR еҚҮиҮі 3вҖ“4гҖҒеҠ  LDAгҖҒжҲ–ж”№ LMWHгҖӮ
    • иЁ»пјҡdual therapyпјҲwarfarin + aspirinпјүеўһеҠ еҮәиЎҖпјӣLA е№Іж“ҫ INR жҷӮж”№з”Ё antiвҖ“factor Xa зӣЈжё¬гҖӮ

    2. Microvascular APSпјҲйқһ CAPSпјү

    • з„ЎзөұдёҖжә–еүҮпјӣйғЁеҲҶзөҰдәҲжҠ—еҮқдҪҶж•ҲзӣҠжңӘжҳҺгҖӮ
    • е…Қз–«жҠ‘еҲ¶пјҡrituximabпјҲRITAPSпјүгҖҒbelimumabгҖҒMMFгҖҒazathioprineгҖҒcyclophosphamideгҖҒsirolimusгҖӮ
    • DAHпјҡglucocorticoid еҸҜиӘҳе°Һз·©и§ЈпјҢзҙ„еҚҠж•ёйңҖ steroid-sparingпјӣcyclophosphamide- жҲ– rituximab-based жңҖжңүж•ҲгҖӮ
    • й–ҖиЁәз¶“й©—ж–№жЎҲпјҡglucocorticoidпјҲиҮі 60 mg/dayпјү+ MMF Вұ rituximabпјӣйӣЈжІ»еҠ  IVIGпјӣйҖҡеёёеҠ  HCQ + statinгҖӮ

    3. Catastrophic APSпјҲCAPSпјү

    • зҷјдҪңзӘҒз„¶гҖҒз«ӢеҚіеҚұеҸҠз”ҹе‘ҪпјҢж—©жңҹиЁәж–·иҮій—ңйҮҚиҰҒгҖӮ
    • й җйҳІ (2019 EULAR)пјҡеҸҠж—©жІ»зҷӮж„ҹжҹ“гҖҒйҒҝе…Қдёӯж–·жҠ—еҮқ/дҪҺ INRгҖӮ
    • з¬¬дёҖз·ҡпјҡglucocorticoid + heparin + plasma exchange жҲ– IVIG дёүиҒҜпјҲе„Әж–је–®и—Ҙ/е…¶д»–зө„еҗҲпјүпјҢеҗҢжҷӮиҷ•зҗҶиӘҳеӣ пјҲж„ҹжҹ“гҖҒеЈһз–ҪгҖҒжғЎжҖ§и…«зҳӨпјүгҖӮ
    • йӣЈжІ»пјҡrituximab жҲ– eculizumabпјҲиЈңй«”жҠ‘еҲ¶пјүпјӣеҗҲдҪө systemic TMA жҷӮе°Өе…¶иҖғж…®иЈңй«”жҠ‘еҲ¶пјҲCAPS Registry 39 дҫӢ eculizumab вҶ’ 74% жҒўеҫ©пјүгҖӮ

    4. Pregnancy Morbidity

    • Warfarin иҮҙз•ё вҶ’ еӯ•жңҹз”Ё UFH жҲ– LMWHпјҲйҰ–йҒёпјүгҖӮ

    • еғ… obstetric APSпјҲз„ЎиЎҖж “пјүпјҡ

    • вүҘ3 ж¬Ў <10 йҖұжөҒз”ўпјҢжҲ– вүҘ1 ж¬Ў вүҘ10 йҖұиғҺе…’жӯ»дәЎ вҶ’ LDA + prophylactic heparinпјҲиғҺе…’еӯҳжҙ»зҺҮ 50%вҶ’80%пјү

    • еӣ еӯҗзҷҮ/йҮҚеәҰеӯҗзҷҮеүҚз—Ү/иғҺзӣӨеҠҹиғҪдёҚе…Ё <34 йҖұз”ҹз”ў вҶ’ LDA жҲ– LDA + prophylactic heparin
    • Non-criteria obstetric APS вҶ’ LDA Вұ heparin еҸҜиҖғж…®

    • High-risk profile дҪҶз„ЎдҪөзҷјз—Ү вҶ’ еӯ•жңҹ LDA еҸҜиҖғж…®

    • Thrombotic APS з—…еҸІ вҶ’ еӯ•жңҹ LDA + therapeutic dose heparinпјҲе…ЁеҠ‘йҮҸж•ҙеҖӢеӯ•жңҹпјүгҖӮ

    • йӣЈжІ»пјҲLDA + prophylactic heparin дёӢд»ҚеҸҚиҰҶпјүпјҡheparin еҚҮиҮі therapeutic doseгҖҒеҠ  HCQгҖҒжҲ–з¬¬дёҖеӯ•жңҹеҠ  low-dose prednisoloneпјӣйҒёж“Үз—…дҫӢеҸҜиҖғж…® IVIGгҖӮ
    • з”ўеҫҢпјҡзәҢ heparin 6 йҖұпјҲthrombotic з—…еҸІиҖ… 8вҖ“12 йҖұпјүйҳІжҜҚй«”иЎҖж “пјҢеҶҚ taperгҖӮ

    дәҢгҖҒOutcome вҖ” й җеҫҢпјҲKellyпјү

    • дёҚиүҜй җеҫҢеӣ еӯҗпјҡpulmonary hypertensionгҖҒзҘһз¶“дҫөзҠҜгҖҒmyocardial ischemiaгҖҒnephropathyгҖҒеӣӣиӮўеЈһз–ҪгҖҒCAPSгҖӮ
    • Primary APS й•·жңҹеҠҹиғҪе·®пјҡ10 е№ҙжҷӮ 1/3 ж°ёд№…еҷЁе®ҳжҗҚе®іпјҢ1/5 з„Ўжі•еҹ·иЎҢж—Ҙеёёжҙ»еӢ•гҖӮ
    • CAPSпјҡеҚідҪҝз©ҚжҘөжІ»зҷӮжӯ»дәЎзҺҮд»Қ >30%пјҲcerebrovascularгҖҒcardiacгҖҒж„ҹжҹ“пјүпјӣй«ҳйҒ” 1/3 еҫ©зҷјгҖӮ
    • й•·жңҹ APSпјҡеҡҙйҮҚз“ЈиҶңз—…йңҖжҸӣз“ЈгҖҒзҪ•иҰӢ TMA иҮҙи…ҺиЎ°з«ӯпјӣз§»жӨҚи…ҺеҸҜеӣ еҚіеҲ»иЎҖж “е–ӘеӨұгҖӮ
    • жүӢиЎ“жңҹпјҡеңҚжүӢиЎ“жңҹиЎҖж “йўЁйҡӘй«ҳпјҢй ҲиЎ“еүҚиҰҸеҠғгҖҒзӣЎйҮҸзё®зҹӯеҒңжҠ—еҮқжҷӮй–“гҖӮ
