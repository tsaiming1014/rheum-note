    APS 致病機轉

    一、正常生理

    磷脂平台是凝血的舞台

    血管受損時，凝血瀑布需要在磷脂平台上組裝複合物才能運作：

    Factor Xa + Va + Ca²⁺ + 磷脂平台
→ Prothrombinase complex
→ Prothrombin → Thrombin → 纖維蛋白 → 血栓

    正常情況下，磷脂在哪裡？

    細胞膜有內外兩層：

    • 外層：帶中性電的磷脂（phosphatidylcholine）→ 平靜
    • 內層：帶負電的磷脂醯絲胺酸（phosphatidylserine, PS）→ 平常藏在裡面

    當細胞活化或凋亡時，PS 從內層翻到外層（flip）→ 帶負電的平台出現在細胞表面 → 凝血複合物可以組裝上去。

    β₂GPI 是天然的守衛

    β₂GPI 平常在血液中流動，功能是：

    • 識別並清除凋亡細胞
    • 輕微抑制凝血（天然抗凝劑）

    當 PS 翻到外層後，β₂GPI 從血液中被吸引過來，貼到外層的 PS 上——aPL 再結合貼在外層的 β₂GPI。這是正常的生理清除機制，也是 aPL 致病的作用位置。

    靜止細胞：PS 在 inner layer（看不到）
         ↓ 活化/凋亡
活化細胞：PS 翻到 outer layer（暴露）
         ↓
β₂GPI 從血液貼上 outer layer 的 PS
         ↓
aPL 結合 outer layer 上的 β₂GPI  ← aPL 作用在這裡
         ↓
補體活化、細胞訊號、促血栓

    抗凝煞車：Protein C pathway

    凝血不能無限制地跑，身體有內建煞車：

    Thrombin + Thrombomodulin（內皮上）
→ 活化 Protein C（APC）
→ APC + Protein S + 磷脂平台
→ 分解 Factor Va / VIIIa → 凝血煞車

    重點：煞車也需要磷脂平台才能運作。

    二、aPL 的作用

    aPL 是什麼？

    aPL（抗磷脂抗體）是一群自體抗體，真正的靶標大多不是磷脂本身，而是貼在磷脂上的 β₂GPI。

    抗體 真正的靶標
    anti-β₂GPI β₂GPI 本身（domain I 最重要）
    anti-cardiolipin（aCL） 貼在 cardiolipin 上的 β₂GPI（自體免疫型）
    Lupus Anticoagulant（LA） β₂GPI 或 prothrombin

    aPL 做了什麼？

    ① 佔據磷脂平台

    aPL 結合 β₂GPI → β₂GPI–aPL 複合物大量佔據磷脂平台

    • 效果一：凝血複合物組不上去 → 試管裡凝血時間延長（LA 的來源）
    • 效果二：Protein C 的煞車也組不上去 → 體內抗凝煞車失效

    ② 直接活化細胞

    aPL–β₂GPI 複合物透過受體（apoER2、TLR4）活化內皮細胞：

    • ↑ Tissue factor → 啟動凝血瀑布
    • ↑ E-selectin、ICAM-1 → 白血球黏附
    • NF-κB 和 p38 MAPK 下游訊號活化

    ③ 活化補體

    aPL–β₂GPI dimer 活化古典補體路徑 → 產生 C5a C5a 大量招募單核球、嗜中性球、血小板 → 釋放 TNF、促凝物質

    ④ 活化嗜中性球 → NETs

    嗜中性球被活化後噴出 NETs（DNA + 組蛋白 + 殺菌蛋白） NETs 再次活化血小板和內皮細胞 → 放大血栓發炎反應

    為什麼 aPL 存在，但不一定發病？

    β₂GPI 平常在血液中濃度低、構型蜷曲，aPL 抓不到它。

    只有當細胞受損、PS 外露、β₂GPI 大量貼上細胞表面時，aPL 才找到目標，一切才開始。

    三、APS 的產生過程

    Two-hit model

    First hit：aPL 持續存在（無症狀期，可長達數年）

Second hit：觸發因子出現
  → 感染、手術、雌激素、懷孕、發炎、惡性腫瘤

細胞活化 → PS 外露 → β₂GPI 貼上去
→ aPL 大量結合
→ 四條火線同時點燃：
    ① 內皮活化（TF、黏附分子）
    ② 補體 C5a（招募效應細胞）
    ③ Protein C 煞車失效
    ④ NETs 放大發炎
→ 血栓形成

    為什麼體外看起來「抗凝」，體內卻「促血栓」？

    體外（試管） 體內（血管）
    看到的 aPL 佔磷脂 → 凝血起始變慢 aPL 同時破壞煞車 + 活化內皮 + 補體
    結果 凝血時間延長 整體環境促血栓
    缺少什麼 活的細胞、補體、動態平衡 —

    體內 aPL 同時影響促凝與抗凝兩側：

    促凝側：
  aPL 活化內皮 → ↑ Tissue factor
  補體 C5a → 招募單核球、血小板
  → 油門踩下去

抗凝側（Protein C pathway）：
  aPL 佔了磷脂平台 → APC 進不去 → 煞車失效

    試管只測到「引擎發動變慢」，卻看不到「煞車線被剪斷、油門被綁住」。

    產科 APS 的特殊機轉

    胎盤的滋養層細胞（trophoblast）天然就有 PS 外露 → β₂GPI 本來就貼在上面 → aPL 不需要觸發就能作用。

    主要傷害不是血栓，而是補體介導的發炎：

    aPL 作用於胎盤
→ 補體活化（C5a）→ 炎症細胞湧入 → 組織損傷
→ STAT5 抑制 → ↓ prolactin、↓ IGFBP-1 → 著床與血管重塑失敗

    C5 knockout 小鼠即使有 aPL，胎兒完全不死 → 補體是產科 APS 的核心機轉

    四、三種檢驗方法

    1. Lupus Anticoagulant（LA）— 功能性檢驗

    原理：不直接測抗體，而是測「磷脂平台是否被佔據」的功能性影響。

    為什麼用 Russell's Viper Venom？ RVV 直接活化 Factor X，繞過前段凝血路徑，讓測試完全依賴磷脂平台——aPL 的效果可以最純粹地被看到。

    dRVVT 流程：

    步驟 做什麼 邏輯
    Screen 低磷脂 + RVV 活化 FX 磷脂少 → aPL 干擾最明顯 → 凝血時間延長
    Confirm 高磷脂 + RVV 活化 FX 磷脂過量 → aPL 被稀釋 → 凝血時間恢復
    Ratio Screen ÷ Confirm >1.2 = LA 陽性

    為什麼不直接做 Confirm？ Screen 和 Confirm 回答不同的問題：

    • Screen = 「有沒有問題？」→ 低磷脂讓 aPL 干擾最大化，製造出可偵測的異常
    • Confirm = 「問題是不是磷脂依賴性的？」→ 用大量磷脂稀釋 aPL，看時間能否恢復

    沒有 Screen，根本不知道有沒有異常；Confirm 在 Screen 正常時不需要做。

    Confirm 不一定會「正常」：

    Confirm 結果 代表
    恢復正常（ratio > 1.2） 磷脂依賴性干擾 → LA 陽性
    仍然延長（ratio ≤ 1.2） 非磷脂依賴性 → 考慮其他 inhibitor（如 Factor VIII inhibitor）

    Mixing test（在 Screen 延長後先做）：

    患者血漿 + 正常血漿（1:1）→ 再測凝血時間

仍延長 → 有 inhibitor 存在（aPL）→ 繼續做 Confirm
恢復正常 → 凝血因子缺乏 → 非 LA

    完整判讀流程：

    Screen 延長？
  → No → LA 陰性，結束
  → Yes ↓

Mixing test：加正常血漿能糾正？
  → Yes → 凝血因子缺乏，非 LA
  → No（仍延長）→ 有 inhibitor ↓

Confirm：加大量磷脂能糾正？
  → No → 非磷脂依賴性抑制物（如 Factor VIII inhibitor）
  → Yes，ratio > 1.2 → LA 陽性 ✓

    臨床地位：三種檢驗中預測血栓風險最高

    2. Anti-cardiolipin（aCL）— 固相免疫檢驗（ELISA）

    原理：直接測血清中的 aCL 抗體濃度（IgG / IgM）

    兩種 aCL 的本質不同：

    類型 真正靶標 致病性
    β₂GPI-dependent（自體免疫型） β₂GPI 有致病性
    β₂GPI-independent（感染/藥物型） Cardiolipin 本身 低（但近年有爭議）

    感染型常見於：梅毒（VDRL）、Borrelia、HIV、Leptospira、寄生蟲感染

    IgG vs IgM：

    IgG IgM
    與血栓相關性 強 較弱（有爭議）
    可過胎盤 是（FcRn receptor） 否
    補體活化效率 高 較低
    單獨陽性臨床意義 明確 不確定

    單獨 IgM 陽性要謹慎解讀，尤其需排除感染誘發的一過性陽性。

    3. Anti-β₂GPI — 固相免疫檢驗（ELISA）

    原理：直接測針對 β₂GPI 的抗體，尤其是 domain I 的抗體最具致病意義

    臨床地位：

    • 最能反映 APS 真正的致病機轉
    • 特異性最高（尤其 anti-domain I）
    • 2023 ACR/EULAR 新標準中權重提升

    三種檢驗比較

    LA（dRVVT） aCL aβ₂GPI
    方法 功能性 ELISA ELISA
    靶標 磷脂平台功能 aCL（多為β₂GPI-dependent） β₂GPI 直接
    血栓預測力 最強 中 中高
    特異性 高 中 最高

    Triple positive（LA + aCL + aβ₂GPI 全陽性）= 最高風險

    整體架構一覽

    正常：PS 外露 → β₂GPI 貼上 → 清除凋亡細胞（生理）

APS：
  β₂GPI 被 aPL 認錯（自體免疫）
  → Two-hit（second hit 觸發）
  → aPL 大量結合 β₂GPI
  → 內皮活化 + 補體 C5a + Protein C 煞車失效 + NETs
  → 血栓（血管）/ 補體炎症（胎盤）

檢驗：
  LA（dRVVT）→ 測功能（磷脂平台被佔）→ 血栓預測力最強
  aCL → 測抗體量（β₂GPI-dependent 才致病，IgG > IgM）
  aβ₂GPI → 測真正靶標（特異性最高）
