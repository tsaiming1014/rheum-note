    RA 治療總整理

    Kelly Ch073 + 2025 EULAR Update

    一、治療核心概念

    Treat-to-Target（T2T）是最重要的典範轉移

    治療目標：持續緩解（remission，首選）或低疾病活性（LDA）

    哪種 DMARD 的重要性，遠不如「確實讓病人達標」。TICORA 試驗是奠基性研究：只用 conventional DMARDs，密集治療組（目標 DAS ≤ 2.4）vs. 常規照護組，18 個月後 ACR70 達成率 71% vs. 18%（p < 0.0001），X 光侵蝕進展 0.5 vs. 3.0（p = 0.002），且不增加不良事件。

    T2T 的操作定義（2025 EULAR）： 治療後 3 個月內需改善 ≥ 50%；6 個月內需達到目標。若 3 個月無改善或 6 個月未達標，必須調整治療。 (2025 EULAR)

    關於緩解目標的補充：

    • 早期 RA：以 remission 為主要目標
    • 長期病程或曾多次 DMARD 失敗者：LDA 可作為替代目標
    • 任何高於 LDA 的狀態都應視為不可接受，除非共病症或患者強烈偏好阻礙升級治療 (2025 EULAR)

    疾病活性評估工具

    常規使用複合指標（composite indices），其可靠性優於單一指標：

    工具 組成 緩解 LDA 中度 高度
    DAS28 28 TJC、28 SJC、PtGA、ESR 或 CRP ≤ 2.6 ≤ 3.2 3.2–5.1 > 5.1
    SDAI 28 TJC、28 SJC、PrGA、PtGA、CRP ≤ 3.3 ≤ 11 11–26 > 26
    CDAI 28 TJC、28 SJC、PrGA、PtGA ≤ 2.8 ≤ 10 10–22 > 22
    RAPID3 HAQ-II、Pain、PtGA ≤ 3 ≤ 6 7–12 ≥ 13

    ❌ 不建議用 DAS28 定義 IL-6i 或 JAKi 患者的緩解：這兩類藥會壓低 CRP 和 ESR，導致 DAS28 偽高估緩解頻率，不少患者實際上仍有活動性疾病。 (2025 EULAR)

    緩解定義（2022 ACR/EULAR 修訂版）

    Boolean 2.0（需全部滿足）：

    項目 閾值
    Tender joint count ≤ 1
    Swollen joint count ≤ 1
    CRP ≤ 1 mg/dL
    Patient global assessment ≤ 2 cm（0–10 cm VAS）

    或 SDAI ≤ 3.3 / CDAI ≤ 2.8

    注意：Boolean 1.0 的 PtGA 門檻為 1 cm，2022 年修訂為 2 cm（改善與 SDAI/CDAI 的一致性）

    ❌ 不建議以超音波緩解或 MRI 緩解為治療目標：多項試驗（TaSER、ARCTIC、IMAGINE-RA）顯示，以影像緩解為目標，結果不優於臨床緩解，反而帶來更多不良事件與更高費用。 (2025 EULAR)

    共病症對疾病活性評估的干擾

    纖維肌痛、OA、感染等共病症可使疾病活性分數偏高，誤判為發炎活動，此時應以腫脹關節數（SJC）為核心評估真實滑膜炎程度。使用 IL-6i 或 JAKi 時，CRP 下降可能獨立於臨床改善，更應聚焦在關節腫脹。 (2025 EULAR)

    二、監測頻率

    • 活動期：每 1–3 個月評估疾病活性（含關節計數、CRP）
    • 達到持續緩解後：可降低頻率（如每 6 個月或更少） (2025 EULAR)
    • 實驗室監測（安全性）：開始新藥後前 3–6 個月較頻繁；耐受後可減為半年一次或更少（近年大型觀察研究支持此更寬鬆頻率） (2025 EULAR)
    • 過度監測對患者、醫師、醫療資源均是負擔

    三、DMARD 分類與藥物選擇

    第一線：Conventional synthetic DMARDs（csDMARDs）

    MTX — 所有治療的基石（anchor therapy）

    MTX 應為絕大多數患者的起始藥物，應作為第一線治療策略的核心。 (2025 EULAR)

    • 便宜、有效、耐受性良好、可延長壽命
    • 與所有 bDMARD（尤其 TNFi）合用均更有效；MTX 能抑制抗藥物抗體形成
    • 頭對頭試驗中，MTX 與多數 biologic 療效相當（JAKi 在劑量充足時可能略優，但 MTX 劑量不足時數據易被低估）

    MTX 最佳化原則：

    • 起始劑量 ≥ 15 mg/week
    • 盡快升至 20–25 mg/week（若目標未達）
    • 口服 MTX 生物利用率在 15–20 mg 後出現 plateau → 改用皮下注射（SC）以提升生物可用率和耐受性
    • 或採 split-dose oral MTX（同一天分兩次服用）改善吸收 (2025 EULAR)
    • 全程補充葉酸（可提高耐受性，需要時增加葉酸劑量）
    • 約 60% 有良好反應；約 30% 可僅用 MTX 達 LDA

    非 MTX csDMARDs

    MTX 禁忌症或早期耐受不良時，考慮 leflunomide 或 sulfasalazine。 (2025 EULAR)

    ❌ 不建議 Triple therapy（MTX + SSZ + HCQ）作為第一線

    • CareRA、NORD-STAR、2010 年以來多項試驗均顯示 triple therapy 無優於 MTX + GC（尤其在早期 RA）
    • NORD-STAR 子分析：結構破壞進展在 MTX + oral GC 組甚至較少（5% vs. 27%） (2025 EULAR)
    藥物 定位 注意事項
    Leflunomide（LEF） MTX 替代或併用 RA-ILD 需謹慎；妊娠禁忌
    Sulfasalazine（SSZ） MTX 替代或 triple 的組成 懷孕相對較安全
    Hydroxychloroquine（HCQ） 輕症輔助或 triple 組成 每年眼科追蹤（視網膜病變）
    Azathioprine（AZA） MTX 禁忌時替代（如 RA-ILD） 確認 TPMT 多型性（1/300 同型突變者骨髓毒性↑）
    Cyclosporine（CSA） MTX 上 add-on，已少用 高血壓、腎毒性（18 個月持續率僅 22%）
    Minocycline/Doxycycline 利基使用（無法監測實驗室、頻繁感染、牙周病合併） 皮膚色素沉著（minocycline 較明顯）

    Glucocorticoids（GC）

    短期 GC 應在開始或更換 csDMARD 時一同考慮使用（橋接療法），不同劑量方案和給藥途徑皆可，但應盡快漸退並停用。 (2025 EULAR)

    • 有顯著臨床和 X 光療效（COBRA 試驗、BeSt 試驗 group 3 均驗證）
    • GC 是橋接 DMARD 起效空窗期的工具，目標是零長期 GC
    • 無法停 GC = DMARD 方案失敗的強指標，需升級治療
    • 長期使用 > 5–7.5 mg/day prednisone equivalent → DMARD 必須升級
    • GC 幾乎不應在沒有 DMARD 的情況下使用
    • 98% 任務小組委員贊成此建議（史上 GC 建議最高共識） (2025 EULAR)

    BeSt 試驗：初始高劑量 prednisone 組，2 年後 92% 已成功停用全部 prednisone（只在 DAS ≤ 2.4 時才漸退）。

    Biologic DMARDs（bDMARDs）

    類別 藥物 靶點
    TNFi adalimumab, etanercept, infliximab, golimumab, certolizumab TNF-α
    IL-6Ri tocilizumab, sarilumab IL-6R
    CD80/86 抑制 abatacept T cell co-stimulation
    Anti-CD20 rituximab B cells

    所有 bDMARD 幾乎均應與 MTX 合用 — 效果顯著優於單藥，尤其 TNFi（MTX 可抑制抗藥物抗體形成）。 (2025 EULAR)

    例外：無法使用 csDMARD 時，IL-6Ri 和 JAKi 的單藥效果優於其他 bDMARD。 (2025 EULAR)

    Targeted synthetic DMARDs：JAK inhibitors（JAKi）

    目前核准用於 RA 者：tofacitinib、baricitinib、filgotinib、upadacitinib

    特點：

    • 口服小分子，起效快
    • 有效於 MTX 失敗、bDMARD 失敗後
    • 單藥即有顯著效果（MTX 加入並不顯著增強 JAKi 療效）
    • 部分試驗中 DAS28/CRP 反應優於 adalimumab，但主要受 CRP blunting 影響；臨床意義上 JAKi 並非明確優於 bDMARD

    安全疑慮（ORAL Surveillance，tofacitinib）：

    • 對象：MTX 中 + 年齡 > 50 + ≥ 1 個 CV 風險因子
    • Tofacitinib vs. TNFi：MACE HR 1.8（超過非劣效界）；惡性腫瘤風險 +48%
    • MACE 風險主要集中在有動脈粥狀硬化心血管疾病（ASCVD）病史者
    • ⭐ 新資料（post-hoc ORAL Surveillance）：使用 statins 可能減輕 tofacitinib 的 CV 風險 (2025 EULAR)
    • 真實世界廣泛 RA 族群中，此風險並未一致複製

    JAKi 使用前需評估的風險因子： 年齡 > 65、吸菸史、CV 疾病史、惡性腫瘤史、靜脈血栓史

    JAKi 的定位（2025 EULAR 維持 2022 立場）： 與 bDMARD 並列於 csDMARD 失敗後的第二線，但必須評估並考量上述風險因子後方可使用。不要求先失敗 bDMARD 才能用；但也不支持在 bDMARD 之前優先使用。 (2025 EULAR)

    ❌ JAKi + bDMARD 組合：禁忌（無增益且感染風險大增）

    四、治療流程與升級策略

    Phase 1：確診即啟動 DMARD

    一旦診斷 RA，立即開始 DMARD 治療（「診斷標準 ≠ 分類標準」，無需等待分類標準確認）。 (2025 EULAR)

    診斷 RA
    ↓
MTX（± 短期 GC 橋接）
    ↓
每 1–3 個月評估
    ↓
3 個月無改善 或 6 個月未達標
    ↓
升級治療（Phase 2）

    Phase 2：csDMARD 失敗後

    ❌ 重要改變（2025 EULAR）：刪除依預後因子分層的舊建議

    舊版（2022）在 MTX + GC 失敗後，會依有無不良預後因子分流：

    • 有預後因子 → 加 bDMARD/JAKi
    • 無預後因子 → 嘗試第二種 csDMARD

    2025 版直接刪除此分層，統一建議加 bDMARD（JAKi 評估風險後亦可）。 (2025 EULAR)

    刪除原因：

    1. MTX + GC 失敗本身即為不良預後訊號
    2. 第二種 csDMARD 在 MTX 失敗後療效有限，持續率低
    3. Biosimilar 降價 + generic tofacitinib 上市，費用考量已改變

    刪除分層並不表示預後因子不重要，而是 MTX 失敗本身已隱含高風險

    Phase 2 選擇：

    MTX + GC 失敗
    ↓
加入 bDMARD（首選 TNFi，長期記錄最豐富）
  或
加入 JAKi（需評估 CV、惡性腫瘤、血栓風險）
    ↓
繼續合用 MTX（若可耐受）

    從 Kelly 的實務視角補充：

    若患者費用考量明顯或 biosimilar/TNFi 仍有顧慮，Triple therapy（MTX + SSZ + HCQ）仍是可考慮的替代選擇：

    • RACAT 試驗：Triple not inferior to ETAN（MTX 失敗後加藥）
    • TEAR 試驗：Step-up triple ≈ step-up ETAN（2 年後 DAS28、HAQ、X 光均相同）
    • 費用效益：先用 bDMARD 策略 ≈ 每 QALY 增益多花 100–200 萬美元

    Phase 3：第一種 b/tsDMARD 失敗後

    可換任何其他 bDMARD 或 JAKi（考量風險後）。 (2025 EULAR)

    • 1 種 TNFi 或 IL-6Ri 失敗 → 可換另一種 MOA，或換第二種 TNFi / IL-6Ri
    • 同類藥物不應使用第三次

    TNFi 失敗後換第二種 TNFi：

    • 繼發性失敗（曾有效後失效）比原發性失敗（從未有效）更可能再次有效
    • 法國 ROC 試驗：TNFi 失敗後，non-TNFi 組 EULAR good response 優於換第二種 TNFi（69% vs. 52%）

    Non-TNFi 選擇（療效相近，ACR20 約 50%）：

    藥物 目標 特殊考量
    Rituximab CD20（B cell） ACPA-/RF- 患者反應較差；施打後 B cell 抑制 6–12 個月
    Abatacept CD80/86（T cell co-stimulation） 感染率低於 infliximab；abatacept ≈ adalimumab（MTX-IR）
    Tocilizumab IL-6R 單藥最佳（ADACTA：TCZ 單藥 > adalimumab 單藥）；LDL↑、GI 穿孔需注意
    Sarilumab IL-6R 同 TCZ 類別

    ❌ TDM（Therapeutic Drug Monitoring）不建議：ADDORA-switch 試驗顯示 adalimumab 濃度監測無助於改善預後，不應常規用於 RA。 (2025 EULAR)

    抗藥物抗體（anti-drug antibodies，ADA）的臨床意義（Kelly）：

    • 28% adalimumab 患者在 3 年內產生 ADA（67% 在前 6 個月）
    • ADA 與療效喪失高度相關（HR 3.0），也與靜脈血栓事件相關（HR 7.6）
    • 雖 TDM 並不建議常規化，ADA 的發展機制仍是重要的病理概念

    七、達標後的 DMARD 減量

    減量順序

    第一優先：停 GC 和 NSAIDs

    因毒性最明確，應最優先退出。有效 DMARD 下，GC 漸退是預期應達到的結果，而非例外。

    GC 停用後，患者達持續緩解 → 繼續使用 DMARD，但可考慮劑量減少。 (2025 EULAR)

    減量的前提條件

    • GC 已停用
    • 達到緩解（越深越好，Boolean/SDAI remission > LDA）
    • 穩定維持目標狀態至少 6–12 個月
    • 2025 EULAR 特別強調：大多數停藥者在一年內復發，應繼續使用 DMARD，停藥並非目標 (2025 EULAR)

    成功預測因子

    • 達到深度（Boolean）緩解
    • 血清陰性（ACPA-、RF-）
    • 超音波無滑膜炎及 Power Doppler 訊號
    • 穩定維持達標夠久

    關鍵原則

    • 減量（降頻次/劑量）勝於停藥
    • 至少保留一種 DMARD
    • 優先選擇延長注射間隔（如 ETN 50 mg Q2–4 週；adalimumab 40 mg Q3–6 週）
    • 教育患者：若症狀加重，立即恢復原劑量，不需等待下次門診
    • 減量期間復發者，恢復原劑量後通常可快速再度控制

    八、難治性 RA（Difficult-to-Treat RA）

    EULAR 定義（需全部符合）：

    1. ≥ 2 種 bDMARD 或 JAKi 失敗（在 csDMARD 失敗後）
    2. 仍有活動性疾病或進行中病變的跡象
    3. 醫師和/或患者均認為控制有困難

    處理流程：

    1. 重新確認診斷（排除誤診）
    2. 評估共病症（纖維肌痛、OA 等造成的假性高活性分數）
    3. 確認影像是否真的有活動性炎症
    4. 評估治療遵從性（adherence）
    5. 加強非藥物介入（衛教、運動、自我管理）
    6. 確認 MTX 是否已充分優化，Triple therapy 是否已嘗試（Kelly）

    九、RA-ILD 補充 (2025 EULAR)

    • Nintedanib（INBUILD 試驗子分析）和 pirfenidone 早期資料顯示可能減緩 RA-ILD 肺功能下滑
    • 但證據尚不充分，且這些藥物不是 RA 的 DMARD
    • 建議諮詢胸腔科醫師共同處理 (2025 EULAR)
    • RA-ILD 患者若 MTX/LEF 有顧慮，AZA 常作為 csDMARD 替代選擇（Kelly）
