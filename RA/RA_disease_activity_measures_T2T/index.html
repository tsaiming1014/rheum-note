    дёҖгҖҒеҗ„е·Ҙе…·иЁҲз®—ж–№ејҸ

    DAS28-ESR

    е…¬ејҸпјҲйңҖиЁҲз®—ж©ҹпјүпјҡ

    0.56 Г— вҲҡ(TJC28) + 0.28 Г— вҲҡ(SJC28) + 0.70 Г— ln(ESR) + 0.014 Г— Patient Global (0вҖ“100)

    • зҜ„еңҚпјҡ0вҖ“9.4
    • йңҖиҰҒпјҡ28 й—ңзҜҖиЁҲж•ёпјҲtender + swollenпјү+ ESR + з—…дәәж•ҙй«”и©•дј°
    • зү№иүІпјҡжӯ·еҸІжңҖжӮ д№…пјҢз”Ёж–ј TICORA зӯүй—ңйҚөи©Ұй©—пјӣжңү online calculator

    SDAI

    е…¬ејҸпјҲзӣҙжҺҘеҠ зёҪпјүпјҡ

    TJC28 + SJC28 + Patient Global (0вҖ“10) + Physician Global (0вҖ“10) + CRP (mg/dL)

    • зҜ„еңҚпјҡ0.1вҖ“86
    • йңҖиҰҒпјҡ28 й—ңзҜҖиЁҲж•ё + йӣҷж–№ж•ҙй«”и©•дј° + CRP
    • зү№иүІпјҡиЁҲз®—жҜ” DAS28 з°Ўе–®пјҲзӣҙжҺҘеҠ пјүпјӣremission жҜ” DAS28 жӣҙеҡҙж ј

    • зҜ„еңҚпјҡ0вҖ“76

    • зү№иүІпјҡй–ҖиЁәеҸҜеҚіжҷӮиЁҲз®—пјӣз ”з©¶ tocilizumab зӯүеҪұйҹҝ CRP зҡ„и—Ҙзү©жҷӮзӮәйҰ–йҒё
