# Sjögren's Disease — 治療

> 來源：Kelly's Textbook of Rheumatology Ch74（St. Clair & Leverenz）；  
> Marques-Gomes et al. *RMD Open* 2026;12:e004310

---

## 一、整體原則

- 目前**缺乏核准的疾病修飾療法（DMARD）**，治療以**症狀控制**為主
- 多科合作：風濕科、眼科、牙科、耳鼻喉科
- 所有有**抗膽鹼效果**的藥物（抗組織胺、抗憂鬱藥、肌肉鬆弛劑）應盡量避免或停用

---

## 二、乾眼（KCS）

| 措施 | 細節 |
|---|---|
| 環境 | 避免暖氣、冷氣、風 |
| 人工淚液 | OTC；低黏度頻繁用，高黏度睡前用；>4次/天須用**無防腐劑**配方 |
| **Cyclosporine 0.05%**（Restasis） | FDA 核准；改善 Schirmer test；副作用：instillation burning |
| **Cyclosporine 0.09%**（Cequa） | 奈米微胞配方，刺激感較低 |
| **Lifitegrast**（Xiidra） | LFA-1 抑制劑，FDA 核准；副作用類似 cyclosporine |
| **Varenicline 鼻噴劑**（Tyrvaya） | 刺激三叉神經副交感路徑促淚液分泌 |
| **Punctal occlusion** | 症狀持續者考慮；先用可溶性 collagen plug 試驗，再考慮矽膠 plug 或燒灼 |
| 眼瞼炎 | 溫敷 + 眼瞼清潔；難治性 → 口服 doxycycline |

## 三、乾口（Xerostomia）

| 措施 | 細節 |
|---|---|
| 人工唾液 | OTC（hypromellose/methylcellulose），效果短暫 |
| 無糖糖果/口香糖 | 刺激殘存唾液分泌 |
| **Pilocarpine** 5 mg TID–QID | M3R 促分泌劑；副作用：出汗、潮紅；禁忌：窄角青光眼、中重度氣喘、虹膜炎 |
| **Cevimeline** | 同 M3R 促分泌；RCT 顯示改善乾眼與乾口 |
| 牙齒保護 | 每日氟化物、定期牙科追蹤 |
| **口腔念珠菌** | Nystatin / Clotrimazole troches / Fluconazole（10–14 天）；角唇炎 → clotrimazole cream |

## 四、其他腺外乾燥

- **皮膚**：保濕劑、Vitamin E 局部、含 urea/lactate 乳劑
- **陰道乾燥**：moisturizer、lubricant、局部 estrogen cream

## 五、腺外病變（Extraglandular Disease）

| 嚴重度 | 治療 |
|---|---|
| **輕度**（疲勞、肌痛、關節痛/炎、purpura 無潰瘍） | NSAIDs |
| **中度** | **Hydroxychloroquine**（≤5 mg/kg/day；每年眼科追蹤）⚠️ RCT（JOQUER）顯示 vs placebo 無顯著差異，但臨床仍常用 |
| **重度**（NSIP、LIP、間質性腎炎、運動性神經病變、壞死性血管炎、CNS focal deficit） | 高劑量類固醇 ± **AZA / MMF / CYC / RTX** |

**個別器官補充：**

- **肺部（NSIP/LIP）**：高劑量類固醇 ± AZA/MMF/CYC/RTX；Bronchiolitis → 類固醇單用通常有效
- **感覺性周邊神經病變**：Gabapentin、pregabalin；類固醇效果短暫
- **運動性神經病變**：積極免疫抑制；嚴重者考慮 **IVIG**
- **Purpura**：支持治療（彈性褲襪 + NSAIDs/抗組織胺）為主；有潰瘍才優先類固醇（停藥易復發）

## 六、生物製劑（截至 Kelly 版本）

| 藥物 | 結果 |
|---|---|
| **TNF inhibitors**（infliximab、etanercept） | ❌ 小型 RCT 無效 |
| **Rituximab（RTX）** | ⚠️ 大型 Phase III（TEARS、TRACTISS）均**未達主要終點**；目前僅用於系統性病變（病例系列支持） |
| **Baminercept**（LTβR fusion protein） | ❌ Phase II RCT 無效 |
| **Abatacept** | ❌ Phase III RCT 無效 |
| **Tocilizumab**（IL-6Ri） | ❌ RCT 無效 |

## 七、新興治療展望（2026）

![SjD 新興治療路徑圖](../images/SjD%20future%20therapies.png)

### B 細胞標靶

| 藥物 | 機轉 | 試驗結果 |
|---|---|---|
| **Ianalumab**（VAY736） | Anti-BAFF-R；ADCC + BAFF-R 訊號抑制 | ✅ **Phase III NEPTUNUS-1 & -2 均達主要終點**（ESSDAI↓）→ SjD 史上首個 Phase III 成功 |
| **Telitacicept** | TACI-Fc；同時阻斷 BAFF + APRIL | ✅ Phase III（中國，n=381）：ESSDAI 及 ESSPRI 均顯著改善 |
| **Belimumab + RTX** | 雙重 B 細胞打擊 | Phase II：生物效應良好，PRO 未改善 |
| **Sibeprenlimab** | Anti-APRIL（plasma cell survival factor） | Phase II 進行中（ENVISAGE） |

### IgG 標靶（FcRn 抑制劑）

| 藥物 | 機轉 | 試驗結果 |
|---|---|---|
| **Nipocalimab** | Anti-FcRn；降低 anti-Ro/La 等病理性 IgG | ✅ Phase II（DAHLIAS）：ClinESSDAI 顯著改善；IgG / anti-Ro60/La 劑量依賴性降低 |
| **Efgartigimod** | FcRn 抑制劑 fragment | Phase II proof-of-concept：IgG 降約 60%，達 CRESS 主要終點 |

### 細胞內訊號路徑

| 藥物 | 機轉 | 試驗結果 |
|---|---|---|
| **Remibrutinib** | 口服高選擇性 BTK 抑制劑 | ✅ Phase II（LOUiSSE）：ESSDAI 顯著改善；CXCL13 降約 50%；無 Phase III 計畫 |
| **Deucravacitinib** | 選擇性 TYK2 抑制劑（阻斷 IFN-I、IL-12/23 訊號） | Phase III 進行中（POETYK SjS-1） |
| Filgotinib / Tirabrutinib / Lanraplenib | JAK1i / BTKi / SYKi | ❌ Phase II 均未達主要終點 |

### T 細胞標靶 / 共刺激阻斷

| 藥物 | 機轉 | 試驗結果 |
|---|---|---|
| **Low-dose IL-2** | 優先擴增 Treg（高親和力 IL-2R） | ✅ Phase II 單中心：ESSDAI 改善、Treg↑；需多中心確認 |
| **Dazodalibep** | Anti-CD40L；阻斷 T-B cell 交互作用 | ✅ Phase II crossover：系統性（ESSDAI）與症狀為主（ESSPRI）**兩個終點均達標**；Phase III 開發中 |
| Iscalimab | Anti-CD40 | Phase IIb（TWINSS）達主要終點；但廠商評估後**停止開發** |

### 舊藥合併新用

- **HCQ + Leflunomide**：Phase II（RepurpSS-I & II）顯示 ESSDAI 顯著下降；目前大型歐洲多中心試驗（**NECESSITY**）中驗證，同時評估 HCQ + MMF

### 新興治療小結

- **最大突破**：Ianalumab（anti-BAFF-R）及 Telitacicept（BAFF/APRIL 雙阻斷）是首批 Phase III 成功藥物，有望成為 **SjD 首個核准 DMARD**
- **共同限制**：系統性疾病（ESSDAI）改善明顯，但**患者自述的乾燥、疲勞、疼痛（ESSPRI）改善仍有限**
  - 可能原因：不可逆腺體破壞、症狀由非免疫機轉（神經、中樞敏化）主導、安慰劑反應高
- **CAR-T 細胞**：個案報告顯示合併 SjD 的 B 細胞淋巴瘤患者在 anti-CD19 CAR-T 後腺體疾病緩解；SjD 專屬試驗登記中（NCT07041099）

## 八、疾病活性評估工具

- **ESSDAI**（EULAR SjD Disease Activity Index）：評估系統性病變，共 12 個器官域
- **ESSPRI**（EULAR SjD Patient Reported Index）：評估乾燥、疲勞、疼痛（0–10 分）
- **STAR / CRESS**：整合客觀與主觀的新複合指標，用於臨床試驗
