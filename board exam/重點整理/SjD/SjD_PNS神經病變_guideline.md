# SjD 周邊神經系統病變 — 評估與治療

![SjD PNS 神經病變分類](../images/SjD%20PNS.jpg)

## 一、為什麼 SjD 會傷害周邊神經

SjD 不只是「乾燥症」，而是全身性自體免疫病，神經系統透過三個機轉受損：

**1. 直接免疫攻擊神經元細胞體**

背根神經節（DRG）無血腦屏障保護 → CD4+ T 細胞、B 細胞、漿細胞直接浸潤 → 殺死感覺神經元 → **Ganglionopathy**

**2. 血管炎**

冷球蛋白血症（尤其 type II）→ 免疫複合體沉積於神經滋養血管（vasa nervorum）→ 缺血性軸突損傷 → **Vasculitic neuropathy**

**3. 小纖維直接損傷**

Anti-Ro/SSA 抗體及 Type I IFN → 直接損傷小纖維神經 → **SFN**（NCS 正常，但燒灼痛 + 自主神經症狀）

**臨床重要性：**

- PNS 病變盛行率：8–60%，且**嚴重被低估**
- **高達 90% 的神經病變發生在 SjD 確診之前** → 不明原因神經病變需主動考慮 SjD
- 30–40% SjD 患者 anti-SSA 陰性，血清學陰性不能排除診斷
- 自主神經症狀常被誤診為焦慮症或功能性疾病

## 二、臨床表現速查表

| 神經病變類型 | 臨床表現 |
|---|---|
| **三叉神經痛（Trigeminal neuralgia）** | 臉部單側陣發性劇烈電擊樣疼痛，有扳機點（吃飯、說話可誘發） |
| **三叉神經病變（Trigeminal neuropathy）** | 臉部麻木、感覺喪失，可雙側；無劇烈疼痛 |
| **顏面神經麻痺（Facial neuropathy）** | 單側上下臉無力（眉毛抬不起來 + 嘴角歪斜）；可伴味覺喪失、聲音過敏 |
| **腕隧道症候群（Carpal tunnel）** | 大拇指到無名指橈側麻木疼痛，夜間加重，大魚際肌萎縮 |
| **尺神經病變（Ulnar neuropathy）** | 小指及無名指尺側麻木，手內肌萎縮，爪狀手 |
| **大纖維神經病變（Large fiber）** | 振動覺/本體感覺喪失、腱反射消失、步態不穩；NCS/EMG 異常 |
| **小纖維神經病變（SFN）** | 燒灼痛、刺痛、感覺異常；NCS/EMG **正常**；常合併 POTS/腸胃/泌尿症狀 |
| **CIDP（慢性脫髓鞘性）** | 對稱性近端+遠端無力，緩慢進展，腱反射消失；CSF 蛋白高；NCS 脫髓鞘型 pattern |
| **Ganglionopathy（感覺神經節病變）** | 深部感覺喪失 + 感覺性共濟失調；**不對稱、不從遠端開始**；可快速致殘 |
| **血管炎性神經病變／Mononeuritis multiplex** | 多條不相鄰神經**急性、疼痛性、不對稱**先後受損；可快速進展；冷球蛋白/低補體常陽性 |
| **POTS（姿勢性心搏過速）** | 站立後心跳增加 ≥30 bpm，頭暈、心悸、近暈厥；臥下改善 |
| **姿勢性低血壓** | 站立後收縮壓降 ≥20 mmHg 或舒張壓降 ≥10 mmHg，頭暈、暈厥 |
| **GI 自主神經病變** | 吞嚥困難、噁心、早飽、腹脹、夜間腹瀉（腸道蠕動失調） |
| **泌尿生殖自主神經病變** | 排尿困難、頻尿、夜尿、性功能障礙 |
| **分泌運動異常** | 出汗過多、過少或熱不耐受 |

> ⚠️ **MM vs 血管炎性神經病變**：Mononeuritis multiplex 是臨床**分布模式**（多條不相鄰神經先後受損），血管炎是其最常見**原因**。在 SjD 脈絡下幾乎可以畫等號，但血管炎性神經病變也可呈對稱分布。

## 三、診斷與評估重點

### 大纖維 vs 小纖維鑑別

| | 大纖維 | 小纖維（SFN） |
|---|---|---|
| 症狀 | 無力、步態不穩、腱反射消失 | 燒灼痛、刺痛、POTS、腸胃/泌尿症狀 |
| NCS/EMG | **異常** | **正常** |
| 確診工具 | NCS/EMG | 皮膚打孔切片（epidermal nerve fiber density） |

### 「免疫媒介大纖維神經病變」診斷步驟

為排除性診斷：

1. NCS/EMG 確認大纖維受損
2. 排除：糖尿病、甲狀腺低下、B12 缺乏、單株球蛋白血症、毒素/藥物
3. 確認 SjD 診斷
4. 排除血管炎（無急性疼痛快速進展、無 mononeuritis multiplex、冷球蛋白陰性）
5. 排除 CIDP

### 血管炎性神經病變高危情境（需積極處理）

- 急性、疼痛性、快速或階梯式進展
- EMG 顯示 mononeuritis multiplex 模式
- 冷球蛋白+、低補體 C4、RF+、hypocomplementemia

### 自主神經病變評估工具

- 初篩問卷：**COMPASS-31**
- 正式測試：傾斜床試驗、**QSART**（定量汗腺軸突反射試驗）、心律變異性、Valsalva maneuver
- 懷疑 ANS SFN：皮膚切片測**汗腺神經纖維密度**
- 需排除其他原因：B12/B6、HbA1c、血清免疫固定電泳、自免性自律神經異常抗體組合（ACh 受體節神經抗體、ANNA-1、CASPR-2、CRMP-5、LGI1 等）

## 四、治療 — 以藥物為導向

### Gabapentinoid / SNRI（神經痛症狀治療）

**適用：所有類型神經病變的疼痛控制**（Strong recommendation）

- 一線：Gabapentin、Pregabalin、SNRI
- 二線：Na channel blocker（Carbamazepine、Oxcarbazepine、Lamotrigine）或 TCA（Nortriptyline、Desipramine）
- 三叉神經痛：一線直接用 **Carbamazepine / Oxcarbazepine**

### 類固醇

| 情境 | 用法 |
|---|---|
| 三叉神經病變（急性發炎） | 短療程口服 |
| 顏面神經麻痺 | ≥10 天 + 考慮加 Valacyclovir |
| 進展性多發性神經病變 | 試驗性治療 |
| **Ganglionopathy** | **高劑量，立即，搭配免疫抑制劑** |
| **血管炎性神經病變** | **高劑量，立即，搭配 RTX** |

### IVIG（免疫媒介、非血管炎）

| 適應症 | 說明 |
|---|---|
| 大纖維神經病變（免疫媒介） | 排除血管炎、CIDP 後考慮 |
| **小纖維神經病變（SFN）** | 近期發病、重症、進展中，症狀治療無效 → IVIG **≥3 個月** |
| **CIDP** | 按標準 CIDP 治療方案 |
| 重症 ANS 功能障礙 | 生活型態 + 藥物無效後使用 |

❌ **血管炎性神經病變不建議 IVIG 一線**

### Rituximab

| 適應症 | 說明 |
|---|---|
| **血管炎性神經病變（一線）** | 高劑量類固醇 + RTX 立即使用 |
| 重症 ANS 功能障礙 | IVIG 無效後的二線 |

### Cyclophosphamide

- 血管炎性神經病變 RTX 無效 → 類固醇 + CYC

### 自主神經病變（POTS）的特殊藥物

| 目的 | 藥物 |
|---|---|
| 心跳控制 | 非選擇性 Beta blocker 或 **Ivabradine**（HCN channel blocker） |
| 低血壓合併 POTS | **Midodrine、Droxidopa、Fludrocortisone** |
| 穩定心律+血壓 | **Pyridostigmine**（膽鹼酯酶抑制劑，促進自律神經傳導） |
| 非藥物 | 多喝水、增加鹽分、腹部/下肢壓力衣、坐姿/臥姿漸進運動（Strong） |

## 五、治療決策樹（快速索引）

```
神經病變性疼痛（任何類型）
└─ 一律先給 Gabapentinoid / SNRI

↓ 依機轉分流

血管炎性神經病變（急性疼痛性、MM、冷球蛋白+）
└─ 高劑量類固醇 + RTX（立即）→ 無效 → CYC

Ganglionopathy（感覺性共濟失調、不對稱深部感覺喪失）
└─ 高劑量類固醇 + 免疫抑制劑（緊急）

CIDP（對稱性近遠端無力、脫髓鞘 NCS）
└─ 標準 CIDP 治療（IVIG / 類固醇 / Plasmapheresis）

大纖維（免疫媒介，排除血管炎 & CIDP）
└─ 考慮 IVIG

SFN（NCS 正常，燒灼痛，進展中）
└─ 症狀治療無效 → IVIG ≥3 個月

ANS 功能障礙（POTS/姿勢性低血壓）
└─ 非藥物 → 藥物（Ivabradine/Midodrine 等）→ IVIG → RTX
```
