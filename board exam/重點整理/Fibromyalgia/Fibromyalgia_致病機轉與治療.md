# 纖維肌痛症（FM）— 致病機轉與治療

## 一、致病機轉（考試重點精簡版）

核心概念：**基因易感性 × 環境觸發 → 中樞疼痛處理失調**，而非組織破壞。FM 屬「中樞致敏化（central sensitization）」型慢性疼痛。

最重要的幾個機轉：

1. **中樞致敏化（central sensitization）= 核心**
   - **Allodynia**（無害刺激→痛）、**hyperalgesia**（痛刺激→更痛）。
   - **Temporal summation / wind-up**：疼痛神經元「預先致敏」，刺激間隔較長仍疊加，且回復延遲（aftersensations）。

2. **下行抑制不足（descending inhibition ↓）**
   - serotonin / norepinephrine / opioid 抑制路徑活性下降。
   - → 解釋 **SNRI（duloxetine、milnacipran）有效**的機轉。

3. **神經傳遞物失衡**
   - 興奮性 **glutamate/glutamine ↑**（後腦島等，MRS 證實）。
   - 抑制性 serotonin/NE 下行訊號 ↓。

4. **基因易感性**
   - 家族聚集（一等親 OR 8.5）、遺傳率約 14%。
   - 兩大路徑：**腎上腺素能（COMT）** 與 **血清素（HTR2A、5HT1a、SLC6A4）**。

5. **周邊疼痛來源 + 壓力系統**
   - **小纖維神經病變**等可作持續傷害性輸入（觸發/維持）。
   - **HPA 軸**與**自律神經失調（交感優勢、HRV 異常）**。
   - 心理：童年創傷/虐待、災難化思考（catastrophizing）。

> 神經影像佐證：腦血流對應「病人主訴疼痛」而非刺激強度 → 證明病人自述為真；PET 顯示 microglia/膠質活化。

## 二、治療（依 EULAR 2016/2017）

### 總原則

- **及早診斷＋衛教＋讓病人參與**；目標是改善 HRQL（生活品質）。
- **先以非藥物為主**；採漸進式、個別化。
- 藥物**低劑量起始、緩慢加量**，依「主要症狀」選藥。
- **多模式治療**（運動＋CBT＋藥物）效果最佳。
- 須讓病人知道：**沒有任何藥物能完全消除 FM 症狀**；盡量精簡用藥。

### 非藥物治療

| 強度 | 項目 |
|---|---|
| **Strong for（最強推薦）** | **有氧運動**（aerobic / 阻力訓練）— 唯一強推薦 |
| Weak for | CBT、多元療法、針灸、水療、冥想式運動（太極／瑜珈／氣功）、正念減壓（MBSR） |
| FDA 核准裝置 | TENS（Quell）、光生物調節（FibroLux） |
| 證據不足 | biofeedback、催眠、按摩 |
| **Strong against** | 整脊（chiropractic）、順勢療法（homeopathy） |

> 運動是基石；陸上 vs 水中無明確優劣（陸上對肌力較佳）；找到病人「能持續」的運動最重要。

### 藥物治療

**FDA 核准 3 種：pregabalin、duloxetine、milnacipran。**

| 藥物（類別） | 起始 | 目標 / 有效劑量 | 給法 | 副作用 / 備註 |
|---|---|---|---|---|
| **Duloxetine**（SNRI, FM核准） | 20–30 mg 早上 | **60 mg/day**（可至 120） | QD | 噁心、性功能障礙、體重增加、睡眠障礙；**血清素症候群**風險；肝功能不全禁用；起效 2–4 週 |
| **Milnacipran**（SNRI, FM核准） | 12.5 mg | **100 mg/day**（50 mg BID），可至 200 | BID | 噁心、心悸、血壓上升；血清素症候群風險；對睡眠障礙較有利 |
| **Pregabalin**（gabapentinoid, FM核准） | 75 mg | 一般 150–300；**FM 仿單可至 450 mg/day** | BID | 頭暈、嗜睡、口乾、**周邊水腫**、視力模糊、體重增加、思考遲鈍；自殺意念風險；欣快/濫用（**Schedule V**）；停藥需緩慢遞減 |
| **Gabapentin**（off-label） | ~300 mg | 疼痛 **1800 mg/day（上限）**；FM 試驗 1200–2400 | TID | 同 pregabalin（頭暈、嗜睡、水腫、體重增加、思考遲鈍、自殺意念）；非線性、滴定 10–14 天；癲癇才可至 3600 |
| **Amitriptyline**（TCA, off-label） | 5–10 mg 睡前 | **25–50 mg 睡前**（少數至 200） | QHS | **抗膽鹼**：口乾、便秘、鎮靜、體重增加；助眠、改善多症狀 |
| **Cyclobenzaprine**（肌鬆, off-label） | 5 mg 睡前 | **5–10 mg 睡前** | QHS | 鎮靜、口乾（結構近 TCA）；助眠 |
| **Tramadol ± acetaminophen**（弱鴉片） | — | **上限 400 mg/day** | — | 噁心、頭暈、**癲癇發作**、**血清素症候群**、依賴/濫用風險；效益存疑 |
| **Venlafaxine**（SNRI, off-label） | 37.5 mg | 37.5–225 mg/day | QD | 噁心、血壓上升（劑量相關）、血清素症候群；證據較弱 |
| SSRI（off-label） | — | 依藥而定 | QD | 止痛效果差；助憂鬱/焦慮；噁心、性功能障礙、體重增加 |

**不建議 / 無效：**

- **強鴉片類（strong opioids）不建議** — FM 患者 mu-opioid 受體已被佔據/下調，CSF enkephalin 升高；oxycodone 無證據。
- **NSAID 對中樞痛無效**（僅對周邊機械性/發炎性疼痛有幫助）。
- Cannabinoids：中度效果。

### EULAR 階梯式處置（Fig 52.3）

1. 診斷、衛教、個別化分級**有氧/肌力運動**。
2. 重新評估 → 加冥想式運動或其他物理治療。
3. 依主要問題加：
   - **嚴重疼痛/憂鬱焦慮/災難化** → CBT、心理治療。
   - **嚴重疼痛** → duloxetine、pregabalin、tramadol±acetaminophen。
   - **嚴重睡眠問題** → 低劑量 amitriptyline、cyclobenzaprine、睡前 pregabalin。
4. **嚴重失能/須請假** → 多模式復健計畫。
