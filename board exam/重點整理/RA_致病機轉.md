# RA 致病機轉完整整理

> 來源：Kelley's Textbook of Rheumatology, Ch.70 Etiology of RA、Ch.71 Pathogenesis of RA、Ch.56 Autoantibodies in RA

---

## 口試答題架構（開場定錨）

> 「RA 的致病機轉分三個階段：**前臨床期的免疫啟動、second hit 後的關節炎引發、以及慢性期的自我維持與組織破壞**。整個過程是遺傳易感性、環境因素、自體免疫三者交互作用的結果。」

---

## 第一步：遺傳 × 環境 → 打破免疫耐受

### 遺傳因素

| 基因 | 機制 | 臨床意義 |
|---|---|---|
| **HLA-DR Shared Epitope（SE）** | peptide-binding groove（位點 11、13）高效率呈現 citrullinated peptide 給 T cells；autoreactive T cells 在胸腺未被刪除 | 主要與 ACPA 陽性 RA 相關；gene dosage effect（2 個 SE allele → OR 11–13）|
| **PTPN22（1858 C/T）** | R620W 突變 → gain of function → 降低 TCR 訊號閾值；亦干擾 PAD4 的調控，促進 NETosis | OR ~2；與 RF 無關；PTPN22 + SE + anti-CCP 組合特異性接近 100% |
| **PADI4** | susceptible haplotype → mRNA 穩定性增加 → PADI4 蛋白量上升 → 更多 citrullination | 主要在亞洲族群；與 anti-PAD4 抗體相關 |

### 環境因素

| 因素 | 機制 |
|---|---|
| **吸菸**（最重要）| 促進肺部 PAD2 表達 → 黏膜 hypercitrullination；與 SE allele 協同作用（OR 20–40 倍）；至少需 20 pack-years；戒菸後風險緩慢下降（>10 年）|
| **黏膜假說** | RA 免疫反應最初在**肺、口腔、腸道**啟動，非關節；支持證據：sputum 中可偵測 anti-CCP IgA；早期 RA 有大量黏膜免疫相關 IgA ACPA |
| **感染** | *A. actinomycetemcomitans*：leukotoxin A → 嗜中性球 Ca²⁺ 內流 → hypercitrullination；CMV：誘導 PAD2/4 過度表達；*P. gingivalis*：爭議中（PPAD 為 exodeiminase，與 ACPA 辨識的 endocitrullination 不符）|
| **Epigenetics** | 吸菸改變 DNA methylation 模式；FLS 的 epigenetic imprinting（IL6、CXCL12 hypomethylation）使 FLS 持續促發炎；miRNA-155 過度表達促進 MMP 與細胞激素分泌 |

## 第二步：前臨床期 — ACPA 形成，但尚無關節症狀

- ACPA（IgG）可在發病前 **2–6 年**出現；IgA 約 14 年前、IgM 約 7 年前
- Dendritic cells 在黏膜將 citrullinated peptide 呈現給 T cells → T cells 活化 B cells → 產生 ACPA
- 隨時間發生 **epitope spreading**：ACPA 可辨識的 citrullinated antigens 種類逐漸增加
- 在 B cell 的 germinal center 中發生 **somatic hypermutation** → affinity maturation → 效價逐漸升高
- **為何有抗體卻無症狀？**
  - 效價低、特異性少
  - 尚未獲得 **pro-inflammatory Fc glycosylation**
  - 需要局部發炎觸發才能跨越閾值

| ACPA 特性 | 說明 |
|---|---|
| Multireactivity | 單一 ACPA 可辨識多種 citrullinated antigen，識別共同 motif（Cit-Gly、Arg-Cit-Asp）|
| Somatic hypermutation | 受 citrullinated antigen 持續驅動，有抗原驅動的 affinity maturation |
| Variable domain N-glycosylation | SHM 過程中出現；幫助 ACPA-producing B cells 逃脫 GC 的 negative selection |
| Cross-reactivity | 可交叉辨識 homocitrulline（carbamylation 產物）及 acetylated lysine |

## 第三步：Second Hit — 關節炎正式引發

### Mast Cell 作為 Second Hit 的執行者

```
循環中非特異性 Immune Complexes（不需要是 ACPA-specific）
        ↓
與滑膜 Mast Cells 上的 FcγR 結合
        ↓
Mast cells 脫顆粒 → histamine、tryptase 等血管活性物質釋放
        ↓
血管通透性急速上升
        ↓
血中大量 ACPA 進入關節腔
        ↓
ACPA 遇到軟骨表面 / 中性球 NETosis 釋出的 citrullinated proteins
        ↓
Immune Complexes 在關節內形成 → 正式引發 synovitis
```

### IC 啟動的三條主要路徑

| 路徑 | 機制 | 結果 |
|---|---|---|
| **Complement（Classical + Alternative）** | IC 活化 C1q → C3a、C5a 產生 | 血管通透性增加；嗜中性球與單核球大量招募 |
| **FcγR → 巨噬細胞活化** | IC 與巨噬細胞 FcγR 結合 + TLR4 協同 | TNF、IL-6、IL-1β 大量分泌 |
| **NLRP3 Inflammasome → IL-1β** | Danger signals（uric acid、ATP、細菌 MDP）或 TNF 活化 NLRP3 → caspase-1 → 切割 pro-IL-1β | IL-1β、IL-18 大量釋放；驅動發燒、軟骨 proteoglycan 降解、全身急性期反應 |

## 第四步：慢性期 — Feed-forward Loop 自我維持

### 核心循環

```
ACPA + Citrullinated proteins → Immune Complexes
        ↓
Complement 活化 + FcγR 活化 → TNF、IL-6、IL-1β
        ↓
嗜中性球大量浸潤（synovial fluid 中可達 50–75%）
        ↓
NETosis → 釋放更多 PAD enzymes + Citrullinated proteins
        ↓
Anti-PAD4 抗體在細胞外穩定並增強 PAD4 活性 → 更多 citrullinated antigens
        ↓
更多 ACPA 產生、更多 IC 形成 ↩（循環持續）
```

### 關節局部的自我維持機制

- **Tph Cells（T peripheral helper）**：CD4⁺ PD-1⁺ CXCR5⁻，位於滑膜淋巴聚集中，分泌 **IL-21** 支持 B cells 局部分化為 plasma cells → 持續局部產生 ACPA 與 RF
- **RF 作為 Amplifier**：RF 與含 ACPA 的 ICs 結合 → 透過 FcγR 放大巨噬細胞發炎反應 → 解釋 RF + ACPA 雙陽性患者預後更差
- **IFN Signature**：約 50% 早期 RA 患者有 type I IFN gene signature（由 TNF → mtDNA 釋出 → cGAS-STING pathway 觸發）→ 與初始治療反應下降相關

### Th17 / Treg 失衡

| | **正常** | **RA** |
|---|---|---|
| STAT3 | 適度活化 | 過度活化 → **促進 Th17 分化**、抑制 Treg |
| STAT5 | 促進 Treg 分化 | 相對不足 |
| Treg 功能 | 正常抑制自體免疫 | **功能受損**（anergy 表型；無法抑制 T cell / 巨噬細胞的細胞激素產生）|
| 結果 | Th17/Treg 平衡 | **IL-17↑、Treg 抑制功能↓** → 發炎持續 |

> TNF inhibitor 治療可部分恢復 Treg 功能；JAKi 透過抑制 JAK-STAT 軸調整 STAT3/STAT5 平衡

## 第五步：組織破壞

### 軟骨破壞 — FLS → Pannus → MMP

- **FLS（Fibroblast-like Synoviocytes）** 在 RA 中獲得**類腫瘤自主侵入特性**（epigenetic imprinting，即使移除發炎環境仍維持）
- TNF 與 FLS 上的 TNFR1 結合 → NF-κB + AP-1 活化 → 上調 MMP-1/3/13、IL-6、RANKL
- FLS 以 **Cadherin-11** 互相黏附 → 形成侵入性 **Pannus** 組織
- MMP-1（Collagenase-1）、MMP-13（Collagenase-3）直接降解 type II collagen；MMP-3（Stromelysin）降解 proteoglycan

| MMP | 別名 | 主要受質 |
|---|---|---|
| MMP-1 | Collagenase-1 | Type I, II, III collagen |
| MMP-3 | Stromelysin-1 | Proteoglycan、fibronectin、活化其他 MMP |
| MMP-13 | Collagenase-3 | Type II collagen（軟骨中最重要）|

### 骨侵蝕 — RANK/RANKL/OPG 軸

| 分子 | 來源 | 功能 |
|---|---|---|
| **RANKL** | Th17 T cells、FLS、滑膜巨噬細胞 | 與 osteoclast precursor 上的 RANK 結合 → 驅動 osteoclastogenesis |
| **RANK** | Osteoclast precursors | 接受 RANKL 信號 → 分化為成熟 osteoclast |
| **OPG（Osteoprotegerin）** | FLS、成骨細胞 | Decoy receptor，競爭性抑制 RANKL-RANK 結合 |

- RA 中 **RANKL/OPG 比值升高** → 過度 osteoclastogenesis → 骨侵蝕
- **TNF**：直接刺激 FLS、巨噬細胞表達 RANKL；可繞過 RANKL，直接活化 RANK 下游 NF-κB
- **IL-17（Th17）**：協同 TNF 增強 RANKL 表達；促進中性球招募放大局部發炎
- **ACPA**：直接與 osteoclast precursor 表面的 citrullinated vimentin 結合 → 活化 FcγR → 加速 osteoclast 前驅分化 → 解釋 ACPA 陽性患者骨侵蝕更嚴重

## 細胞激素網絡總覽

| 細胞激素 | 主要來源 | 主要效果 | 對應治療 |
|---|---|---|---|
| **TNF** | 巨噬細胞 | 活化 FLS（MMP↑、RANKL↑）、血管新生、全身發炎 | TNF inhibitors（etanercept、adalimumab 等）|
| **IL-6** | 巨噬細胞、FLS | 急性期反應（CRP↑）、Th17 分化↑、B cell 活化、RANKL↑ | Tocilizumab（anti-IL-6R）|
| **IL-1β** | 巨噬細胞（NLRP3 → caspase-1）| 軟骨 proteoglycan 降解、發燒、急性期反應 | Anakinra（IL-1Ra）|
| **IL-17（Th17）** | CD4⁺ T cells | 協同 TNF 增強 MMP 與 RANKL；中性球招募 | IL-17 inhibitors（ixekizumab 等）|
| **IL-21** | Tph cells | 支持 B cell 分化為 plasma cells → 持續 ACPA/RF 產生 | — |
| **GM-CSF** | 多種細胞 | 促進 DC、巨噬細胞成熟與活化 | Mavrilimumab（anti-GM-CSFR）|

## Seronegative RA vs. Seropositive RA

| | **Seropositive（ACPA+）** | **Seronegative（ACPA−）** |
|---|---|---|
| 主要驅動機制 | Adaptive immunity（ACPA-IC-T/B cell axis）| **Innate immunity**（TLR、巨噬細胞）主導 |
| HLA-SE 關聯 | 強 | 弱 |
| 滑膜病理 | 淋巴聚集多、B cells 多、plasma cells 多 | Myeloid cell 主導、淋巴聚集少 |
| 前臨床期 | 有抗體前驅期（數年）| 無，onset 較突然 |
| 骨侵蝕 | 較嚴重 | 相對輕微 |
| Rituximab 反應 | **較好** | **較差**（非 B cell 依賴）|
| TNFi / IL-6i 反應 | 有效 | 同樣有效 |

## 口試常見追問與口袋答案

| 追問 | 簡答 |
|---|---|
| ACPA 單獨能引發關節炎嗎？ | 不行。Monoclonal ACPA 在健康小鼠只能引起 arthralgia、bone loss，需要 second hit（如 LPS）才能引發完整 synovitis |
| Second hit 是什麼？ | 非特異性 IC 活化滑膜 mast cells → 血管通透性↑ → ACPA 進入關節 → 遇到 citrullinated antigen → IC 形成 → 補體活化 |
| RF 在 RA 中的角色？ | 與含 ACPA 的 ICs 結合 → 透過 FcγR 放大巨噬細胞發炎反應，是 ACPA 致病的 amplifier |
| 為何前臨床期有抗體卻無症狀？ | 效價低、特異性少、尚未獲得 pro-inflammatory Fc glycosylation；需 epitope spreading + 局部發炎觸發才跨越閾值 |
| 為何 ACPA 陽性患者骨侵蝕更嚴重？ | ACPA 直接活化 osteoclast precursor（FcγR pathway）；ACPA 陽性患者有更完整的 RANKL-driven osteoclastogenesis |
| Tocilizumab 為何能快速降 CRP 但臨床效果可能不完全？ | IL-6 阻斷後急性期反應快速消失，但 TNF、IL-17、FLS 自主活化等路徑仍持續，RA 是多細胞激素疾病 |
| JAKi 的機轉連結？ | JAK-STAT 抑制 → STAT3↓ → Th17 分化減少、Treg 功能恢復；同時抑制多個細胞激素（IL-6、JAK1/2 下游的 TNF、IFN 等）的信號傳遞 |
| 為何 seronegative RA 對 rituximab 反應差？ | 其致病以 innate immunity / 巨噬細胞為主，不依賴 B cell-produced ACPA；B cell depletion 無法打斷主要致病路徑 |
| Filaggrin 為何不是真正的 RA autoantigen？ | 關節腔中無 filaggrin 表達，它只是 surrogate antigen；真正致病靶點是 fibrin、vimentin、α-enolase |
| Smoking 怎麼連進 RA？ | 肺部 PAD2↑ → 黏膜 hypercitrullination → 誘發 IgA ACPA；在帶 SE allele 者中協同作用 → OR 升高 20–40 倍 |

## 整體流程圖（口試心智地圖）

```
遺傳（HLA-SE、PTPN22、PADI4）× 環境（吸菸、感染、黏膜）
                ↓
        黏膜 citrullination
                ↓
    DC 呈現 citrullinated peptide → T cell 活化
                ↓
        B cell 活化 → ACPA 形成
        （前臨床期：2–6 年，無症狀）
                ↓
    Second hit（Mast cells + IC → 血管通透性↑）
                ↓
    ACPA 進入關節 → IC 形成
    ┌──────────────────────────────┐
    │ Complement 活化（C3a、C5a）  │
    │ FcγR → 巨噬細胞活化          │→ TNF / IL-6 / IL-1β
    │ NLRP3 Inflammasome → IL-1β   │
    └──────────────────────────────┘
                ↓
        急性關節炎引發
                ↓
    Feed-forward loop
    ├── NETosis → 更多 citrullinated Ag → 更多 ACPA
    ├── Tph → B cell → 局部 ACPA 持續產生
    ├── RF 放大 IC → 巨噬細胞持續活化
    └── Th17↑ / Treg 功能↓（STAT3 過度活化）
                ↓
    慢性組織破壞
    ├── FLS → Pannus → MMP-1/3/13 → 軟骨破壞
    └── RANKL/OPG 失衡 + ACPA 直接活化 → Osteoclast → 骨侵蝕
```
