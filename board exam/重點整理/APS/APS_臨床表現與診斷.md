# APS 臨床表現與診斷 Criteria

## 一、臨床表現

### 總覽：APS 的表現譜

```
無症狀 aPL 陽性
    ↓
微血管病變（Microvascular APS）
    ↓
大血管血栓（Macrovascular thrombosis）
    ↓
CAPS（多器官血栓，數天內發生）
```

---

### 1. 大血管血栓（Macrovascular）

**靜脈**
- **DVT + PE** 最常見
- 不尋常位置：Budd-Chiari syndrome、矢狀竇血栓（sagittal sinus）、上肢靜脈

**動脈**
- **Stroke** 最常見（尤其年輕患者）
- 特徵：反覆發作、不尋常位置、發生在年輕人
- 腎動脈血栓 → 嚴重高血壓
- 周邊動脈血栓

---

### 2. 微血管病變（Microvascular）

| 器官 | 表現 |
|------|------|
| 皮膚 | **Livedo racemosa**（病理性）；livedoid vasculopathy（下肢潰瘍）|
| 腎臟 | aPL nephropathy（TMA）→ 蛋白尿、血尿、高血壓（~3% aPL陽性）|
| 肺 | Diffuse alveolar hemorrhage（DAH）（2–12%）|
| 心臟 | 冠狀動脈微血管血栓 → 心肌梗塞（無明顯阻塞性冠心病）|
| 腎上腺 | 腎上腺梗塞/出血 → 急性腎上腺功能不全（CAPS 的早期表現）|

---

#### Livedo Reticularis vs Livedo Racemosa

兩者外觀相似，但意義完全不同：

| | Livedo Reticularis | Livedo Racemosa |
|--|---|---|
| 網狀圖案 | **完整封閉**的圓圈 | **破碎、不完整**的弧形 |
| 分布 | 對稱 | 不對稱 |
| 顏色 | 均勻 | 不均勻 |
| 遇熱/復溫後 | **消失**（可逆）| **不消失**（不可逆）|
| 意義 | 生理性血管收縮，**非病理** | 真正的微血管栓塞，**病理性** |

**記憶方式**：
- Reticularis → 完整網格 → 冷縮熱張，加溫消失 → 生理
- Racemosa → 破碎弧形 → 血管已受損，不可逆 → APS、SLE、Sneddon syndrome

> 看到下肢**破碎、不規則、不會消退**的網狀紫斑 → 想到 APS

---

### 3. 非血栓性表現（Nonthrombotic）

- **血小板低下**：~20% aPL 陽性患者，通常 ≥50,000/mm³，輕中度
- **心臟瓣膜病變**：Libman-Sacks endocarditis（vegetations，<4mm，無菌性）；瓣膜增厚
- **肺高壓**：反覆肺栓塞或小血管血栓所致
- **神經系統**（罕見且有爭議）：認知障礙、舞蹈症、癲癇、MRI 白質病變
- **LA hypoprothrombinemia syndrome**（極罕見）：高親和力 anti-prothrombin Ab → 耗盡 prothrombin → **出血**（APS 中例外的出血表現）

---

### 4. 產科表現（Obstetric）

| 表現 | 特點 |
|------|------|
| **胎兒死亡（≥10週）** | 最具特異性，尤其伴隨血管功能不全徵象 |
| 嚴重子癇前症（<34週）| 尤其早發性 |
| 嚴重胎盤功能不全（<34週）| 胎兒生長遲滯、羊水過少 |
| 連續 ≥3 次早期流產（<10週）| 特異性較低，需排除染色體等原因 |

> <10 週的單次流產特異性低，常見原因是染色體異常，非 APS 首要診斷  
> PROMISSE 研究：**LA 陽性**是 >12 週後不良妊娠結果的最強預測因子

---

### 5. CAPS（Catastrophic APS）

- 發生率 <1% APS 患者，但**死亡率高**
- 特徵：**數天內**出現多器官血栓（腎、肺、腦、心、皮膚最常見）
- 常伴隨 TMA（血小板低下 + schistocytes）
- 誘發因素：感染、手術、藥物、**停用抗凝藥**
- 急性腎上腺功能不全 = CAPS 的高度提示表現

**CAPS 鑑別診斷**：
- Sepsis + DIC
- TTP / HUS
- Heparin-induced thrombocytopenia（HIT）
- 心房黏液瘤栓塞
- 動脈粥樣硬化斑塊栓塞

---

## 二、診斷 Criteria

### 2023 ACR/EULAR APS Classification Criteria

**設計理念**：加權積分系統，取代舊版 Sapporo（2006）的二元是/否判斷  
**目的**：研究用分類，特異性提升至 99%（vs Sapporo 86%），敏感性 84%

---

#### Entry Criteria（必須先符合）

> 至少一項臨床 criterion，**加上**在該臨床事件發生前後 3 年內有至少一次 aPL 陽性

---

#### 臨床 Domains（共 6 個，需累計 ≥3 分）

| Domain | 項目 | 分數 |
|--------|------|------|
| **1. 靜脈血栓（VTE）** | 高 VTE 風險背景下的 VTE | 1 |
| | 無高 VTE 風險背景 | 3 |
| **2. 動脈血栓（AT）** | 高 CVD 風險背景下的 AT | 2 |
| | 無高 CVD 風險背景 | 4 |
| **3. 微血管** | 疑似（suspected）| 2 |
| | 確立（established）| 5 |
| **4. 產科** | ≥3 次連續 <10 週或 10–15w6d 的流產 | 1 |
| | 16–33w6d 胎兒死亡（無嚴重 PEC/PI）| 1 |
| | <34w 嚴重 PEC 或嚴重 PI | 3 |
| | <34w 嚴重 PEC **且** 嚴重 PI | 4 |
| **5. 心臟瓣膜** | 瓣膜增厚 | 2 |
| | Vegetation | 4 |
| **6. 血液** | 血小板低下（20–130×10⁹/L）| 2 |

---

#### 檢驗 Domains（共 2 個，需累計 ≥3 分）

| Domain | 項目 | 分數 |
|--------|------|------|
| **7. LA（功能性）** | 單次陽性 | 1 |
| | 持續陽性（persistent）| **5** |
| **8. aCL / aβ₂GPI（ELISA，需 persistent）** | 中高效價 IgM（aCL 或 aβ₂GPI）| 1 |
| | 中效價 IgG（aCL 或 aβ₂GPI）| 4 |
| | 高效價 IgG（aCL 或 aβ₂GPI，擇一）| 5 |
| | 高效價 IgG（aCL **且** aβ₂GPI）| **7** |

**確診條件**：臨床 domains ≥3 分 **且** 檢驗 domains ≥3 分

---

#### aPL 效價定義

| 效價 | 定義（GPL/MPL units）|
|------|---------------------|
| 低效價 | <40 |
| 中效價 | 40–80 |
| 高效價 | >80 |

> **≥40 = moderate-to-high**，才算 criteria aPL（2023 ACR/EULAR 標準）  
> 考試重點：記住 **40 是門檻**，高效價 IgG aCL 且 aβ₂GPI 同時陽性 = 最高分（7分）

---

#### 重要概念

- 風險分層：同樣是「血栓」，有無傳統危險因子，**分數不同**（強調 aPL 本身的致病貢獻）
- IgM 單獨陽性分數極低 → 反映其臨床意義不確定
- aPL 需持續陽性（≥12 週間隔兩次）

---

### CAPS Classification Criteria

| 條件 | |
|------|--|
| 1 | ≥3 個器官/系統/組織受累 |
| 2 | 同時或 <1 週內發生 |
| 3 | 至少 1 處組織切片證實小血管阻塞 |
| 4 | aPL 實驗室確認陽性 |

**Definite CAPS**：符合全部 4 項  
**Probable CAPS**：
- 符合 2–4 項且僅 2 器官受累
- 符合 1–3 項但患者早死無法等待 6 週確認
- 符合 1、2、4 項
- 符合 1、3、4 項但第三個事件在 1 週–1 個月後才出現

---

## 三、診斷注意事項

### aPL 陽性不等於 APS

| 情況 | 解讀 |
|------|------|
| 低效價、一過性 | 感染誘發（梅毒、HIV、Borrelia）→ 通常 IgM，不算 criteria |
| 藥物誘發 | chlorpromazine、procainamide、phenytoin → β₂GPI-independent |
| 孤立 IgM 陽性 | 臨床意義不確定，謹慎解讀 |
| 健康人群 | aPL 盛行率約 10%，但持續中高效價 <1% |

### LA 假陽性 / 假陰性

- 接受抗凝治療時（warfarin、heparin、DOAC）**LA 結果不可靠**
- 嚴重發炎狀態也可干擾結果

### Seronegative APS

少數患者臨床表現典型但 criteria aPL 全陰 → 可考慮 non-criteria aPL：
- Anti-Domain I aβ₂GPI
- aPS/PT（anti-phosphatidylserine/prothrombin）
- IgA isotypes

目前仍主要用於研究，尚未納入常規檢測
