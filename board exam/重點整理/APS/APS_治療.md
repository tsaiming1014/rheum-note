# APS 治療與預後（Kelly Ch83 + 2019 EULAR）

> 來源：Kelly's Textbook of Rheumatology Ch83；Tektonidou MG, et al. *Ann Rheum Dis* 2019;78:1296–1304（2019 EULAR APS management recommendations）。
> EULAR 提供或強調的內容標註 **(2019 EULAR)**。

---

## 〇、aPL profile 風險分層 (2019 EULAR)

- **Medium-high titre**：aCL > 40 GPL/MPL（或 > 99th percentile）；anti-β2GPI > 99th percentile
- **High-risk profile**（間隔 ≥12 週、≥2 次）：LA 陽性、double 或 triple 陽性、或持續高效價
- **Low-risk profile**：單獨 aCL 或 anti-β2GPI、low-medium titre，尤其暫時性陽性
- 治療強度依 **aPL profile + 事件類型（venous/arterial）+ 出血/復發風險** 決定

---

## 一、Management — 治療

### 1. Macrovascular Thrombosis

#### (1) Primary prevention

- 無症狀 aPL 陽性者不建議常規抗凝。
- **Low-dose aspirin (LDA)**：唯一 RCT 顯示對無症狀者預防首次血栓證據不強；但 **(2019 EULAR)** 仍建議以下族群用 **LDA 75–100 mg/day**：

  - 無症狀 aPL carrier + **high-risk profile**
  - SLE + high-risk profile（low-risk profile 則「可考慮」）
  - 非孕期、僅 obstetric APS 病史（評估風險效益後）

- **DOAC**：無試驗，不建議。
- **HCQ**：對 aPL 陽性 SLE 具保護作用；單純 aPL 陽性者角色未明。
- **Statin**：下調 prothrombotic biomarker，高 CVD risk 者可能有角色。
- **風險分層工具**：aPL-S、GAPSS。
- **一般措施 (2019 EULAR)**：嚴格控制 CV risk（戒菸、控血壓/血脂/血糖、運動）；**高風險情境（手術、住院、長期臥床、產褥期）用 prophylactic LMWH**。
- 避免 estrogen 及含 estrogen 口服避孕藥。

#### (2) Secondary prevention — Venous

- **急性期**：heparin（UFH 或 LMWH）。
- **長期**：VKA，target **INR 2–3**（INR 3–4 無額外效益）。
- **療程 (2019 EULAR)**：unprovoked 首次 → 長期抗凝；provoked 首次 → 依一般族群 guideline 療程（low-risk + 可逆誘因可僅 3–6 個月），high-risk profile 可延長。
- **DOAC**：**Rivaroxaban 禁用於 triple-positive**（TRAPS 因動脈血栓過多提早中止）；ASTRO-APS（apixaban）stroke 較多 → DOAC 目前不建議用於 thrombotic APS，僅 **(2019 EULAR)** 在 VKA 順從下仍無法達標或 VKA 禁忌時考慮（不可因順從性差而換 DOAC）。
- **復發（INR 2–3 下仍復發）(2019 EULAR)**：先查順從性、頻繁監測 INR；達標仍復發 → 加 **LDA**、INR 升至 **3–4**、或改 **LMWH**。

#### (3) Secondary prevention — Arterial

- **首次 arterial (2019 EULAR)**：VKA 優於單用 LDA；VKA **INR 2–3 或 INR 3–4**（依風險權衡），或 INR 2–3 + LDA。
- **Rivaroxaban 禁用於 triple-positive + 動脈事件**；不建議 DOAC 用於 APS 動脈事件。
- stroke 依 aPL profile 與 CV risk 分層；老年、單一 low-titer aCL 的 stroke → aspirin 單用可能等效（APASS，適用性有限）。
- **復發 arterial（VKA 下仍復發）(2019 EULAR)**：排除其他病因後 → INR 升至 **3–4**、加 **LDA**、或改 **LMWH**。
- 註：dual therapy（warfarin + aspirin）增加出血；LA 干擾 INR 時改用 anti–factor Xa 監測。

### 2. Microvascular APS（非 CAPS）

- 無統一準則；部分給予抗凝但效益未明。
- 免疫抑制：**rituximab**（RITAPS）、belimumab、MMF、azathioprine、cyclophosphamide、sirolimus。
- **DAH**：glucocorticoid 可誘導緩解，約半數需 steroid-sparing；cyclophosphamide- 或 rituximab-based 最有效。
- 門診經驗方案：glucocorticoid（至 60 mg/day）+ MMF ± rituximab；難治加 IVIG；通常加 HCQ + statin。

### 3. Catastrophic APS（CAPS）

- 發作突然、立即危及生命，早期診斷至關重要。
- **預防 (2019 EULAR)**：及早治療感染、避免中斷抗凝/低 INR。
- **第一線**：**glucocorticoid + heparin + plasma exchange 或 IVIG** 三聯（優於單藥/其他組合），同時處理誘因（感染、壞疽、惡性腫瘤）。
- **難治**：**rituximab** 或 **eculizumab**（補體抑制）；合併 systemic TMA 時尤其考慮補體抑制（CAPS Registry 39 例 eculizumab → 74% 恢復）。

### 4. Pregnancy Morbidity

- **Warfarin 致畸** → 孕期用 UFH 或 **LMWH（首選）**。
- **僅 obstetric APS（無血栓）**：

  - ≥3 次 <10 週流產，或 ≥1 次 ≥10 週胎兒死亡 → **LDA + prophylactic heparin**（胎兒存活率 50%→80%）
  - 因子癇/重度子癇前症/胎盤功能不全 <34 週生產 → LDA 或 LDA + prophylactic heparin
  - Non-criteria obstetric APS → LDA ± heparin 可考慮
  - High-risk profile 但無併發症 → 孕期 LDA 可考慮

- **Thrombotic APS 病史** → 孕期 **LDA + therapeutic dose heparin**（全劑量整個孕期）。
- **難治（LDA + prophylactic heparin 下仍反覆）**：heparin 升至 **therapeutic dose**、加 **HCQ**、或第一孕期加 **low-dose prednisolone**；選擇病例可考慮 **IVIG**。
- **產後**：續 heparin 6 週（thrombotic 病史者 8–12 週）防母體血栓，再 taper。

## 二、Outcome — 預後（Kelly）

- **不良預後因子**：pulmonary hypertension、神經侵犯、myocardial ischemia、nephropathy、四肢壞疽、CAPS。
- **Primary APS 長期功能差**：10 年時 1/3 永久器官損害，1/5 無法執行日常活動。
- **CAPS**：即使積極治療死亡率仍 >30%（cerebrovascular、cardiac、感染）；高達 1/3 復發。
- 長期 APS：嚴重瓣膜病需換瓣、罕見 TMA 致腎衰竭；移植腎可因即刻血栓喪失。
- **手術期**：圍手術期血栓風險高，須術前規劃、盡量縮短停抗凝時間。
