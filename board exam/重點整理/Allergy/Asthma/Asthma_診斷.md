# Asthma 診斷（GINA 2026）

> 資料來源：GINA 2026 Strategy Report Section 1（[Ch01_Definition_Description_Diagnosis.md](../../../../Non-autopush/GINA/chapters_md/Ch01_Definition_Description_Diagnosis.md)）

---

## 一、診斷核心邏輯

診斷奠基於兩大支柱，**缺一不可**：

1. **典型且會變動的呼吸道症狀病史**（wheeze、呼吸困難、胸悶、咳嗽）
2. **客觀證實的可變呼氣氣流限制（variable expiratory airflow）**——通常需要 bronchodilator responsiveness test 或其他肺功能檢查佐證

> **重要原則：Test before treating**——盡量在開始 ICS-containing 治療**之前**確診，因為治療後症狀與肺功能變異性會下降，之後才想確診會更困難。

---

## 二、Box 1-2：成人（≥18歲）與兒童（6–17歲）初診斷標準

### 1. 典型可變呼吸道症狀病史

支持氣喘診斷的症狀特徵：

- 症狀隨時間變動、強度也會變化
- 常在夜間或清晨較嚴重
- 常由運動、大笑、過敏原、冷空氣誘發
- **運動停止後症狀反而惡化**（非常具特異性）
- 常因病毒感染而出現或惡化

**降低氣喘可能性**的特徵：慢性咳痰、合併頭暈/肢體麻木的呼吸困難、胸痛、運動誘發呼吸困難合併吸氣時有雜音、局部性（單側）wheeze。

### 2. 確認可變呼氣氣流限制（符合以下任一項）

| 檢查方式 | 陽性標準 |
|---|---|
| **Bronchodilator responsiveness（BD reversibility）** | 給予 200–400mcg salbutamol 後10–15分鐘測量：**成人** FEV1或FVC較基礎值增加 ≥12% 且 ≥200mL（≥15%且≥400mL更具信心）；無 spirometry 時 PEF↑≥20%。**兒童** FEV1 ≥12% predicted（或PEF↑≥15%） |
| **兩週內每日兩次 PEF 變異度過大** | 成人：diurnal PEF variability 平均 >10%；兒童 >13% |
| **4週 ICS 治療後肺功能上升** | 成人 FEV1↑≥12%且≥200mL（或PEF↑≥20%）；兒童 FEV1↑≥12% predicted |
| **Bronchial provocation test 陽性** | 成人：methacholine使FEV1↓≥20%，或標準化hyperventilation/高滲食鹽水/mannitol使↓≥15%，或標準化運動誘發試驗↓>10%且>200mL；兒童：methacholine↓≥20%，或運動試驗↓>12% predicted |
| **不同回診間肺功能變異過大**（specificity高、sensitivity低） | 成人：FEV1變異≥12%且≥200mL（或PEF≥20%）；兒童：FEV1≥12%（或PEF≥15%） |

> BD test 執行細節：需停用支氣管擴張劑（SABA≥4小時；formoterol/salmeterol 24小時；indacaterol/vilanterol 36小時；tiotropium/umeclidinium/aclidinium/glycopyrronium 36–48小時）再測。

### Type 2 Biomarker 輔助角色

若無法做 spirometry/PEF，或檢查為陰性，但症狀典型：

- **FeNO**：成人/青少年 >50 ppb、兒童 >35 ppb → 支持 Type 2 asthma 診斷
- **血中 eosinophil** 高於當地參考值上限 → 支持診斷

⚠️ 這兩者**可因非氣喘疾病而升高**，**數值低也不能排除氣喘**（因會受晝夜、性別、年齡、裝置等多重因素影響）。

## 三、診斷流程（Box 1-1 概念）

適用於**慢性或反覆呼吸道症狀**病人（急性惡化的診斷/處置見 Box 9-4、9-6）：

```
病史+身體檢查 → 症狀型態是否典型氣喘？
    ↓ 是
做 spirometry（無法做則用PEF）測 BD responsiveness
    ├─ 陽性 → 確診氣喘，開始治療
    └─ 陰性但高度懷疑 → 重複檢查、加做 bronchial provocation test，
                        或考慮 4週ICS治療trial後複測肺功能
```

## 四、身體檢查與其他輔助檢查

- **理學檢查通常正常**；最常見異常為吐氣時 wheezing（expiratory rhonchi），但可能只在用力吐氣時聽到；嚴重惡化時反而可能因氣流嚴重受限而**無 wheeze（"silent chest"）**
- **Crackles 與吸氣時 wheeze 不是氣喘特徵**（提示其他診斷）
- **Bronchial provocation test**（methacholine等）：sensitivity中等但specificity有限——陰性可排除氣喘（未用ICS病人），陽性不等於氣喘
- **過敏檢測**（SPT或specific IgE）：支持 allergic asthma，但非氣喘特異性
- **Imaging**：不用於常規診斷，僅在 difficult-to-treat asthma 疑似其他共病時使用（HRCT可見mucus plugging、鼻竇CT可評估CRSwNP）

## 五、鑑別診斷重點（Box 1-3，依年齡分層）

| 年齡層 | 需考慮的替代診斷（依症狀特徵） |
|---|---|
| 6–11歲 | 上呼吸道咳嗽症候群、異物吸入（單側wheeze+突發）、支氣管擴張症、原發性纖毛不動症、先天性心臟病、囊狀纖維化 |
| 12–39歲 | 上呼吸道咳嗽症候群、GERD/喉部誘發阻塞（inducible laryngeal obstruction）、過度換氣、支氣管擴張症、α1-antitrypsin缺乏 |
| ≥40歲 | COPD、心衰竭、藥物性咳嗽（ACEi）、肺實質病變、肺栓塞（急性呼吸困難+胸痛）、中央氣道阻塞 |
| 所有年齡 | 肺結核（慢性咳嗽+咳血+體重減輕）、百日咳 |

## 六、已經在用 ICS 的病人如何確診？（Box 1-4）

臨床上約 **25–35%** 曾被診斷氣喘的病人，之後無法用客觀檢查再確認診斷——這是 GINA 特別強調「治療前先確診」的原因。若病人已在用藥：

- 症狀典型+已有可變氣流限制證據 → 直接確診
- 症狀典型但無氣流限制證據 → 停藥後重測BD test；FEV1>70%可考慮**降階ICS**後2–4週再評估（Box 1-5：25–50%減量或停用，2–4週後重新評估，症狀惡化+氣流限制重現=確診）；FEV1<70%則考慮先升階治療1–2個月
- 症狀不典型+無氣流限制 → 改查其他鑑別診斷

## 七、特殊族群診斷考量

| 族群 | 重點 |
|---|---|
| **慢性咳嗽為唯一症狀**（cough variant asthma） | Spirometry常正常，唯一異常可能只有 bronchial provocation test 顯示 AHR；多合併痰中eosinophilia、FeNO升高 |
| **職業性氣喘** | 詢問「離開工作環境（週末/休假）症狀是否改善」是關鍵篩檢問題；需客觀確診（法律/社經影響大），常需專科轉介+上下班PEF監測 |
| **懷孕婦女** | 初步BD test未確診但臨床高度懷疑時，直接以ICS治療氣喘處置，其餘檢查延至產後；孕期**禁做** bronchial provocation test，也不建議降階ICS |
| **老年人** | 常被低診斷；需與心血管疾病/左心衰竭鑑別（心電圖、胸部X光、BNP、心臟超音波可能有幫助） |
| **吸菸者/戒菸者** | 需與COPD/asthma+COPD overlap鑑別；現行吸菸者FeNO偏低、血中eosinophil偏高 |
| **肥胖病人** | 務必以客觀肺功能檢查確認，避免過度診斷（研究顯示非肥胖與肥胖病人被過度診斷比例相近，約30%） |
| **低資源地區/中低收入國家** | Spirometry常不可得，改用PEF meter；GINA**不建議**單純以症候群式臨床判斷確診，仍建議至少用PEF佐證 |
