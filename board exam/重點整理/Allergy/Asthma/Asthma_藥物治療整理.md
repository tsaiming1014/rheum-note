## 一、治療階梯總覽（Track 1 vs Track 2, Step 1–5）

成人／青少年氣喘治療分兩條 **Track**，差別在 reliever（緩解藥）選擇：

| | **Track 1（優先）** | **Track 2（替代）** |
|---|---|---|
| Reliever | 低劑量 ICS-formoterol（AIR，抗發炎緩解藥） | ICS-SABA 或單純 SABA |
| 適用時機 | GINA 建議優先路徑，降低嚴重惡化風險 | Track 1 不可行，或病人已穩定、adherence良好、過去一年無惡化 |

| Step | Track 1（ICS-formoterol為主） | Track 2（ICS-SABA/SABA為主） |
|---|---|---|
| **1–2** | As-needed 低劑量 ICS-formoterol（無需每日 maintenance） | Step1: as-needed ICS-SABA；Step2: 每日低劑量 ICS + as-needed ICS-SABA/SABA |
| **3** | 低劑量 ICS-formoterol **MART**（maintenance 1吸BID + as-needed） | 每日低劑量 ICS-LABA + as-needed ICS-SABA/SABA |
| **4** | 中劑量 ICS-formoterol MART（maintenance 2吸BID） | 中劑量 ICS-LABA + as-needed ICS-SABA/SABA |
| **5** | 轉介專科，考慮 add-on：高劑量ICS-LABA試用、LAMA、**生物製劑**（見第四節） | 同左 |

## 二、ICS（Inhaled Corticosteroids）— 治療核心

### 機轉

脂溶性 corticosteroid 穿過細胞膜 → 結合細胞質 **glucocorticoid receptor (GR)** → 複合體入核 →

- **Transactivation**：促進抗發炎蛋白轉錄（β2-receptor↑、annexin-1↑）
- **Transrepression（主要機轉）**：抑制 **NF-κB、AP-1**，下調 IL-4、IL-5、IL-13、GM-CSF、TNF-α 等發炎基因表現，促進 eosinophil 凋亡

**下游效果**：減少氣道發炎細胞浸潤、降低氣道高反應性、改善症狀肺功能、降低惡化風險。

> 對比：全身性類固醇（OCS）作用機轉相同，但因全身暴露劑量大，副作用（骨鬆、糖尿病、白內障、憂鬱、肥胖等）遠高於局部吸入劑型。

### 臨床原則

- 診斷後應**盡早開始**（延遲>2–4年才治療，肺功能改善較差、後續需要劑量較高）
- 不建議 SABA-only；輕症也建議 as-needed low-dose ICS-formoterol（AIR）取代單純 SABA
- 多數療效在**低劑量**即可達成；adherence 差是最大挑戰，這是 GINA 偏好 **MART** 的理由之一

### 劑量分級（Box 4-2，每日 metered dose，mcg）

**成人／青少年（≥12歲）**

| ICS | Low | Medium | High |
|---|---|---|---|
| Beclometasone dipropionate（standard particle pMDI） | 200–500 | >500–1000 | >1000 |
| Beclometasone dipropionate（extrafine particle DPI/pMDI） | 100–200 | >200–400 | >400 |
| Budesonide（DPI/pMDI） | 200–400 | >400–800 | >800 |
| Ciclesonide（extrafine pMDI） | 80–160 | >160–320 | >320 |
| Fluticasone furoate（DPI） | 100 | — | 200 |
| Fluticasone propionate（DPI/pMDI） | 100–250 | >250–500 | >500 |
| Mometasone furoate（pMDI） | 200–400 | — | >400 |

**兒童 6–11歲**

| ICS | Low | Medium | High |
|---|---|---|---|
| Beclometasone dipropionate（standard particle） | 100–200 | >200–400 | >400 |
| Beclometasone dipropionate（extrafine particle） | 50–100 | >100–200 | >200 |
| Budesonide（DPI/pMDI） | 100–200 | >200–400 | >400 |
| Ciclesonide（extrafine pMDI） | 80 | >80–160 | >160 |
| Fluticasone propionate（DPI/pMDI） | 50–100 | >100–200 | >200 |
| Mometasone furoate（pMDI） | 100 | — | 200 |

> ⚠️ 此表非等效劑量對照表；同級劑量換藥仍可能有 potency 落差，換藥後須觀察病人穩定度。實際劑量選擇需依當地藥品仿單與健保給付條件為準。

### 副作用

| 類型 | 內容 |
|---|---|
| 局部（Local） | 口咽念珠菌感染、聲音沙啞（spacer+漱口可降低風險） |
| 全身性（Systemic，長期高劑量） | 骨質疏鬆、白內障、青光眼、腎上腺抑制 |
| 交互作用 | CYP3A4抑制劑（ketoconazole、ritonavir、itraconazole、macrolides）↑ICS全身濃度及副作用風險 |

給藥裝置：優先 DPI，或 pMDI + spacer（避免 facemask/nebulizer 增加皮膚眼周暴露）。

## 三、支氣管擴張劑（Bronchodilators）— SABA / LABA / SAMA / LAMA

### β2-agonist（SABA、LABA）

β2-agonist → **β2-adrenergic receptor**（Gs-coupled）→ ↑adenylyl cyclase → **cAMP↑** → PKA活化 → 抑制MLCK →平滑肌鬆弛（bronchodilation）；並輕度抑制 mast cell mediator釋放。

| | SABA | LABA |
|---|---|---|
| 代表藥物 | Salbutamol、Terbutaline | Formoterol、Salmeterol、Vilanterol、Indacaterol |
| 起效 | 快（~5分鐘） | Formoterol快；Salmeterol慢（20–30分） |
| 時效 | 4–6小時 | 12–24小時 |
| 臨床角色 | Reliever，不建議單獨使用 | 僅能與ICS併用；formoterol因起效快可用於MART/AIR |
| 副作用 | 顫抖、心搏過速 | 同左，較少見 |

### Antimuscarinic（SAMA、LAMA）

ACh作用於氣道平滑肌 **M3 receptor**（Gq-coupled）→ PLC → IP3/Ca²⁺↑ → 收縮。Antimuscarinic藥物**競爭性阻斷M3** → 阻止收縮、減少黏液分泌。

| | SAMA | LAMA |
|---|---|---|
| 代表藥物 | Ipratropium、Oxitropium | Tiotropium、Glycopyrronium、Umeclidinium |
| 時效 | 6–8小時 | 24小時（tiotropium：對M3解離慢、M2解離快） |
| 臨床角色 | 急性嚴重惡化時加在SABA之後使用 | Step 5 add-on（triple therapy ICS-LABA-LAMA），需先確保ICS達medium–high dose |
| 副作用 | 口乾、苦味 | 口乾為主 |

**重點比較**：β2-agonist走**Gs→cAMP↑**促進鬆弛；antimuscarinic走**阻斷Gq**路徑阻止收縮，兩者機轉互補，可合併使用（急性期SABA+ipratropium；Step5 triple therapy）。**SABA/LABA皆不可單獨使用。**

## 四、生物製劑（Step 5 Add-on Biologics）

### 使用前提

Step 4治療（≥high-dose ICS-LABA）後仍未控制，且確認為 **allergic/eosinophilic Type 2-high** 或需要 maintenance OCS。

**Type 2 inflammation 評估標準**（high-dose ICS或daily OCS下）：blood eosinophils ≥150/μL、FeNO ≥20 ppb、sputum eosinophils ≥2%、或臨床allergen-driven。

### 四大類比較

| 類別 | 藥物 | 機轉 | 劑量 | Response predictors | 其他適應症 | 試用期 |
|---|---|---|---|---|---|---|
| **Anti-IgE** | Omalizumab | 結合游離IgE Fc端，阻止與FcεR1結合，下調受體表現 | SC Q2–4W依體重/IgE | 兒童期發病、allergen-driven病史；基礎IgE不能預測反應 | CRSwNP、CSU、食物過敏 | ≥4個月 |
| **Anti-IL5/5Rα** | Mepolizumab、Depemokimab（中和IL-5）；Benralizumab（結合IL-5Rα，ADCC清除eosinophil） | 阻斷IL-5存活訊號 vs 直接清除eosinophil | Mepo 100mg SC Q4W；Benra 30mg Q4W×3後Q8W | 高eosinophil（強）、過去一年惡化次數多（強）、鼻息肉、baseline OCS | Mepo: EGPA/HES/CRSwNP；Benra: EGPA | ≥4個月 |
| **Anti-IL4Rα** | Dupilumab | 結合IL-4Rα，同時阻斷IL-4與IL-13訊號 | 200–300mg SC Q2W | 高eosinophil、高FeNO（皆強） | AD、CSU、CRSwNP、eos-COPD | ≥4個月 |
| **Anti-TSLP** | Tezepelumab | 結合TSLP，作用在Th2/ILC2活化之前 | 210mg SC Q4W | 高eos/FeNO（強，但biomarker低者仍可能有效）、鼻息肉 | CRSwNP | ≥4個月 |

**選藥考量**：payer eligibility、Type 2共病（AD/鼻息肉可一藥兩用）、allergen病史、上表predictors、藥價/頻率/途徑、病人偏好。

**反應評估**：3–4個月評估（不明確可延至6–12個月）；無反應→換用其他Type2製劑；有反應→先減OCS，其他藥物依序調整，ICS不可完全停用。

## 五、整體用藥邏輯圖

```
Step 1–2：ICS-formoterol as-needed（或ICS-SABA as-needed）
    ↓ 症狀增加
Step 3：低劑量 MART / ICS-LABA maintenance
    ↓
Step 4：中劑量 MART / ICS-LABA maintenance
    ↓ 仍未控制（已排除inhaler technique/adherence/暴露問題）
Step 5：轉介專科 → 評估Type 2 biomarker
    ├─ 無Type2證據 → LAMA、azithromycin、非藥物選項
    └─ 有Type2證據 → 選擇生物製劑（IgE/IL5/IL4Rα/TSLP）
                    ± add-on LAMA（triple therapy）
                    ± 短期高劑量ICS-LABA trial
```

## 六、Step-up 前必查項目（避免不必要的升階治療）

GINA 反覆強調：**症狀未控制不等於需要加藥**，升階前必須先排查以下可逆因子：

- Inhaler technique 是否正確
- Adherence（遵從性）是否良好
- 是否有持續暴露：過敏原、菸害、室內/室外空污、藥物（beta-blocker、部分病人的 NSAIDs）
- Comorbidity 是否造成類似症狀
- 診斷是否正確（differential diagnosis）

### 依危險因子分類的處置策略（Box 3-5 節錄）

| 危險因子 | 處置策略 |
|---|---|
| 任何惡化風險因子（含症狀控制不佳） | 確保用 ICS-containing治療；換用 AIR（ICS-formoterol/ICS-SABA）reliever；提供書面 action plan；更頻繁回診；常規檢查 inhaler technique/adherence |
| 過去一年 ≥1次嚴重惡化 | 換 AIR reliever；若無可逆因子，考慮加 LAMA、增加ICS劑量（尤其Type2 biomarker高者）、轉介專科評估生物製劑 |
| 菸/電子菸暴露 | 鼓勵戒菸；若控制不佳可考慮提高ICS劑量 |
| FEV1低（尤其<60%預測值） | 先處理adherence/technique；可考慮3個月高劑量ICS trial；排除COPD等其他肺病 |
| 肥胖 | 提供減重策略；區分氣喘症狀與 deconditioning/睡眠呼吸中止症狀 |
| 重大心理問題 | 轉介精神科評估；協助區分焦慮症狀與氣喘症狀 |
| 社經弱勢 | 選擇當地成本最低的ICS方案；優化吸入技巧以最大化現有藥物效益 |
| 確診食物過敏 | 避免特定食物；備妥 anaphylaxis action plan/腎上腺素；轉介專科 |
| 職業/居家刺激物暴露 | 儘速脫離暴露源；轉介專科 |
| 致敏後過敏原暴露無法避免 | 考慮升階治療；HDM致敏且部分控制者可考慮加 SLIT（FEV1>70%預測值者） |
| 中/高劑量ICS下FeNO仍高 | 先確認並改善adherence（直接觀察治療下約2/3病人FeNO會被壓下，顯示先前adherence不佳） |

## 七、非藥物治療總覽（Box 3-6）

| 介入措施 | 建議重點 |
|---|---|
| 戒菸/戒電子煙、避免二手菸 | 每次回診皆應鼓勵戒菸/戒電子煙；提供諮商/戒菸計畫資源；家長不應在小孩使用的房間/車內吸菸；有>10 pack-year菸史者需評估COPD/asthma+COPD overlap |
| 規律運動 | 鼓勵規律運動（一般健康益處，非氣喘專屬效益，游泳對青少年例外有肺功能助益）；運動誘發支氣管收縮可用維持性ICS或運動前 as-needed ICS-formoterol預防；突破性症狀可用暖身、SABA或低劑量ICS-formoterol於運動前使用 |
| 肺復健（Pulmonary rehabilitation） | 4–12週結構化計畫，對症狀控制效果有限，但可改善短期運動耐受性與生活品質；運動耐受度差或有持續性氣流受限相關呼吸困難者可轉介 |
| 避免職業/居家過敏原或刺激物暴露 | 詢問所有成人發病氣喘病人之工作史；儘速排除職業致敏物暴露；疑似職業性氣喘應儘速轉介專科 |
| 避免使可能使氣喘惡化的藥物 | 開立NSAIDs前應詢問氣喘病史，惡化即停用；Aspirin/NSAIDs非一般禁忌（除非有過敏反應病史，見AERD）；口服/眼用beta-blocker需個案評估，由專科醫師密切監督下起始；心血管急症需要cardioselective beta-blocker時氣喘非絕對禁忌 |
| 健康飲食 | 鼓勵高蔬果飲食（一般健康效益） |
| 避免室內過敏原 | 不建議作為一般策略；單一策略對多數病人臨床效益有限；居家除濕/除霉可減少成人症狀與用藥量；HDM/寵物致敏之避免策略證據有限（僅兒童部分有效）；策略通常複雜昂貴且缺乏預測誰會受益的驗證方法 |
| 減重 | 納入肥胖病人治療計畫；減重+每週兩次有氧/肌力訓練優於單純減重；減重成效最佳者為減重手術 |
| 呼吸訓練（Breathing exercises） | 可作為藥物治療的有用輔助，改善症狀與生活品質，但不降低惡化風險、對肺功能無一致效果 |
| 避免室內空污 | 使用非污染性暖氣/烹飪設備，污染源儘量排至室外 |
| 避免室外過敏原 | 花粉/黴菌濃度高時關窗、留在室內、使用空調（致敏病人） |
| 處理情緒壓力 | 協助病人找出因應策略；放鬆訓練/呼吸訓練可能有幫助；有焦慮/憂鬱症狀者安排心理衛生評估 |
| 處理社會風險因子 | 美國研究顯示綜合性社會風險介入可減少兒童ED就診與住院 |
| 避免室外空污/極端天氣 | 空污嚴重或極端低溫時儘量留在室內、避免劇烈室外運動；病毒感染期間避免污染環境 |
| 避免特定食物/食品化學物 | 除非經口服激發試驗確診過敏/敏感，否則不建議常規避免食物；確診食物過敏者避免可能減少惡化 |

**疫苗接種**：流感、RSV、COVID-19疫苗除了降低一般族群重症風險外，也可降低病毒誘發的氣喘惡化，建議常規接種。

## 八、轉介專科時機（Box 3-8）

| 情境 | 細節 |
|---|---|
| 診斷確認困難 | 疑似慢性感染/心因性或其他非肺部病因（建議立即轉介）；ICS或全身性類固醇治療後診斷仍不明確；同時具氣喘與COPD特徵且治療優先順序不確定 |
| 疑似職業性氣喘 | 轉介確認測試、鑑定致敏/刺激物、脫離暴露建議 |
| 持續或嚴重未控制/頻繁惡化 | Step 4治療（medium-dose ICS-LABA）且inhaler technique正確、adherence良好下，症狀/惡化/肺功能仍未改善；頻繁使用急性醫療資源 |
| 氣喘相關死亡危險因子 | 過去曾近乎致命氣喘發作（ICU/使用呼吸器）；疑似或確診anaphylaxis/食物過敏合併氣喘 |
| 治療副作用證據或風險 | 顯著治療副作用；需要長期口服類固醇；一年內≥2次OCS療程 |
| 疑似氣喘亞型/併發症 | 如 AERD（aspirin-exacerbated respiratory disease）、ABPA |
| 兒童6–11歲額外轉介理由 | 早產兒診斷存疑；medium-dose ICS加正確技巧/adherence仍未控制；疑似治療副作用（如生長遲滯）；對兒童福祉/健康有疑慮 |
