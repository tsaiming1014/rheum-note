# Systemic Sclerosis 治療策略 — EULAR 2026 HOT Session 重點整理

> 講者:Prof. Oliver Distler(Zurich)|主題:器官導向治療、生物標記能否選藥、新興療法

## 一、核心架構:三大主軸

1. **器官導向治療(Organ-based treatment)** — 除自體幹細胞移植(HSCT)外,SSc 目前**無系統性疾病修飾藥物(DMARD)**,治療仍以個別器官為單位。
2. **治療策略選擇** — 生物標記(血清 + 影像)能否協助選藥?
3. **新興療法(Emerging therapies)** — 尚未納入指引的最新試驗數據。

---

## 二、器官導向評估與治療(EULAR Recommendations)

逐一評估各器官:①是否存在 → ②是否需治療(與病人共享決策)→ ③選擇對應藥物。

| 器官表現 | 治療要點 | 證據/陷阱 |
|---|---|---|
| **雷諾現象 (Raynaud's)** | 僅在病人受困擾且接受藥物時治療;鈣離子通道阻斷劑為首選 | 無證據顯示雷諾本身促進纖維化,不需一律治療 |
| **指端潰瘍 (Digital ulcers)** | 曾發生者復發風險最高;預防用 bosentan;已存在用 PDE5i ± bosentan;部分國家用 IV iloprost | — |
| **肺動脈高壓 (PAH)** | 用 DETECT 等多參數評分篩檢 → 右心導管確診;確診後**首選上游合併治療**(PDE5i + ERA) | 單靠心臟超音波偽陽性高;合併治療對死亡率優於單一治療 |
| **肌肉骨骼(關節炎/肌炎)** | 缺乏高證據;以嘗試錯誤為主 | 無 SSc 專屬臨床試驗證據 |
| **皮膚纖維化** | 早期、進展中或長期仍嚴重者需治療 | — |
| **間質性肺病 (ILD)** | 證據最充分(見下) | — |
| **胃腸道 (GI)** | 逆流→ PPI(強專家意見);促腸蠕動劑證據低(C 級) | — |
| **腎危象 (Renal crisis)** | 一旦發生盡早給 ACEi | **不可預防性使用**(會掩蓋高血壓徵象、延誤診斷) |

> **類固醇警語:** SSc 應避免高劑量類固醇 — 增加腎危象風險。

## 三、SSc-ILD 第一線藥物(ACR/CHEST 跨指引一致)

| 藥物 | 地位 |
|---|---|
| MMF(mycophenolate) | ILD 與肺外表現的泛用良好選擇 |
| Tocilizumab | 特定表現型證據極強(見下) |
| Rituximab | 對皮膚 + 肺皆有效 |
| Cyclophosphamide | 對皮膚 + 肺皆有效 |
| Nintedanib | 抗纖維化,SSc-ILD 證據佳 |
| (Azathioprine) | 僅 ACR 因美國保險可近性納入,**無療效試驗證據** |

- ACR 以「head-to-head 投票」將 MMF 列為首選,但 **EULAR 不採用「一體適用」排序**,主張依個別病人選藥。

## 四、生物標記能否選藥?→ 尚未成熟(Not ready for prime time)

**高證據推薦使用某標記選藥,須同時滿足:**

1. Responder vs non-responder 在 baseline 有不同數值
2. 多變項回歸中具預測性(贏過其他預測因子)
3. 獨立世代驗證(single study is no study)
4. 前瞻性策略試驗證實依標記選藥結果較佳

| 候選標記 | 結果 |
|---|---|
| Whitfield 基因表現分型(inflammatory / fibroproliferative / normal-like) | SCOT 試驗中 fibroproliferative 對 HSCT 反應明顯,但僅滿足條件①,未達多變項/獨立世代/前瞻驗證 |
| HRCT 上的 UIP / 纖維化 pattern | **不**預測抗纖維化反應 |
| 血清發炎標記(CRP) | **不**作為排除 nintedanib 的依據 |

**關鍵反駁「發炎型不該用抗纖維化藥」的迷思:**

- **SENSCIS** 次分析:nintedanib 在早期/發炎型病人**效應量更大**(發炎標記高者 52 vs 41 mL;更嚴重者 95 mL vs 整體 41 mL)。
- **INBUILD**:UIP vs non-UIP 的 95% CI 大幅重疊 → UIP pattern 不影響 SSc-ILD 反應(RA-ILD 則不同)。
- 結論:**SSc-ILD 不應依 UIP 或發炎標記決定是否用抗纖維化藥**。

## 五、EULAR/ERS 建議的選藥策略(取代生物標記選藥)

**三步驟:**

1. **風險分層(差預後風險因子)** — 越多 → 越積極 / 越早合併治療
2. **評估肺外表現** — 是否影響肺部用藥選擇
3. **證據可外推性** — 試驗收案表現型能否套用到眼前病人

### SSc-ILD 差預後風險因子
- 年長、男性、非裔美國人
- 持續性發炎
- anti-Topo I / anti-Scl-70 陽性
- baseline PFT(較高者進展機率高;極低者連結死亡率)
- HRCT 上 ILD 範圍較廣
- 近期皮膚快速進展(預測未來肺部進展)

### 各藥物的表現型與外推
| 藥物 | 表現型 / 證據 |
|---|---|
| **Nintedanib** | SENSCIS(600 人,NEJM)、INBUILD 支持;**對皮膚纖維化無效** → 若需治皮膚須加藥 |
| **Tocilizumab(focuSSced)** | 收案僅占真實世界 SSc-ILD 約 **1.6%**(早期瀰漫型、發炎/CRP或ESR↑、近期皮膚進展);此表現型效果極佳,但**無法外推至無發炎型** |
| **Cyclophosphamide / Rituximab** | 對皮膚 + 肺皆有效(多器官受累時選用) |
| **MMF** | ILD 與肺外泛用 |

### 治療建議總結
- 高進展風險(多項風險因子)→ **nintedanib + MMF 上游合併**(耐受性佳、療效更好);或免疫抑制劑合併。
- 皮膚 + 肺多器官受累 → **cyclophosphamide 或 rituximab**。
- 纖維化 >10% 且無肺外受累(含發炎亞群)→ **nintedanib**。
- 早期瀰漫型 + 發炎 + 近期皮膚進展 → **tocilizumab**(較強推薦)。

## 六、新興療法(>20 項 RCT 進行中)

### 1. Nerandomilast(PDE4B 抑制劑)
- **FIBRONEER 試驗**:Phase 3,1,000 人,三臂(兩劑量 vs placebo);主要終點 1 年 FVC 變化。
- 收 PPF(進展型肺纖維化)病人,40% 併用 nintedanib;**不允許免疫抑制劑**。
- 結果:整體 +80/+60 mL,顯著;自體免疫 ILD 亞群一致顯著。
- **次要終點:死亡訊號**(自體免疫亞群死亡率較 placebo 低,未跨越零線)— 此為與其他藥物的關鍵區別。
- **機轉契合 SSc 三大主軸**:抑制細胞激素釋放(免疫)、降低肌纖維母細胞分化/ECM 合成(纖維化)、穩定內皮屏障(血管)→ 有潛力成為疾病修飾藥。
- **靶點驗證**:PDE4B 在 SSc 皮膚/肺/PBMC 的髓系發炎細胞高表現;FAP+ 肌纖維母細胞約 90% 表現 PDE4B。
- 進行中 3 項 RCT(SSc-ILD 平台試驗、整體疾病修飾、cTD-ILD)。
- (註:小分子 avenciguat 試驗為**陰性**。)

### 2. CAR-T(CD19)
- **BREAKFREE-SSc(BMS),Phase 1b**(>29 人):早期(<5 年)、高 MRSS、近期皮膚進展、進展型 ILD、≥1 種免疫抑制劑反應不佳。
- 不良反應:多為輕度 CRS;ICANS 26%,皆非嚴重。
- 療效(10 名 SSc-ILD):FVC **3 個月 +2.3%、6 個月 +6%**;MRSS 3 個月降 7.5%、6 個月降 13%(但高基線 MRSS 易自然回歸,須謹慎)。
- 已啟動含對照組 RCT(BMS 80 人、Novartis 類似)。

### 3. T-cell engagers(雙特異抗體,如 teclistamab,BCMA × CD3)
- 優勢:可 IV/SC 給藥,**不需 CAR-T 的細胞製備基礎設施**(CAR-T 昂貴、僅限特定中心)。
- BCMA 涵蓋更廣 B 細胞(含 plasmablast、漿細胞)。
- Düsseldorf(Jörg Distler)5 名病人:皮膚改善、肺穩定;**抗體(含 anti-topoisomerase)明顯下降**(CD19 CAR-T 對此較難);肌腱摩擦音與 EUSTAR 活動指數降低。
- 因合併 rituximab,**感染事件較多**。全球約 50 名 SSc 接受治療(多為摘要/海報)。

## 七、Q&A 重點

| 問題 | Distler 回應 |
|---|---|
| Anti-centromere 限制型、輕微 ILD/皮膚、難治雷諾的血管病變病人,要免疫抑制嗎? | 依器官評估;anti-centromere 為**負向**風險因子。ILD 在此族群較少見但出現時仍會進展,是限制型 SSc 第三大死因。有近期進展或 >10% ILD 才治療;不建議用免疫抑制治療血管病變(僅觀察性數據)。 |
| MMF 血中濃度監測判斷反應? | SSc 反應難定義(自然病程 25% 改善、50% 穩定、25% 進展);以**有無進展**判斷,每年重新評估風險因子。 |
| FAPI/PET 選 nintedanib? | 看似有潛力(Erlangen 數據),但可近性受限。 |
| FIBRONEER 為何不允許免疫抑制劑? | 肺科醫師想看單獨療效;Distler 同意此設計使結果難解讀,故後續三試驗開放併用免疫抑制劑。 |
| nintedanib 能否與 cyclophosphamide/rituximab 併用? | **僅** nintedanib + MMF 有證據;其他組合並非不可行,但無證據。 |
| CAR-T 皮膚改善是逆轉纖維化或發炎? | MRSS 測皮膚厚度;SSc 皮膚發炎不足以解釋增厚 → 為**纖維化回歸**(分子數據支持);但需與安慰劑自然回歸區分。 |
| 為何 TCE 比 CD19 CAR-T 更能降 anti-topoisomerase? | BCMA 涵蓋更廣 B 細胞(含 plasmablast);但 B 細胞作用不僅止於抗體,勿僅以抗體下降解讀療效。 |
| 是否提早同時針對三主軸治療? | 須權衡效益與毒性;合併血管+發炎+抗纖維化缺乏 RCT 證據,目前不建議常規使用。 |

## 重點結論

- SSc 治療目前為**器官導向**,逐一評估治療需求並協調各器官用藥。
- **生物標記(CRP、UIP pattern)尚不足以選藥** — 無真實證據。
- 替代策略:看**證據等級 + 臨床表現型 + 多器官需求 + 風險因子**;風險因子多則更積極。
- 新興療法重點:**PDE4B 抑制劑(nerandomilast,含死亡訊號)** 與 **B 細胞清除療法(CAR-T / T-cell engager)**,皆待 RCT 驗證。
