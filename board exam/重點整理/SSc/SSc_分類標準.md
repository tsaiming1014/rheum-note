# SSc 分類標準演進與詳解（1980 ACR → 2013 ACR/EULAR；含亞型分類）

## 總覽

| 標準 | 年份 | 計分方式 | 達標條件 | 特點 |
|------|------|----------|----------|------|
| ACR（preliminary） | 1980 | 1 主 + 3 次，無權重 | **1 主要 或 ≥2 次要** | 對**早期/侷限型敏感度差**（漏 lcSSc、早期病人） |
| **ACR/EULAR** | **2013** | **加權計分** | **總分 ≥ 9**（或單一充分條件） | 納入**抗體、甲褶微血管、Raynaud**，敏感度大幅提升 |
| LeRoy/Medsger 亞型 | 1988 | — | 依**皮膚侵犯範圍** | 分 **lcSSc / dcSSc**（加 sine scleroderma），決定預後與監測 |

> 演進邏輯：1980 標準只看皮膚與肺纖維化，**早期、limited、sine scleroderma 都會漏**；2013 把 **Raynaud、甲褶微血管異常、SSc 專一抗體、PAH/ILD** 納入加權，敏感度與特異度都顯著提升。

---

## 一、1980 ACR Preliminary Criteria

| 類型 | 項目 |
|------|------|
| **主要（major）** | 近端硬皮（skin thickening **proximal to MCP/MTP**，即超過掌/蹠指關節） |
| **次要（minor）** | ① 指端硬化（sclerodactyly）② 指尖凹陷性疤痕或指腹組織流失 ③ 兩側肺底纖維化 |

**計分規則：** 符合 **1 項主要** **或** **≥2 項次要** 即分類為 SSc。每項等值、無權重。**缺點**：必須已有明顯皮膚變化，對 **limited type、早期病人、sine scleroderma 敏感度低**。

## 二、2013 ACR/EULAR Criteria（現行主流）

### 加權計分表

| 項目（category） | 子項 | 分數 |
|------------------|------|------|
| **充分條件（單項即達標）** | **雙手手指皮膚增厚並延伸至 MCP 近端（proximal to MCP）** | **9（足以分類，不必再加總）** |
| 手指皮膚增厚（取**最高分**一項） | 手指浮腫（puffy fingers） | 2 |
| | 指端硬化（sclerodactyly，MCP 遠端至 PIP 近端） | 4 |
| 指尖病灶（取**最高分**一項） | 指尖潰瘍（digital tip ulcers） | 2 |
| | 指尖凹陷性疤痕（pitting scars） | 3 |
| 微血管擴張（telangiectasia） | | 2 |
| 甲褶微血管異常（abnormal nailfold capillaries） | | 2 |
| 肺侵犯（此類**最多 2 分**） | 肺動脈高壓（PAH） | 2 |
| | 間質性肺病（ILD） | 2 |
| Raynaud 現象 | | 3 |
| SSc 專一抗體（此類**最多 3 分**） | anti-centromere、anti-topoisomerase I（Scl-70）、anti-RNA polymerase III | 3 |

**計分規則：**

- **充分條件**：雙手手指皮膚增厚延伸至 MCP 近端 = **9 分**，**單獨即可分類為 SSc**，不必再加總其他項目。
- 否則加權加總，**總分 ≥ 9 → 分類為 SSc**。
- 某些類別（手指皮膚、指尖病灶）**同類只取最高分一項**。
- **肺侵犯類最高 2 分、抗體類最高 3 分**（不論幾個抗體陽性）。
- **不適用**於：皮膚增厚**未犯手指**者，或有更合理解釋之 scleroderma-like 疾病（如腎源性全身硬化症、嗜伊紅筋膜炎等）。

> 考點：**Raynaud（3）+ 任一 SSc 抗體（3）+ 甲褶異常（2）+ puffy fingers（2）= 10 分**即達標，凸顯 2013 對**早期病人**的捕捉力。

## 三、亞型分類（LeRoy & Medsger 1988）— 預後關鍵

| 亞型 | 皮膚侵犯範圍 | 代表抗體 | 內臟特徵 |
|------|--------------|----------|----------|
| **lcSSc**（侷限型） | 限**肘/膝遠端 + 臉**（軀幹不犯） | **anti-centromere (ACA)** | CREST、**晚發 PAH**、ILD 較少；含舊稱 CREST |
| **dcSSc**（瀰漫型） | 延伸至**肘/膝近端、軀幹** | **anti-Scl-70、anti-RNA pol III** | **早期 ILD、腎危象（renal crisis）**、進展快 |
| **SSc sine scleroderma** | **無皮膚增厚** | 各 SSc 抗體 | 有內臟侵犯 + Raynaud + 甲褶異常/抗體 |

> CREST = Calcinosis、Raynaud、Esophageal dysmotility、Sclerodactyly、Telangiectasia，屬 lcSSc 範疇。

## 四、SSc 抗體與臨床關聯（高頻考點）

### 主要抗體一覽表

| 抗體 | 靶抗原 | 盛行率（SSc 中）| 疾病亞型 | 主要器官受累 | 預後 |
|------|--------|:--------------:|---------|------------|------|
| **Anti-Scl-70（Anti-Topo I）** | DNA topoisomerase I | ~25–40% | **dcSSc** | 嚴重 ILD、皮膚、SRC、關節炎、心臟 | 差（尤其非裔） |
| **ACA（Anti-centromere）** | CENP-A/B/C | ~20–40% | **lcSSc** | CREST、晚期 PAH；ILD 少 | 相對良好 |
| **Anti-RNA pol III** | RNA polymerase III | ~15–25% | **dcSSc** | 快速皮膚病變、SRC（25–40%）、GAVE、癌症相關 | 差 |
| Anti-U3-RNP（fibrillarin） | U3-RNP | ~4–8% | dcSSc/lcSSc | PAH、嚴重 GI、SRC；多見於非裔 | 差 |
| Anti-Th/To | Th/To RNP | ~2–5% | **lcSSc** | 嚴重 ILD、PAH | 差 |
| Anti-PM/Scl | PM/Scl complex | ~3–8% | Overlap | Myositis、ILD | 中等 |
| Anti-Ku | Ku protein | ~1–4% | Overlap | 肌肉、關節 | 中等 |
| Anti-U1-RNP | U1-RNP | ~5–10% | Overlap/MCTD | 多發性關節炎、myositis | 中等 |

> 註：ACA 在 lcSSc 中盛行率可達 40–60%；Anti-Scl-70 在 dcSSc 中可達 ~40%；各文獻因族群不同略有差異

## 五、補充：VEDOSS（極早期診斷）

- **VEDOSS**（Very Early Diagnosis of SSc）概念：當病人有 **Raynaud + puffy fingers + ANA 陽性**，應進一步查 **SSc 專一抗體 + 甲褶微血管鏡（nailfold capillaroscopy）**。
- 若抗體陽性或甲褶出現 scleroderma pattern（巨大微血管、出血、稀疏）→ **高度懷疑早期 SSc**，可在器官受損前介入監測。

---

## 六、一句話總整理

- **分類** → **2013 ACR/EULAR**：手指越 MCP 近端 = 9 分（單項達標）；否則加權 **總分 ≥9**（抗體最多 3、肺最多 2、同類取最高）。
- **亞型** → **lcSSc（ACA、PAH）vs dcSSc（Scl-70→ILD、RNA pol III→腎危象/癌）**。
- **早期** → **VEDOSS**：Raynaud + puffy + ANA → 查抗體 + 甲褶微血管鏡。
