# Neonatal Lupus (NLE)

> 依 Kelley's Textbook of Rheumatology, Ch110 整理

## 定義與分類

- NLE 是**獨立於 SLE 的疾病實體**，是少數會在**新生兒期/嬰兒早期**表現的風濕疾病之一
- **被動自體免疫疾病**：母體循環中的自體抗體透過胎盤傳給胎兒
- **⚠️ 目前沒有 NLE 的診斷標準（no diagnostic criteria available）**

## 基因與致病機轉

- 絕大多數 NLE 與 **anti-Ro/SS-A、anti-La/SS-B** 相關；**anti-U1RNP 抗體**也有關聯
- 帶有這些抗體的母親，只有 **1-2%** 的孩子會發生 NLE——代表光有抗體並不足以致病
- 主流假說：胎兒心臟發育過程中，Ro/La 抗原會暴露在**心肌細胞表面**，成為母體抗體攻擊的標的，引發發炎反應損傷傳導系統，部分甚至導致心肌病變
- **心臟的病變不可逆**，但其他發炎相關變化會隨著母體抗體從嬰兒體內清除而緩解，**通常在 6 個月內**

## 臨床表現

- **皮疹**：non-scarring malar/annular rash，好發臉、頭皮、軀幹，通常在出生後**前 6 週內**出現，可被 UV 光誘發，組織學上類似 **subacute cutaneous lupus**
- 其他：免疫性血球低下、肝炎（transaminitis、肝腫大）
- **最常見且最嚴重的併發症：心臟侵犯**
  - 傳導異常最早可在**懷孕 16 週**（母體 IgG 開始可穿越胎盤時）就出現在子宮內，或出生後才發現
  - 範圍從 **PR interval 延長 → complete heart block (CHB)**
  - 也可能發生 **endocardial fibroelastosis、進行性心肌病變**
  - **再發率：17%**

## 治療

### 非心臟表現

通常自行緩解不需治療；皮疹可用外用類固醇；嚴重血球低下可用 IVIG。

### 心臟侵犯的偵測與預防

- 建議 anti-Ro/SSA 或 anti-La/SSB 陽性孕婦，從**懷孕 16 週開始**監測胎兒 PR interval 與心臟功能
- **孕期使用 HCQ 可降低胎兒 cardiac NLE 風險**——前瞻性開放性研究顯示可將 CHB 發生率降至歷史發生率的 **50%**
- **IVIG 預防性使用（400mg/kg，每 3 週一次，第二孕期）證據不明確（unconvincing）**
- 抗發炎劑量 IVIG（1-2 g/kg）預防 NLE 尚未被研究

### 胎兒已發生 NLE 的治療

- **Fluorinated corticosteroid（如 dexamethasone、betamethasone，母體使用）對 2nd degree heart block 有一定療效證據**——因為 fluorinated steroid 可穿越胎盤
- IVIG 也可用於 2nd degree heart block（相關臨床試驗進行中）
- **一旦進展到 3rd degree heart block，抗發炎治療已無效**（通常不可逆）
- **β-agonist** 可用於提升胎兒心跳（若合併 fetal bradycardia）
- Fluorinated CS ± IVIG（產前、產後）可改善 NLE cardiomyopathy/endocardial fibroelastosis 的存活率
- 部分完全性心傳導阻滯的孩子需要**心律調節器**

## 重點速記

| 項目 | 數值/重點 |
|---|---|
| 相關抗體 | Anti-Ro/SSA、anti-La/SSB（最主要）、anti-U1RNP |
| 抗體陽性母親的 NLE 發生率 | 1-2% |
| CHB 佔新生兒 CHB 比例 | 80-95% |
| 再發率（後續妊娠） | 17% |
| 監測起始時間 | 懷孕 16 週 |
| HCQ 預防效果 | 降低 CHB 至歷史發生率的 50% |
| 2nd degree CHB 治療 | Fluorinated corticosteroid（dexamethasone/betamethasone） |
| 3rd degree CHB | 通常不可逆，抗發炎治療無效，可能需心律調節器 |
