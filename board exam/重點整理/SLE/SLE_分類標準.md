# SLE 分類標準演進與詳解（1997 ACR → 2012 SLICC → 2019 EULAR/ACR）

## 總覽

| 標準 | 年份 | 計分方式 | 達標條件 | 敏感度 / 特異度 |
|------|------|----------|----------|------------------|
| ACR（revised） | 1997 | 11 項，**每項 1 分、無權重** | **≥ 4 項**（可不同時發生） | 敏感度較低（早期/輕症易漏）、特異度佳 |
| SLICC | 2012 | 臨床 11 + 免疫 6，**每項 1 分、無權重** | **≥ 4 項，且 ≥1 臨床 + ≥1 免疫**；或**腎切片證實狼瘡腎炎 + ANA 或 anti-dsDNA 陽性** | 敏感度↑（94%）、特異度略↓（92%） |
| **EULAR/ACR** | **2019** | **入門條件 + 加權計分** | **ANA≥1:80 入門**，加權**總分 ≥10、≥1 臨床、同領域取最高** | 敏感度 96%、特異度 93%（最佳平衡） |

> 演進邏輯：1997 太不敏感（漏早期病人）→ 2012 SLICC 提高敏感度並納入更多神經/血液項目、要求臨床+免疫兼具 → 2019 改用「入門門檻 + 權重」兼顧敏感與特異，並讓單一重大表現（如腎 Class III/IV）能直接達標。

## 一、1997 ACR Revised Criteria

| # | 項目 | 定義 / 切點 |
|---|------|-------------|
| 1 | 蝴蝶斑（malar rash） | 顴部固定紅斑，不犯鼻唇溝 |
| 2 | 盤狀紅斑（discoid） | 紅斑性鱗屑斑塊、可疤痕 |
| 3 | 光敏感 | 日曬後異常皮疹（病史或醫師觀察） |
| 4 | 口腔潰瘍 | 口腔或鼻咽潰瘍，通常無痛 |
| 5 | 關節炎 | **非侵蝕性**，**≥2 個周邊關節**，有壓痛、腫脹或積液 |
| 6 | 漿膜炎 | 肋膜炎 **或** 心包炎 |
| 7 | 腎病變 | 蛋白尿 **>0.5 g/day**（或 >3+），**或**細胞圓柱 |
| 8 | 神經病變 | **癲癇 或 精神病**（排除其他原因） |
| 9 | 血液病變 | 溶血性貧血；**或** 白血球 <4,000（≥2 次）；**或** 淋巴球 <1,500（≥2 次）；**或** 血小板 <100,000 |
| 10 | 免疫病變 | anti-dsDNA；**或** anti-Sm；**或** 抗磷脂抗體（aCL IgG/IgM 異常、LA、或梅毒血清偽陽性） |
| 11 | ANA | titre 異常 |

**計分規則：** 11 項中**符合 ≥ 4 項**即可分類為 SLE。**每項權重相同（各 1 分）**，可在病程中**先後或同時**出現，不須一次到位。沒有「入門條件」、也不要求臨床與免疫兼具——這正是它**對早期或輕症敏感度不足**的原因（只有皮膚幾項就可能湊到 4，或反之輕症難達 4）。

## 二、2012 SLICC Criteria

### 臨床項目（11）

| # | 項目 | 定義 / 切點 |
|---|------|-------------|
| 1 | 急性皮膚狼瘡 | 蝴蝶斑、大疱型、TEN 變異、斑丘疹、光敏疹；含亞急性皮膚狼瘡 |
| 2 | 慢性皮膚狼瘡 | 盤狀狼瘡等 |
| 3 | 口腔/鼻腔潰瘍 | |
| 4 | 非疤痕性禿髮 | |
| 5 | 滑膜炎 | **≥2 關節**腫脹/積液，**或** ≥2 關節壓痛 + **晨僵 ≥30 分鐘** |
| 6 | 漿膜炎 | 肋膜炎或心包炎（>1 天等） |
| 7 | 腎病變 | UPCR 或 24h 蛋白 **≥500 mg/day**，**或**紅血球圓柱 |
| 8 | 神經病變 | 癲癇、精神病、**多發性單神經炎、脊髓炎、周邊/腦神經病變、急性意識混亂**（範圍比 1997 廣） |
| 9 | 溶血性貧血 | |
| 10 | 白血球/淋巴球低下 | 白血球 **<4,000**（≥1 次）**或** 淋巴球 **<1,000**（≥1 次） |
| 11 | 血小板低下 | **<100,000**（≥1 次） |

### 免疫項目（6）

| # | 項目 | 定義 / 切點 |
|---|------|-------------|
| 1 | ANA | 高於參考值 |
| 2 | anti-dsDNA | 高於參考值（ELISA 須 >2 倍） |
| 3 | anti-Sm | |
| 4 | 抗磷脂抗體 | LA、梅毒偽陽性、**中–高效價 aCL**、anti-β2GPI |
| 5 | 低補體 | C3、C4 **或** CH50 |
| 6 | 直接 Coombs（DAT） | 在**無溶血性貧血**情況下陽性才算 |

**計分規則：** **每項 1 分、無權重**。分類為 SLE 須符合 **總計 ≥ 4 項，且其中至少 1 項臨床 + 至少 1 項免疫**；**或**單獨走捷徑——**腎切片證實狼瘡腎炎 + ANA 或 anti-dsDNA 陽性**即可（不必湊滿 4 項）。相較 1997，SLICC **要求臨床與免疫兼具**（避免純皮膚診斷）、**擴充神經與血液項目**、並把**腎切片**單獨列為達標路徑，敏感度因此明顯提升。

## 三、2019 EULAR/ACR Criteria（現行主流）

### 基本架構

**入門條件（一票否決）：HEp-2 ANA ≥ 1:80（曾經一次即可）**；未達者不能分類為 SLE。

**計分七規則：**

1. ANA 達標後加權加總，**總分 ≥ 10** → 分類為 SLE。
2. **7 臨床領域 + 3 免疫領域**，各項固定權重。
3. **同一領域只取最高分項**。
4. **至少 1 項臨床**領域（不能純靠免疫湊 10）。
5. 各表現**曾出現一次即可**，不須同時。
6. 有**更合理其他解釋**（感染/藥物）→ 不計分。
7. 採**有/無二元**判定，**不因 titre 高低加減分**（titre 只在 ANA 入門有意義）。

## 四、細節 Q&A

**Q1. 血小板低下要多少？**

- 三套標準一致：**PLT < 100,000/µL（<100 ×10⁹/L）**。
- 1997 與 SLICC 計 1 項；2019 計 **4 分**。皆須排除其他原因。

**Q2. 抗體數值高低會影響計分嗎？**

- **原則上不會**，三套皆為「有/無」判定，**titre 高低不改分數**。例外：
  - **ANA**：2019 唯一看 titre 之處，**入門 ≥1:80**。
  - **anti-dsDNA**：不看 titre，但 2019 要求檢測**特異度 ≥90%**；SLICC 若用 ELISA 須 **>2 倍**參考值。
  - **aCL**：三套皆要求**中–高效價**才算。
  - **補體**：2019 唯一「程度」影響分數處——C3+C4 同低（4）> 只低一個（3）。

**Q3. 關節炎要滿足什麼條件？**

- **1997**：非侵蝕性、**≥2 個周邊關節**有壓痛/腫脹/積液。
- **2012 SLICC / 2019**（定義相同，2019 計 6 分）符合任一：
  - **滑膜炎 ≥2 關節**（腫脹或積液）；**或**
  - **≥2 關節壓痛 ＋ 晨僵 ≥30 分鐘**。
- 重點：**至少 2 個關節**；走壓痛路徑**一定要加晨僵 ≥30 分**。

## 五、考試陷阱

- **2019：ANA 陰性 → 幾乎排除**（入門條件）。
- **2019：單一腎 Class III/IV = 10 分**自己達標。
- **2019：同領域只取最高分**，別把皮膚多項相加。
- **2019：至少 1 項臨床**，純免疫湊 10 不算。
- **SLICC 捷徑：腎切片狼瘡腎炎 + ANA/anti-dsDNA** 即達標。
- **1997 無權重、無入門**，所以對早期/輕症**敏感度最差**。
- 三套**血小板切點一致（<100k）**；**淋巴球切點不同**（1997 <1,500、SLICC <1,000）。
