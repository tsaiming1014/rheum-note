# RA 口試完整補充

---

## 1. Pregnancy 情境下的藥物選擇

### 可以使用（相對安全）

| 藥物 | 備註 |
|------|------|
| **HCQ** | 整個孕期皆可，可降低 neonatal lupus 風險 |
| **SSZ** | 可用，需補充 folic acid |
| **低劑量 prednisolone**（< 10 mg/d） | 可用，盡量短期 |
| **Azathioprine** | 可用，常用於懷孕期間維持 |
| **Certolizumab pegol** | **首選 TNF inhibitor**，無 Fc 段，不過胎盤，哺乳也安全 |

### 禁用

| 藥物 | 原因 |
|------|------|
| **MTX** | 強力致畸胎（葉酸拮抗劑），需停藥 ≥3 個月才可懷孕 |
| **LEF** | 致畸胎，半衰期極長，需 cholestyramine washout 後確認血中濃度 < 0.02 mg/L |
| **JAK inhibitor** | 動物實驗致畸胎，禁用 |
| **Rituximab** | 妊娠中晚期可能造成新生兒 B cell 耗竭 |
| **其他 TNF inhibitor**（Adalimumab、Infliximab、Etanercept） | 第三孕期有 Fc 段主動轉運，新生兒體內濃度高，出生後 6 個月內不可打活疫苗 |

> **哺乳：** HCQ、SSZ、低劑量 pred 皆可；MTX、LEF 禁用

## 2. CV Risk / Old Age 情境

### JAKi 的 CV 風險（ORAL Surveillance Trial）

- 2021 年 FDA Black Box Warning
- Tofacitinib vs Adalimumab（在 CV risk ≥ 50 歲、有吸菸史的 RA 病人）
- 結果：tofacitinib **MACE（心肌梗塞、腦中風）、cancer、VTE、死亡率** 均較高

**FDA 因此規定 JAKi 的使用限制：**
> TNF inhibitor 治療反應不佳，且無法使用 TNF inhibitor 替代方案時才使用

### 老年人用藥考量

| 考量點 | 說明 |
|------|------|
| **感染風險** | 老年人免疫力下降，JAKi > TNFi 感染風險更高 |
| **腎功能** | MTX 需調整劑量（eGFR < 30 禁用） |
| **骨鬆** | 長期 prednisolone → 需評估 GIOP |
| **多重用藥** | 注意 MTX + NSAIDs → 減少腎排泄，毒性↑ |
| **Herpes zoster** | JAKi 顯著增加帶狀皰疹，建議用藥前先打 **Shingrix（非活性疫苗）** |

## 3. Monotherapy 情境

### 什麼情況下考慮 monotherapy？

- 病人無法耐受 combination therapy
- 早期 RA、低疾病活性
- 懷孕（盡量單藥）

### MTX monotherapy vs combination

| | 效果 |
|--|------|
| **MTX monotherapy** | 對 low-moderate disease activity 效果良好 |
| **MTX + HCQ + SSZ（triple）** | 效果接近 MTX + TNFi，但副作用少、費用低 |
| **MTX + bDMARD** | High disease activity 或 poor prognostic factors 的首選 |

### bDMARD monotherapy（不加 MTX）

| 藥物 | Monotherapy 效果 |
|------|----------------|
| **Tocilizumab（IL-6Ri）** | Monotherapy 效果**優於** TNFi monotherapy（ADACTA trial：TCZ > ADA monotherapy） |
| **Abatacept** | 可 monotherapy |
| **TNFi** | Monotherapy 效果較差，建議搭配 MTX |
| **JAKi** | 可 monotherapy（Upadacitinib SELECT-MONOTHERAPY trial 有效） |

> 重點：**TCZ 是最適合 monotherapy 的 bDMARD**（MTX 不耐受時首選）

## 4. ACR vs EULAR Guideline 差異

| 比較點 | ACR 2021 | EULAR 2022 |
|--------|---------|-----------|
| **csDMARD 首選** | MTX（若無禁忌） | MTX + short-term GC bridge |
| **bDMARD 順序** | TNFi、IL-6Ri、Abatacept 並列 | TNFi 仍為首選，但 IL-6Ri、JAKi 可早期考慮 |
| **JAKi 使用時機** | 須先試過 TNFi 才可用（FDA 警告後） | 有 CV/cancer/VTE risk 者，避免 JAKi |
| **T2T 目標** | Remission 或 LDA | Remission 優先，無法達到則 LDA |
| **GC 使用** | 盡量短期 bridge，< 3 個月 | 同左，但強調盡快 taper |

## 5. RA Image 表現

### X-ray（由早到晚）

| 時期 | 表現 |
|------|------|
| 早期 | **Juxtaarticular osteopenia**（關節周圍骨質疏鬆）、soft tissue swelling |
| 中期 | **Joint space narrowing**（均勻性，全面性）、**marginal erosion** |
| 晚期 | **Mutilans deformity**、**ankylosis**、subluxation |

### 好發部位
- **MCP、PIP、wrist**（DIP 通常不侵犯，可與 OA 鑑別）
- 頸椎：**Atlantoaxial subluxation**（neck flexion > 3 mm）、periodontoid pannus

### MRI
- 早期偵測 **bone marrow edema、synovitis、erosion**（比 X-ray 早 2–3 年）

### 超音波（OMERACT 定義）
- **Synovial hypertrophy**：灰階 → 增厚的滑膜
- **Power Doppler**：活動性發炎的血流信號
- **Erosion**：兩個切面都看到 bone cortex 不連續

## 6. RA Extra-articular Manifestation

### 系統性整理

| 器官 | 表現 |
|------|------|
| **皮膚** | **Rheumatoid nodule**（皮下、好發伸側、與 RF 相關）、vasculitis |
| **眼睛** | **Scleritis**（嚴重，需積極治療）、episcleritis（輕微）、keratoconjunctivitis sicca（secondary Sjogren's） |
| **肺** | **ILD**（UIP/NSIP pattern，最常見）、pleuritis、pleural effusion（低 glucose 為特徵）、rheumatoid nodule、Caplan syndrome |
| **心臟** | **Pericarditis**（最常見心臟表現）、myocarditis、CV risk ↑ |
| **血液** | **Felty syndrome**（RA + splenomegaly + neutropenia）、anaemia of chronic disease |
| **神經** | Carpal tunnel syndrome、cervical myelopathy（atlantoaxial subluxation） |
| **腎臟** | 較少直接侵犯，多為藥物副作用（NSAIDs、gold）或 AA amyloidosis（晚期） |
| **血管炎** | Rheumatoid vasculitis（高 RF、長病程、皮膚潰瘍、mononeuritis multiplex） |

## 7. DAS28 公式

$$DAS28 = 0.56\sqrt{TJC28} + 0.28\sqrt{SJC28} + 0.70\ln(ESR) + 0.014 \times GH$$

| 縮寫 | 意義 |
|------|------|
| TJC28 | Tender joint count（28 個關節） |
| SJC28 | Swollen joint count（28 個關節） |
| ESR | 紅血球沉降速率（mm/hr） |
| GH | Global health，病人自評 0–100 mm VAS |

**28 個關節：** 雙側 MCP×5、PIP×5、wrist、elbow、shoulder、knee（各×2）= 28

### DAS28 判讀

| 分數 | 疾病活性 |
|------|---------|
| < 2.6 | **Remission** |
| 2.6–3.2 | Low disease activity |
| 3.2–5.1 | Moderate disease activity |
| > 5.1 | High disease activity |

## 8. 2010 ACR/EULAR RA Classification Criteria

**需 ≥ 6 分**（前提：至少一個關節有 synovitis，且排除其他診斷）

| 項目 | 選項 | 分數 |
|------|------|------|
| **關節侵犯** | 1 large joint | 0 |
| | 2–10 large joints | 1 |
| | 1–3 small joints | 2 |
| | 4–10 small joints | 3 |
| | > 10 small joints（含至少一個 small joint） | **5** |
| **血清學** | RF 和 anti-CCP 皆陰性 | 0 |
| | 低陽性（≤ 3× ULN） | 2 |
| | 高陽性（> 3× ULN） | **3** |
| **急性期反應物** | CRP 和 ESR 皆正常 | 0 |
| | 異常 | 1 |
| **症狀持續時間** | < 6 週 | 0 |
| | ≥ 6 週 | 1 |

## 9. Remission 定義細節

| 工具 | Remission 標準 |
|------|--------------|
| **DAS28** | < 2.6 |
| **CDAI**（Clinical DAI） | ≤ 2.8（TJC28 + SJC28 + PGA + EGA） |
| **SDAI**（Simplified DAI） | ≤ 3.3（CDAI + CRP） |
| **Boolean remission**（ACR/EULAR 2011） | TJC ≤ 1 **且** SJC ≤ 1 **且** CRP ≤ 1 mg/dL **且** PGA ≤ 1（0–10 scale） |

> **注意：** DAS28 remission（< 2.6）比 Boolean 或 CDAI remission **寬鬆**，部分病人 DAS28 < 2.6 但仍有影像上的進展

## 10. JIA（Juvenile Idiopathic Arthritis）

### 定義
- 16 歲以前發病
- 關節炎持續 ≥ 6 週
- 排除其他診斷

### 7 大分類（ILAR criteria）

| 類型 | 特徵 |
|------|------|
| **Systemic JIA（sJIA）** | 每日發燒 spike、皮疹（salmon-colored）、肝脾淋巴結腫大；與 IL-1、IL-18 相關；MAS 風險高 |
| **Oligoarticular（persistent/extended）** | ≤ 4 關節；**ANA 陽性、uveitis 風險高**；多為女孩 |
| **Polyarticular RF(-)** | ≥ 5 關節，RF 陰性 |
| **Polyarticular RF(+)** | ≥ 5 關節，RF 陽性；**= 成人 RA**，預後較差 |
| **Psoriatic JIA** | 關節炎 + 牛皮癬（或指甲病變 + 一等親 PsO） |
| **Enthesitis-related arthritis（ERA）** | 多為男孩、HLA-B27、enthesitis；與 AS 相關 |
| **Undifferentiated** | 不符合上述或符合 ≥ 2 類 |

### 與成人 RA 差異

| 比較 | JIA | 成人 RA |
|------|-----|---------|
| RF | 多陰性 | 多陽性 |
| Uveitis | **常見**（尤其 oligoarticular ANA+），多**無症狀** | 少見 |
| 全身症狀 | sJIA 明顯（發燒、皮疹） | 少見 |
| MAS | sJIA 特有風險 | 無 |
| 治療 | IL-1i（Anakinra、Canakinumab）用於 sJIA | 不常用 |
