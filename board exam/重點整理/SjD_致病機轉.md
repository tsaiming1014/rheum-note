# Sjögren's Syndrome：致病機轉完整解析

## 一、核心概念

Sjögren's syndrome 是一個 **autoimmune epithelitis**（自體免疫上皮炎），也稱為 **autoimmune exocrinopathy**。這個命名本身就揭示了兩個核心概念：第一，它是全身性自體免疫疾病；第二，上皮細胞不是無辜的旁觀者，而是主動驅動發炎的參與者。臨床上以 keratoconjunctivitis sicca（乾眼）和 xerostomia（口乾）為表徵，但實際上是一個牽涉多器官的系統性疾病。

---

## 二、致病機轉

### （一）易感因素

**遺傳背景**
- 最強遺傳風險：**HLA class II alleles**（DRB1\*03:01、DQA1\*05:01、DQB1\*02:01）
- 與 anti-Ro/SS-A、anti-La/SS-B 抗體陽性高度相關
- 非 HLA 風險基因（GWAS）：IRF5、STAT4（type I IFN 路徑）、BLK、EBF1（B cell 發育）

**表觀遺傳**
- Naive CD4+ T cells 及 B cells 的 IFN 相關基因 → **hypomethylation** → 促發炎基因過度表現

**環境觸發**
- **EBV** 最受關注：腺體內可偵測到 EBV 基因組；anti-Ro/SS-A 陽性者對 EBV early antigen 抗體陽性率更高
- Sjögren's 顯著的 **type I IFN signature** 間接支持病毒觸發假說

---

### （二）先天免疫：Type I IFN——最上游的開關

- 病毒感染 → **pDC** 透過 **TLR7/TLR9** 感應病毒核酸 → 大量分泌 **type I IFN**
- Type I IFN 下游效應：
  - 活化唾液腺上皮細胞
  - 誘導 **BAFF** 大量分泌
  - 促進 Th1/Th17 分化
  - 解釋 IRF5、STAT4 等遺傳風險基因的致病意義

---

### （三）上皮細胞活化——致病核心

上皮細胞本身表現 **TLR3/7/9**，感應危險訊號後，同時扮演三個角色：

- **Antigen presenter**：上調 HLA-II、CD40、CD80/86 → 直接活化 CD4+ T cells
- **Cytokine factory**：分泌 IL-1、IL-6、IL-12、IL-18、TNF、**BAFF**
- **Lymphocyte recruiter**：分泌 **CXCL12、CXCL13、CCL21** → 招募 T/B cells 聚集於導管周圍

> 切片上炎症圍繞導管，正因上皮細胞是招募淋巴球的磁鐵。

---

### （四）適應性免疫

**T cells**
- 浸潤以 **CD4+ memory T cells（CD45RO+）** 為主，TCR repertoire 受限
- 主導細胞激素：**Th1（IFN-γ、IL-2）** 和 **Th17（IL-17）**
- FoxP3+ Treg cells 存在，但抑制功能不確定

**B cells 與 BAFF**
- **BAFF** 由 pDC、macrophage、T cells、B cells、上皮細胞共同產生
- Type I IFN → 誘導 BAFF → **連結先天與適應性免疫的樞紐**
- 高濃度 BAFF 使 B cells 逃脫自我耐受、持續活化、產生自體抗體

**Ectopic Germinal Center（GC）——疾病升級的標誌**
- LT-α/β + CXCL13 → 腺體內形成有組織的 ectopic GC
- 具備完整 GC 功能：Ki-67+ 增殖細胞、CD21/CD35+ follicular DC network、CD31+ HEV
- Anti-Ro/SS-A 在此局部產生；亦是 **MALT lymphoma** 的發源地

---

### （五）腺體功能喪失：兩種機轉

**Model A：免疫破壞（Immune-destructive）**
- 免疫攻擊 → Fas/FasL、Bax 上調 → acinar cell apoptosis → 纖維化
- 弱點：臨床觀察中 apoptosis 並不常見，無法解釋「形態尚存但功能喪失」

**Model B：免疫抑制（Immune-inhibited）**
- **Anti-M3R 抗體** → 阻斷乙醯膽鹼受體 → 分泌訊號中斷
- **AQP5（aquaporin 5）** 表現下調 → 水分無法通過頂端細胞膜
- Ca²⁺ signaling 缺損、粒線體功能失調
- 暗示功能喪失具**可逆性**，為 pilocarpine（M3R agonist）治療提供機轉基礎

> 兩模型不互斥，可能在不同疾病階段共存。

## 三、致病機轉與唾液腺切片的對應

切片上的每一個發現，都直接對應一個致病機轉：

| 切片所見 | 對應機轉 |
|---------|---------|
| Periductal lymphocytic infiltrate | 上皮細胞活化 → CXCL12/13 招募 T/B cells |
| CD4+ T cells 為主（90%） | Th1/Th17 response，memory phenotype |
| B cell 比例隨嚴重度增加 | BAFF 過度表現 → B cell 存活延長 |
| Ectopic GC-like structure | LT-α/β + CXCL13 → 腺體內 antigen-driven B cell 選擇 |
| Focus score ↑ | 更多 B cells、更活躍的 GC → 自體抗體產量增加、NHL 風險 ↑ |
| 腺泡形態保存但功能差 | Model B：anti-M3R + AQP5↓，而非大量 apoptosis |

診斷標準（2016 ACR/EULAR）要求 focus score ≥1/4 mm²（即每 4 mm² 組織至少有 50 個淋巴球的聚集灶），正是量化這個病理過程的嚴重程度。

## 四、為什麼有腺體外症狀？

腺外症狀並非獨立的病理，而是同一套失控免疫機轉透過三條路徑蔓延至全身：

**路徑一：上皮細胞不只在腺體**
肺部、腎臟的上皮細胞同樣表現 TLR，可被同一套機轉活化，引發淋巴球浸潤。肺部的 NSIP/LIP 和腎臟的 tubulointerstitial nephritis，其病理切片幾乎與唾液腺如出一轍。

**路徑二：B cell 失控 → 循環中的免疫武器**
Ectopic GC 產生的 B cells 和自體抗體進入血液，以 cryoglobulin 和 immune complex 的形式沉積於小血管，造成 palpable purpura、glomerulonephritis 及系統性血管炎。Cryoglobulinemia + hypocomplementemia 的病人，NHL 風險上升 ≥5 倍。

**路徑三：全身性細胞激素**
Type I IFN 和 IL-6 循環至全身，直接作用於神經內分泌系統，造成約 70% 病人出現的**慢性疲勞**；小血管炎影響神經滋養血管，引發 small fiber neuropathy；自律神經功能異常則解釋了 Raynaud's phenomenon（13–33%）往往在 sicca 症狀出現前數年就已發生。

---

## 五、Classification Criteria 與 Disease Activity 評估的邏輯差異

Classification criteria（2016 ACR/EULAR）聚焦於腺體，而 disease activity 評估（ESSDAI）涵蓋全身，這並非矛盾，而是**目的不同**：

**Classification criteria 聚焦腺體**，是因為腺體症狀（focal lymphocytic sialadenitis + anti-Ro/SS-A）最為普遍且最具特異性，能夠與 SLE、SSc 等疾病劃清邊界。Extraglandular 症狀特異性不足，放入診斷標準只會增加誤診。

**ESSDAI 評估全身**，是因為腺體嚴重度本身與預後幾乎不相關。真正決定治療強度和死亡風險的，是 vasculitis、cryoglobulinemia、lymphoma、severe neuropathy 等腺外表現。若僅評估腺體，永遠無法判斷病人需要 hydroxychloroquine 還是 cyclophosphamide。

值得注意的是，**anti-Ro/SS-A** 同時出現在 classification criteria 和臨床預後評估中，扮演橋樑角色：它既是診斷成立的依據，也預測腺外疾病的風險，連結了兩個看似分離的評估框架。

## 六、口試核心三角

```
Type I IFN（最上游的開關）
        ↓
上皮細胞（發動機：antigen presenter + cytokine factory + lymphocyte recruiter）
        ↓
BAFF（樞紐：連結先天與適應性免疫，驅動 B cell 失控）
        ↓
Ectopic GC → 自體抗體 + Lymphoma 風險
        ↓
腺體功能喪失（破壞 + 抑制）＋ 全身症狀
```

這三個角色構成一個自我強化的循環：病毒觸發 IFN，IFN 活化上皮細胞，上皮細胞分泌 BAFF，BAFF 維持 B cells 失控，B cells 又持續刺激上皮細胞產生更多 BAFF。
