## 筆試

### 1. 下列配對何者錯誤
- A IL-17a & f 配對的 receptor
- B IL-4/13 配對的 receptor
- C BAFF 配對 TACI、BCMA
- D IL-12/23 配對的 receptor

**⚠️ 選項記錄不完整**：原始題庫只記錄了「配對的類別」，沒有記錄題目實際給出的具體錯誤配對內容，無法判斷正確答案。以下整理相關機轉作為複習參考：

- **IL-17A/F receptor**：IL-17A 與 IL-17F 皆透過 **IL-17RA + IL-17RC** 異二聚體受體傳訊。
- **IL-4/13 receptor**：
  - **Type I receptor**（IL-4Rα + γc）→ 只有 **IL-4** 能結合，主要在造血細胞表現
  - **Type II receptor**（IL-4Rα + IL-13Rα1）→ **IL-4 與 IL-13 皆可結合**，在非造血細胞（如上皮細胞）表現，也是 Dupilumab（anti-IL-4Rα）能同時阻斷兩者訊息的原因
- **BAFF receptor**：BAFF 可結合 **三個** receptor：**BAFF-R (BR3)、TACI、BCMA**；**APRIL 只結合 TACI、BCMA，不結合 BAFF-R**，是兩者常見的鑑別考點。
- **IL-12/23 receptor**：無「IL-12Rα」此正式名稱，正確為 **IL-12Rβ1（IL-12 與 IL-23 共用）+ IL-12Rβ2（IL-12 專屬）**；IL-23 則為 **IL-12Rβ1 + IL-23R**。訊息傳遞上 **IL-12Rβ1 走 TYK2、IL-12Rβ2 走 JAK2**，下游主要活化 **STAT4**。

### 2. Inflammasome、caspase-1 會使細胞內前驅物產生下列何者？
- A IL-1-alpha
- B IL-18
- C （未記錄）
- D （未記錄）

**答案：B**

> ⚠️ 原始題庫記錄標記答案為 A，與此處判斷不同，請自行核對。

**解析：**

- **B（對）** Inflammasome 活化後，**caspase-1** 會切割 **pro-IL-1β → mature IL-1β**、**pro-IL-18 → mature IL-18**，這是 caspase-1 最經典的兩個受質（連同 gasdermin D，切割後造成 pyroptosis）。
- **A（錯，題庫原記錄可能有誤）** **IL-1α 不是 caspase-1 的受質**。Pro-IL-1α 本身即具有生物活性（不需要切割就能作用），主要是由 **calpain**（而非 caspase-1）在細胞壞死時切割釋放，屬於 alarmin 性質的 cytokine，與 inflammasome/caspase-1 路徑機轉不同，是常見的考試陷阱（IL-1α vs IL-1β 混淆）。

**速記：**

| Cytokine | 是否需 caspase-1 切割才有活性 | 主要切割酵素 |
|---|---|---|
| IL-1β | 是 | Caspase-1（inflammasome） |
| IL-18 | 是 | Caspase-1（inflammasome） |
| IL-1α | 否（前驅物即有活性） | Calpain（壞死釋放時） |

### 3. 下列哪個 cytokine 不會與 JAK1, JAK2 反應
- A IL-2
- B IL-5
- C TNF-alpha
- D IL-10

**答案：C**

**解析：**

- **A（錯，會與 JAK1 反應）** IL-2 屬於 **common gamma chain (γc) family**（IL-2, 4, 7, 9, 15, 21），receptor 由 **IL-2Rβ（結合 JAK1）+ γc（結合 JAK3）** 組成，走 **JAK1-JAK3**，不是 JAK2，但仍與 JAK1 有關。
- **B（錯，會與 JAK2 反應）** IL-5 屬於 **common beta chain (βc) family**（IL-3, IL-5, GM-CSF），receptor 走 **JAK2 同型二聚體（JAK2-JAK2）**。
- **C（對）** **TNF-α** 完全不透過 JAK-STAT 路徑傳訊，而是結合 **TNF receptor (TNFR1/TNFR2)**，經由 **TRADD、TRAF、RIP1** 等 adaptor 分子活化 **NF-κB** 與 **MAPK** pathway，與 JAK1、JAK2 皆無關。
- **D（錯，會與 JAK1 反應）** IL-10 receptor 走 **JAK1（IL-10R1）+ TYK2（IL-10R2）**，不是 JAK2，但仍與 JAK1 有關。

**速記表：**

| Cytokine family | 代表成員 | JAK 組合 |
|---|---|---|
| Common γ chain (γc) | IL-2, 4, 7, 9, 15, 21 | JAK1 + JAK3 |
| Common β chain (βc) | IL-3, IL-5, GM-CSF | JAK2 + JAK2 |
| gp130 family | IL-6, IL-11 | JAK1 + JAK2 + TYK2 |
| IL-10 family | IL-10, 19, 20, 22 | JAK1 + TYK2 |
| Type I/II IFN | IFN-α/β, IFN-γ | JAK1/JAK2/TYK2（依型別） |
| TNF superfamily | TNF-α | 不走 JAK-STAT，走 TNFR → NF-κB |

### 5. 下列何者跟鐵離子最相關
- A Apoptosis
- B Necroptosis
- C Pyroptosis
- D Ferroptosis

**答案：D**

**解析：**

- **D（對）** **Ferroptosis** 是一種**鐵離子依賴**的細胞死亡形式，核心機轉是細胞內鐵離子過量、觸發 **Fenton reaction** 產生活性氧自由基，加上 **glutathione peroxidase 4 (GPX4)** 功能不足（無法清除脂質過氧化物），導致細胞膜上的多元不飽和脂肪酸大量**脂質過氧化 (lipid peroxidation)**，最終使細胞膜崩解死亡。與鐵離子代謝、GPX4、脂質過氧化直接相關，是四種死亡機轉中唯一以鐵離子為核心驅動力的。

**四種細胞死亡機轉比較：**

| 機轉 | 觸發方式 | 關鍵分子 | 是否促發炎 | 風濕科相關疾病 |
|---|---|---|---|---|
| Apoptosis | 程式性死亡（內在/外在路徑） | Caspase-3, -8, -9；Bcl-2 家族 | 通常**不**促發炎（細胞膜維持完整，形成 apoptotic body 被吞噬清除） | 清除自體反應性淋巴球失敗與 SLE 等自體免疫病發生相關 |
| Necroptosis | 程式性壞死（TNFR 活化但 caspase-8 被抑制時啟動） | RIPK1、RIPK3、MLKL（MLKL 造成細胞膜穿孔） | **促發炎**（細胞膜破裂，釋出 DAMP） | 慢性發炎性疾病、IBD 相關關節炎的機轉之一 |
| Pyroptosis | Inflammasome 活化 | Caspase-1（或 caspase-4/5/11）切割 Gasdermin D，形成細胞膜孔洞 | **強烈促發炎**（釋出成熟 IL-1β、IL-18） | 痛風（MSU 結晶活化 NLRP3 inflammasome）、CAPS 等 autoinflammatory disease |
| Ferroptosis | 鐵依賴脂質過氧化 | 鐵離子、GPX4 功能不足、脂質過氧化 | 可促發炎（釋出 DAMP），機轉與前三者不同 | 近年研究與 SLE、RA 滑膜細胞死亡、器官損傷有關（仍在研究中） |

**Pyroptosis 在痛風的完整路徑：** MSU 結晶被 macrophage 吞噬 → 活化 NLRP3 inflammasome（NLRP3 + ASC + pro-caspase-1）→ caspase-1 活化 → 切割 pro-IL-1β → IL-1β、pro-IL-18 → IL-18，同時切割 Gasdermin D 使其 N 端片段插入細胞膜形成孔洞 → 細胞爆裂 → 大量 IL-1β、IL-18、DAMP 釋出，放大發炎反應，是急性痛風紅腫熱痛的核心機轉，也是 Anakinra、Canakinumab（anti-IL-1）在頑固性痛風可發揮療效的理論基礎。

### 7. 50 歲男性，反覆鼻竇炎、中耳炎，最近血痰、腎功能惡化，兩足下垂及感覺異常。除了 ANA，下列何者有助於鑑別診斷？
- A （未記錄）
- B Procalcitonin
- C ANCA
- D dsDNA

**答案：C**

**解析：**

- **C（對）** 除了排除 SLE（ANA），這個病人的臨床表現組合——**上呼吸道（鼻竇炎、中耳炎）+ 下呼吸道/腎臟（血痰、腎功能惡化，pulmonary-renal syndrome）+ 周邊神經（雙足下垂、感覺異常 = mononeuritis multiplex）**——是典型 **ANCA-associated vasculitis (AAV)** 的三大器官侵犯模式，**ANCA（含 PR3-ANCA/c-ANCA、MPO-ANCA/p-ANCA）**是這類疾病最關鍵的鑑別診斷檢查。
- **B（錯）** Procalcitonin 主要用於區分細菌感染 vs 非感染性發炎，跟這個以血管炎表現為主的臨床圖像鑑別價值有限。
- **D（錯）** dsDNA 是 SLE 的特異性抗體，這個病人臨床表現（鼻竇炎、中耳炎、肺出血、周邊神經侵犯）較不符合 SLE 典型表現，優先度低於 ANCA。

**AAV 三型鑑別速記：**

| | GPA | MPA | EGPA |
|---|---|---|---|
| ANCA 型別 | c-ANCA/PR3 為主 | p-ANCA/MPO 為主 | p-ANCA/MPO（約半數陽性） |
| 上呼吸道侵犯 | **典型**（鼻竇炎、中耳炎、鞍鼻） | 少見 | 可有（鼻息肉、慢性鼻竇炎） |
| 氣喘/嗜酸性球增高 | 無 | 無 | **必要條件** |
| 肉芽腫 | 有（granulomatous） | 無 | 有 |
| 肺侵犯 | 結節、空洞病灶、肺出血 | 肺出血、ILD | 遊走性肺浸潤、氣喘 |
| Mononeuritis multiplex | 中等 | 中等 | **最常見** |

### 8. SSA/Ro52 & Ro60 相關整理

**⚠️ 選項記錄不完整（僅記錄關鍵字，非完整敘述句），無法判斷「何者錯誤」，以下整理相關考點作為參考：**

**1. Ro52 與 Ro60 是兩個不同的抗原**

| | Ro60 (SSA1/TROVE2) | Ro52 (TRIM21) |
|---|---|---|
| 分子功能 | 結合 RNA（hY RNA、pre-5S rRNA），參與 RNA quality control | E3 ubiquitin ligase，胞內抗體受體，參與先天免疫調控 |
| 好發疾病 | SLE、SjD（原發性修格連氏症候群） | SjD、IIM（肌炎）、SSc、SLE，常與其他抗體並存（如 anti-Jo-1） |
| 單獨陽性意義 | 較具疾病特異性 | 常為 non-specific co-positive marker，單獨陽性臨床意義較弱 |

**1-1. 盛行率數據（引用 Kelly's Textbook of Rheumatology）**

| 疾病 | Anti-Ro/SS-A 盛行率 | 備註 |
|---|---|---|
| SLE | ~40%（Table 55.3） | |
| 原發性 SjD | 40–95%（Table 55.6，範圍寬因各實驗室檢測方式異質性大） | 與 NLS、CHB 相關 |
| SSc | anti-Ro52 約 35.6%（Ch085 Scleroderma） | 未特別與特定臨床表現相關，但 anti-Ro52/TRIM21 被列為與 PAH 相關的 SSc 抗體之一 |
| IIM | 未給明確百分比，列為 myositis-associated antibody (MAA) | 常與 anti-synthetase syndrome 並存 |

原文（Kelly Ch055）："The prevalence of anti-Ro52 versus anti-Ro60 appear similar in SLE and SjS but appear to differ in other autoimmune diseases (e.g., with **anti-Ro52 more prevalent than anti-Ro60 in IIM, SSc, and MCTD**). In general, anti-Ro52 antibodies have been suggested to associate with **more severe disease and/or interstitial lung disease**."

- **SLE、SjD**：Ro52 與 Ro60 盛行率**差不多**
- **IIM、SSc、MCTD**：**Ro52 > Ro60**
- Ro52 陽性普遍被認為跟**較嚴重疾病 / ILD** 相關

**2. Neonatal lupus / Congenital heart block (CHB)**

- 母親帶有 **anti-Ro (Ro52 和/或 Ro60)、anti-La** 抗體，經胎盤傳遞至胎兒，攻擊胎兒心臟傳導系統，導致 **third-degree AV block**，是 neonatal lupus 最嚴重且不可逆的表現。
- 機轉上，**anti-Ro52** 被認為與胎兒心臟 Cav1.2 calcium channel 及 5-HT4 receptor 有交叉反應，被視為造成 CHB 較關鍵的抗體，但 **anti-Ro60、anti-La 也都與 CHB 風險相關**，不是只有單一抗體。
- 曾生過 CHB 病兒的母親，下一胎再發生的風險約 **12–20%**，是抗 Ro/La 陽性孕婦產檢需密切胎兒心臟監測的原因。

**3. ANA 陰性但 Ro52/Ro60 陽性的 SLE**

- 部分 SLE（尤其 **subacute cutaneous lupus, SCLE**）病人可以 **ANA 陰性但 anti-Ro60 陽性**。
- 原因是常規 **HEp-2 cell-based IFA** 檢測 ANA 時，Ro60 抗原在細胞固定過程中容易流失或表現量低，導致 IFA 偵測不到，但用**特異性 anti-Ro/SSA ELISA** 卻可測到陽性，造成「ANA(-) 但 anti-Ro(+)」的現象（Kelly Ch055 指出這可能占「ANA 陰性 SLE」超過一半的原因）。
- 臨床意義：**懷疑 SCLE 或 Sjogren 相關疾病時，即使 ANA 陰性也應加做 anti-Ro/La**，不能單靠 ANA 陰性排除診斷。

**4. Ro52 與 ILD、PAH 的關聯**

- **Anti-Ro52** 常與 **anti-synthetase syndrome（尤其合併 anti-Jo-1）** 及 **systemic sclerosis** 並存，並被多篇研究認為是 **ILD 較嚴重、進展較快、預後較差**的預測因子。
- 在 SSc 族群，**anti-Ro52/TRIM21** 被 Kelly 教科書列為與 **PAH** 相關的抗體之一（與 anti-centromere、anti-U3 RNP、anti-Th/To、anti-Ku、anti-U1 RNP 並列）。
