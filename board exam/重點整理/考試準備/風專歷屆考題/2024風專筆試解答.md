# 2024 風濕科專科考試 筆試解答

## 筆試

### 1. 下列配對何者錯誤
- A IL-17a & f 配對的 receptor
- B IL-4/13 配對的 receptor
- C BAFF 配對 TACI、BCMA
- D IL-12/23 配對的 receptor

**⚠️ 選項記錄不完整**：原始題庫只記錄了「配對的類別」，沒有記錄題目實際給出的具體錯誤配對內容，無法判斷正確答案。以下整理相關機轉作為複習參考：

- **IL-17A/F receptor**：IL-17A 與 IL-17F 皆透過 **IL-17RA + IL-17RC** 異二聚體受體傳訊。
- **IL-4/13 receptor**：
  - **Type I receptor**（IL-4Rα + γc）→ 只有 **IL-4** 能結合，主要在造血細胞表現
  - **Type II receptor**（IL-4Rα + IL-13Rα1）→ **IL-4 與 IL-13 皆可結合**，在非造血細胞（如上皮細胞）表現，也是 Dupilumab（anti-IL-4Rα）能同時阻斷兩者訊息的原因
- **BAFF receptor**：BAFF 可結合 **三個** receptor：**BAFF-R (BR3)、TACI、BCMA**；**APRIL 只結合 TACI、BCMA，不結合 BAFF-R**，是兩者常見的鑑別考點。
- **IL-12/23 receptor**：無「IL-12Rα」此正式名稱，正確為 **IL-12Rβ1（IL-12 與 IL-23 共用）+ IL-12Rβ2（IL-12 專屬）**；IL-23 則為 **IL-12Rβ1 + IL-23R**。訊息傳遞上 **IL-12Rβ1 走 TYK2、IL-12Rβ2 走 JAK2**，下游主要活化 **STAT4**。

### 2. Inflammasome、caspase-1 會使細胞內前驅物產生下列何者？
- A IL-1-alpha
- B IL-18
- C （未記錄）
- D （未記錄）

**答案：B**

> ⚠️ 原始題庫記錄標記答案為 A，與此處判斷不同，請自行核對。

**解析：**

- **B（對）** Inflammasome 活化後，**caspase-1** 會切割 **pro-IL-1β → mature IL-1β**、**pro-IL-18 → mature IL-18**，這是 caspase-1 最經典的兩個受質（連同 gasdermin D，切割後造成 pyroptosis）。
- **A（錯，題庫原記錄可能有誤）** **IL-1α 不是 caspase-1 的受質**。Pro-IL-1α 本身即具有生物活性（不需要切割就能作用），主要是由 **calpain**（而非 caspase-1）在細胞壞死時切割釋放，屬於 alarmin 性質的 cytokine，與 inflammasome/caspase-1 路徑機轉不同，是常見的考試陷阱（IL-1α vs IL-1β 混淆）。

**速記：**

| Cytokine | 是否需 caspase-1 切割才有活性 | 主要切割酵素 |
|---|---|---|
| IL-1β | 是 | Caspase-1（inflammasome） |
| IL-18 | 是 | Caspase-1（inflammasome） |
| IL-1α | 否（前驅物即有活性） | Calpain（壞死釋放時） |
