# 2025 風濕科專科考試 筆試解答

## 筆試

### 1. 生物製劑阻斷細胞激素作用, 何者正確?
- A Mepolizumab 可以認識 IL-5 receptor 來阻斷 IL-5 作用
- B Ixekizumab 可以阻斷 IL-17A 和 IL-17B
- C Guselkumab, risankizumab, ustekinumab 皆認識 IL-23 的 p19
- D Anifrolumab 可以阻斷 IFN-α 和 IFN-β 的作用

**答案：D**

**解析：**

- **A（錯）** Mepolizumab 認識的是 **IL-5 本身（ligand）**，不是 IL-5 receptor。真正認識 IL-5Rα 的是 **Benralizumab**（同時會透過 ADCC 清除嗜酸性球）。
- **B（錯）** Ixekizumab 只選擇性認識 **IL-17A**，不會阻斷 IL-17B。同時阻斷 IL-17A + IL-17F 的是 **Bimekizumab**（注意是 F 不是 B，選項刻意混淆）。
- **C（錯）** Guselkumab、Risankizumab 認識 IL-23 的 p19（選擇性），但 **Ustekinumab 認識的是 p40**（IL-12 與 IL-23 共用次單元），並非 p19。
- **D（對）** Anifrolumab 認識 **type I interferon receptor (IFNAR1)**，透過阻斷受體，阻斷所有 type I IFN（IFN-α、IFN-β、IFN-ω 等）的下游訊息傳遞。

**速記表：**

| 藥物 | 標靶 | Ligand 或 Receptor |
|---|---|---|
| Mepolizumab / Reslizumab | IL-5 | Ligand |
| Benralizumab | IL-5Rα | Receptor |
| Secukinumab / Ixekizumab | IL-17A | Ligand |
| Bimekizumab | IL-17A + IL-17F | Ligand |
| Guselkumab / Risankizumab / Tildrakizumab | IL-23 (p19) | Ligand |
| Ustekinumab | IL-12/23 (p40) | Ligand |
| Anifrolumab | IFNAR1 | Receptor |

### 2. 人體 germinal center 中 B cell 進行 affinity maturation
- A Somatic hypermutation
- B VDJ recombination
- C Class switch
- D Antibody glycosylation

**答案：A**

**解析：**

- **A（對）** **Somatic hypermutation (SHM)** 發生在 germinal center 的 dark zone，由 **AID (activation-induced cytidine deaminase)** 催化免疫球蛋白可變區基因產生點突變，之後在 light zone 由 Tfh cell 與 follicular dendritic cell (FDC) 篩選出對抗原親和力較高的 B cell clone，這個「突變 + 篩選」的循環就是 **affinity maturation** 的核心機轉。
- **B（錯）** VDJ recombination 發生在**骨髓**的 B cell 發育早期（pro-B → pre-B 階段），由 RAG1/RAG2 催化，屬於產生**初始抗體多樣性**的機制，跟 germinal center 內的親和力成熟無關。
- **C（錯）** Class switch recombination (CSR) 也發生在 germinal center，同樣需要 AID，但它改變的是抗體的 **isotype（如 IgM → IgG/IgA/IgE）**，不改變抗原結合的親和力，是與 affinity maturation **平行但獨立**的過程。
- **D（錯）** Glycosylation 是抗體轉譯後修飾，與親和力成熟無直接因果關係。

**速記：**

| 機制 | 發生位置 | 作用 |
|---|---|---|
| VDJ recombination | 骨髓 | 產生抗體初始多樣性 |
| Somatic hypermutation | Germinal center (dark zone) | 產生點突變 → 親和力篩選（affinity maturation） |
| Class switch recombination | Germinal center | 改變 isotype，不影響親和力 |

### 3. 關於 T cell 和 B cell 活化的相關敘述何者正確?
- A CTLA4 與 B7 的親和力大約等於 CD28 與 B7 的親和力
- B Follicular helper T cell 透過 CD40L 與 CD40 結合來刺激 B cell
- C T cell 活化時 T cell receptor 直接和 calcineurin 結合來傳導訊息
- D B cell 活化時 B cell receptor 會直接和 Syk 結合來傳導訊息

**答案：B**

**解析：**

- **A（錯）** **CTLA4 對 B7 (CD80/CD86) 的親和力遠高於 CD28**，並非「大約相等」。正因為 CTLA4 親和力較高，才能有效與 CD28 競爭 B7 結合，優先搶到共同的 ligand，藉此傳遞抑制訊號（這也是 CTLA4-Ig / abatacept 設計的藥理基礎）。
- **B（對）** **Tfh cell 透過 CD40L (CD154)** 與 B cell 表面的 **CD40** 結合，加上分泌 **IL-21**，共同促進 B cell 在 germinal center 中的存活、增殖與分化（包括前一題提到的 SHM、affinity maturation 及 class switch），是 Tfh-B cell help 的核心共同刺激路徑。
- **C（錯）** TCR **不會直接**與 calcineurin 結合。正確路徑是：TCR/CD3 複合體活化 → ZAP-70 → PLCγ 活化 → 產生 IP3 → 細胞內鈣離子上升 → 鈣離子結合 **calmodulin** → calmodulin-calcium 複合體才去活化 **calcineurin**（一種磷酸酶）→ calcineurin 去磷酸化 **NFAT**，使其進入細胞核。中間隔了好幾個步驟，並非直接結合。
- **D（錯）** BCR 本身胞內尾端很短，不直接結合 Syk。正確路徑是：抗原結合 BCR → 與 BCR 共同表現的 **Igα/Igβ (CD79a/CD79b)** 上的 ITAM 被 Src family kinase（如 Lyn）磷酸化 → 磷酸化的 ITAM 才去招募並活化 **Syk**（透過 Syk 的 SH2 domain 結合磷酸化 ITAM）。所以是 Igα/Igβ 中介，而非 BCR 直接結合 Syk。

**觀念重點：** C、D 兩個選項的錯誤模式相同——都是把「受體活化後一連串訊息傳遞的下游分子」誤植成「受體直接結合」，這是免疫學考題常見的陷阱句型，遇到「A 直接與 B 結合」的敘述要特別留意中間是否還有其他分子。

### 4. TLR 何者正確?
- A TLR 可專一識別每一種 viral RNA
- B TLR 在細胞膜辨識細菌胜肽
- C TLR9 識別未甲基化的 CpG DNA
- D 所有 TLR 皆在同一種免疫細胞上

**答案：C**

**解析：**

- **A（錯）** TLR 辨識的是**結構性的 PAMP（pathogen-associated molecular pattern）**，例如 TLR7/8 辨識 single-stranded RNA 的結構特徵、TLR3 辨識 double-stranded RNA，但這是廣泛辨識「RNA 的結構型態」，並非針對「每一種病毒」的 RNA 序列做專一性辨識，TLR 不具有像抗體或 TCR 那樣的體突變/重組產生的高度序列專一性。
- **B（錯）** 這句過度概化。TLR2（配合 TLR1 或 TLR6）辨識的是**細菌脂胜肽 (lipopeptide)**，位於細胞膜沒錯；但 TLR4 辨識的是 **LPS（脂多醣，不是胜肽）**，也在細胞膜；而許多重要的細菌/病毒核酸類 PAMP（如 TLR9 辨識的 CpG DNA、TLR3/7/8 辨識的病毒 RNA）是在**胞內體 (endosome)** 被辨識，並非都在細胞膜、也不是都辨識「胜肽」。
- **C（對）** **TLR9** 位於 endosome 內，專一辨識細菌與病毒 DNA 中**未甲基化的 CpG motif (unmethylated CpG DNA)**——這是因為脊椎動物基因體的 CpG 大多被甲基化，未甲基化的 CpG 在演化上被當作「非自我」的辨識訊號。
- **D（錯）** 不同免疫細胞表現的 TLR repertoire 不同。例如 **plasmacytoid dendritic cell (pDC)** 高量表現 TLR7、TLR9；**macrophage** 則以 TLR2、TLR4 為主；並非所有 TLR 都表現在同一種細胞上。

**TLR 分布速記：**

| TLR | 主要 ligand | 細胞位置 |
|---|---|---|
| TLR1/2/6 | Lipopeptide | 細胞膜 |
| TLR4 | LPS | 細胞膜 |
| TLR5 | Flagellin | 細胞膜 |
| TLR3 | dsRNA | Endosome |
| TLR7/8 | ssRNA | Endosome |
| TLR9 | Unmethylated CpG DNA | Endosome |

### 5. Pattern recognition receptor (PRR) 敘述何者正確?
- A PRR 專門識別宿主 DNA
- B TLR 位在細胞膜或內體內, 識別外來微生物分子
- C 所有的 PRR 專一性極高
- D PRR 由 T cell 與 B cell 表達

**答案：B**

**解析：**

- **A（錯）** PRR 辨識的主要對象是 **PAMP（pathogen-associated molecular pattern，外來病原體分子）**，而非宿主自身 DNA。雖然某些特殊情境下（如 cGAS 辨識到外洩至胞質的自身 DNA）PRR 也能感應 **DAMP（damage-associated molecular pattern）**，但「專門識別宿主 DNA」並非 PRR 的一般定義或主要功能。
- **B（對）** 呼應第 4 題內容：TLR 依種類不同，分布在**細胞膜**（如 TLR1/2/4/5/6）或**胞內體 endosome**（如 TLR3/7/8/9），共同功能是辨識外來微生物的保留性分子結構（PAMP），啟動先天免疫反應。
- **C（錯）** PRR 是**germline-encoded**（生殖系基因直接編碼，不經過體細胞重組），辨識的是廣泛且保留性高的分子結構模式（如 LPS、flagellin、CpG DNA），專一性遠低於 TCR/BCR（後者透過 VDJ recombination 與 somatic hypermutation 產生高度多樣性與專一性）。這是 innate immunity（PRR）與 adaptive immunity（TCR/BCR）在辨識機制上的根本差異。
- **D（錯）** PRR 是**先天免疫**的辨識受體，主要表現在 macrophage、dendritic cell、neutrophil、epithelial cell 等多種細胞上，並非由 T cell、B cell（適應性免疫細胞）表達為其代表性特徵；T/B cell 的抗原辨識受體是 TCR/BCR，走的是完全不同的辨識邏輯。

**觀念重點：** 第 4、5 題都是圍繞「Innate（PRR/TLR，廣泛/低專一性/germline-encoded）vs. Adaptive（TCR/BCR，高專一性/體細胞重組）」這個核心對比，是免疫學考題的高頻主軸，遇到「專一性極高」或「專門識別」這類絕對化敘述描述 PRR 時要提高警覺。

### 6–10（部分，來不及抄）之一：Rheumatoid factor 的自體抗原為?
- A Fab of IgG
- B Fab of IgM
- C Fc gamma chain of IgG
- D Fc alpha chain of IgG

**答案：C**

**解析：**

Rheumatoid factor (RF) 是一種**自體抗體**（本身多半是 IgM isotype，但也可以是 IgG 或 IgA），辨識的自體抗原是**已結合抗原之 IgG 的 Fc 恆定區**，也就是 IgG 重鏈（γ chain）的 Fc 部分（CH2-CH3 domain）。

- **A、B（錯）** Fab 是抗體的**可變區**（辨識外來抗原的那一端），不是 RF 攻擊的目標；RF 攻擊的是 Fc（恆定區）。
- **D（錯）** 這個選項本身是個**名詞錯誤配對**：免疫球蛋白重鏈依 isotype 各有專屬名稱——**IgG 重鏈是 γ (gamma) chain、IgM 是 μ (mu) chain、IgA 是 α (alpha) chain、IgD 是 δ (delta) chain、IgE 是 ε (epsilon) chain**。所以「Fc **alpha** chain of **IgG**」這個組合本身就自相矛盾——alpha chain 是 IgA 重鏈的命名，不會用在 IgG 身上。這題其實同時在考兩件事：(1) RF 認的是 Fc 不是 Fab；(2) IgG 重鏈的正確名稱是 gamma chain，不是 alpha chain。

**重鏈命名速記：**

| Isotype | 重鏈名稱 |
|---|---|
| IgG | γ (gamma) |
| IgM | μ (mu) |
| IgA | α (alpha) |
| IgD | δ (delta) |
| IgE | ε (epsilon) |

### 6–10（部分，來不及抄）之二：SLE 的小鼠模型，何者錯誤?
- A （忘記）
- B 近親繁殖會造成基因變異，也是一種 SLE 小鼠模型
- C SLE 小鼠模型可以用來測試新藥的效果
- D New Zealand Black/White 是現在最常用的 SLE 小鼠模型（考生註記：應該錯誤）

**⚠️ 答案未確認**（原始考生記錄「來不及抄」，選項 A 缺失，無法確定正確答案）

**參考分析：**

- **B（較可能是錯誤敘述）** 邏輯上有問題：**近親繁殖（inbreeding）本身是讓基因走向同質化（homozygosity），並不會「造成」新的基因變異**。經典 SLE 小鼠模型（NZB、NZB/W F1、MRL/lpr、BXSB/Yaa）是在近親繁殖品系中，因**自發性突變**被篩選、固定下來（如 MRL 的 *Fas^lpr* 突變、BXSB 的 *Yaa* duplication），而不是「近親繁殖」這個行為本身產生變異。這句話把「近親繁殖」跟「突變來源」的因果講反了。
- **C（應為正確敘述）** SLE 小鼠模型（自發性如 NZB/W、MRL/lpr，或誘導性如 pristane-induced lupus）本來就是藥物研發測試新藥效果的重要平台，敘述正確。
- **D（考生標記可能錯誤，但保留討論空間）** NZB/W F1 是**最早、最經典**的自發性 SLE 小鼠模型（prototype），但是否為「現在最常用」見仁見智——MRL/lpr 因病程快、症狀重，在許多實驗室也很常用。這句話的對錯較主觀，無法單憑現有資訊確定。

### 11. Scleritis 相關敘述何者正確？
- A 常合併 AAV, RA
- B 常是 RA 的初始表現
- C AAV 有 Scleritis 的話比較不會攻擊 lung, kidney
- D IBD, RP, AS 都可能有 Scleritis

**答案：A**

**解析：**

- **A（對）** Scleritis 約有近一半的病例會合併**全身性疾病（systemic disease）**，其中最常見的兩大類就是 **RA** 與 **AAV（尤其 GPA）**。尤其是 **necrotizing scleritis**（壞死性鞏膜炎）幾乎是這兩類疾病的招牌合併症，也是預後不良、需要積極全身性免疫抑制治療的警訊（可能進展成 scleromalacia perforans）。
- **B（錯）** Scleritis 通常出現在 **病程較久、已建立診斷、blood RF/anti-CCP 陽性的嚴重侵蝕性 RA** 病人身上，屬於**晚期的 extra-articular manifestation**，而非 RA 的初始表現。
- **C（錯）** 恰恰相反：AAV（尤其 GPA）出現眼部侵犯（包括 scleritis）時，通常代表疾病較活躍、systemic involvement 較廣泛，**與 lung、kidney 受侵犯的風險增加相關**，而非減少。
- **D（部分正確但不夠精確）** IBD、relapsing polychondritis (RP)、AS 這幾個疾病確實都可能有眼部發炎表現，但需要區分 **episcleritis（鞏膜外層炎）vs. scleritis（鞏膜炎）**：
  - **RP**：episcleritis 與 scleritis 都可能出現，是這題裡唯一與 scleritis 本身關聯性較強的。
  - **IBD**：眼睛表現以 **episcleritis** 較常見，scleritis 相對少見。
  - **AS**：典型的眼睛表現是 **acute anterior uveitis**（與 HLA-B27 相關），而非 scleritis。

  這個選項把「episcleritis 常見的疾病群」跟「scleritis」混在一起講，精確度不如 A，所以 A 是更佳答案。

**Episcleritis vs. Scleritis 常見合併疾病速記：**

| | 常見合併疾病 | 嚴重度 |
|---|---|---|
| Episcleritis | IBD、SpA（含 AS）、RA | 較輕微，通常自限 |
| Scleritis | RA（尤其晚期、seropositive）、AAV（尤其 GPA）、Relapsing polychondritis | 較嚴重，necrotizing type 可能致視力喪失，需全身性免疫抑制 |

### 12. DISH 相關敘述何者錯誤？
- A 與 DM, Obesity 相關
- B 與 HLA-B27 無關
- C 脊椎鈣化產生慢性黏連
- D 30 歲以前很少出現

**答案：C**

**解析：**

DISH (Diffuse Idiopathic Skeletal Hyperostosis) 的病理特徵是脊椎**前縱韌帶（anterior longitudinal ligament）及其他 entheses 的骨化**，形成典型的「**flowing osteophytes（流動狀骨贅）**」，橋接相鄰椎體。

- **A（正確敘述）** DISH 與代謝症候群相關的因子高度相關，包括 **糖尿病 (DM)、肥胖 (obesity)**、高胰島素血症、高尿酸血症、血脂異常等，是公認的危險因子。
- **B（正確敘述）** DISH **與 HLA-B27 無關聯**，這也是它與 AS 的重要鑑別點之一——AS 與 HLA-B27 高度相關，DISH 則不然。
- **C（錯誤敘述，本題答案）** 這句話把 DISH 的機轉錯誤地描述成類似 AS 的「**關節沾黏 / ankylosis**」。實際上 DISH 的病理是**韌帶／enthesis 的骨化增生（新骨形成）**，橋接椎體前外側，但**椎間盤空間（disc space）與 facet joint、SI joint 通常維持正常、不會被侵蝕或融合**——這正是 DISH 與 AS 在影像學上最重要的鑑別點：
  - **AS**：sacroiliitis → 關節本身被侵蝕、發炎、最終**融合（true joint ankylosis）**，合併 disc space 的 syndesmophyte 形成 "bamboo spine"。
  - **DISH**：SI joint 與 disc space 大致**保留正常**，只是韌帶外側形成大塊、流動狀的骨贅橋接，造成活動度下降，但機轉是**骨化增生**而非「發炎侵蝕後的關節沾黏」。
- **D（正確敘述）** DISH 好發於**中老年族群**（多在 50 歲以上診斷，盛行率隨年齡增加，80 歲以上可達 25–35%），**30 歲以前極為少見**。

**DISH vs. AS 鑑別速記：**

| | DISH | AS |
|---|---|---|
| HLA-B27 | 無關 | 高度相關 |
| SI joint | 保留正常 | 侵蝕、發炎、融合 |
| Disc space | 保留正常 | Syndesmophyte、bamboo spine |
| 骨贅型態 | Flowing osteophytes（前外側） | Marginal syndesmophytes（垂直生長） |
| 好發族群 | 中老年、代謝症候群相關 | 年輕成人、男性略多 |

### 13. 36 y/o AS for 12 years，近三個月多處疼痛（neck, upper and lower trunk, shoulders, elbows, hips, knee），目前使用 Etoricoxib，以下敘述何者正確？
- A nociceptive pain, Etoricoxib 要加量
- B neuropathic pain, 要加 gabapentin
- C nociplastic pain, 要加 Tramadol
- D nociplastic pain, IL-17 在 central sensitization 有角色

**答案：D**

**解析：**

**先判斷疼痛型態：** 病人是已診斷 AS 12 年的病人，近三個月出現**廣泛、多處、跨區域**的疼痛（頸部、上下軀幹、雙肩、雙肘、雙髖、膝關節），這種分布模式**不符合典型 AS 發炎性關節/中軸疼痛**的局部型態，反而符合 **nociplastic pain（傷害感受可塑性疼痛，即中樞敏感化 central sensitization 主導的疼痛）** 的臨床特徵——這是長期發炎性關節炎病人（包括 AS、RA）常見的共病現象，類似 **fibromyalgia overlap**。

三種疼痛型態定義複習：

| 型態 | 機轉 | 典型例子 |
|---|---|---|
| Nociceptive pain | 組織實質發炎/損傷刺激傷害受器 | AS 的 sacroiliitis 發炎痛 |
| Neuropathic pain | 神經系統本身受損 | Diabetic neuropathy |
| Nociplastic pain | 中樞神經痛覺處理異常，無明顯組織損傷或神經病灶 | Fibromyalgia、長期發炎疾病合併的廣泛疼痛 |

- **A（錯）** 若歸類為 nociceptive pain 並加量 NSAID（Etoricoxib），對這種中樞敏感化主導的廣泛疼痛效果有限，且長期使用/加量 NSAID 有腸胃道、心血管、腎臟風險，並非對症處置。
- **B（錯）** 病人的疼痛分布廣泛、對稱性跨多個區域，並無神經病變常見的**燒灼感、麻刺感、皮節分布 (dermatomal distribution)** 等特徵，歸類為 neuropathic pain 並不恰當。
- **C（錯）** 雖然 nociplastic pain 的判斷方向正確，但治療建議錯誤——**Tramadol（類鴉片藥物）並非 nociplastic pain / fibromyalgia 的建議一線用藥**，現行指引不建議使用類鴉片藥物治療此類疼痛，因為長期療效證據不足，且有成癮、耐受性等風險，甚至可能加重中樞敏感化。Fibromyalgia/nociplastic pain 建議的藥物是 **duloxetine、pregabalin、milnacipran** 等，並強調非藥物治療（運動、認知行為治療）。
- **D（對）** 正確歸類為 nociplastic pain，且點出近年研究關注的機轉：**IL-17 除了驅動周邊關節/附著點的發炎反應外，也被認為在 central sensitization（中樞敏感化）的路徑中扮演角色**，可能參與解釋為何部分 SpA 病人即便發炎控制良好，仍持續有廣泛性疼痛（類似 fibromyalgia 表現）。這也呼應這份考卷第 1 題對 IL-17 pathway 的重視。

### 15. 35 歲張先生，下背痛及髖關節疼痛多年，早上起床最嚴重，隨著日常活動疼痛會改善，關於張先生的影像檢查描述，何者錯誤?
- A Pelvis X 光看到 sacroiliac joint 有 pseudowidening
- B Pelvis MRI 在 STIR sequence 看到 sacroiliac joint 有 bone marrow edema
- C L-spine MRI 在 STIR sequence 看到 vertebral corners with increased signal indicating fat metaplasia
- D T-spine X 光看到右側有 large, flowing, osteophytes

**答案：C**

**臨床情境：** 35 歲男性，多年下背痛合併髖痛，**晨僵明顯、活動後改善**——典型 **inflammatory back pain (IBP)**，指向 **axial SpA / AS**。

**解析：**

- **A（正確敘述）** X 光上 sacroiliitis 早期因關節邊緣被侵蝕（erosion），造成關節間隙看起來變寬的假象，稱為 **pseudowidening**，是 AS 早期 SI joint X 光的典型表現之一。
- **B（正確敘述）** **STIR (Short Tau Inversion Recovery)** 是壓脂序列，對水分/水腫訊號極敏感，是偵測 **active sacroiliitis（bone marrow edema）** 的標準序列，也是 ASAS MRI 定義 active sacroiliitis 的核心依據。
- **C（錯誤敘述，本題答案）** MRI 序列的物理特性被講反了。**STIR 序列會把脂肪訊號壓掉（fat suppression）**，STIR 上呈現**高訊號**代表的是**水腫（bone marrow edema，發炎活性）**，而不是脂肪。相反地，**fat metaplasia（脂肪化生，慢性結構性變化，即 Romanus lesion / fatty corner lesion）** 是在**未壓脂的 T1-weighted 序列**上呈現高訊號；若真的做在 STIR 上，脂肪化生反而會因壓脂而**訊號被抑制（變暗）**，不會是「increased signal」。STIR + 高訊號只對應 bone marrow edema（急性發炎），不會是 fat metaplasia（慢性變化），敘述自相矛盾。

  **脊椎 MRI 序列判讀速記：**

  | 序列 | 高訊號代表 | 意義 |
  |---|---|---|
  | STIR / T2 fat-sat | 水腫（edema） | Active inflammation |
  | T1（非壓脂） | 脂肪 | Fat metaplasia（慢性結構變化） |
  | T1 + Gd 顯影劑 + 壓脂 | 增強訊號 | Active inflammation（另一種偵測方式） |

- **D（敘述本身正確，但非本題答案）** 這句話描述的其實是 **DISH（而非 AS）的典型影像特徵**——**右側優位（right-sided predominance）的大塊流動狀骨贅**是 DISH 經典教學重點：主動脈在胸椎左側的搏動會抑制左側骨贅形成，所以 DISH 骨贅通常好發於**右側**（呼應第 12 題 DISH 內容）。此敘述作為獨立的影像學事實**本身正確**，只是與這位病人（臨床圖像較像 AS）不一定吻合；但題目問的是敘述本身「對不對」，而非「是否符合此病人」，故 D 不是錯誤選項。

### 16. 67 歲王女士，最近兩個月開始出現雙側肩膀及髖關節疼痛，尤其是早上起床最嚴重，持續時間超過一小時，抽血 ESR:80 mm/hr，RF 和 anti-CCP 數值正常，請問下列何者超音波影像上的發現對於診斷較無幫助?
- A 雙側肩膀有 subdeltoid bursitis
- B 雙側肩膀有 subscapularis tenosynovitis
- C 雙側肩膀有 glenohumeral synovitis
- D 雙側髖關節有 trochanteric bursitis

**答案：B**

**臨床情境：** 67 歲女性，急性發作（2 個月內）的**雙側**肩、髖疼痛，晨僵 **>1 小時**，ESR 顯著上升（80），RF/anti-CCP 陰性——典型 **Polymyalgia Rheumatica (PMR)** 表現。

**解析（對照 2012 EULAR/ACR PMR 分類標準的超音波項目）：**

2012 EULAR/ACR PMR provisional classification criteria 特別把超音波列為加分項目，其中定義的典型超音波發現為：

- **肩膀**：subdeltoid bursitis、**biceps tenosynovitis（長頭肱二頭肌肌腱鞘炎）**、glenohumeral synovitis（三者任一）
- **髖關節**：synovitis 和/或 trochanteric bursitis

分類標準的組合規則：**至少一側肩膀**有上述任一發現 + **至少一側髖**有 synovitis/trochanteric bursitis；或**雙側肩膀**都有上述任一發現，即可加分（有助於與 RA 鑑別）。

- **A（有幫助）** Subdeltoid bursitis 是 PMR 分類標準列出的典型肩部發現。
- **B（無幫助，本題答案）** 標準中提到的肌腱鞘炎是 **biceps（長頭肱二頭肌）tenosynovitis**，並非 **subscapularis tenosynovitis**。Subscapularis 是旋轉肌袖的一部分（附著於 lesser tuberosity），其肌腱鞘炎並非 PMR 分類標準定義的典型超音波表現，對診斷 PMR 較無幫助。
- **C（有幫助）** Glenohumeral synovitis 是 PMR 分類標準列出的典型肩部發現。
- **D（有幫助）** Trochanteric bursitis 是 PMR 分類標準列出的典型髖部發現。

**速記：PMR 超音波三合一（肩）+ 二選一（髖）**

| 部位 | 典型超音波發現 |
|---|---|
| 肩 | Subdeltoid bursitis、**Biceps (long head) tenosynovitis**、Glenohumeral synovitis |
| 髖 | Synovitis、Trochanteric bursitis |

### 17. 一個 RA + SLE overlapping syndrome 的病人，給一張手的 X 光照片問是下列哪個選項？
- A Swan-neck deformity
- B Jaccoud-like deformity
- C Boutonnière deformity
- D Sausage digits

**答案：B**（考生判讀依據：diffuse MCP joints dislocation 但 erosion 與 ankylosis 不嚴重）

**解析：**

這題的關鍵鑑別點在於**「有明顯的關節脫位/半脫位變形，但骨頭本身的侵蝕（erosion）與融合（ankylosis）並不嚴重」**——這正是 **Jaccoud（-like）arthropathy** 影像學上最經典的特徵，用來和 RA 的侵蝕性變形做區分。

- **B（對）** **Jaccoud arthropathy** 原本是描述 SLE 病人的一種**非侵蝕性（non-erosive）、可逆性（reducible）**的手部變形——因為關節囊、韌帶鬆弛（capsular/ligamentous laxity）造成 **MCP joint 尺偏（ulnar deviation）、半脫位（subluxation）**，外觀上很像 RA 的變形，但**骨頭本身沒有明顯侵蝕或融合**，且將手掌平放於 X 光板上時，變形通常可以被「矯正回去」（臨床身體檢查也可觀察到可用手動方式復位）。在 **RA + SLE overlap（rhupus）** 的病人身上，常見這種「外觀像 RA 變形、但影像上骨本身破壞程度遠低於典型 RA」的混合表現，稱為 **Jaccoud-like deformity**，是這類 overlap syndrome 很具代表性的影像特徵。
- **A（錯）** Swan-neck deformity（PIP hyperextension + DIP flexion）雖然也可能出現在 SLE 或 Jaccoud arthropathy 的變形譜系中，但它描述的是**單一關節層面的特定變形型態**，並非「diffuse MCP joint dislocation 但骨頭破壞輕微」這個整體影像描述的最佳答案。
- **C（錯）** Boutonnière deformity（PIP flexion + DIP hyperextension）是 **RA** 較典型的變形型態之一，通常伴隨較明顯的關節侵蝕，與題目描述的「erosion 不嚴重」不符。
- **D（錯）** Sausage digits（dactylitis，整根手指瀰漫性腫脹）是 **PsA、其他 SpA 或痛風**的特徵性表現，與 RA/SLE overlap 的變形機轉無關。

**RA vs. Jaccoud arthropathy 鑑別速記：**

| | RA | Jaccoud arthropathy（SLE / rhupus） |
|---|---|---|
| 機轉 | 滑膜發炎侵蝕骨頭 | 關節囊/韌帶鬆弛，骨頭本身較少被破壞 |
| 變形性質 | 固定、不可逆 | 可逆、可被動矯正 |
| X 光侵蝕 | 明顯 erosion | erosion 輕微或無 |

### 18.（影像題）給 T1 SI joint MRI，問 SI joint 旁邊白色一塊是什麼?

**答案：Fat metaplasia**

**解析：**

呼應第 15 題的 MRI 序列觀念——**T1-weighted（非壓脂）序列上，脂肪本身就會呈現高訊號（白色）**。SI joint 旁邊骨頭因為慢性發炎反覆修復，局部骨髓被脂肪組織取代（**fat metaplasia / fatty infiltration**），在 T1 序列上就會呈現這種鄰近關節面的白色亮訊號，是 axSpA **慢性結構性變化**的指標之一（與 STIR 上呈現的 bone marrow edema 屬於「急性發炎」訊號相對應，兩者合併判讀可以同時掌握疾病的「活性」與「慢性化程度」）。

### 19.（影像題）一個 RA 的病人, 給 wrist 的 MRI, 在 radial ulnar bone 之間有白色亮起來的地方是什麼?
- A Bone marrow edema
- B Synovitis
- C Bone erosion
- D Bone sclerosis

**答案：B**

**解析：**

關鍵在於病灶的**位置**——題目描述的白色亮訊號是在 **radius 和 ulna 之間**，也就是**關節腔/滑膜隱窩（joint space / synovial recess）**，而不是在骨頭本身內部。

- **A（錯）** Bone marrow edema 是骨頭**內部（骨髓腔）**的訊號變化，會出現在 radius 或 ulna 的骨髓裡，而非兩骨之間的關節腔。
- **B（對）** 兩骨之間的關節腔內出現高訊號，代表**滑膜增生/關節積液（synovial proliferation/effusion, pannus）**，也就是 **synovitis**——這是 RA 最核心的病理表現，也是 MRI 偵測 RA 早期病灶最敏感的發現之一（甚至早於 X 光可見的骨侵蝕）。
- **C（錯）** Bone erosion 是骨頭皮質/骨小樑本身的**缺損**，會呈現骨頭結構本身的中斷，而非單純關節腔內的高訊號。
- **D（錯）** Bone sclerosis 在 MRI 上因為骨質緻密，訊號通常是**偏低（暗）**，不會呈現高訊號（白色）。

**病灶位置判讀速記：**

| 病灶位置 | 代表意義 |
|---|---|
| 骨頭內部高訊號 | Bone marrow edema |
| 關節腔內高訊號 | Synovitis（滑膜炎/積液） |
| 骨頭皮質缺損 | Bone erosion |
| 骨頭訊號偏低（暗） | Sclerosis |

### 20.（影像題）給一個類似正常孩童手部的圖片，問是什麼？
- A （忘記）
- B Ankylosing spondylitis
- C Psoriatic arthritis
- D JIA

**答案：D**

**⚠️ 此題無實際圖片可核對，以下為依教學重點的推論分析**

**解析：**

這題的關鍵教學重點在於：**兒童發炎性關節炎（尤其是 JIA）的手部 X 光，早期常常「看起來幾乎正常」或只有非常細微的變化**，跟成人 RA 那種明顯的侵蝕性破壞完全不同，這正是這題想考的陷阱——如果沒有仔細看，很容易誤判為「正常手部」。

- **D（對）** JIA（尤其是 **oligoarticular JIA**）早期的手部 X 光變化通常很細微，包括：
  - **周邊軟組織腫脹（soft tissue swelling）**——常是唯一線索
  - **關節周圍骨質流失（periarticular osteopenia）**
  - **骨骺過度生長（epiphyseal overgrowth / ballooning）**——因為兒童的生長板（growth plate）還未閉合，慢性滑膜炎造成的局部充血反而會刺激骨骼**局部生長加速**，導致受影響關節的骨骺比對側大、骨齡（bone age）超前，而不是像成人 RA 那樣出現明顯的侵蝕
  - 因為沒有戲劇化的侵蝕或關節間隙狹窄，整體外觀確實可能被誤認為「接近正常」的孩童手部
- **B（錯）** AS 是中軸關節為主的疾病，通常不會用「手部 X 光」作為主要診斷影像，且 AS 極少在兒童期以手部病灶表現。
- **C（錯）** PsA 的手部影像通常有較 specific 的表現（如 DIP 關節侵犯、periostitis、"pencil-in-cup" 變形、dactylitis 造成的軟組織腫脹型態），不太會呈現「類似正常」的樣貌。

**兒童 vs. 成人發炎性關節炎影像鑑別重點：**

| | 成人 RA | 兒童 JIA |
|---|---|---|
| 骨骼狀態 | 生長板已閉合 | 生長板未閉合 |
| 慢性發炎後果 | 骨侵蝕、關節間隙狹窄 | 骨骺過度生長、骨齡超前、局部巨指(趾)化 |
| 早期影像 | 軟組織腫脹 + 骨質流失 | 軟組織腫脹（常是唯一線索，容易被忽略） |

### 21. 病人使用 adalimumab 治療一陣子後, 出現手腳紅色發癢, 病理切片診斷為 paradoxical psoriasis. 請問以下何者錯誤？
- A 病理切片 T cell 比 vulgaris 少
- B 成因是 Type I interferon overproduction
- C 主要是因為抑制 pDC cell maturation
- D 是 TNFi 特有的現象, 其他生物製劑不會

**答案：D**

**解析（對照 Conrad et al. 2018, *Nat Commun*；Maronese et al. 2024, *J Invest Dermatol*）：**

- **A（正確敘述）** Conrad et al. (2018) 證實 paradoxical psoriasis (PP) 病灶切片中的 **CD8+ T cell 與傳統 dendritic cell 數量，比傳統乾癬（psoriasis vulgaris）病灶明顯較少**，並用小鼠模型證明清除 αβ-T cell 與 γδ-T cell 並不影響 PP 的發生——代表 PP 是**T cell 非依賴性**的先天免疫過度活化反應，與傳統乾癬（T cell 依賴）機轉不同。
- **B（正確敘述）** 核心機轉正是 **Type I interferon (IFN-α) 過度分泌**。
- **C（正確敘述）** 機轉細節：**TNF-α 正常會促進 pDC 走向成熟**，成熟的 pDC 會停止產生 IFN-α；使用 anti-TNF 藥物後，這個成熟過程被阻斷，pDC 停留在**未成熟、持續分泌 IFN-α** 的狀態，造成 IFN-α 過度累積。所以說「主要是因為抑制 pDC maturation」是精確描述機轉上游的原因。
- **D（錯誤敘述，本題答案）** 這句話**不正確**。矛盾反應（paradoxical psoriasis / paradoxical reaction）**並非 TNFi 專屬現象**，其他生物製劑也有明確報告：
  - **Rituximab、Tocilizumab、Ustekinumab、Abatacept** 都有誘發或惡化乾癬的案例報告
  - **Dupilumab（anti-IL-4/13）** 透過 Th2→Th1/Th17 反轉機轉，誘發矛盾性乾癬
  - **Anti-IL-17A（secukinumab、ixekizumab 等）** 反而透過 TSLP 上升誘發**矛盾性濕疹（方向相反）**
  - Anti-IL-12/23、JAK inhibitor 也都有零星案例報告

  只是因為 **anti-TNF 上市最早、適應症最廣、使用時間最久**，所以文獻報告數量遠多於其他藥物類別，並非機轉上「TNFi 獨有」。

**銜接：** 這題把「矛盾反應不是 TNFi 專屬」講得很清楚，呼應獨立筆記〈[Paradoxical_Psoriasis.md](../../跨疾病主題/Paradoxical_Psoriasis.md)〉裡的三角機轉框架（anti-TNF / anti-IL-4-13 / anti-IL-17 三種藥物各自誘發不同方向的矛盾反應）。

### 22. 下列何者敘述錯誤？
- A TNFi 治療 RA, 加入 methotrexate 有益處
- B TNFi 對 GPA, GCA 都有療效
- C Anti-IL6R 對 GCA 有效果
- D Anti-IL17A/F 對 AS 有效果

**答案：B**

**解析：**

- **A（正確敘述）** TNFi 治療 RA 時**併用 methotrexate** 已證實有益處——不只加強療效，還能**降低 anti-drug antibody（免疫原性）的產生**，提升藥物的 drug retention rate，這是臨床常規做法（尤其對單株抗體類 TNFi，如 infliximab、adalimumab 更明顯）。
- **B（錯誤敘述，本題答案）** TNFi **對 GPA 與 GCA 其實都沒有證實療效**，甚至有負面試驗結果：
  - **GPA**：**WGET trial** 顯示在標準治療上加上 **etanercept** 並無法有效維持疾病緩解，且合併使用還增加了 solid malignancy 的風險，因此 TNFi **不建議**用於 GPA。
  - **GCA**：TNFi（infliximab、etanercept）的臨床試驗同樣呈現 **negative results**，未證實對 GCA 有效，目前並非 GCA 的治療選項。
- **C（正確敘述）** **Tocilizumab（anti-IL-6R）** 已由 **GiACTA trial** 證實對 GCA 有效，並獲得核准用於 GCA 治療，是目前 GCA 生物製劑治療的標準選項。
- **D（正確敘述）** **IL-17A（或 IL-17A/F 雙標）抑制劑**（如 secukinumab、ixekizumab，以及雙標的 bimekizumab）已證實對 **AS** 有效，是 AS 生物製劑治療的核准選項之一。

**血管炎生物製劑治療速記（避免和 RA/SpA 混淆）：**

| 疾病 | 有效的生物製劑 | 無效/不建議 |
|---|---|---|
| GPA/MPA (AAV) | Rituximab（anti-CD20） | TNFi（WGET trial 陰性） |
| GCA | Tocilizumab（anti-IL-6R） | TNFi（negative trials） |
| AS | TNFi、IL-17A(/F)i | — |

### 23. MMF 的藥理機轉？

**答案：Inhibit IMPDH**

**解析：**

**Mycophenolate mofetil (MMF)** 是 **inosine monophosphate dehydrogenase (IMPDH)** 的抑制劑。IMPDH 是**purine（嘌呤）de novo 合成路徑**中的關鍵限速酶，負責將 IMP 轉換為 XMP，最終合成 guanosine nucleotide（鳥嘌呤核苷酸）。

**選擇性作用在淋巴球的原因：** 大多數細胞可以透過 **salvage pathway（補救途徑）** 回收利用嘌呤，不完全依賴 de novo 合成；但 **T cell 與 B cell 的增生高度依賴 de novo purine 合成路徑**，salvage pathway 相對不足，因此 MMF 抑制 IMPDH 後，會**優先抑制淋巴球的增生**，達到免疫抑制效果，相對其他細胞受影響較小。

### 25. 關於 RF 抗體何者正確
- A 專一性強, 數值高代表病況嚴重
- B ACPA 代表病程慢, 在疾病晚期出現
- C anti-PAD4 IgA 可以在病程晚期出現, 並在肺部表現
- D citrulline 可透過 PADI 轉成 arginine

**答案：C**（採排除法推論）

**解析：**

- **A（錯）** RF（rheumatoid factor）的**專一性其實不高**——許多感染、其他自體免疫疾病、甚至健康老年人都可能有 RF 陽性，是敏感度尚可但專一性較差的抗體。雖然「titer 越高、病況/erosive disease 傾向越嚴重」這部分的關聯性大致成立，但整句話因為「專一性強」這個前提錯誤而不成立。
- **B（錯）** **ACPA（anti-citrullinated protein antibody，如 anti-CCP）通常在疾病「早期」甚至臨床發病「之前數年」就可以偵測到**（pre-clinical phase），並且 ACPA 陽性通常預示**較嚴重、較具侵蝕性（erosive）的病程**，而非「病程慢、晚期才出現」。這句話把時序和嚴重度都講反了。
- **C（對，本題答案）** **Anti-PAD4 IgA** 抗體與 **RA-ILD（類風濕性關節炎相關間質性肺病）** 的關聯性是近年研究重點之一，呼應「RA 致病機轉的肺部起源假說（mucosal/lung origin hypothesis）」——citrullination 反應可能最先發生在肺部（尤其吸菸者），透過 NETosis 等機轉產生 citrullinated protein，進而誘發 ACPA/anti-PAD4 免疫反應，這個機轉與肺部侵犯（ILD）的關聯性、以及疾病進程中抗體 repertoire 的擴展（epitope spreading）皆有文獻支持。
- **D（錯）** 這句話把 **PADI（peptidylarginine deiminase）** 的作用方向講反了。PADI 的功能是把蛋白質中的 **arginine（精胺酸）轉換成 citrulline（瓜胺酸）**（這個過程稱為 **citrullination / deimination**），而不是「citrulline 轉成 arginine」。方向完全相反。

**PADI 反應方向速記：** Arginine **→（PADI/PAD 酵素）→** Citrulline（這就是 RA 中 ACPA 辨識的目標抗原被製造出來的方式）

### 26. RA 的病人出現紫斑、甲床出血點、手麻和腹痛，下面哪個是最可能的診斷
- A SLE
- B 病毒感染
- C 硬皮症
- D RA vasculitis

**答案：D**

**解析：**

這是**多重器官同時出現血管炎表現**的典型組合，在已知 **RA** 病人身上出現，最符合 **Rheumatoid vasculitis (RV)**：

- **紫斑（purpura）**：皮膚小血管炎的表現
- **甲床出血點（nail fold infarcts / splinter hemorrhages）**：**digital vasculitis** 的經典徵象，是 RV 很具特色的周邊表現
- **手麻**：提示**周邊神經血管炎**，造成 **mononeuritis multiplex**（多發性單神經病變，血管炎侵犯滋養神經的血管導致神經缺血）
- **腹痛**：提示**腸繫膜血管炎（mesenteric vasculitis）**，是 RV 較嚴重、可能導致腸缺血甚至穿孔的表現

**Rheumatoid vasculitis 好發族群：** 通常發生在**病程長、seropositive（RF/ACPA 陽性）、有侵蝕性關節破壞**的 RA 病人身上，是 RA 的嚴重 extra-articular manifestation 之一。

- **A（錯）** SLE 雖然也可以有皮膚血管炎、周邊神經侵犯，但題目明確是「RA 的病人」出現這些新表現，最直接合理的診斷是 RA 本身的併發症，而非另外診斷 SLE overlap（除非有額外 SLE 特異性的抗體或臨床表現佐證）。
- **B（錯）** 病毒感染（如 parvovirus B19、hepatitis）可能引起關節痛與皮膚疹，但不會同時精準符合「甲床出血點 + 手麻 + 腹痛」這種多器官血管炎的組合。
- **C（錯）** 硬皮症（scleroderma）的皮膚表現是**皮膚增厚、硬化**，並非紫斑；其血管病變也以 Raynaud's、digital ulcer 為主，臨床圖像與此不符。

### 27. anti-MDA5 抗體同時合併下列何者抗體, 比較不會臨床惡化
- A Jo-1
- B Ro-52
- C SAE
- D RNA polymerase 1

（考生註記：北榮弘正老師的研究）

**答案：C**

**⚠️ 此題引用特定研究（北榮），我沒有原始論文可逐字核對，以下為依既有文獻脈絡的推論，信心度中等**

**解析：**

- **B（會使病況惡化，非本題答案）** 目前文獻中較確立的關聯是：**anti-MDA5 合併 anti-Ro52（anti-Ro52/TRIM21）陽性**，與**較差的預後**相關——包括更高比例的 **rapidly progressive ILD (RP-ILD)** 與更高死亡率。這是多篇文獻反覆驗證的組合，代表 Ro-52 co-positivity 是**惡化**因子，而非「比較不會惡化」。
- **C（對，本題答案）** **Anti-SAE（anti-SAE1）** 抗體相關的 dermatomyositis，臨床上通常以**皮膚表現為主、肌肉侵犯較輕**，且 ILD 的嚴重度、進展速度相對**較溫和**，與典型 anti-MDA5 associated rapidly progressive ILD 的兇猛病程形成對比。因此當 anti-MDA5 合併 anti-SAE 陽性時，相對於合併 Ro-52 陽性的病人，臨床病程**較不易快速惡化**。
- **A（不典型組合）** anti-Jo-1 屬於 antisynthetase antibody，典型對應 **antisynthetase syndrome**（肌肉炎、機械手、Raynaud's、慢性 ILD 為主），與 anti-MDA5 associated **clinically amyopathic DM**（少肌肉侵犯，但 ILD 兇猛）在臨床表現上本質不同，兩者同時陽性較不典型，較難直接類比。
- **D（不典型組合）** anti-RNA polymerase III 是**硬皮症（尤其 scleroderma renal crisis 高風險型）** 的標誌性抗體，與 DM/anti-MDA5 的疾病譜較不相關，同時陽性也是少見的 overlap 情境。

**MDA5 相關研究小提醒：** 這類「哪個 co-existing antibody 影響 anti-MDA5 DM 預後」的題目，近年在台灣風濕科研究社群（尤其北榮團隊）是熱門主題，建議之後準備時可以針對 anti-MDA5 + Ro-52（惡化）、anti-MDA5 + anti-SAE（相對溫和）這組對比額外找原始文獻核對確認。

### 28. 下列小分子藥物何者含有重氫 (deuterium, H2)
- A Ritlecitinib
- B Baricitinib
- C Upadacitinib
- D Deucravacitinib

**答案：D**

**解析：**

**Deucravacitinib**（商品名 Sotyktu，2022 年核准用於乾癬的 **TYK2 inhibitor**）的名字本身就是關鍵記憶點——字首 **"Deu-"** 就是在暗示 **Deuterium（重氫，²H）**。

這個藥物的設計刻意在其中一個**甲基（methyl group）**上，把原本的氫原子替換成重氫（這個技術稱為 **deuterium substitution / "deuterium switch"**）。因為 C-D（碳-重氫）鍵比 C-H 鍵更穩定、更不容易被 **CYP450 酵素氧化代謝**，所以這個取代可以**減緩藥物在該位置的代謝速率**，進而改變藥物的暴露量/藥物動力學特性（但不影響其藥效學作用本身）。這是近年小分子藥物設計的一個新穎技巧，也是 Deucravacitinib 上市時被特別強調的技術亮點，因此變成很好考的「冷知識」考點。

- **A、B、C（都不含重氫）** Ritlecitinib（JAK3/TEC family inhibitor）、Baricitinib（JAK1/2i）、Upadacitinib（JAK1i）都是傳統的氫原子結構，沒有這個特殊的 deuterium substitution 設計。

**記憶口訣：** 看到藥名開頭是 **"Deu-"**，直接聯想 **"Deuterium"**——這題本質上是在考「有沒有注意到這個藥名本身就是提示」。

### 29. 病人有下背痛, HLA-B27 陽性, X 光檢查是正常的, 下一步建議安排甚麼檢查?

**答案：MRI**

**解析：**

這是典型的 **non-radiographic axial spondyloarthritis (nr-axSpA)** 臨床情境——病人有慢性下背痛、**HLA-B27 陽性**（強烈提示 axSpA 的免疫遺傳背景），但**X 光正常**。

**為什麼下一步是 MRI，而不是重複 X 光或直接下診斷？**

- X 光要能看到 **sacroiliitis**（關節侵蝕、硬化、融合），代表 SI joint 已經經歷**多年的結構性破壞**才會顯影，屬於**慢性、不可逆**的變化。
- 但發炎反應（active inflammation，即 **bone marrow edema**）在骨頭結構被破壞之前就已經存在，這時候**X 光看不到，但 MRI（尤其 STIR 或 T2 fat-sat 序列）可以偵測到**——這也呼應第 15、18 題討論的序列原理。
- 因此，當臨床高度懷疑 axSpA（年輕、發炎性下背痛型態、HLA-B27 陽性）但 X 光正常時，**MRI 是下一步標準檢查**，目的是尋找**尚未形成結構破壞、但已經有發炎活性**的 sacroiliitis，這正是 **ASAS axSpA 分類標準**中「影像學路徑（imaging arm）」的核心精神：X 光陰性不代表沒有 axSpA，MRI 陽性（active sacroiliitis）合併 ≥1 個 SpA 特徵，一樣可以歸類為 axSpA（非放射學型，nr-axSpA）。

**臨床意義：** 這代表 axSpA 的診斷不再需要等到病人出現不可逆的 X 光變化才能確立，MRI 讓臨床可以在疾病**更早期**就介入治療，避免延誤診斷（過去平均延遲診斷可達 8-10 年，MRI 的應用是縮短這個延遲的關鍵工具之一）。

### 30. 承上題, 病人幾年後出現 anterior uveitis, 應該選擇甚麼治療?

**答案：Monoclonal antibody（抗體類）anti-TNF-α**（詳細藥物選擇見下方）

**解析：**

延續第 29 題，這位病人已確立 axSpA 診斷，數年後出現 **anterior uveitis**（axSpA 最常見的 extra-articular manifestation 之一），需要調整治療策略。

**第一線：Anti-TNF-α「抗體類分子」，而非 soluble receptor**

- **首選藥物：Adalimumab、Infliximab、Golimumab、Certolizumab pegol**
  - Adalimumab、Golimumab 為 humanized monoclonal antibody；Infliximab 為 chimeric monoclonal antibody
  - **Certolizumab pegol** 雖然是 **PEGylated Fab' fragment**（缺少 Fc 區域，非完整單株抗體），但同樣屬於「直接中和 TNF-α 的抗體類分子」，且有專門試驗數據支持可**降低 uveitis flare 復發率**，因此也是合理選項
- **關鍵原則：不選 Etanercept**——Etanercept 是 **p75 TNF soluble receptor fusion protein**，並非抗體類分子。多篇註冊登錄研究（包括我們前面讀過的 Toussirot 2016 文獻）都顯示 **etanercept 對預防/治療 uveitis 的效果較差**，甚至有研究顯示 etanercept 使用者的 uveitis 發生率/復發率**反而高於** infliximab、adalimumab。這是因為 uveitis 的發炎機轉可能需要**更完整、更持續地中和 TNF-α**，soluble receptor 的中和效果相對較弱、較不完全。

**若 anti-TNF 效果不佳或不適用，次線選擇：**

- **IL-17 inhibitor：Bimekizumab**（IL-17A/F 雙重抑制劑）——與傳統 IL-17A 選擇性抑制劑（secukinumab、ixekizumab）不同，這兩者過去對 uveitis 的證據較不一致、甚至有疑慮（IL-17 路徑抑制在部分研究中與 uveitis 控制不佳相關），但 **bimekizumab 有較新的證據支持可用於 acute anterior uveitis (AAU)**（呼應這份考卷第 32 題）。
- **JAK inhibitor：Upadacitinib**——JAK1 選擇性抑制劑，在 axSpA 臨床試驗（如 SELECT-AXIS 系列）的次要指標中，顯示可以**降低 uveitis flare 的發生率**，是另一個有實證支持的選項。

**治療階梯總整理：**

| 優先順序 | 藥物類別 | 具體藥物 | 備註 |
|---|---|---|---|
| 1st line | Anti-TNF 抗體類分子 | Adalimumab、Infliximab、Golimumab、Certolizumab pegol | **避開 Etanercept**（soluble receptor，效果較差/uveitis 風險可能較高） |
| 2nd line | IL-17A/F 雙重抑制 | Bimekizumab | 優於傳統 IL-17A 選擇性抑制劑 |
| 2nd line | JAK1 inhibitor | Upadacitinib | 有降低 uveitis flare 的證據 |

### 31. AS 的病人有 IBD, 不能有下列哪一種藥物?

**答案：IL-17i（如 ixekizumab、secukinumab）**

**解析：**

**核心原則：IBD 病人（或有 IBD 病史者）應避免使用 IL-17 inhibitor。**

**機轉：** IL-17 在腸道黏膜其實扮演**保護性角色**——維持腸道上皮屏障完整性、協助對抗腸道菌叢與病原體。當使用 IL-17 inhibitor（如 secukinumab、ixekizumab）阻斷這條路徑時，**反而可能誘發或惡化 IBD**，這點在臨床試驗中已有明確證據：

- Secukinumab 在 **Crohn's disease** 的臨床試驗中，不僅沒有療效，反而觀察到**病情惡化**的訊號，導致試驗提前終止
- 因此 IL-17i 在藥物仿單上，對於**合併 IBD 或有 IBD 病史**的病人，屬於**應避免使用（相對禁忌）**的藥物類別

**臨床意義：** 這也是為什麼在 AS 治療選擇上，若病人同時有 IBD，治療選擇應優先考慮：

| 藥物類別 | 對 IBD 的影響 | AS 治療角色 |
|---|---|---|
| Anti-TNF（monoclonal antibody，如 infliximab、adalimumab） | 對 IBD 也有效 | 首選（一藥兩治） |
| IL-17i（secukinumab、ixekizumab、bimekizumab） | **可能誘發/惡化 IBD** | **避免使用** |
| JAK inhibitor（如 upadacitinib） | 對 IBD 也有適應症 | 可考慮 |

（呼應第 30 題：Etanercept 雖然是 anti-TNF，但因為是 soluble receptor，**對 IBD 沒有療效**，若病人同時有 IBD，也不會選 etanercept，應選 monoclonal antibody 類的 anti-TNF）

### 32. 關於 SpA 併發 Acute anterior uveitis 下列敘述何者正確
- A Etanercept 效果優於 Sulfasalazine
- B 各種 TNFi 效果相似
- C IL-17A/Fi (Bimekizumab) 可用於治療 AAU
- D JAKi 沒有用於 AAU 的臨床實證

**答案：C**

**解析：**

- **A（錯）** Etanercept 對 uveitis 的效果其實**不如**其他 anti-TNF 單株抗體，甚至有研究顯示其 uveitis 復發率**較高**（見第 30 題）；而 **sulfasalazine** 過去在 HLA-B27 相關的 recurrent anterior uveitis 預防上，其實有一定的實證基礎。這句話的比較方向是錯的。
- **B（錯）** 呼應第 30 題：TNFi 之間效果**並不相似**——**monoclonal antibody 類**（adalimumab、infliximab、golimumab、certolizumab）對 uveitis 有明確效果，但 **etanercept（soluble receptor）效果明顯較差**，兩者不能一概而論。
- **C（對，本題答案）** **Bimekizumab（IL-17A/F 雙重抑制劑）** 有較新的證據支持可用於治療 **acute anterior uveitis (AAU)**，這點與傳統 IL-17A 選擇性抑制劑（secukinumab、ixekizumab）過去證據不一致、甚至有疑慮的情況不同，是近年來的重要進展。
- **D（錯）** **Upadacitinib（JAK1i）** 在 axSpA 相關臨床試驗（SELECT-AXIS 系列）的次要指標中，已顯示可以**降低 uveitis flare 的發生率**，所以 JAKi 並非「沒有臨床實證」，而是已有初步證據支持。

### 33. SLE 的生物製劑治療下列何者正確
- A 相比於 Belimumab, Rituximab 更需要定期監測 IgG level
- B Anifrolumab 目前健保核准用於狼瘡腎炎的治療
- C Belimumab 比 Anifrolumab 更容易產生 herpes zoster
- D Belimumab 作用位點是 BAFF

**答案：A、D**（⚠️ 此題經核對後 A、D 皆為正確敘述，可能是原題允許多選，或考生重建考題時選項細節有落差，僅供參考）

**解析：**

- **A（正確）** **Rituximab（anti-CD20，B cell depletion）** 使用上，尤其是**多次療程後**，有明確的 **hypogammaglobulinemia（低免疫球蛋白血症）** 風險，臨床上會**定期監測 IgG level**（並在治療前先建立 baseline）。相對地，**Belimumab（anti-BAFF）** 雖然理論上也可能影響免疫球蛋白，但臨床上對 IgG 監測的重視程度與常規性遠不如 rituximab。
- **B（錯）** **Anifrolumab** 的關鍵臨床試驗（TULIP-1、TULIP-2）**排除了嚴重活動性狼瘡腎炎**的病人，目前核准適應症是**中重度、非腎臟侵犯的 SLE**。狼瘡腎炎的療效仍在另外的試驗中評估（如 IRIS study），**目前並未核准、健保也未給付**於狼瘡腎炎治療。
- **C（錯，方向講反）** 方向相反——**Anifrolumab（阻斷 type I IFN receptor）** 因為阻斷了對抗病毒（尤其 herpesvirus）很重要的 type I IFN 路徑，臨床試驗中 **herpes zoster 發生率明顯較高**，是已知的不良反應信號；相對地 Belimumab 的 herpes zoster 風險並沒有這麼突出。應該是「Anifrolumab 比 Belimumab 更容易產生 herpes zoster」才對。
- **D（正確）** **Belimumab** 就是 **anti-BAFF（B-cell activating factor，也稱 BLyS）** 單株抗體，是最基本、無爭議的藥理機轉事實。

### 34. SLE 病患準備懷孕, 下列哪個 DMARD 不能使用
- A Cyclosporine
- B Azathioprine
- C Sulfasalazine
- D Mycophenolate

**答案：D**

**解析：**

**Mycophenolate（MMF/MPA）是明確的致畸胎藥物（teratogen）**，懷孕禁用。已知會造成特定的胎兒畸形模式，包括：**唇顎裂（cleft lip/palate）、外耳畸形（microtia/耳道閉鎖）、小眼症**等特徵性的 "**mycophenolate embryopathy**"。臨床建議備孕婦女應在**停藥後至少 6 週至 3 個月**，確認轉換為懷孕安全藥物並疾病穩定後，才嘗試懷孕。

- **A（可用）** Cyclosporine 屬於懷孕相容（pregnancy-compatible）藥物，臨床上可持續使用於懷孕的風濕病人（例如合併使用於高風險妊娠的免疫抑制）。
- **B（可用）** Azathioprine 是懷孕期間**可以使用**的免疫抑制劑之一，是 SLE 病人備孕/懷孕期間最常用的替代藥物，安全性資料最充足。
- **C（可用）** Sulfasalazine 也是懷孕相容的 DMARD（需搭配補充 folic acid），常用於懷孕期間需要疾病控制的風濕病人。

**懷孕用藥速記：**

| 懷孕禁用（致畸胎） | 懷孕相對安全（可持續使用） |
|---|---|
| **Mycophenolate (MMF)** | Azathioprine |
| Methotrexate | Sulfasalazine |
| Leflunomide（需 washout） | Cyclosporine |
| Cyclophosphamide | Hydroxychloroquine（建議繼續使用） |

### 35. 下列哪個自體抗體與 neonatal lupus 最有關
- A Anti-dsDNA
- B Anti-Ribosomal P
- C Anti-SSA
- D Anti-Sm

**答案：C**

**解析：**

**Neonatal lupus (NL)** 是母體自體抗體透過胎盤傳給胎兒所導致的被動免疫疾病，與母親體內的 **anti-SSA/Ro（尤其 anti-Ro52）** 及 **anti-SSB/La** 抗體關聯性最強。

**機轉：** 這兩種抗體會**穿過胎盤**，進入胎兒循環，與胎兒心臟傳導系統的抗原產生交叉反應，導致**心臟傳導系統發炎、纖維化**，最終造成 **congenital heart block (CHB)**——這是 neonatal lupus 最嚴重、最具代表性的表現（通常是不可逆的三度房室傳導阻滯）。其他表現還包括**皮膚病灶（cutaneous NL，類似 SCLE 的環狀紅斑）**、短暫性肝功能異常、血球低下，這些通常會隨著母體抗體被代謝而自行緩解，唯獨心臟傳導阻滯多半是永久性的。

- **A（錯）** Anti-dsDNA 主要與 **lupus nephritis** 的活動性相關，與 neonatal lupus 的關聯性不強。
- **B（錯）** Anti-Ribosomal P 主要與 **neuropsychiatric lupus（尤其 lupus psychosis）** 相關。
- **D（錯）** Anti-Sm 是 SLE 高度專一性的抗體，但與 neonatal lupus 的關聯性遠不如 anti-SSA/SSB。

**臨床意義：** 這也是為什麼 anti-SSA/SSB 陽性的孕婦（無論是否已確診 SLE 或 Sjögren's），懷孕期間都需要**定期胎兒心臟超音波監測**，並在必要時考慮使用 hydroxychloroquine 來降低 CHB 風險（呼應第 67 題口試提到的 pregnancy guideline 內容）。

### 36. 一位 SLE 病人, 最近一週視力惡化, 以下敘述何者錯誤
- A HCQ retinopathy 須列入鑑別診斷, 特別是已經使用 > 5mg/kg/day 超過五年以上的人
- B SLE retinopathy 也是一個鑑別診斷, 可以加強免疫治療
- C 若病患同時出現癲癇與高血壓, 要小心是否為 PRES (Posterior reversible encephalopathy syndrome)
- D 可以加量類固醇, 類固醇對眼睛完全沒有副作用

**答案：D**

**解析：**

- **A（正確敘述）** **Hydroxychloroquine (HCQ) retinopathy** 是 SLE 病人視力惡化的重要鑑別診斷之一，風險因子包括**劑量 >5 mg/kg/day（依實際體重計算）**、**累積使用時間 >5 年**，以及腎功能不全、合併其他視網膜/黃斑部疾病等，這也是目前建議 HCQ 使用者需定期做眼科篩檢（視野、OCT）的原因。
- **B（正確敘述）** **SLE retinopathy（lupus retinopathy，屬於 CNS/vascular lupus 的表現之一）** 本身也是視力惡化的重要鑑別診斷，若確認是疾病活動造成，治療方向是**加強免疫抑制治療**，而非停用 HCQ。
- **C（正確敘述）** SLE 病人若同時出現**癲癇 + 高血壓**，合併視力/視覺症狀，需要警覺 **PRES (Posterior Reversible Encephalopathy Syndrome)** 的可能性——PRES 常與嚴重高血壓、免疫抑制劑（如 cyclophosphamide、CNI）、腎功能惡化等因素相關，影像學上以枕葉（occipital lobe）為主的血管性水腫為特徵，是需要緊急處理的情況（控制血壓、調整藥物）。
- **D（錯誤敘述，本題答案）** 這句話犯了「**絕對化敘述**」的錯誤——類固醇**並非**對眼睛「完全沒有副作用」。類固醇已知的眼科副作用包括：**白內障（cataract，尤其後囊下型）**、**眼壓上升/類固醇誘發性青光眼（steroid-induced glaucoma）**、**中心性漿液性脈絡膜視網膜病變（central serous chorioretinopathy）**等。因此「可以加量類固醇」這部分需視臨床情境（例如確認是 SLE retinopathy 活動性發炎導致才考慮加量），但「類固醇對眼睛完全沒有副作用」這個陳述本身就是錯的。

**臨床意義：** 這題其實是在整合「SLE 病人視力惡化」的完整鑑別診斷思路，涵蓋**藥物性（HCQ）、疾病本身（lupus retinopathy）、續發性（PRES）**三個面向，也提醒不能忽略治療本身（類固醇）也可能是造成眼科問題的原因之一。

### 38. 何者不是 SSc 常見的腸胃道相關表現
- A GAVE
- B Pneumatosis intestinalis
- C Barrett's esophagus
- D Malabsorption

**答案：B**

**解析：**

SSc 的腸胃道侵犯範圍廣泛，從食道到大腸都可能受影響，機轉主要是**平滑肌萎縮、纖維化造成的腸胃道蠕動異常**，以及血管病變。

- **A（常見）** **GAVE (Gastric Antral Vascular Ectasia)**，俗稱「**watermelon stomach**」，是 SSc 相對具代表性的胃部血管病變，會造成慢性腸胃道出血、貧血，是相對常見且重要的併發症。
- **B（相對少見，本題答案）** **Pneumatosis intestinalis（腸壁囊樣積氣）** 雖然在 SSc 病人身上確實有被報告過（因腸道蠕動異常、腸壁萎縮，氣體滲入腸壁形成囊泡），但相較於其他三個選項，這是**遠遠少見、屬於偶發個案報告等級**的表現，多半是**偶然影像學發現、無症狀**，並非 SSc 腸胃道侵犯的典型或常見表現。
- **C（常見）** 因**下食道括約肌功能不良、慢性胃酸逆流（GERD）**，SSc 病人發生 **Barrett's esophagus** 的風險明顯 higher 於一般人群，是很具代表性、常規需要監測的併發症。
- **D（常見）** **Malabsorption（吸收不良）** 主要因**小腸蠕動異常造成細菌過度增生（SIBO, small intestinal bacterial overgrowth）**，是 SSc 病人常見且需要處理的問題（可能需要輪替抗生素治療）。

**SSc 腸胃道侵犯全貌（由近到遠）：**

| 部位 | 常見表現 |
|---|---|
| 食道 | Dysmotility、GERD、Barrett's esophagus |
| 胃 | GAVE、delayed gastric emptying |
| 小腸 | Dysmotility、SIBO、malabsorption、wide-mouth diverticula |
| 大腸 | Wide-mouth diverticula、便秘、（少見）pneumatosis intestinalis |

### 39. 問 systemic sclerosis macrophage, Th2 cell 和 B cells 之間分泌的 cytokine 哪個是錯的?
選項：TGF-beta, TNF, IL-6, IL-4/13

**答案：TNF**

**解析（根據 Kelley 教科書原圖核對）：**

原圖描繪的路徑是：

- **Trigger**（細胞損傷、proteolysis、neoantigens、ROS/oxidation、DAMPs、環境因子）→ 活化 **Dendritic cell**（經由 TLR）
- **Dendritic cell** 分泌 **Type I IFN**，同時作用在 **Th2 T cell、Activated M2 macrophage、B cell activation** 三者上
- **Th2 T cell** 分泌 **IL-4、IL-13** → 作用在 **Activated M2 macrophage**
- **Activated M2 macrophage** 最終分泌 **TGF-β**（促進纖維化）
- **B cell activation** 分泌 **IL-6**
- 產生的 **autoantibodies** 又回過頭刺激 dendritic cell 的 TLR（虛線回饋迴路）

整張圖從頭到尾都**沒有出現 TNF**——核心驅動的 cytokine 是 **Type I IFN（dendritic cell 為中心）、IL-4/IL-13（Th2）、TGF-β（macrophage 輸出）、IL-6（B cell 輸出）**，共同組成 SSc 這個以 **type I IFN 與 Th2 為軸心的促纖維化網絡**，TNF 並不在這個特定機轉圖裡，因此是本題答案。

**圖示重點整理：**

| 細胞 | 接收的訊號 | 分泌的 cytokine |
|---|---|---|
| Dendritic cell | TLR 活化（trigger） | Type I IFN |
| Th2 T cell | Type I IFN | IL-4、IL-13 |
| Activated M2 macrophage | Type I IFN、IL-4、IL-13 | TGF-β |
| B cell | Type I IFN | IL-6、autoantibodies |

### 43. 問 IgG4 related disease pathology

**答案：Dense lymphoplasmacytic infiltrate, Storiform fibrosis, Obliterative phlebitis**

**解析：**

**IgG4-related disease (IgG4-RD)** 的病理診斷仰賴三個核心組織學特徵（無論侵犯哪個器官都共通）：

1. **Dense lymphoplasmacytic infiltrate**：大量淋巴球與漿細胞浸潤，其中漿細胞需要用 IHC 染色確認 **IgG4+ plasma cell** 的比例（通常要求 IgG4+/IgG+ plasma cell ratio > 40%，且每高倍視野 IgG4+ plasma cell 數量達到特定閾值，依器官不同標準略異）
2. **Storiform fibrosis（席紋狀纖維化）**：呈現特徵性的「**車輪狀 / 漩渦狀（storiform pattern）**」纖維化排列，是 IgG4-RD 相對特異的組織學型態，有別於一般的瀰漫性纖維化
3. **Obliterative phlebitis（阻塞性靜脈炎）**：靜脈管腔被發炎細胞浸潤與纖維化阻塞，但**動脈通常不受侵犯**（這點也是與血管炎的重要鑑別點——IgG4-RD 侵犯靜脈而非動脈，機轉上並非典型的 vasculitis）

**三個特徵不需要同時出現**在同一切片上，但**同時具備這三者，對 IgG4-RD 的病理診斷專一性最高**。

### 44. 哪些是 PSS 患者得 lymphoma 的 risk factors

**答案：Persistent salivary gland swelling, lymphadenopathy, low C3/C4**（另有 Kelley 教科書版本清單，詳見下方兩組對照）

**解析：**

**Primary Sjögren's syndrome (PSS/SjD)** 患者發生 lymphoma（多為 **MALT lymphoma**，好發於腮腺）的危險因子，文獻上有兩組經典清單，出處不同、互補：

**Kelley's Textbook of Rheumatology 原文（Ch74）明確列出：NHL 風險增加至少 5 倍**
- Neutropenia（嗜中性白血球低下）
- Cryoglobulinemia（冷凝球蛋白血症）
- Splenomegaly（脾腫大）
- Lymphadenopathy（淋巴結腫大）
- Low C4 levels（低補體 C4）

**其他經典文獻常整理的臨床預測因子**（如 Theander et al. 2006；Baimpa et al. 2009，即 Kelley 本章引用文獻 #131）：
- **Persistent parotid/salivary gland swelling（持續性唾液腺腫大，尤其腮腺）**
- Palpable purpura（可觸摸紫斑）
- 低補體 **C3**/C4
- Monoclonal gammopathy（單株免疫球蛋白血症）
- 高 ESSDAI

**兩組清單有重疊（cryoglobulinemia、lymphadenopathy、低補體），但各自強調的項目不同**——Kelley 版本多了 neutropenia、splenomegaly；另一組則多強調 persistent parotid swelling、purpura、monoclonal gammopathy，準備考試建議兩組都熟悉。

**機轉關聯：** 這些因子多與腺體內 **ectopic germinal center 形成、慢性抗原刺激驅動 B cell clonal expansion** 的機轉相關，最終可能演變為 MALT lymphoma。

- **C（正確敘述）** Sjögren's 病人的腎臟侵犯，最經典的表現就是 **type I（distal）RTA**——因為間質性腎炎侵犯遠端腎小管，造成 H+ 分泌障礙，即使全身處於代謝性酸中毒狀態，尿液仍**無法酸化到 pH < 5.5**，故呈現**尿液 pH 偏高**的矛盾現象，這是 type I RTA 的招牌特徵。
- **D（錯誤敘述，本題答案）** **Cevimeline 和 pilocarpine** 是治療 Sjögren's 口乾症狀的**蕈毒鹼受體（muscarinic receptor）促效劑**，但主要作用位點是 **M3（cevimeline 對 M1、M3 皆有相對選擇性）**，而非 M2。**M3 受體**主要表現在唾液腺、淚腺等外分泌腺體，活化後促進腺體分泌；**M2 受體**主要與**心臟**（降低心率）相關——兩個藥物的設計目標其實是盡量**避開/減少 M2 作用**（降低心搏過緩等副作用），而非「作用在 M2」。

**Muscarinic receptor M1–M5 速記：**

| 受體亞型 | 主要分布 | 功能 |
|---|---|---|
| M1 | 中樞神經系統（認知記憶）、胃壁細胞、唾液腺（部分） | 促進胃酸分泌、協同唾液腺分泌 |
| M2 | **心臟**、平滑肌突觸前受體 | **降低心率** |
| **M3** | **唾液腺、淚腺**、平滑肌（支氣管、腸胃道、膀胱）、眼睛 | **促進腺體分泌**、平滑肌收縮（Cevimeline/Pilocarpine 主要作用位點） |
| M4 | 中樞神經系統（紋狀體） | 突觸前調控 |
| M5 | 中樞神經系統（黑質）、腦血管 | 較少臨床相關性 |

**Cevimeline vs. Pilocarpine：** Cevimeline 對 M1/M3 有相對選擇性，減少 M2（心臟）副作用；Pilocarpine 則是非選擇性促效劑，M1–M5 皆會作用，心臟相關副作用相對較明顯。

### 46. Large vessel vasculitis 影像
考 Takayasu's arteritis 跟 GCA 各可以用那些 image modality 診斷

**解析（依 2018 EULAR imaging recommendations in large vessel vasculitis 為主要架構）：**

兩者都屬於 **large vessel vasculitis**，可用的影像工具高度重疊，差異主要在於**血管侵犯部位不同**，導致優先選擇的工具略有不同。

**GCA（侵犯 cranial arteries 為主，也可侵犯 aorta 及分支）**

| 影像工具 | 角色/發現 |
|---|---|
| Temporal artery ultrasound | 第一線建議；經典發現是 **"halo sign"**（血管壁水腫造成的環狀低回音影），也可評估 axillary artery |
| High-resolution MRI（cranial vessel） | 可看血管壁水腫、增厚、顯影劑增強，用於超音波無法確認時 |
| CT angiography (CTA) | 適合評估 aorta 及其主要分支是否受侵犯（extracranial GCA） |
| MR angiography (MRA) | 同上，適合評估大血管，且無輻射 |
| 18F-FDG PET-CT | 對 aorta、subclavian 等大血管的血管壁發炎活性偵測敏感，但對 temporal/cranial artery 因血管太細、腦部生理性 FDG 攝取干擾，效果較差 |
| Temporal artery biopsy | 病理金標準（非影像學，但常一併列入診斷工具比較） |

**Takayasu's Arteritis（侵犯 aorta 及其主要分支為主，好發年輕女性）**

| 影像工具 | 角色/發現 |
|---|---|
| CT angiography (CTA) | 常用第一線，可見血管壁增厚、管腔狹窄/阻塞/動脈瘤 |
| MR angiography (MRA) | 年輕病人優先考慮（避免輻射暴露） |
| 18F-FDG PET-CT | 偵測血管壁發炎活性，對早期、結構變化尚未出現前的活動性疾病特別有幫助 |
| Doppler ultrasound | 適用於可及部位的血管（carotid、subclavian、axillary），經典發現 **"macaroni sign"** |
| 傳統導管血管攝影 | 傳統金標準，現多為非侵入性影像取代，但介入治療時仍會使用 |

**兩者比較速記：**

| | GCA | Takayasu |
|---|---|---|
| 好發族群 | >50 歲 | <40 歲，年輕女性 |
| 主要侵犯 | Temporal/cranial artery（也可侵犯大血管） | Aorta 及主要分支 |

明確列入 2022 criteria 的影像工具：**Ultrasound（halo sign）、PET-CT（aorta activity）**，以及泛指的影像學（CTA/MRA/US 皆可）用於偵測 axillary artery 侵犯。

### 47. 四個情境哪一個不是 Behçet:
- A Oral ulcer + genital ulcer
- B Oral ulcer + pathergy test positive
- C Palmoplantar pustulosis + lower limbs arthritis
- D Recurrent colitis, colonoscopy 看起來有 ulcer

**答案：C**

**解析：**

- **A（是 Behçet）** **反覆口腔潰瘍 + 生殖器潰瘍**是 Behçet's disease 最核心的診斷特徵（International Study Group criteria 的必要條件之一就是反覆口腔潰瘍，再加上其他條件如生殖器潰瘍、眼睛病灶、皮膚病灶、pathergy test 陽性）。
- **B（是 Behçet）** 反覆口腔潰瘍合併 **pathergy test 陽性**，同樣是 Behçet 分類標準中的經典組合。
- **C（不是 Behçet，本題答案）** **Palmoplantar pustulosis（掌蹠膿疱病）合併下肢關節炎**，這個組合並非 Behçet 的典型表現，反而更像：
  - **SAPHO syndrome**（Synovitis, Acne, Pustulosis, Hyperostosis, Osteitis）——掌蹠膿疱病是這個疾病的招牌皮膚表現，合併骨關節侵犯
  - 或是 **Reactive arthritis（ReA）**——經典的 **keratoderma blennorrhagicum**（外觀類似掌蹠膿疱疹）合併**下肢不對稱寡關節炎**，是 ReA 的典型表現之一（合併尿道炎、結膜炎）

  Behçet 的皮膚表現典型是 **erythema nodosum、pseudofolliculitis（毛囊炎樣病灶）**，並非掌蹠膿疱病；關節炎型態也多是**非侵蝕性、大關節（膝、踝、腕）為主**，與這裡描述的型態不完全吻合。
- **D（是 Behçet）** **反覆腸炎、大腸鏡看到潰瘍**，符合 **Intestinal（entero）Behçet's disease** 這個亞型——好發於東亞族群，好發部位是迴盲部（ileocecal region），臨床上有時需要與 Crohn's disease 做鑑別，但這個情境本身仍屬於 Behçet 的表現譜。

**Behçet 皮膚黏膜表現速記：**

| 常見於 Behçet | 不典型於 Behçet（提示其他診斷） |
|---|---|
| Oral ulcer、genital ulcer | Palmoplantar pustulosis（提示 SAPHO/ReA/PsO） |
| Erythema nodosum | Keratoderma blennorrhagicum（提示 ReA） |
| Pseudofolliculitis / papulopustular lesion | — |
| Pathergy test (+) | — |

### 50. 考痛風的關節以及腎臟表現（原始筆記過於簡略，以下為推測完整涵蓋範圍）

**一、關節表現**

急性期：**Acute gouty arthritis**——突發、劇痛、單一關節為主，典型是第一蹠趾關節（podagra），其次是踝、中足、膝；紅腫熱痛，未治療約 1-2 週可自行緩解。

Intercritical gout（發作間歇期）：無症狀，但尿酸鹽結晶仍持續沉積。

慢性期：**Chronic tophaceous gout**——長期未控制的高尿酸血症 → **Tophus（痛風石）**形成，好發於耳輪、鷹嘴滑囊、關節周圍；可造成慢性破壞性多關節炎，外觀可能與 RA 混淆；關節侵犯分布會隨病程從下肢（第一蹠趾）逐漸擴展到上肢、變成多關節型態。

**影像學表現：**

| 工具 | 發現 |
|---|---|
| X 光 | **"Punched-out" erosion**，具 **"overhanging edge"（懸垂邊緣）**，與 RA 的邊緣性侵蝕不同；關節間隙相對維持到疾病後期 |
| 超音波 | **Double contour sign**（軟骨表面尿酸鹽沉積）、tophus |
| DECT（雙能量電腦斷層） | 可偵測尿酸鹽結晶沉積，並以顏色區分尿酸鹽 vs. 鈣化，是 2015 ACR/EULAR 分類標準中的輔助工具 |

**二、腎臟表現**

1. **尿酸腎結石（Uric acid nephrolithiasis）**：痛風病人最常見的腎臟併發症，好發於尿液持續偏酸性（low urine pH）的病人。
2. **慢性尿酸腎病變（Chronic urate nephropathy）**：長期尿酸鹽結晶沉積在腎髓質、間質，造成慢性間質性腎炎，逐漸進展為 CKD。
3. **急性尿酸腎病變（Acute uric acid nephropathy）**：大量尿酸結晶在腎小管內急速沉積，造成阻塞性急性腎損傷；經典情境是 **Tumor lysis syndrome (TLS)**，化療後腫瘤細胞大量崩解釋放尿酸（呼應這份考卷口試李惠婷老師考的 rasburicase 情境）。
4. **高尿酸血症與 CKD 的因果關係爭議**：目前仍在討論高尿酸血症是 CKD 進展的原因還是結果（雙向關係），並非單純因果。

### 51. Cr 2.5 gout treatment, 可以用哪些藥物組合進行 acute phase treatment
Colchicine, NSAIDs, steroid 的不同組合

**解析：**

**關鍵前提：Cr 2.5 代表腎功能已受損（約當 CKD stage 3-4）**，急性痛風三大類藥物中，每一種在腎功能不全的病人身上都需要個別評估安全性：

| 藥物 | 腎功能不全時的考量 |
|---|---|
| NSAIDs | **相對禁忌/應避免**——本身有腎毒性，可能進一步惡化腎功能 |
| Colchicine | **需要調整劑量**（依 CrCl 減量），嚴重腎功能不全（如 CrCl <30）應避免或大幅減量，主要經腎臟排除，蓄積可能造成 myopathy、neuropathy，甚至骨髓抑制 |
| Corticosteroid（口服、肌肉注射或關節內注射） | **腎功能不全病人的首選**——不經腎臟代謝造成腎毒性，是最安全的選擇 |

**建議的藥物組合／選擇順序：**

1. 單一關節侵犯：優先考慮 **intra-articular corticosteroid injection**（局部注射，全身性副作用最少）
2. 多關節侵犯或無法關節內注射：**口服或全身性 corticosteroid**（如 prednisolone 短期療程）
3. 若需加強效果或 steroid 效果不夠：可考慮**低劑量 colchicine**（依腎功能調整劑量）合併使用，但需密切監測 colchicine 相關副作用
4. 應避免：NSAIDs（尤其長期或高劑量使用）

**核心答案：** Cr 2.5 的病人，急性痛風治療應避開 NSAIDs，優先使用 corticosteroid（口服/肌肉/關節內注射皆可），colchicine 需減量後謹慎使用，必要時可以 steroid + 低劑量 colchicine 合併治療。

### 52. Allopurinol hypersensitivity syndrome
發生的 risk factor, 發生的時機點在使用 allopurinol 多久之後

**詳解：**

**定義：** Allopurinol Hypersensitivity Syndrome (AHS) 是 allopurinol 少見但致命性極高的藥物過敏反應，屬於 **SJS/TEN、DRESS syndrome** 這個嚴重皮膚藥物反應（SCAR）光譜之一，死亡率可達 20-25%。臨床表現：發燒、皮疹（可進展至 SJS/TEN）、嗜伊紅性球增多、肝炎、急性腎損傷。

**Risk Factors：**

1. **HLA-B*58:01 基因型**——最強的遺傳危險因子，在漢人、泰國人等亞洲族群中盛行率明顯較高。這也是為什麼台灣臨床建議在開始使用 allopurinol 前，先做 HLA-B*5801 基因篩檢，尤其漢人族群。
2. **腎功能不全（CKD）**——活性代謝物 oxypurinol 主要經腎臟清除，腎功能不好會造成蓄積，增加過敏反應風險。
3. **併用利尿劑（尤其 thiazide diuretics）**——降低 oxypurinol 腎臟清除率，進一步增加蓄積風險。
4. **起始劑量過高**——可避免的危險因子：建議從低劑量開始（如 100 mg/day 或更低，腎功能不全者需再更低），再緩慢逐步調整劑量（"start low, go slow"）。
5. 女性（部分研究顯示風險略高，證據不一）
6. 併用 ACE inhibitor（部分文獻提及風險增加，證據強度較弱）

**發生時機：** AHS 典型發生在開始使用 allopurinol 後的 **2-8 週內**，中位數約在 **4-6 週**，多數案例集中在治療的**前 2 個月內**。

**臨床意義：** 開始 allopurinol 前考慮 HLA-B*5801 篩檢（尤其漢人族群）；從低劑量開始緩慢titrate；腎功能不全病人起始劑量需再下修；用藥前 2 個月內是風險最高的觀察窗。

### 54. FRAX 看的是那些骨頭, 以及骨折風險怎麼算?

**FRAX（Fracture Risk Assessment Tool，WHO 發展）** 使用的骨密度輸入值是 **femoral neck（股骨頸，即髖部）BMD**（若有測量，屬於選填加分項，沒有 BMD 也可以單靠臨床因子計算）。

- Sex（性別）

- Height（身高，用來計算 BMI）
- Previous fracture（過去骨折病史）
- Parent fractured hip（父母髖部骨折史）
- Current smoking（目前吸菸）
- Glucocorticoid use（類固醇使用）
- Rheumatoid arthritis
- Secondary osteoporosis（次發性骨鬆病因）
- Alcohol ≥3 units/day（酒精攝取）
- Femoral neck BMD（選填，有測會更準）

**計算出兩種 10 年骨折風險：**

1. **Major osteoporotic fracture risk**（髖部、臨床脊椎、前臂、肱骨，四個部位合併計算的整體骨折風險）
2. **Hip fracture risk**（單獨計算髖部骨折風險）

**治療門檻（Treatment threshold，美國 NOF guideline 常用切點）：**

- Major osteoporotic fracture risk ≥ 20% 或
- Hip fracture risk ≥ 3%

→ 符合任一項，建議考慮啟動治療

**⚠️ 重要澄清：** FRAX **不是用來「診斷」骨質疏鬆症**的工具（骨鬆的診斷仍是靠 **T-score ≤ -2.5** 或發生脆弱性骨折）。FRAX 的核心用途是在 **osteopenia 範圍（T-score 介於 -1 到 -2.5 之間）**，單靠 BMD 無法明確判斷是否需要治療時，用來輔助決定是否該啟動藥物治療。

### 55. 甲狀腺異常在骨頭上看會到?
→ Pretibial myxedema；會有哪些感覺異常?（會有 carpal tunnel syndrome?）

**詳解：**

這題橫跨甲狀腺功能亢進與低下兩種狀態的骨骼肌肉/神經表現，分開來看比較清楚：

**甲狀腺功能亢進（Hyperthyroidism / Graves' disease）相關骨骼肌肉表現**

- **Pretibial myxedema（脛前黏液水腫）**：Graves' disease 的皮膚表現，glycosaminoglycan 沉積在脛前皮膚真皮層，造成非凹陷性水腫/斑塊，是 Graves' 三合一表現之一（合併 ophthalmopathy 眼病變與 acropachy 肢端肥厚）
- **Thyroid acropachy（甲狀腺肢端肥厚）**：少見但具特異性，表現為手指/腳趾杵狀變化 + 骨膜下新骨形成，通常合併嚴重的眼病變與 pretibial myxedema 一起出現
- **骨質疏鬆（Osteoporosis）**：甲亢會加速骨轉換，骨吸收 > 骨生成，長期骨量流失、骨折風險增加
- **Proximal myopathy（近端肌肉病變）**：甲亢型肌肉病變，近端無力
- **Thyrotoxic periodic paralysis**：好發於亞洲男性，甲亢誘發低血鉀造成的週期性癱瘓

**甲狀腺功能低下（Hypothyroidism）相關表現——Carpal tunnel syndrome 屬於這裡**

- **Carpal tunnel syndrome (CTS)**：經典與 hypothyroidism（而非甲亢）關聯的感覺異常表現——機轉是黏液性物質（mucopolysaccharide）沉積在腕隧道周圍軟組織，造成正中神經受壓迫，導致手部感覺異常（麻、刺痛，尤其第 1-3 指）
- **近端肌肉病變**：合併 CK 上升，少數可有肌肉假性肥大（Hoffman syndrome）
- **深部肌腱反射遲緩**：經典的「hung-up reflex」（反射放鬆期延遲）
- **關節病變**：可合併關節積液，甲狀腺功能低下也是 CPPD（假性痛風）的次發性病因之一
- **Adhesive capsulitis（五十肩/冰凍肩）**：與甲狀腺低下、糖尿病都有關聯

**⚠️ 容易混淆的地方：** Pretibial myxedema 是甲亢（Graves'）的表現，但 Carpal tunnel syndrome 是甲狀腺功能低下更經典的關聯，兩者分屬甲狀腺功能的兩個極端，這題應該是要考「能不能分清楚甲亢 vs. 甲低各自對應的骨骼肌肉/神經表現」。

**速記表：**

| | 甲狀腺功能亢進（Hyper） | 甲狀腺功能低下（Hypo） |
|---|---|---|
| 皮膚 | Pretibial myxedema | 皮膚乾燥、黏液水腫（全身性） |
| 骨頭 | Acropachy、osteoporosis（骨轉換加速） | 較少特異性骨變化 |
| 神經 | 週期性癱瘓（亞洲男性、低血鉀） | Carpal tunnel syndrome |
| 肌肉 | 近端肌肉無力 | 近端肌肉無力、CK 上升、反射遲緩 |
| 關節 | 較少特異性 | CPPD、adhesive capsulitis |

### 56. DM 病人的骨頭會有哪些異常?
特別是骨生成的變化, 以及 type I and II 是哪種? 糖尿病用藥 (metformin, PPARγ) 對骨頭的影響?

**詳解：**

**Type 1 DM vs. Type 2 DM 骨骼表現對比**

| | Type 1 DM | Type 2 DM |
|---|---|---|
| BMD（骨密度） | 偏低 | 正常，甚至偏高（肥胖、高胰島素血症的機械負荷與同化作用） |
| 骨折風險 | 明顯增加 | 也明顯增加（矛盾點） |
| 骨轉換（bone turnover） | 低骨生成（osteocalcin、P1NP 偏低，因胰島素缺乏 + IGF-1 低，成骨細胞活性下降） | 骨形成與骨吸收皆偏低（low turnover state），但骨密度正常 |
| 骨折風險的機轉 | 骨量本身不足（低 BMD 直接反映骨量減少） | 骨質「量」正常但「質」變差：AGEs 沉積在膠原蛋白造成交聯異常、皮質骨孔隙度增加、sclerostin 上升抑制 Wnt 訊息（抑制成骨），加上周邊神經病變/視網膜病變造成跌倒風險增加 |

**核心觀念：**
- **Type 1 DM**：低骨密度 + 低骨生成——胰島素缺乏直接影響成骨細胞功能與 IGF-1 分泌，屬於典型的「量不足」
- **Type 2 DM**：骨密度正常/偏高，但骨折風險仍增加——這是「**Diabetic bone disease paradox**」，關鍵在於骨質量正常但骨質「品質」下降（AGEs 交聯、骨微結構異常），DXA 測不出這種「品質」問題

**糖尿病用藥對骨頭的影響：**

- **Metformin**：整體被認為對骨頭中性偏保護——活化 **AMPK pathway**，可能促進成骨細胞分化，臨床上沒有增加骨折風險的疑慮。
- **Thiazolidinediones (TZDs，如 pioglitazone、rosiglitazone)**：作用受體是 **PPAR-γ（gamma）**，並非 PPAR-α（PPAR-α 是 fibrate 類降血脂藥的作用位點，與骨頭無關）。PPAR-γ 活化會讓 mesenchymal stem cell 傾向分化成脂肪細胞而非成骨細胞，造成骨髓內脂肪增加、骨形成減少，**明確增加骨折風險**（尤其停經後女性）。
- **SGLT2 inhibitor**：證據較不一致，其中 canagliflozin 在 CANVAS trial 中觀察到骨折風險增加的訊號。
- **Insulin**：對骨頭是同化（anabolic）作用——直接透過 insulin receptor 及 IGF-1 pathway 促進成骨細胞活性。

**藥物對骨頭影響速記：**

| 藥物 | 對骨頭的影響 |
|---|---|
| Metformin | 中性偏保護（AMPK pathway） |
| TZD（PPAR-γ agonist） | 增加骨折風險（促進脂肪生成、抑制成骨） |
| SGLT2i（尤其 canagliflozin） | 部分證據顯示骨折風險增加 |
| Insulin | 同化作用，促進成骨 |
