# 2025 風濕科專科考試 筆試解答

## 筆試

### 1. 生物製劑阻斷細胞激素作用, 何者正確?
- A Mepolizumab 可以認識 IL-5 receptor 來阻斷 IL-5 作用
- B Ixekizumab 可以阻斷 IL-17A 和 IL-17B
- C Guselkumab, risankizumab, ustekinumab 皆認識 IL-23 的 p19
- D Anifrolumab 可以阻斷 IFN-α 和 IFN-β 的作用

**答案：D**

**解析：**

- **A（錯）** Mepolizumab 認識的是 **IL-5 本身（ligand）**，不是 IL-5 receptor。真正認識 IL-5Rα 的是 **Benralizumab**（同時會透過 ADCC 清除嗜酸性球）。
- **B（錯）** Ixekizumab 只選擇性認識 **IL-17A**，不會阻斷 IL-17B。同時阻斷 IL-17A + IL-17F 的是 **Bimekizumab**（注意是 F 不是 B，選項刻意混淆）。
- **C（錯）** Guselkumab、Risankizumab 認識 IL-23 的 p19（選擇性），但 **Ustekinumab 認識的是 p40**（IL-12 與 IL-23 共用次單元），並非 p19。
- **D（對）** Anifrolumab 認識 **type I interferon receptor (IFNAR1)**，透過阻斷受體，阻斷所有 type I IFN（IFN-α、IFN-β、IFN-ω 等）的下游訊息傳遞。

**速記表：**

| 藥物 | 標靶 | Ligand 或 Receptor |
|---|---|---|
| Mepolizumab / Reslizumab | IL-5 | Ligand |
| Benralizumab | IL-5Rα | Receptor |
| Secukinumab / Ixekizumab | IL-17A | Ligand |
| Bimekizumab | IL-17A + IL-17F | Ligand |
| Guselkumab / Risankizumab / Tildrakizumab | IL-23 (p19) | Ligand |
| Ustekinumab | IL-12/23 (p40) | Ligand |
| Anifrolumab | IFNAR1 | Receptor |

### 2. 人體 germinal center 中 B cell 進行 affinity maturation
- A Somatic hypermutation
- B VDJ recombination
- C Class switch
- D Antibody glycosylation

**答案：A**

**解析：**

- **A（對）** **Somatic hypermutation (SHM)** 發生在 germinal center 的 dark zone，由 **AID (activation-induced cytidine deaminase)** 催化免疫球蛋白可變區基因產生點突變，之後在 light zone 由 Tfh cell 與 follicular dendritic cell (FDC) 篩選出對抗原親和力較高的 B cell clone，這個「突變 + 篩選」的循環就是 **affinity maturation** 的核心機轉。
- **B（錯）** VDJ recombination 發生在**骨髓**的 B cell 發育早期（pro-B → pre-B 階段），由 RAG1/RAG2 催化，屬於產生**初始抗體多樣性**的機制，跟 germinal center 內的親和力成熟無關。
- **C（錯）** Class switch recombination (CSR) 也發生在 germinal center，同樣需要 AID，但它改變的是抗體的 **isotype（如 IgM → IgG/IgA/IgE）**，不改變抗原結合的親和力，是與 affinity maturation **平行但獨立**的過程。
- **D（錯）** Glycosylation 是抗體轉譯後修飾，與親和力成熟無直接因果關係。

**速記：**

| 機制 | 發生位置 | 作用 |
|---|---|---|
| VDJ recombination | 骨髓 | 產生抗體初始多樣性 |
| Somatic hypermutation | Germinal center (dark zone) | 產生點突變 → 親和力篩選（affinity maturation） |
| Class switch recombination | Germinal center | 改變 isotype，不影響親和力 |

### 3. 關於 T cell 和 B cell 活化的相關敘述何者正確?
- A CTLA4 與 B7 的親和力大約等於 CD28 與 B7 的親和力
- B Follicular helper T cell 透過 CD40L 與 CD40 結合來刺激 B cell
- C T cell 活化時 T cell receptor 直接和 calcineurin 結合來傳導訊息
- D B cell 活化時 B cell receptor 會直接和 Syk 結合來傳導訊息

**答案：B**

**解析：**

- **A（錯）** **CTLA4 對 B7 (CD80/CD86) 的親和力遠高於 CD28**，並非「大約相等」。正因為 CTLA4 親和力較高，才能有效與 CD28 競爭 B7 結合，優先搶到共同的 ligand，藉此傳遞抑制訊號（這也是 CTLA4-Ig / abatacept 設計的藥理基礎）。
- **B（對）** **Tfh cell 透過 CD40L (CD154)** 與 B cell 表面的 **CD40** 結合，加上分泌 **IL-21**，共同促進 B cell 在 germinal center 中的存活、增殖與分化（包括前一題提到的 SHM、affinity maturation 及 class switch），是 Tfh-B cell help 的核心共同刺激路徑。
- **C（錯）** TCR **不會直接**與 calcineurin 結合。正確路徑是：TCR/CD3 複合體活化 → ZAP-70 → PLCγ 活化 → 產生 IP3 → 細胞內鈣離子上升 → 鈣離子結合 **calmodulin** → calmodulin-calcium 複合體才去活化 **calcineurin**（一種磷酸酶）→ calcineurin 去磷酸化 **NFAT**，使其進入細胞核。中間隔了好幾個步驟，並非直接結合。
- **D（錯）** BCR 本身胞內尾端很短，不直接結合 Syk。正確路徑是：抗原結合 BCR → 與 BCR 共同表現的 **Igα/Igβ (CD79a/CD79b)** 上的 ITAM 被 Src family kinase（如 Lyn）磷酸化 → 磷酸化的 ITAM 才去招募並活化 **Syk**（透過 Syk 的 SH2 domain 結合磷酸化 ITAM）。所以是 Igα/Igβ 中介，而非 BCR 直接結合 Syk。

**觀念重點：** C、D 兩個選項的錯誤模式相同——都是把「受體活化後一連串訊息傳遞的下游分子」誤植成「受體直接結合」，這是免疫學考題常見的陷阱句型，遇到「A 直接與 B 結合」的敘述要特別留意中間是否還有其他分子。

### 4. TLR 何者正確?
- A TLR 可專一識別每一種 viral RNA
- B TLR 在細胞膜辨識細菌胜肽
- C TLR9 識別未甲基化的 CpG DNA
- D 所有 TLR 皆在同一種免疫細胞上

**答案：C**

**解析：**

- **A（錯）** TLR 辨識的是**結構性的 PAMP（pathogen-associated molecular pattern）**，例如 TLR7/8 辨識 single-stranded RNA 的結構特徵、TLR3 辨識 double-stranded RNA，但這是廣泛辨識「RNA 的結構型態」，並非針對「每一種病毒」的 RNA 序列做專一性辨識，TLR 不具有像抗體或 TCR 那樣的體突變/重組產生的高度序列專一性。
- **B（錯）** 這句過度概化。TLR2（配合 TLR1 或 TLR6）辨識的是**細菌脂胜肽 (lipopeptide)**，位於細胞膜沒錯；但 TLR4 辨識的是 **LPS（脂多醣，不是胜肽）**，也在細胞膜；而許多重要的細菌/病毒核酸類 PAMP（如 TLR9 辨識的 CpG DNA、TLR3/7/8 辨識的病毒 RNA）是在**胞內體 (endosome)** 被辨識，並非都在細胞膜、也不是都辨識「胜肽」。
- **C（對）** **TLR9** 位於 endosome 內，專一辨識細菌與病毒 DNA 中**未甲基化的 CpG motif (unmethylated CpG DNA)**——這是因為脊椎動物基因體的 CpG 大多被甲基化，未甲基化的 CpG 在演化上被當作「非自我」的辨識訊號。
- **D（錯）** 不同免疫細胞表現的 TLR repertoire 不同。例如 **plasmacytoid dendritic cell (pDC)** 高量表現 TLR7、TLR9；**macrophage** 則以 TLR2、TLR4 為主；並非所有 TLR 都表現在同一種細胞上。

**TLR 分布速記：**

| TLR | 主要 ligand | 細胞位置 |
|---|---|---|
| TLR1/2/6 | Lipopeptide | 細胞膜 |
| TLR4 | LPS | 細胞膜 |
| TLR5 | Flagellin | 細胞膜 |
| TLR3 | dsRNA | Endosome |
| TLR7/8 | ssRNA | Endosome |
| TLR9 | Unmethylated CpG DNA | Endosome |

### 5. Pattern recognition receptor (PRR) 敘述何者正確?
- A PRR 專門識別宿主 DNA
- B TLR 位在細胞膜或內體內, 識別外來微生物分子
- C 所有的 PRR 專一性極高
- D PRR 由 T cell 與 B cell 表達

**答案：B**

**解析：**

- **A（錯）** PRR 辨識的主要對象是 **PAMP（pathogen-associated molecular pattern，外來病原體分子）**，而非宿主自身 DNA。雖然某些特殊情境下（如 cGAS 辨識到外洩至胞質的自身 DNA）PRR 也能感應 **DAMP（damage-associated molecular pattern）**，但「專門識別宿主 DNA」並非 PRR 的一般定義或主要功能。
- **B（對）** 呼應第 4 題內容：TLR 依種類不同，分布在**細胞膜**（如 TLR1/2/4/5/6）或**胞內體 endosome**（如 TLR3/7/8/9），共同功能是辨識外來微生物的保留性分子結構（PAMP），啟動先天免疫反應。
- **C（錯）** PRR 是**germline-encoded**（生殖系基因直接編碼，不經過體細胞重組），辨識的是廣泛且保留性高的分子結構模式（如 LPS、flagellin、CpG DNA），專一性遠低於 TCR/BCR（後者透過 VDJ recombination 與 somatic hypermutation 產生高度多樣性與專一性）。這是 innate immunity（PRR）與 adaptive immunity（TCR/BCR）在辨識機制上的根本差異。
- **D（錯）** PRR 是**先天免疫**的辨識受體，主要表現在 macrophage、dendritic cell、neutrophil、epithelial cell 等多種細胞上，並非由 T cell、B cell（適應性免疫細胞）表達為其代表性特徵；T/B cell 的抗原辨識受體是 TCR/BCR，走的是完全不同的辨識邏輯。

**觀念重點：** 第 4、5 題都是圍繞「Innate（PRR/TLR，廣泛/低專一性/germline-encoded）vs. Adaptive（TCR/BCR，高專一性/體細胞重組）」這個核心對比，是免疫學考題的高頻主軸，遇到「專一性極高」或「專門識別」這類絕對化敘述描述 PRR 時要提高警覺。

### 6–10（部分，來不及抄）之一：Rheumatoid factor 的自體抗原為?
- A Fab of IgG
- B Fab of IgM
- C Fc gamma chain of IgG
- D Fc alpha chain of IgG

**答案：C**

**解析：**

Rheumatoid factor (RF) 是一種**自體抗體**（本身多半是 IgM isotype，但也可以是 IgG 或 IgA），辨識的自體抗原是**已結合抗原之 IgG 的 Fc 恆定區**，也就是 IgG 重鏈（γ chain）的 Fc 部分（CH2-CH3 domain）。

- **A、B（錯）** Fab 是抗體的**可變區**（辨識外來抗原的那一端），不是 RF 攻擊的目標；RF 攻擊的是 Fc（恆定區）。
- **D（錯）** 這個選項本身是個**名詞錯誤配對**：免疫球蛋白重鏈依 isotype 各有專屬名稱——**IgG 重鏈是 γ (gamma) chain、IgM 是 μ (mu) chain、IgA 是 α (alpha) chain、IgD 是 δ (delta) chain、IgE 是 ε (epsilon) chain**。所以「Fc **alpha** chain of **IgG**」這個組合本身就自相矛盾——alpha chain 是 IgA 重鏈的命名，不會用在 IgG 身上。這題其實同時在考兩件事：(1) RF 認的是 Fc 不是 Fab；(2) IgG 重鏈的正確名稱是 gamma chain，不是 alpha chain。

**重鏈命名速記：**

| Isotype | 重鏈名稱 |
|---|---|
| IgG | γ (gamma) |
| IgM | μ (mu) |
| IgA | α (alpha) |
| IgD | δ (delta) |
| IgE | ε (epsilon) |

### 6–10（部分，來不及抄）之二：SLE 的小鼠模型，何者錯誤?
- A （忘記）
- B 近親繁殖會造成基因變異，也是一種 SLE 小鼠模型
- C SLE 小鼠模型可以用來測試新藥的效果
- D New Zealand Black/White 是現在最常用的 SLE 小鼠模型（考生註記：應該錯誤）

**⚠️ 答案未確認**（原始考生記錄「來不及抄」，選項 A 缺失，無法確定正確答案）

**參考分析：**

- **B（較可能是錯誤敘述）** 邏輯上有問題：**近親繁殖（inbreeding）本身是讓基因走向同質化（homozygosity），並不會「造成」新的基因變異**。經典 SLE 小鼠模型（NZB、NZB/W F1、MRL/lpr、BXSB/Yaa）是在近親繁殖品系中，因**自發性突變**被篩選、固定下來（如 MRL 的 *Fas^lpr* 突變、BXSB 的 *Yaa* duplication），而不是「近親繁殖」這個行為本身產生變異。這句話把「近親繁殖」跟「突變來源」的因果講反了。
- **C（應為正確敘述）** SLE 小鼠模型（自發性如 NZB/W、MRL/lpr，或誘導性如 pristane-induced lupus）本來就是藥物研發測試新藥效果的重要平台，敘述正確。
- **D（考生標記可能錯誤，但保留討論空間）** NZB/W F1 是**最早、最經典**的自發性 SLE 小鼠模型（prototype），但是否為「現在最常用」見仁見智——MRL/lpr 因病程快、症狀重，在許多實驗室也很常用。這句話的對錯較主觀，無法單憑現有資訊確定。

### 11. Scleritis 相關敘述何者正確？
- A 常合併 AAV, RA
- B 常是 RA 的初始表現
- C AAV 有 Scleritis 的話比較不會攻擊 lung, kidney
- D IBD, RP, AS 都可能有 Scleritis

**答案：A**

**解析：**

- **A（對）** Scleritis 約有近一半的病例會合併**全身性疾病（systemic disease）**，其中最常見的兩大類就是 **RA** 與 **AAV（尤其 GPA）**。尤其是 **necrotizing scleritis**（壞死性鞏膜炎）幾乎是這兩類疾病的招牌合併症，也是預後不良、需要積極全身性免疫抑制治療的警訊（可能進展成 scleromalacia perforans）。
- **B（錯）** Scleritis 通常出現在 **病程較久、已建立診斷、blood RF/anti-CCP 陽性的嚴重侵蝕性 RA** 病人身上，屬於**晚期的 extra-articular manifestation**，而非 RA 的初始表現。
- **C（錯）** 恰恰相反：AAV（尤其 GPA）出現眼部侵犯（包括 scleritis）時，通常代表疾病較活躍、systemic involvement 較廣泛，**與 lung、kidney 受侵犯的風險增加相關**，而非減少。
- **D（部分正確但不夠精確）** IBD、relapsing polychondritis (RP)、AS 這幾個疾病確實都可能有眼部發炎表現，但需要區分 **episcleritis（鞏膜外層炎）vs. scleritis（鞏膜炎）**：
  - **RP**：episcleritis 與 scleritis 都可能出現，是這題裡唯一與 scleritis 本身關聯性較強的。
  - **IBD**：眼睛表現以 **episcleritis** 較常見，scleritis 相對少見。
  - **AS**：典型的眼睛表現是 **acute anterior uveitis**（與 HLA-B27 相關），而非 scleritis。

  這個選項把「episcleritis 常見的疾病群」跟「scleritis」混在一起講，精確度不如 A，所以 A 是更佳答案。

**Episcleritis vs. Scleritis 常見合併疾病速記：**

| | 常見合併疾病 | 嚴重度 |
|---|---|---|
| Episcleritis | IBD、SpA（含 AS）、RA | 較輕微，通常自限 |
| Scleritis | RA（尤其晚期、seropositive）、AAV（尤其 GPA）、Relapsing polychondritis | 較嚴重，necrotizing type 可能致視力喪失，需全身性免疫抑制 |

### 12. DISH 相關敘述何者錯誤？
- A 與 DM, Obesity 相關
- B 與 HLA-B27 無關
- C 脊椎鈣化產生慢性黏連
- D 30 歲以前很少出現

**答案：C**

**解析：**

DISH (Diffuse Idiopathic Skeletal Hyperostosis) 的病理特徵是脊椎**前縱韌帶（anterior longitudinal ligament）及其他 entheses 的骨化**，形成典型的「**flowing osteophytes（流動狀骨贅）**」，橋接相鄰椎體。

- **A（正確敘述）** DISH 與代謝症候群相關的因子高度相關，包括 **糖尿病 (DM)、肥胖 (obesity)**、高胰島素血症、高尿酸血症、血脂異常等，是公認的危險因子。
- **B（正確敘述）** DISH **與 HLA-B27 無關聯**，這也是它與 AS 的重要鑑別點之一——AS 與 HLA-B27 高度相關，DISH 則不然。
- **C（錯誤敘述，本題答案）** 這句話把 DISH 的機轉錯誤地描述成類似 AS 的「**關節沾黏 / ankylosis**」。實際上 DISH 的病理是**韌帶／enthesis 的骨化增生（新骨形成）**，橋接椎體前外側，但**椎間盤空間（disc space）與 facet joint、SI joint 通常維持正常、不會被侵蝕或融合**——這正是 DISH 與 AS 在影像學上最重要的鑑別點：
  - **AS**：sacroiliitis → 關節本身被侵蝕、發炎、最終**融合（true joint ankylosis）**，合併 disc space 的 syndesmophyte 形成 "bamboo spine"。
  - **DISH**：SI joint 與 disc space 大致**保留正常**，只是韌帶外側形成大塊、流動狀的骨贅橋接，造成活動度下降，但機轉是**骨化增生**而非「發炎侵蝕後的關節沾黏」。
- **D（正確敘述）** DISH 好發於**中老年族群**（多在 50 歲以上診斷，盛行率隨年齡增加，80 歲以上可達 25–35%），**30 歲以前極為少見**。

**DISH vs. AS 鑑別速記：**

| | DISH | AS |
|---|---|---|
| HLA-B27 | 無關 | 高度相關 |
| SI joint | 保留正常 | 侵蝕、發炎、融合 |
| Disc space | 保留正常 | Syndesmophyte、bamboo spine |
| 骨贅型態 | Flowing osteophytes（前外側） | Marginal syndesmophytes（垂直生長） |
| 好發族群 | 中老年、代謝症候群相關 | 年輕成人、男性略多 |

### 13. 36 y/o AS for 12 years，近三個月多處疼痛（neck, upper and lower trunk, shoulders, elbows, hips, knee），目前使用 Etoricoxib，以下敘述何者正確？
- A nociceptive pain, Etoricoxib 要加量
- B neuropathic pain, 要加 gabapentin
- C nociplastic pain, 要加 Tramadol
- D nociplastic pain, IL-17 在 central sensitization 有角色

**答案：D**

**解析：**

**先判斷疼痛型態：** 病人是已診斷 AS 12 年的病人，近三個月出現**廣泛、多處、跨區域**的疼痛（頸部、上下軀幹、雙肩、雙肘、雙髖、膝關節），這種分布模式**不符合典型 AS 發炎性關節/中軸疼痛**的局部型態，反而符合 **nociplastic pain（傷害感受可塑性疼痛，即中樞敏感化 central sensitization 主導的疼痛）** 的臨床特徵——這是長期發炎性關節炎病人（包括 AS、RA）常見的共病現象，類似 **fibromyalgia overlap**。

三種疼痛型態定義複習：

| 型態 | 機轉 | 典型例子 |
|---|---|---|
| Nociceptive pain | 組織實質發炎/損傷刺激傷害受器 | AS 的 sacroiliitis 發炎痛 |
| Neuropathic pain | 神經系統本身受損 | Diabetic neuropathy |
| Nociplastic pain | 中樞神經痛覺處理異常，無明顯組織損傷或神經病灶 | Fibromyalgia、長期發炎疾病合併的廣泛疼痛 |

- **A（錯）** 若歸類為 nociceptive pain 並加量 NSAID（Etoricoxib），對這種中樞敏感化主導的廣泛疼痛效果有限，且長期使用/加量 NSAID 有腸胃道、心血管、腎臟風險，並非對症處置。
- **B（錯）** 病人的疼痛分布廣泛、對稱性跨多個區域，並無神經病變常見的**燒灼感、麻刺感、皮節分布 (dermatomal distribution)** 等特徵，歸類為 neuropathic pain 並不恰當。
- **C（錯）** 雖然 nociplastic pain 的判斷方向正確，但治療建議錯誤——**Tramadol（類鴉片藥物）並非 nociplastic pain / fibromyalgia 的建議一線用藥**，現行指引不建議使用類鴉片藥物治療此類疼痛，因為長期療效證據不足，且有成癮、耐受性等風險，甚至可能加重中樞敏感化。Fibromyalgia/nociplastic pain 建議的藥物是 **duloxetine、pregabalin、milnacipran** 等，並強調非藥物治療（運動、認知行為治療）。
- **D（對）** 正確歸類為 nociplastic pain，且點出近年研究關注的機轉：**IL-17 除了驅動周邊關節/附著點的發炎反應外，也被認為在 central sensitization（中樞敏感化）的路徑中扮演角色**，可能參與解釋為何部分 SpA 病人即便發炎控制良好，仍持續有廣泛性疼痛（類似 fibromyalgia 表現）。這也呼應這份考卷第 1 題對 IL-17 pathway 的重視。

### 15. 35 歲張先生，下背痛及髖關節疼痛多年，早上起床最嚴重，隨著日常活動疼痛會改善，關於張先生的影像檢查描述，何者錯誤?
- A Pelvis X 光看到 sacroiliac joint 有 pseudowidening
- B Pelvis MRI 在 STIR sequence 看到 sacroiliac joint 有 bone marrow edema
- C L-spine MRI 在 STIR sequence 看到 vertebral corners with increased signal indicating fat metaplasia
- D T-spine X 光看到右側有 large, flowing, osteophytes

**答案：C**

**臨床情境：** 35 歲男性，多年下背痛合併髖痛，**晨僵明顯、活動後改善**——典型 **inflammatory back pain (IBP)**，指向 **axial SpA / AS**。

**解析：**

- **A（正確敘述）** X 光上 sacroiliitis 早期因關節邊緣被侵蝕（erosion），造成關節間隙看起來變寬的假象，稱為 **pseudowidening**，是 AS 早期 SI joint X 光的典型表現之一。
- **B（正確敘述）** **STIR (Short Tau Inversion Recovery)** 是壓脂序列，對水分/水腫訊號極敏感，是偵測 **active sacroiliitis（bone marrow edema）** 的標準序列，也是 ASAS MRI 定義 active sacroiliitis 的核心依據。
- **C（錯誤敘述，本題答案）** MRI 序列的物理特性被講反了。**STIR 序列會把脂肪訊號壓掉（fat suppression）**，STIR 上呈現**高訊號**代表的是**水腫（bone marrow edema，發炎活性）**，而不是脂肪。相反地，**fat metaplasia（脂肪化生，慢性結構性變化，即 Romanus lesion / fatty corner lesion）** 是在**未壓脂的 T1-weighted 序列**上呈現高訊號；若真的做在 STIR 上，脂肪化生反而會因壓脂而**訊號被抑制（變暗）**，不會是「increased signal」。STIR + 高訊號只對應 bone marrow edema（急性發炎），不會是 fat metaplasia（慢性變化），敘述自相矛盾。

  **脊椎 MRI 序列判讀速記：**

  | 序列 | 高訊號代表 | 意義 |
  |---|---|---|
  | STIR / T2 fat-sat | 水腫（edema） | Active inflammation |
  | T1（非壓脂） | 脂肪 | Fat metaplasia（慢性結構變化） |
  | T1 + Gd 顯影劑 + 壓脂 | 增強訊號 | Active inflammation（另一種偵測方式） |

- **D（敘述本身正確，但非本題答案）** 這句話描述的其實是 **DISH（而非 AS）的典型影像特徵**——**右側優位（right-sided predominance）的大塊流動狀骨贅**是 DISH 經典教學重點：主動脈在胸椎左側的搏動會抑制左側骨贅形成，所以 DISH 骨贅通常好發於**右側**（呼應第 12 題 DISH 內容）。此敘述作為獨立的影像學事實**本身正確**，只是與這位病人（臨床圖像較像 AS）不一定吻合；但題目問的是敘述本身「對不對」，而非「是否符合此病人」，故 D 不是錯誤選項。

### 16. 67 歲王女士，最近兩個月開始出現雙側肩膀及髖關節疼痛，尤其是早上起床最嚴重，持續時間超過一小時，抽血 ESR:80 mm/hr，RF 和 anti-CCP 數值正常，請問下列何者超音波影像上的發現對於診斷較無幫助?
- A 雙側肩膀有 subdeltoid bursitis
- B 雙側肩膀有 subscapularis tenosynovitis
- C 雙側肩膀有 glenohumeral synovitis
- D 雙側髖關節有 trochanteric bursitis

**答案：B**

**臨床情境：** 67 歲女性，急性發作（2 個月內）的**雙側**肩、髖疼痛，晨僵 **>1 小時**，ESR 顯著上升（80），RF/anti-CCP 陰性——典型 **Polymyalgia Rheumatica (PMR)** 表現。

**解析（對照 2012 EULAR/ACR PMR 分類標準的超音波項目）：**

2012 EULAR/ACR PMR provisional classification criteria 特別把超音波列為加分項目，其中定義的典型超音波發現為：

- **肩膀**：subdeltoid bursitis、**biceps tenosynovitis（長頭肱二頭肌肌腱鞘炎）**、glenohumeral synovitis（三者任一）
- **髖關節**：synovitis 和/或 trochanteric bursitis

分類標準的組合規則：**至少一側肩膀**有上述任一發現 + **至少一側髖**有 synovitis/trochanteric bursitis；或**雙側肩膀**都有上述任一發現，即可加分（有助於與 RA 鑑別）。

- **A（有幫助）** Subdeltoid bursitis 是 PMR 分類標準列出的典型肩部發現。
- **B（無幫助，本題答案）** 標準中提到的肌腱鞘炎是 **biceps（長頭肱二頭肌）tenosynovitis**，並非 **subscapularis tenosynovitis**。Subscapularis 是旋轉肌袖的一部分（附著於 lesser tuberosity），其肌腱鞘炎並非 PMR 分類標準定義的典型超音波表現，對診斷 PMR 較無幫助。
- **C（有幫助）** Glenohumeral synovitis 是 PMR 分類標準列出的典型肩部發現。
- **D（有幫助）** Trochanteric bursitis 是 PMR 分類標準列出的典型髖部發現。

**速記：PMR 超音波三合一（肩）+ 二選一（髖）**

| 部位 | 典型超音波發現 |
|---|---|
| 肩 | Subdeltoid bursitis、**Biceps (long head) tenosynovitis**、Glenohumeral synovitis |
| 髖 | Synovitis、Trochanteric bursitis |
