# 2025 過敏免疫科專科考試 筆試解答

## 筆試

### 1. 抗體組合診斷 primary Sjögren syndrome 最不具相關性?
- A anti-Ro52(+), anti-Ro60(+), anti-La(+)
- B anti-Ro52(-), anti-Ro60(+), anti-La(+)
- C anti-Ro52(+), anti-Ro60(-), anti-La(-)
- D anti-Ro52(+), anti-Ro60(-), anti-La(+)

**答案：C**

**解析：**

這題的重點在於區分 **anti-Ro52** 與 **anti-Ro60 / anti-La** 在 pSS 血清學上的**特異性差異**。

- **anti-Ro60（SSA）與 anti-La（SSB）** 是傳統上與 primary Sjögren syndrome **直接相關**的經典抗體組合，也是 2016 ACR/EULAR SjD 分類標準中血清學項目的核心（anti-SSA/Ro 陽性）。這兩者同時陽性、或至少 anti-Ro60 陽性，都高度指向 pSS 的典型血清學圖譜。
- **anti-Ro52（TRIM21）** 則是一個**專一性很低**的抗體，廣泛出現在多種自體免疫疾病中，包括 SLE、systemic sclerosis、**肌炎（尤其 antisynthetase syndrome）**、RA 等，並非 pSS 專屬。它常常是**伴隨 anti-Ro60/anti-La 一起出現**的「協同抗體」，而不是獨立指向 pSS 的標誌物。

逐一看選項：

- **A（Ro52+/Ro60+/La+）**：三者全陽性，是 pSS 血清學上**最典型**的組合，相關性最高。
- **B（Ro52-/Ro60+/La+）**：Ro60 與 La 皆陽性，仍是**經典 pSS 抗體型態**，只是缺少 Ro52，相關性依然很高。
- **D（Ro52+/Ro60-/La+）**：有 La 陽性，仍保留與 pSS 相關的核心抗體，相關性中等偏高。
- **C（Ro52+/Ro60-/La-，本題答案）**：**只有 Ro52 單獨陽性**，Ro60 與 La 皆陰性。因為 Ro52 本身專一性極低，「孤立的 Ro52 陽性」在血清學上**最不能拿來支持 pSS 診斷**，反而更常見於肌炎或其他 CTD overlap 的情境。這個組合是四個選項中**與 pSS 相關性最弱**的。

**Ro52 vs. Ro60/La 專一性速記：**

| 抗體 | 對 pSS 的專一性 | 備註 |
|---|---|---|
| anti-Ro60 | 高 | pSS 經典抗體，分類標準核心項目 |
| anti-La (SSB) | 高 | 常與 Ro60 併存，pSS 經典抗體 |
| anti-Ro52 (TRIM21) | 低 | 廣泛見於多種 CTD/肌炎，孤立陽性時對 pSS 診斷意義不大 |

### 2. 懷疑病人 autoimmune hemolytic anemia, 最關鍵的檢查為何?
- A Serum protein electrophoresis
- B Direct Coombs test
- C Indirect Coombs test
- D Anti-dsDNA

**答案：B**

**解析：**

**Direct Coombs test（Direct antiglobulin test, DAT）** 是診斷 **autoimmune hemolytic anemia (AIHA)** 最關鍵、最直接的檢查。

- **B（對）** DAT 偵測的是**已經附著在病人紅血球表面**的抗體（IgG）或補體成分（C3d）。原理是加入 anti-human globulin（Coombs reagent）使其與附著在紅血球上的抗體/補體產生凝集反應。**DAT 陽性直接證實紅血球表面有自體抗體或補體附著**，是 AIHA 診斷的**核心確診依據**，也可依據染色型態（IgG-only、C3-only、或 IgG+C3）幫助區分 warm AIHA（多為 IgG）與 cold agglutinin disease（多為 C3，IgM 已在體外洗脫）。
- **C（錯）** **Indirect Coombs test** 偵測的是**血清中游離、尚未附著在紅血球上的抗體**，主要用途是**輸血前的交叉配對（crossmatch）、產前 Rh 抗體篩檢**，並非診斷 AIHA 的直接工具——AIHA 的抗體已經結合在病人自己的紅血球上，Indirect test 反而可能測不到。
- **A（錯）** Serum protein electrophoresis 主要用於篩檢單株蛋白（如 multiple myeloma、Waldenström macroglobulinemia），與 AIHA 診斷無直接關係，雖然某些續發性 AIHA（如 lymphoproliferative disease 相關）可能會加做以尋找病因，但不是「最關鍵」的診斷工具。
- **D（錯）** Anti-dsDNA 是 SLE 的特異性抗體，雖然 SLE 病人可以續發 AIHA（也是 SLICC/ACR 分類標準的血液學項目之一），但這是在找**病因/共病**，並非診斷 AIHA 本身的關鍵檢查。

**Direct vs. Indirect Coombs 速記：**

| 檢查 | 偵測對象 | 主要用途 |
|---|---|---|
| Direct Coombs (DAT) | 附著在病人紅血球上的抗體/補體 | 診斷 AIHA |
| Indirect Coombs | 血清中游離抗體 | 輸血前配對、產前篩檢 |

### 3. 何種 virus 最可能透過降低 T cell activation threshold 增加藥物過敏機會?
- A HIV
- B Influenza
- C HSV1
- D RSV

**答案：A**

**解析：**

**HIV 感染者的藥物過敏（尤其是 sulfonamide 類藥物過敏）發生率顯著高於一般人群**，是過敏免疫科教學中非常經典的觀念。

- **A（對）** HIV 病人使用 **sulfamethoxazole/trimethoprim (TMP-SMX)**（常用於 PJP prophylaxis）時，藥物過敏反應發生率可高達一般族群的 **10–100 倍**。機轉上，HIV 感染造成慢性免疫活化（chronic immune activation）、CD4 T cell 數量與功能異常、氧化壓力增加、glutathione 缺乏等多重因素，被認為會**降低 T cell activation 的閾值**，使原本不會活化 T cell 的藥物代謝產物（hapten）更容易誘發 T cell 媒介的過敏反應，導致藥物過敏（尤其 delayed-type，如 maculopapular rash、甚至 SJS/TEN）的機率大幅上升。這也是為什麼 HIV/AIDS 病人是藥物過敏的高危險族群，臨床上使用 sulfonamide 類藥物需特別謹慎。
- **B、C、D（錯）** Influenza、HSV1、RSV 雖然都是常見病毒感染，但**沒有類似 HIV 這種明確、大量文獻支持的「降低 T cell activation threshold 而增加藥物過敏」機轉**。（若題目選項中有 EBV、CMV、HHV-6，則會牽涉另一個經典主題——病毒再活化與 DRESS syndrome 的關聯，但機轉是病毒再活化驅動免疫反應本身，與「降低 T cell 活化閾值」的敘述不同，且此題選項中並未列出。）

**觀念重點：** HIV 相關藥物過敏（尤其 sulfonamide hypersensitivity）是過敏科很愛考的經典臨床關聯，機轉核心是「慢性免疫活化 → T cell 活化閾值下降 → 藥物 hapten 更容易誘發過敏反應」。

### 4. 李小姐 3 年前 CT 打 contrast, 幾分鐘內口吐白沫意識模糊…這次心臟問題做心導管注射 Visipaque 不到 10 分鐘, 血壓量不到 + 臉腫…施打 epinephrine…問以後若要做心臟影像檢查:
- A 做 heart echo
- B MDCT coronary angiography
- C 13, 7, 1 小時前 premedication 再做心導管
- D MRI of heart

**答案：A**

**解析：**

這題的關鍵在於病人有**兩次、由不同碘顯影劑（CT contrast 與心導管用的 Visipaque/iodixanol）引起的嚴重過敏反應**——第一次口吐白沫、意識模糊，第二次血壓量不到（低血壓）+ 臉腫（angioedema），且需要**施打 epinephrine** 才能穩定——這已經達到 **anaphylaxis（過敏性休克）** 的診斷等級，而且是**重複發生、換了不同廠牌/種類的碘顯影劑仍然發生**，代表病人對**碘顯影劑這一整個藥物類別**有真正、嚴重的過敏傾向，並非單一批次或單一廠牌的偶發反應。

逐一分析選項：

- **A（對，本題答案）** **心臟超音波（heart echo）完全不需要使用任何碘顯影劑**，是評估心臟結構、心室功能、瓣膜疾病最安全的第一線非侵入性檢查。對於已經有兩次嚴重碘顯影劑過敏史的病人，**完全避開碘顯影劑**是最安全的策略，echo 是唯一完全不涉及顯影劑暴露風險的選項。
- **B（錯）** MDCT coronary angiography（心臟冠狀動脈電腦斷層）**必須使用碘顯影劑**才能顯影冠狀動脈，對這位病人屬於**高風險、不建議**的檢查選擇，會讓病人再次暴露於已知會誘發 anaphylaxis 的藥物類別。
- **C（錯）** 這是這題的**核心陷阱**：**Premedication（13、7、1 小時前給予類固醇 + 抗組織胺的預防性給藥）主要被證實可以降低「輕度至中度」、非過敏性（anaphylactoid/non-IgE mediated）顯影劑反應的發生率**，但對於已經發生過**嚴重 anaphylaxis（低血壓、需要 epinephrine）** 的病人，**premedication 並不能可靠地預防再次發生嚴重反應**，尤其病人已經在換過顯影劑種類後仍再度發作，代表其致敏傾向對整個碘顯影劑類別都有風險。在這種情況下**再次安排需要碘顯影劑的心導管檢查是不安全的**，即使給予 premedication 也不建議。
- **D（可考量但非最佳）** Cardiac MRI 使用的是 **gadolinium 顯影劑**，與碘顯影劑屬於不同化學類別，交叉反應率低，理論上是碘顯影劑過敏病人的替代選項之一。但若能以**完全不需要任何顯影劑**的 echo 就能提供足夠的臨床資訊，**echo 會是風險更低、優先層級更高的選擇**；MRI 則保留給 echo 資訊不足、仍需更詳細心臟構造/血流評估時的次選方案。

**核心觀念：** 對於已經發生過**嚴重 / anaphylactic** 碘顯影劑反應的病人，**premedication 不是萬靈丹**——它對輕中度反應有效，但不能保證預防嚴重反應復發；最安全的做法是**未來檢查完全避開碘顯影劑**，優先選擇不需顯影劑的替代檢查（如 echo），必要時才考慮不同化學類別的顯影劑（如 MRI 用的 gadolinium）。

### 5. 王先生用 ibuprofen 及 ketorolac 會 urticaria + 眼部 angioedema…這次掛號 for inflammatory back pain, r/o AS, 何種處置最適當?
- A Acetaminophen
- B Naproxen
- C Diclofenac
- D Etoricoxib

**答案：D**

**解析：**

**先判斷過敏型態：** 王先生對 **ibuprofen（propionic acid 類）** 與 **ketorolac（pyrrolizine carboxylic acid 類）** 這兩個**化學結構完全不同**的 NSAID 都出現 urticaria + angioedema，這種「對多種化學結構不相關的 NSAID 都會反應」的型態，是典型的 **cross-reactive type NSAID hypersensitivity（又稱 NSAID-exacerbated cutaneous disease, NECD，或 NSAID-induced urticaria/angioedema, NIUA）**，機轉是**COX-1 抑制導致 arachidonic acid 代謝路徑轉向、leukotriene 過度產生**，屬於**藥理性（非 IgE 專一性）交叉反應**，並非對單一藥物的專一性過敏。這種病人對**任何強效 COX-1 抑制劑**都有高風險再次誘發相同反應。

**因此治療策略是：避開強效 COX-1 抑制劑，改用高度選擇性 COX-2 inhibitor。**

- **D（對，本題答案）** **Etoricoxib** 是高度選擇性的 **COX-2 inhibitor**，對 COX-1 的抑制作用極弱，在 cross-reactive NSAID hypersensitivity 病人身上被證實**耐受性良好**，同時保有足夠的抗發炎效果，適合用來治療 AS 這種**需要有效抗發炎藥物**的 inflammatory back pain，是這類病人的標準替代方案。
- **A（錯）** Acetaminophen 雖然對 COX-1 抑制作用弱、通常安全性較高，但**抗發炎效果非常有限**，對於 AS 這種需要 NSAID 等級抗發炎治療效果的疾病，**療效不足**，並非適當處置（且極少數 cross-reactive 病人在高劑量時仍可能有交叉反應風險）。
- **B（錯）** Naproxen 是**強效、非選擇性 COX-1/COX-2 抑制劑**，與病人已知過敏的 ibuprofen、ketorolac 同樣屬於強力 COX-1 抑制劑，**交叉反應風險高**，不適合使用。
- **C（錯）** Diclofenac 雖然 COX-2 選擇性略高於 ibuprofen/naproxen，但仍**非真正的選擇性 COX-2 inhibitor**，仍保有相當程度的 COX-1 抑制作用，對 cross-reactive 病人**仍有交叉反應風險**，安全性不如 etoricoxib。

**NSAID 過敏處置速記：**

| 過敏型態 | 機轉 | 處置原則 |
|---|---|---|
| Cross-reactive (NECD/NIUA) | COX-1 抑制 → leukotriene 過度產生 | 避開所有強效 COX-1 抑制劑，改用選擇性 COX-2i（如 etoricoxib、celecoxib） |
| Selective/single-drug reaction | 對單一藥物的 IgE-mediated 過敏 | 避開該單一藥物，其他化學結構不同的 NSAID 通常可耐受 |

### 6. 20 歲張先生, 2 年前反覆 popliteal, cubital, 雙手, 軀幹紅 + 癢 + 滲液…治療 target 不包括?
- A PDE4
- B IL-4
- C IFNγ
- D JAK1

**答案：C**

**臨床情境：** 20 歲男性，2 年反覆發作於 **popliteal fossa（膕窩）、cubital fossa（肘窩）、雙手、軀幹**的紅疹 + 搔癢 + 滲液——這是**典型 atopic dermatitis (AD)** 的好發部位（屈側/flexural distribution）與臨床型態（濕疹合併滲液，代表急性/亞急性期病灶）。

**解析：**

AD 的致病機轉核心是 **Th2 driven inflammation**（IL-4、IL-13、IL-31 等），目前已核准或臨床使用的治療標靶皆圍繞這條路徑：

- **A（是治療標靶，非本題答案）** **PDE4 (phosphodiesterase-4) inhibitor**——如**外用 crisaborole**，透過抑制 PDE4 降低 cAMP 分解，間接減少發炎性細胞激素釋放，已核准用於輕中度 AD 的局部治療。
- **B（是治療標靶，非本題答案）** **IL-4** 是 AD 最核心的治療標靶之一——**Dupilumab** 透過結合 **IL-4Rα**，同時阻斷 IL-4 與 IL-13 這兩條共用受體次單元的訊息路徑，是 AD 治療的指標性生物製劑。
- **D（是治療標靶，非本題答案）** **JAK1 inhibitor**（如 **upadacitinib、abrocitinib**）透過阻斷 IL-4/IL-13/IL-31 等多種細胞激素共用的 JAK-STAT 下游訊息路徑，已核准用於中重度 AD 治療。
- **C（不是治療標靶，本題答案）** **IFN-γ** 是 **Th1 pathway** 的代表性細胞激素，並非 AD（Th2-driven disease）的主流治療標靶。事實上歷史上曾經嘗試用**外源性給予 recombinant IFN-γ**（而非「阻斷」它）作為嚴重難治型 AD 的輔助治療，理論基礎是想利用 Th1/Th2 的**反向平衡**來壓制過度活化的 Th2 反應，但這與「以 IFN-γ 作為被阻斷的治療標靶」的邏輯完全相反，且此療法證據有限、非現行主流治療。因此 IFN-γ **不屬於目前 AD 標靶治療（阻斷型)** 的範疇，是四個選項中唯一不適用的。

**AD 治療標靶速記：**

| 標靶 | 藥物類別 | 代表藥物 |
|---|---|---|
| IL-4Rα (IL-4 + IL-13) | 生物製劑 | Dupilumab |
| IL-13 | 生物製劑 | Lebrikizumab, Tralokinumab |
| JAK1 (或 JAK1/2) | 小分子口服藥 | Upadacitinib, Abrocitinib |
| PDE4 | 外用小分子藥 | Crisaborole |
| TSLP | 生物製劑（仍在研究中） | Tezepelumab（AD phase 2） |
| IFN-γ | 非阻斷標靶，歷史上為輔助性給藥 | （非主流治療） |

### 7. 21 歲李先生, r/o AD, topical therapy 不包括?
- A Moisturizer
- B Calcineurin inhibitor
- C Steroid
- D PDE5 inhibitor

**答案：D**

**解析：**

這題延續上一題的觀念，考的是**外用（topical）AD 治療選項**的基本組成：

- **A（是 topical therapy，非本題答案）** **Moisturizer（保濕劑）** 是 AD 治療的**基礎且必要**的第一線處置，維持皮膚屏障功能，減少水分流失與外界過敏原/刺激物穿透，所有 AD 病人都應規律使用。
- **B（是 topical therapy，非本題答案）** **Topical calcineurin inhibitor（如 tacrolimus ointment、pimecrolimus cream）**是外用第二線抗發炎藥物，透過抑制 T cell 內 calcineurin-NFAT 訊息路徑減少發炎，常用於臉部、皮膚皺褶處等**類固醇不適合長期使用**的部位。
- **C（是 topical therapy，非本題答案）** **Topical steroid（外用類固醇）**是 AD 急性發作最基本、最常用的第一線外用抗發炎藥物。
- **D（不是 topical AD therapy，本題答案）** **PDE5 inhibitor**（如 sildenafil）是用於**勃起功能障礙、肺動脈高壓**的藥物，與 AD 治療完全無關。這題是刻意混淆上一題的 **PDE4 inhibitor（crisaborole，真正用於 AD 的外用藥）**，把 4 改成 5 製造陷阱——**PDE4i 才是 AD 的外用治療選項，PDE5i 不是**。

**易混淆提醒：** PDE4 inhibitor（crisaborole）＝ AD 外用藥；PDE5 inhibitor（sildenafil 等）＝ 治療 ED/肺動脈高壓，兩者作用機轉、適應症完全不同，只差一個數字，是很典型的考試陷阱題。

### 8. 57 歲男性, Raynaud phenomenon + 四肢皮膚增厚 + 乾咳 + 喘 + 胃食道逆流…最可能?
- A Sjögren's syndrome
- B Systemic sclerosis
- C Polymyositis
- D Antisynthetase syndrome
- E Multiple sclerosis

**答案：B**

**解析：**

這是 **Systemic sclerosis (SSc)** 教科書等級的典型臨床組合：

- **Raynaud phenomenon**：SSc 幾乎必有的血管病變表現，常是**最早出現**的症狀，可早於皮膚變化數年。
- **四肢皮膚增厚（skin thickening）**：SSc 的核心診斷特徵，代表皮膚纖維化（fibrosis），依侵犯範圍可分為 **limited cutaneous SSc（肢體遠端為主）** 與 **diffuse cutaneous SSc（軀幹＋肢體近端）**。
- **乾咳 + 喘**：提示 **interstitial lung disease (ILD)**，是 SSc 最常見、也是造成死亡率上升最主要的內臟侵犯之一。
- **胃食道逆流（GERD）**：SSc 侵犯**食道平滑肌**造成蠕動異常（esophageal dysmotility）與**下食道括約肌張力下降**，是 SSc 腸胃道侵犯中**最常見**的表現，幾乎見於多數病人。

這四個特徵（**血管病變 + 皮膚纖維化 + 肺纖維化 + 食道侵犯**）合在一起，正是 SSc 多重器官纖維化/血管病變的核心病理生理縮影，是這題最直接指向的診斷。

逐一排除其他選項：

- **A（錯）** Sjögren's syndrome 核心表現是**口乾、眼乾**（sicca symptoms），雖然也可能有 Raynaud 與 ILD，但**不會有皮膚增厚**這種纖維化特徵，皮膚增厚並非 SjD 的表現。
- **C（錯）** Polymyositis 核心是**近端肌肉無力**，題目中並未提到肌肉症狀，且皮膚增厚、GERD 並非 PM 的典型表現。
- **D（錯）** Antisynthetase syndrome 的經典組合是**肌炎 + ILD + 關節炎 + 發燒 + mechanic's hand + Raynaud**，雖然也有 Raynaud 與 ILD，但**不會有廣泛四肢皮膚增厚**這種明顯的皮膚纖維化表現，這是與 SSc 最重要的鑑別點。
- **E（錯）** Multiple sclerosis 是中樞神經系統的脫髓鞘疾病，與 Raynaud、皮膚增厚、ILD、GERD 這些表現完全無關，屬於明顯的誤導選項。

**SSc 核心四聯徵速記：**

| 系統 | 表現 |
|---|---|
| 血管 | Raynaud phenomenon（常最早出現） |
| 皮膚 | 皮膚增厚/纖維化（limited vs. diffuse） |
| 肺臟 | ILD（乾咳、喘）— 主要死因之一 |
| 腸胃道 | 食道蠕動異常、GERD（最常見的內臟侵犯） |

### 9. AD 何者為錯?
- A Dupilumab 結合 IL-4Rα, 也可影響 IL-13
- B Dupilumab 對難治的 COPD 也有效
- C Tezepelumab 治療 AD, phase 2 trial 顯著療效
- D Lebrikizumab, tralokinumab 為 anti-IL-13, 對 AD 有效

**答案：C**

**解析：**

- **A（正確敘述，非本題答案）** **Dupilumab** 結合的標靶是 **IL-4Rα**。因為 IL-4Rα 是 **type I IL-4 receptor（IL-4Rα + γc）** 與 **type II receptor（IL-4Rα + IL-13Rα1，同時也是 IL-13 的訊息受體）** 的**共同次單元**，所以阻斷 IL-4Rα 可以**同時阻斷 IL-4 與 IL-13 兩條細胞激素的下游訊息傳遞**，這也是 dupilumab「一藥雙標」機轉的分子基礎。
- **B（正確敘述，非本題答案）** Dupilumab 除了 AD、氣喘、CRSwNP 之外，近年也已證實對**特定表型（type 2 inflammation / eosinophilic phenotype）的難治性 COPD** 有效——**BOREAS 與 NOTUS 兩項第三期試驗**證實在合併 type 2 發炎特徵（如周邊血 eosinophil 上升）的 COPD 病人身上，dupilumab 可顯著降低急性惡化（exacerbation）發生率，並已於近年獲得 COPD 這項新適應症的核准，是 dupilumab 適應症持續擴展的重要里程碑。
- **C（錯誤敘述，本題答案）** **Tezepelumab（anti-TSLP）** 雖然在**氣喘**（尤其重度氣喘，NAVIGATOR trial）已證實顯著療效並獲核准上市，但用於治療 **AD 的 phase 2 trial 結果並不理想**——並未達到預期中顯著優於安慰劑的療效指標，效果不如在氣喘領域亮眼，目前 tezepelumab **並未**成為 AD 的核准治療選項。這句話把它在氣喘的成功「複製貼上」到 AD，是錯誤的敘述。
- **D（正確敘述，非本題答案）** **Lebrikizumab** 與 **Tralokinumab** 都是**選擇性 anti-IL-13** 單株抗體（不像 dupilumab 同時阻斷 IL-4/IL-13），兩者皆已完成第三期試驗並獲核准用於中重度 AD 治療，是目前 AD 生物製劑家族中除 dupilumab 外的重要成員。

**AD/氣喘生物製劑速記（含跨適應症）：**

| 藥物 | 標靶 | AD | 氣喘 | COPD | CRSwNP |
|---|---|---|---|---|---|
| Dupilumab | IL-4Rα (IL-4+IL-13) | ✓ | ✓ | ✓（type 2 表型） | ✓ |
| Lebrikizumab / Tralokinumab | IL-13 | ✓ | — | — | — |
| Tezepelumab | TSLP | ✗（phase 2 未達顯著療效） | ✓ | — | — |

### 10. 健保 AD biologics 前的免疫抑制劑, 何者不建議?
- A Methotrexate
- B Cyclosporine
- C Azathioprine
- D Mycophenolate

**答案：D（採實證等級推論，信心度中等）**

**解析：**

查詢 [健保給付條件202606更新](../../../../健保給付條件202606更新) 資料夾中的 dupilumab 申請表（附表三十二之一）與 chap8 條文，發現健保條文只寫「至少2種其他系統性(全身性)用藥」，**未明確列出哪幾種藥物名稱**，也沒有排除 mycophenolate 的明文規定，此題答案無法直接從健保條文逐字核對確認。以下改用**臨床實證等級**的角度解析：

AD 傳統全身性免疫抑制劑（非生物製劑）常用的四種藥物，依國際指引（如 AAD guideline）實證等級大致排序：

- **B（首選，非本題答案）** **Cyclosporine** 是**證據等級最強、起效最快**的傳統全身性藥物，多數指引與健保審查邏輯中都將它列為**優先使用/常規要求**的第一線系統性治療（健保條文中 cyclosporine 明確被列為異位性皮膚炎全身性紅皮症、替代性療法無效之嚴重異位性皮膚炎的給付藥物之一）。
- **A（次選，非本題答案）** **Methotrexate** 也有相當證據支持用於中重度 AD，是常見的第二線傳統系統性藥物選擇。
- **C（次選，非本題答案）** **Azathioprine** 同樣有實證支持，但因起效較慢（約需 8–12 週）、使用前需檢測 **TPMT/NUDT15** 活性以評估骨髓抑制風險，仍是被接受的傳統選項之一。
- **D（證據等級最弱，本題答案）** **Mycophenolate mofetil (MMF)** 在 AD 治療的實證等級**明顯弱於前三者**，多數指引將其列為**「其他藥物皆無效或不適用時才考慮」的末線選項**，證據強度不足以支持將它列為健保生物製劑前必須嘗試的常規免疫抑制劑，因此四者之中**最不建議**作為 AD biologics 前優先使用的傳統系統性治療。

**⚠️ 提醒：** 因無法在現有健保條文中找到逐字對應的規定，這題答案是根據**臨床實證等級**推論，若之後查到更明確的官方函釋文字可再進一步核對確認。

### 11. GCA 特色何者正確
- A >50 y/o female 常見
- B 常見 DAH 跟 GN
- C 常見 visual loss 跟 temporal tenderness
- D 常合併 PMR 但需 high dose steroid treatment

**答案：C**

**解析：**

**Giant cell arteritis (GCA)** 是大血管炎中最經典、最需要及早辨識的疾病（因為可能造成不可逆失明），核心臨床特徵圍繞在**顳動脈相關症狀**與**視力威脅**：

- **C（對，本題答案）** **Visual loss（視力喪失）與 temporal tenderness（顳動脈壓痛）** 是 GCA 教科書等級的核心特徵：
  - **Temporal tenderness / headache**：因顳動脈本身發炎、管壁增厚，觸診時有明顯壓痛，常合併新發生的顳部頭痛（new-onset headache），是 GCA 最典型的局部表現。
  - **Visual loss**：GCA 最令人畏懼的併發症，機轉是**anterior ischemic optic neuropathy (AION)**，因供應視神經的後睫狀動脈（posterior ciliary artery）被侵犯造成缺血，一旦發生**通常是不可逆**的，這也是為什麼懷疑 GCA 時要**立即經驗性給予高劑量類固醇**（不需等切片結果），以搶救對側眼睛視力。

逐一排除其他選項：

- **A（不夠精確，非本題答案）** GCA 確實好發於 **>50 歲、女性略多**（female:male 約 2–3:1），這個敘述本身在流行病學上大致正確，但只描述**族群分布**，並非 GCA 最具代表性、最該優先辨識的「特色」（相較於 C 的臨床急迫性與診斷意義，A 較不是這題想考的重點）。
- **B（錯）** **DAH（diffuse alveolar hemorrhage）與 GN（glomerulonephritis）** 是**小血管炎（尤其 AAV：GPA/MPA）** 的典型表現，與 GCA（大血管炎，主要侵犯 aorta 及其分支，尤其顱外分支如顳動脈）的疾病譜完全不同，GCA **並不會**造成肺泡出血或腎絲球腎炎。
- **D（部分正確但論述角度不是本題重點）** GCA 確實常合併 **PMR**（約 40–60% GCA 病人合併 PMR 症狀），而且治療 GCA（因視力風險）本身就需要**高劑量類固醇**（如 prednisone 40–60mg/day，甚至 IV pulse），這點沒錯；但這句話容易讓人誤解成「因為合併 PMR 所以需要高劑量」——實際上**單純 PMR** 本身只需要**低劑量類固醇**（15–20mg/day）就有效，是 GCA 本身（而非 PMR 部分）才需要高劑量治療。因為論述邏輯較容易造成混淆，不如 C 精確、典型。

**GCA vs. PMR 治療劑量速記：**

| | 典型類固醇起始劑量 | 核心風險 |
|---|---|---|
| 單純 PMR | Prednisone 15–20mg/day | 復發 |
| GCA（±合併 PMR） | Prednisone 40–60mg/day，甚至 IV pulse | 視力喪失（不可逆） |

### 12. Behçet's Dz 選對的
- A Skin biopsy 常見 eosinophil infiltration
- B 常見 HLA-B27, erosive urethritis
- C 常見非瘢痕性 ulcer, antibiotics 反應佳
- D 多為 small vessel involvement, 常見陰部及眼部病變

**答案：D**

**解析：**

**Behçet's disease (BD)** 在 2012 Chapel Hill Consensus Conference (CHCC) 血管炎分類中被歸類為 **「variable vessel vasculitis（可變管徑血管炎）」**——因為它是少數可以**同時侵犯小、中、大血管，動脈與靜脈皆可受累**的疾病，這是 BD 血管炎表現上的一大特色（也因此不易被歸入傳統依血管口徑分類的體系）。臨床上，**皮膚黏膜的小血管炎表現（如 oral/genital ulcer）搭配眼部病變（uveitis、retinal vasculitis）**，是 BD 最具代表性、也是分類標準核心的兩大特徵。

逐一解析：

- **D（對，本題答案）** BD 的血管侵犯確實**以小血管表現最常見**（如口腔/生殖器潰瘍的血管炎機轉、皮膚病灶），同時**陰部潰瘍（genital ulcer）與眼部病變（uveitis、視網膜血管炎）**是 BD 國際診斷標準（ISG criteria、ICBD criteria）中最重要的兩項核心表現，此敘述整體符合 BD 的臨床/病理特徵。
- **A（錯）** BD 的皮膚切片病理特徵是 **neutrophilic vascular reaction（嗜中性白血球血管反應）**，甚至可呈現 **leukocytoclastic vasculitis**，而非 eosinophil infiltration。Eosinophil 浸潤並非 BD 的典型病理表現，反而較常見於過敏性疾病或 EGPA 等嗜酸性血管炎。
- **B（錯）** 這句話犯了兩個錯誤：
  1. BD 相關的 HLA 是 **HLA-B51**，而非 HLA-B27（HLA-B27 是 SpA 家族，如 AS、reactive arthritis 的相關基因）。
  2. BD 的生殖器表現是**潰瘍（genital ulcer）**，並非 **erosive urethritis**——urethritis 是 reactive arthritis（Reiter syndrome）的典型表現，兩者容易混淆但機轉、病灶型態完全不同。
- **C（錯）** BD 的口腔/生殖器潰瘍確實多為**非瘢痕性（non-scarring，尤其口腔潰瘍）**（雖然男性陰囊潰瘍偶爾可能留疤），但這句話的錯誤在於**「antibiotics 反應佳」**——BD 的致病機轉是**自體發炎/自體免疫**，並非感染性疾病，潰瘍對抗生素**沒有反應**，標準治療是 **colchicine、局部/全身類固醇**，而非抗生素。這也是臨床上用來與感染性潰瘍鑑別的重要線索。

**BD 血管侵犯與核心表現速記：**

| 項目 | BD 特徵 |
|---|---|
| 血管口徑分類 | Variable vessel vasculitis（動靜脈、大中小血管皆可侵犯） |
| 皮膚病理 | Neutrophilic vascular reaction |
| 相關 HLA | HLA-B51（非 B27） |
| 生殖器表現 | Genital ulcer（非 urethritis） |
| 潰瘍治療 | Colchicine / 類固醇（非抗生素，因非感染性疾病） |

### 13. AS 選對的
- A SSZ for all AS patients
- B Anti-IL17 for AS + uveitis
- C TNFi regular combine steroid to reduce anti-drug antibody
- D Unclear disease activity → arrange MRI

**答案：D**

**解析：**

- **D（對，本題答案）** 當臨床上**難以判斷 AS 疾病活性**（例如症狀不典型、發炎指標與臨床表現不一致、或懷疑合併其他病因造成的疼痛）時，**MRI（尤其 SI joint 或脊椎的 STIR/T2 fat-sat 序列）** 是釐清是否仍有**活動性發炎（active inflammation, bone marrow edema）**的標準工具。這呼應 ASAS 對 axSpA 的建議：MRI 能偵測 X 光看不到的早期/活動性發炎變化，是判斷疾病活性、輔助治療決策（如是否升階治療）的重要影像工具。
- **A（錯）** **Sulfasalazine (SSZ)** 對 AS 的**軸心（axial）症狀療效證據薄弱**，主要證據集中在**周邊關節炎（peripheral arthritis）**的治療。因此「SSZ for all AS patients」的敘述並不正確——SSZ 並非所有 AS 病人的常規用藥，對單純中軸型（無周邊關節炎）的 AS 病人效果有限，並非治療首選。
- **B（錯）** 呼應之前風專題目討論過的觀念：AS 合併 uveitis 時，**IL-17 inhibitor 並非優先選擇**，甚至部分研究對其在葡萄膜炎控制上的證據不一致/存有疑慮；治療首選是**單株抗體類 anti-TNF（如 adalimumab、infliximab、golimumab），並避開 etanercept（soluble receptor，效果較差）**。近年雖有 bimekizumab（IL-17A/F 雙標）在 AAU 有較新證據，但傳統選擇性 anti-IL-17A（如 secukinumab、ixekizumab）並非合併 uveitis 病人的優先建議。
- **C（錯）** 這句話混淆了兩件事：(1) **降低 anti-drug antibody（提升藥物免疫原性耐受度）的標準做法是併用 methotrexate**，而非類固醇；(2) 類固醇在 AS（軸心疾病為主）治療中的角色本來就**有限**，並非 TNFi 常規需要合併使用的藥物。這題的陷阱是把「TNFi + MTX 可降低 anti-drug antibody」的觀念（原本用於 RA）誤植成「TNFi + steroid」，藥物與機轉都對應錯誤。

**AS 治療觀念速記：**

| 項目 | 正確做法 |
|---|---|
| SSZ 適用範圍 | 主要對周邊關節炎有效，非中軸型 AS 的常規用藥 |
| 降低 anti-drug antibody | 併用 Methotrexate（非類固醇） |
| AS + uveitis 首選 | 單株抗體類 Anti-TNF（避開 etanercept、避開傳統 IL-17Ai） |
| 疾病活性不明確 | 安排 MRI（STIR/T2 fat-sat）評估是否有活動性發炎 |

### 14. ASDAS, which is not included?
- A Back pain
- B Peripheral pain
- C Uveitis
- D Morning stiffness duration

**答案：C**

**解析：**

**ASDAS (Ankylosing Spondylitis Disease Activity Score)** 的計算公式由以下**五個項目**組成：

1. **Back pain（背痛，BASDAI Q2）**
2. **Duration of morning stiffness（晨僵持續時間，BASDAI Q6）**
3. **Patient global assessment（病人整體評估）**
4. **Peripheral pain/swelling（周邊關節疼痛/腫脹，BASDAI Q3）**
5. **CRP（ASDAS-CRP版本）或 ESR（ASDAS-ESR版本）**

逐一核對選項：

- **A（是 ASDAS 組成項目，非本題答案）** Back pain 是 ASDAS 公式中的核心項目之一。
- **B（是 ASDAS 組成項目，非本題答案）** Peripheral pain（周邊關節疼痛）也是公式組成項目之一。
- **D（是 ASDAS 組成項目，非本題答案）** Morning stiffness duration（晨僵持續時間）同樣是公式組成項目之一。
- **C（不是 ASDAS 組成項目，本題答案）** **Uveitis（葡萄膜炎）** 是 AS 常見的 **extra-articular manifestation（關節外表現）**，但**並未被納入 ASDAS 疾病活性評分公式**。ASDAS 評估的是「疾病活性」本身（以疼痛、晨僵、病人整體感受、發炎指標為主軸），而 uveitis 屬於獨立的器官侵犯併發症，需要另外追蹤與處置，不會直接反映在 ASDAS 分數中。

**ASDAS 組成速記：**

| 項目 | 來源 |
|---|---|
| Back pain | BASDAI Q2 |
| Duration of morning stiffness | BASDAI Q6 |
| Patient global assessment | 病人自評 |
| Peripheral pain/swelling | BASDAI Q3 |
| CRP 或 ESR | 抽血 |

**不包含：** Uveitis、其他 extra-articular manifestation（如 IBD、psoriasis）皆非 ASDAS 公式成分。

### 15. 42 y/o male, sinusitis, asthma, peripheral neuropathy. Lab: eos > 2300/μL, ↑↑ IgE. What's the most useful test to confirm diagnosis?
- A Allergen specific IgE
- B Anti-MPO
- C Nerve or tissue biopsy showing eosinophilic vasculitis
- D Detection of FIP1L1-PDGFRA fusion gene

**答案：C**

**解析：**

**臨床情境：** 42 歲男性，**鼻竇炎 + 氣喘 + 周邊神經病變**，合併**顯著嗜酸性球增多（eos > 2300/μL）與 IgE 顯著升高**——這是 **EGPA (Eosinophilic Granulomatosis with Polyangiitis，舊稱 Churg-Strauss syndrome)** 教科書等級的典型臨床三部曲：

1. **前驅期（prodromal phase）**：氣喘、過敏性鼻炎/鼻竇炎、鼻息肉
2. **嗜酸性球浸潤期（eosinophilic phase）**：周邊血與組織嗜酸性球顯著增多（常 >1500/μL，此例 >2300）
3. **血管炎期（vasculitic phase）**：全身性小血管炎表現，最經典的就是 **peripheral neuropathy（尤其 mononeuritis multiplex）**，另可侵犯皮膚、心臟、腎臟等。

**確診的關鍵：**

- **C（對，本題答案）** 雖然 EGPA 有 2022 ACR/EULAR 分類標準（結合臨床特徵 + 檢驗 + 影像的積分系統），但**確診的金標準仍是病理**——取受侵犯的組織（此病人有周邊神經病變，最適合做 **nerve biopsy**，或皮膚/肺組織切片）,顯示**嗜酸性球浸潤合併血管炎（eosinophilic vasculitis）**，甚至合併 **extravascular granuloma**，是最直接、最具確定性的診斷依據。
- **B（輔助但非最佳）** **Anti-MPO（p-ANCA）** 在 EGPA 病人中陽性率約僅 **40–60%**，且陽性與否與臨床表型有關（ANCA 陽性者較常有腎臟/周邊神經侵犯，屬於「vasculitic phenotype」；ANCA 陰性者則較常以心臟侵犯為主，屬於「eosinophilic phenotype」）。因為並非所有 EGPA 病人都會 ANCA 陽性，anti-MPO **陰性不能排除診斷**，屬於**支持性**而非**確診性**的檢查，不如組織切片直接。
- **A（錯）** Allergen specific IgE 只能反映病人是否有特定過敏原致敏（atopy），與 EGPA 這種系統性血管炎的診斷無直接關聯。
- **D（錯，但為重要鑑別診斷）** **FIP1L1-PDGFRA fusion gene** 是 **PDGFRA-associated clonal hypereosinophilic syndrome / chronic eosinophilic leukemia (CEL)** 的分子標記，並非 EGPA 的診斷工具。但這是這題**很重要的鑑別診斷**——當病人有顯著嗜酸性球增多合併器官侵犯時，臨床上必須先排除**myeloid neoplasm 相關的 clonal HES**（此類病人對類固醇反應差，需用 imatinib 治療，治療策略完全不同），所以檢驗 FIP1L1-PDGFRA 仍有其臨床意義，只是目的是**排除另一個疾病**，而非**確認 EGPA 本身**。

**EGPA 診斷邏輯速記：**

| 檢查 | 角色 |
|---|---|
| Nerve/tissue biopsy（eosinophilic vasculitis） | **確診金標準** |
| Anti-MPO (p-ANCA) | 支持診斷，陽性率約 40–60%，可預測表型 |
| FIP1L1-PDGFRA | 排除 clonal HES/CEL 這個重要鑑別診斷 |
| Allergen specific IgE | 僅反映 atopy，非診斷工具 |

### 16. Autoinflammatory 敘述選錯的
- A Proinflammatory cytokine ↑
- B Neutrophil NETosis ↑
- C Macrophage activation ↑
- D Type I IFN ↓

**答案：D**

**解析：**

**Autoinflammatory disease（自體發炎疾病）** 是一群因**先天免疫系統（innate immunity）失調**造成的疾病，核心機轉是不需要自體抗體或抗原專一性 T/B cell 參與，而是**innate immune cell（尤其 macrophage、monocyte、neutrophil）過度活化**，導致促發炎細胞激素大量釋放：

- **A（正確敘述，非本題答案）** **Proinflammatory cytokine（尤其 IL-1β、IL-18、IL-6、TNF-α）顯著上升**是 autoinflammatory disease 的核心生化特徵，多數經典疾病（FMF、CAPS、TRAPS、MKD 等）都圍繞 **inflammasome 過度活化 → IL-1β 大量釋放**這個核心機轉。
- **B（正確敘述，非本題答案）** **Neutrophil NETosis（嗜中性球胞外捕捉網形成）增加**近年被認為是 autoinflammatory disease 病理機轉中日益重要的一環，NETs 本身可作為 damage-associated molecular pattern (DAMP)，進一步活化 innate immune response，形成惡性循環。
- **C（正確敘述，非本題答案）** **Macrophage activation 增加**是 autoinflammatory disease 的核心細胞層級表現，inflammasome（如 NLRP3）主要就是在 macrophage/monocyte 內活化，驅動下游細胞激素釋放。
- **D（錯誤敘述，本題答案）** 這句話**方向講反了**。Autoinflammatory disease 這個大家族中，有一群特別的疾病稱為 **interferonopathy（干擾素病變）**，代表性疾病如 **SAVI（STING-associated vasculopathy with onset in infancy）、AGS（Aicardi-Goutières syndrome）、CANDLE syndrome**，這些疾病的核心機轉正是 **Type I interferon（IFN-α/β）訊息路徑過度活化，IFN signature 顯著「上升」**，而非降低。因此把 autoinflammatory disease 概括描述成「Type I IFN ↓」是錯誤的，事實上部分 autoinflammatory disease 亞型正是以 **Type I IFN ↑** 為特色。

**Autoinflammatory disease 分類速記：**

| 機轉分類 | 代表疾病 | 核心細胞激素 |
|---|---|---|
| IL-1 mediated（inflammasomopathy） | FMF, CAPS, TRAPS, MKD | IL-1β ↑ |
| Interferonopathy | SAVI, AGS, CANDLE | Type I IFN ↑ |
| NF-κB mediated | Blau syndrome | TNF, IL-6 ↑ |

### 17. Autoinflammatory 治療選錯的
- A Acute phase 常用 steroid
- B MTX 為常用 immunosuppressive
- C JAK 為目前治療主流
- D Anti-IL1 跟 anti-IL6 為常見 biologics

**答案：C**

**解析：**

- **A（正確敘述，非本題答案）** **Autoinflammatory disease 急性發作期（acute flare）常使用類固醇**快速控制發炎症狀（如發燒、關節痛、漿膜炎等），是急性期常見的輔助治療。
- **B（正確敘述，非本題答案）** **Methotrexate (MTX)** 是常用的傳統免疫抑制劑之一，尤其在合併關節炎表現明顯的 autoinflammatory disease（如 AOSD 合併多關節炎）時，常作為類固醇減量後的維持治療選擇。
- **D（正確敘述，非本題答案）** **Anti-IL-1（如 anakinra、canakinumab、rilonacept）與 anti-IL-6（如 tocilizumab）** 是目前 autoinflammatory disease **最主流、最核心**的生物製劑類別，直接針對 inflammasome 活化後下游的關鍵促發炎細胞激素（IL-1β、IL-6），是這類疾病治療的標準骨幹藥物。
- **C（錯誤敘述，本題答案）** **JAK inhibitor 目前並非 autoinflammatory disease 的治療主流**。雖然近年在**interferonopathy（如 SAVI、AGS、CANDLE）** 這個亞型中，因為機轉牽涉 **type I IFN → JAK-STAT** 訊息路徑，JAK inhibitor（如 baricitinib）確實展現出治療潛力並有臨床試驗證據支持，但這僅限於**特定亞型**，並不能代表**整個 autoinflammatory disease 類別的治療主流**。目前絕大多數經典 autoinflammatory disease（IL-1 mediated 為主的 FMF、CAPS、TRAPS、MKD 等）**治療主流仍是 anti-IL-1/anti-IL-6 生物製劑**，JAKi 只是新興、針對特定亞型（interferonopathy）的治療選項，尚非「主流」。

**Autoinflammatory disease 治療速記：**

| 治療 | 角色 |
|---|---|
| 類固醇 | 急性期快速控制 |
| MTX | 傳統免疫抑制劑，合併關節炎時常用 |
| Anti-IL-1 / Anti-IL-6 | **目前治療主流**（生物製劑骨幹） |
| JAK inhibitor | 新興治療，主要用於 interferonopathy 亞型，尚非全面主流 |

### 18. 有關 IIM, 何者不正確?
- A EMG: high amplitude, short duration, polyphasic
- B MRI: STIR view show bright signal in inflamed muscle
- C Pulmonary function test: decreased TLC and DLCO
- D CT: GGO or reticular pattern

**答案：A**

**解析：**

- **A（錯誤敘述，本題答案）** IIM（Idiopathic Inflammatory Myopathy，發炎性肌肉病變）的典型 **EMG（肌電圖）myopathic pattern** 應為：**low amplitude（低振幅）、short duration（短時距）、polyphasic（多相波）** 的 motor unit action potential (MUAP)，同時常合併**自發性纖顫電位（fibrillation potential）與正銳波（positive sharp wave）**。這題把 **amplitude 講反了**——「**high amplitude**」實際上是**neurogenic pattern（神經性病變）**的特徵（如周邊神經病變、運動神經元疾病），並非肌肉本身發炎受損的表現。**肌肉病變（myopathic）的 EMG 特徵是振幅「降低」而非「增高」**，這是與神經性病變鑑別的關鍵。
- **B（正確敘述，非本題答案）** **MRI STIR（壓脂序列）** 對水腫訊號敏感，發炎中的肌肉因肌纖維水腫、發炎細胞浸潤，會在 STIR 序列上呈現**高訊號（bright signal）**，是評估 IIM 肌肉發炎活性、選定切片部位的重要影像工具。
- **C（正確敘述，非本題答案）** IIM 常合併 **interstitial lung disease (ILD)**，造成**限制性通氣障礙（restrictive pattern）**，肺功能檢查典型表現為 **TLC（total lung capacity）與 DLCO（diffusion capacity）皆下降**，是評估 IIM 肺部侵犯嚴重度的常規檢查。
- **D（正確敘述，非本題答案）** IIM 合併 ILD 的胸部 CT 典型影像為 **ground-glass opacity (GGO)** 或 **reticular pattern**，反映肺部發炎/纖維化變化，與前一項 PFT 限制性變化互相呼應。

**EMG Myopathic vs. Neurogenic pattern 速記：**

| | Myopathic（肌肉病變，如 IIM） | Neurogenic（神經性病變） |
|---|---|---|
| Amplitude | **低（low）** | 高（high） |
| Duration | 短（short） | 長（long） |
| Polyphasic | 常見 | 常見 |

### 19. 哪個抗體跟 congenital heart block 有關?
- A ANCA
- B Anti-SSA
- C Anti-phospholipid
- D Anti-dsDNA

**答案：B**

**解析：**

**Congenital heart block (CHB)** 是 **neonatal lupus syndrome** 最嚴重、最具代表性的表現，與母體的 **anti-SSA (Ro)、anti-SSB (La)** 抗體經胎盤傳遞至胎兒有直接因果關係。

- **B（對，本題答案）** **Anti-SSA（Ro，尤其 anti-Ro52）抗體**（常與 anti-SSB/La 併存）可經由胎盤主動運輸進入胎兒循環，結合胎兒心臟傳導系統（尤其 AV node）的心肌細胞表面抗原，誘發**局部發炎反應與纖維化**，最終破壞正常的房室傳導路徑，導致**不可逆的 complete heart block（third-degree AV block）**，好發於懷孕 **18–24 週**（心臟傳導系統發育的關鍵敏感期）。這也是為什麼 SjD 或 SLE 病人（尤其 anti-SSA/SSB 陽性者）懷孕時需要密切追蹤胎兒心律，必要時給予 **hydroxychloroquine** 作為預防性治療。
- **A（錯）** ANCA（anti-neutrophil cytoplasmic antibody）與 AAV（GPA、MPA、EGPA）相關，與胎兒心臟傳導系統無關。
- **C（錯）** Anti-phospholipid antibody 與 APS 相關的**血栓、流產、胎盤功能不全**有關，但**不會直接造成 congenital heart block**，機轉完全不同（APS 造成的是血管栓塞相關的胎盤病變，而非抗體直接攻擊心臟傳導組織）。
- **D（錯）** Anti-dsDNA 是 SLE 疾病活性與 lupus nephritis 的指標性抗體，與 CHB 的致病機轉無直接關聯。

**Neonatal lupus / CHB 速記：**

| 項目 | 內容 |
|---|---|
| 致病抗體 | Anti-SSA (Ro)（常合併 anti-SSB/La） |
| 好發週數 | 懷孕 18–24 週 |
| 病理機轉 | 抗體經胎盤 → 攻擊胎兒 AV node → 發炎/纖維化 → 不可逆傳導阻斷 |
| 預防/治療 | Hydroxychloroquine（高風險孕婦預防性使用） |

### 20. 根據 EULAR Sjögren criteria, 幾分可以診斷?
- A 3
- B 4
- C 5
- D 6

**答案：B**

**解析：**

**2016 ACR/EULAR primary Sjögren syndrome classification criteria** 採計分制，五個項目依權重給分，總分 **≥4 分**即可分類診斷為 primary SjD（需先符合基本納入條件：至少有一項 sicca 症狀或於 ESSDAI 中有懷疑 SjD 之系統性表現）。

**五個計分項目：**

| 項目 | 分數 |
|---|---|
| Labial salivary gland biopsy 呈現 focal lymphocytic sialadenitis，focus score ≥1 foci/4mm² | 3 分 |
| **Anti-SSA/Ro 陽性** | 3 分 |
| Ocular Staining Score ≥5（或 van Bijsterveld score ≥4），至少一眼 | 1 分 |
| **Schirmer's test ≤5mm/5min**，至少一眼 | 1 分 |
| **Unstimulated whole saliva flow rate ≤0.1mL/min** | 1 分 |

**總分 ≥4 分 → 分類為 primary Sjögren syndrome。**

值得注意的是，光是 **anti-SSA/Ro 陽性（3分）+ 唾液腺切片陽性（3分）** 這兩項中任一項成立，加上其他任一客觀項目（1分），就能輕易達到 4 分門檻——這也反映出這兩項是整份分類標準中**權重最高、診斷意義最強**的兩個核心項目。

### 21. Polymyositis 的典型特色？
- A 全身無力
- B 近端無力
- C 遠端無力
- D 頸部無力

**答案：B**

**解析：**

**Polymyositis (PM)** 與其他 IIM（如 dermatomyositis）共同的核心臨床特徵是 **symmetric proximal muscle weakness（對稱性近端肌肉無力）**——侵犯的是**近端肌群**，如肩帶（shoulder girdle）、骨盆帶（pelvic girdle）肌肉，典型症狀包括**爬樓梯困難、從坐姿站起困難、舉手過肩困難（如梳頭、曬衣）**。

- **B（對，本題答案）** 對稱性**近端**肌肉無力是 PM/DM 的診斷核心特徵，也是分類標準（如 Bohan and Peter criteria、2017 EULAR/ACR IIM criteria）中最重要的臨床項目之一。
- **A（錯）** 「全身無力」過於籠統，並非 PM 的專一性描述——PM 特別侵犯的是**近端**肌群，遠端肌力通常**相對保留**（尤其疾病早期），這個「近端 > 遠端」的分布型態才是關鍵鑑別點。
- **C（錯）** **遠端無力（distal weakness）** 反而是 **inclusion body myositis (IBM)** 的特徵性表現（IBM 常合併遠端肌肉無力，尤其手指屈肌、足下垂），與 PM 的近端為主分布相反，是鑑別 PM/DM 與 IBM 的重要線索。
- **D（錯）** 頸部無力（尤其頸屈肌無力，neck flexor weakness）雖然也可以出現在 PM/DM 病人身上（甚至是評估疾病嚴重度的項目之一），但**並非其最典型、最具代表性的核心特徵**，近端肢體肌肉無力才是主軸。

**IIM 肌無力分布速記：**

| 疾病 | 典型無力分布 |
|---|---|
| Polymyositis / Dermatomyositis | 對稱性**近端**肌肉無力 |
| Inclusion body myositis (IBM) | **遠端**肌肉無力（常不對稱），尤其手指屈肌、足下垂 |

### 22. 哪個抗體跟皮下鈣化最有關係?
- A TIF1γ
- B NXP2
- C Jo-1
- D Mi-2

**答案：B**

**解析：**

**Anti-NXP2（nuclear matrix protein 2）抗體**是與 **calcinosis cutis（皮下鈣化）** 關聯性最強的肌炎特異性抗體（myositis-specific antibody, MSA），這在**兒童型皮肌炎（juvenile dermatomyositis, JDM）** 的文獻中尤其明確，anti-NXP2 陽性病人發生皮下鈣化的比例顯著較高，也常合併較嚴重的肌肉無力與血管病變（vasculopathy）表現。

- **B（對，本題答案）** **Anti-NXP2** 與**皮下鈣化（calcinosis）**高度相關，是這個抗體最具代表性的臨床關聯，尤其在 JDM 病人中，calcinosis 甚至可作為疾病嚴重度與病程慢性化的指標之一。
- **A（錯）** **Anti-TIF1γ（p155/140）** 主要與**成人惡性腫瘤相關皮肌炎（cancer-associated dermatomyositis）**高度相關，是篩檢潛在惡性腫瘤的重要標記，並非以皮下鈣化為主要關聯特徵。
- **C（錯）** **Anti-Jo-1** 是最常見的 **antisynthetase antibody**，與 **antisynthetase syndrome**（ILD、機械手、關節炎、發燒、Raynaud）相關，與皮下鈣化無特別關聯。
- **D（錯）** **Anti-Mi-2** 與**典型皮肌炎皮膚表現（如 Gottron's papules、heliotrope rash）**及**對治療反應較好、預後較佳**的皮肌炎表型相關，並非皮下鈣化的代表性抗體。

**肌炎特異性抗體（MSA）臨床關聯速記：**

| 抗體 | 主要臨床關聯 |
|---|---|
| Anti-NXP2 | **皮下鈣化（calcinosis）**，兒童型 DM 尤其明顯 |
| Anti-TIF1γ | 惡性腫瘤相關 DM（成人） |
| Anti-Jo-1 | Antisynthetase syndrome（ILD、機械手、關節炎） |
| Anti-Mi-2 | 典型皮膚表現，預後較佳 |
| Anti-MDA5 | Clinically amyopathic DM，Rapidly progressive ILD |
| Anti-SAE | 皮膚表現為主，病程較溫和 |

### 23. Anti-Jo-1 是 AC 多少?
- A AC-17
- B AC-18
- C AC-19
- D AC-20

**答案：A**

**解析：**

ICAP 對 **cytoplasmic pattern** 的分類中：

| AC 編號 | Pattern | 相關抗體/抗原 |
|---|---|---|
| **AC-17** | **Cytoplasmic speckled（細胞質顆粒狀）** | **Anti-synthetase antibodies**（包括 **Jo-1**、PL-7、PL-12、EJ、OJ）、anti-SRP |
| AC-18 | Discrete dots / GW body-like | Anti-GW182 |
| AC-19 | Dense fine speckled（cytoplasmic） | Anti-ribosomal P protein |
| AC-20 | Reticular / AMA pattern | Anti-mitochondrial antibody (PDC-E2，原發性膽道膽管炎標記) |

**Anti-Jo-1（histidyl-tRNA synthetase）** 屬於 **antisynthetase antibody** 家族，這類抗體辨識的抗原位於**細胞質**（tRNA synthetase 是細胞質蛋白，並非細胞核蛋白），在 HEp-2 細胞 IIF 染色下會呈現**cytoplasmic speckled pattern**，對應 ICAP 編號 **AC-17**，而非 AC-18。

**AC-18（Discrete dots/GW body-like）** 是另一種細胞質型態，特徵是細胞質內**少量、離散的點狀訊號**（類似 P-body/GW-body 結構），對應的是 **anti-GW182** 抗體，與 anti-Jo-1 是完全不同的抗原與染色型態。

**易混淆提醒：** AC-17 到 AC-20 都屬於細胞質染色型態，编號相近很容易混淆，記憶口訣可以抓：**AC-17 speckled → synthetase（Jo-1家族）；AC-19 dense fine speckled → ribosomal P；AC-20 reticular → mitochondrial (AMA)**。

### 24. HAE 的病人, C4 低下, C1-INH 低下, 是 Type 幾?
- A I
- B II
- C III
- D IV

**答案：A（Type I）**

**解析：**

**Hereditary angioedema (HAE)** 依 C1-inhibitor (C1-INH) 的異常型態分為三型，鑑別的關鍵是要同時看 **C1-INH 抗原量（antigenic level）** 與 **C1-INH 功能（functional level）**，並搭配 **C4** 作為輔助篩檢：

| 型別 | C1-INH 抗原量 | C1-INH 功能 | C4 | 機轉 |
|---|---|---|---|---|
| **Type I（約 85%，最常見）** | **低下** | 低下 | **低下** | C1-INH 基因突變 → 蛋白質**量不足**（quantitative deficiency） |
| **Type II（約 15%）** | **正常或升高** | **低下** | 低下 | C1-INH 基因突變 → 蛋白質量正常但**功能異常**（dysfunctional protein，qualitative defect） |
| **Type III（HAE with normal C1-INH）** | 正常 | 正常 | 正常 | 與 C1-INH 無關，與 Factor XII、plasminogen、angiopoietin-1 等基因變異相關，常與雌激素（estrogen）有關，好發於女性 |

**這題的關鍵：** 題目給的是 **「C4 低下 + C1-INH 低下」**——這裡的 **C1-INH 低下指的是抗原量（antigenic level）本身就不足**，同時 C4 也偏低（因為 C1-INH 功能不足，導致 classical complement pathway 的 C1 esterase 活性失控，持續消耗 C4）。這個組合（**抗原量與功能都低**）正是 **Type I HAE** 的定義特徵。

若題目給的是「C4 低下，但 C1-INH **抗原量正常/偏高**，功能檢測才異常」，那才會是 **Type II**。

**記憶口訣：**
- **C4 篩檢**：HAE Type I / II 病人 C4 幾乎都是持續偏低（很好的初步篩檢工具，靈敏度高）。
- **確診分型靠 C1-INH**：量不夠 → Type I；量夠但不會動（功能異常）→ Type II；量跟功能都正常 → Type III（要找 C1-INH 以外的基因）。

### 25. 承上題, 病人急性腹痛去急診, CT 切腸壁腫脹, 急性處置為下列何者?
- A Danazol
- B Transamin
- C Icatibant
- D FFP

**答案：C**

**解析：**

**臨床情境：** 已知 HAE（Type I）病人，急性發作**腹痛**，CT 顯示**腸壁腫脹（bowel wall edema）**——這是 HAE **腹部發作（abdominal attack）** 的典型表現，機轉與皮膚/喉部血管性水腫相同，都是**bradykinin 過度堆積**造成的血管通透性增加，只是這次侵犯的是腸道黏膜下層，可造成劇烈腹痛、甚至類似腸阻塞的影像表現，是 HAE 病人常被誤診為外科急腹症的經典陷阱。

**急性發作的標準處置是 bradykinin 路徑的急性阻斷藥物：**

- **C（對，本題答案）** **Icatibant** 是 **bradykinin B2 receptor antagonist**，直接阻斷 bradykinin 在受體端的作用，是 HAE **急性發作（acute attack）** 的標準第一線治療藥物之一（與 C1-INH concentrate、ecallantide 並列急性期用藥），可快速緩解症狀，不論發作部位是皮膚、喉部或腹部皆適用。
- **A（錯）** **Danazol** 是雄性素衍生物，機轉是促進肝臟增加 C1-INH 的合成，屬於**長期預防性治療（long-term prophylaxis）**藥物，起效慢（需數天），**不適合用於急性發作**的處置。
- **B（錯）** **Tranexamic acid（Transamin）** 是抗纖溶藥物，效果較弱，主要用於**輕度預防性治療**（尤其兒童或危險性較低的病人），並非急性發作的標準治療，起效也不夠快。
- **D（次選，非首選）** **FFP（新鮮冷凍血漿）** 內含 C1-INH，理論上可作為急性期補充來源，但因為同時也含有其他補體成分與 kininogen（bradykinin 的前驅物質），**理論上有可能因提供更多受質而讓症狀短暫惡化**，因此只在**專一性 C1-INH concentrate 或 icatibant 等藥物無法取得時，作為次選的替代方案**，並非首選急性期治療。

**HAE 急性期 vs. 預防性治療速記：**

| 用途 | 藥物 |
|---|---|
| **急性發作（acute treatment）** | C1-INH concentrate、**Icatibant**、Ecallantide |
| 長期預防（prophylaxis） | Danazol（雄性素）、Tranexamic acid、C1-INH concentrate（長期使用） |
| 次選/替代（急性藥物無法取得時） | FFP |

### 26. 65 歲男性, 咳血腎衰竭到 ICU, creatinine 3 且 urine protein 2+, 下列何者錯誤?
- A ANCA vasculitis 要列入鑑別診斷
- B APS syndrome 要列入鑑別診斷
- C Anti-GBM 要列入鑑別診斷
- D 檢驗結果出來前不先用類固醇

**答案：D**

**解析：**

**臨床情境：** 65 歲男性，**咳血 + 急性腎衰竭（creatinine 3、蛋白尿 2+）**，需要 ICU 等級照護——這是典型的 **pulmonary-renal syndrome（肺腎症候群）**，代表同時有**瀰漫性肺泡出血（DAH）與快速進行性腎絲球腎炎（RPGN）**，屬於**內科急症**，死亡率與致殘率都高，必須快速鑑別診斷並**及早介入治療**。

- **A（正確敘述，非本題答案）** **ANCA-associated vasculitis（尤其 GPA、MPA）** 是 pulmonary-renal syndrome **最常見**的病因，務必列入鑑別診斷首位。
- **C（正確敘述，非本題答案）** **Anti-GBM disease（Goodpasture syndrome）** 是 pulmonary-renal syndrome 的經典病因之一，典型表現正是**肺泡出血合併 RPGN**，也必須列入鑑別。
- **B（正確敘述，非本題答案）** **Catastrophic APS** 雖然相對少見，但也可能造成**多重器官微血栓（thrombotic microangiopathy）**，包括肺部（造成出血/梗塞）與腎臟侵犯，在廣泛性鑑別診斷中仍應列入考量，尤其若病人合併其他血栓病史或抗磷脂抗體陽性。
- **D（錯誤敘述，本題答案）** 這句話違反了 **pulmonary-renal syndrome 的臨床處置原則**。由於此症候群**病程進展快速、死亡率高**（未及時治療可能短時間內進展成呼吸衰竭或末期腎病），標準做法是**在確診檢驗（ANCA、anti-GBM antibody、腎切片等）結果尚未出來前，就應該先經驗性給予高劑量類固醇（甚至 pulse therapy）**，必要時合併 plasmapheresis，以搶救器官功能，絕不能等到檢驗報告確認病因才開始治療——這與這份考卷第 4 題「懷疑 GCA 時要立即經驗性給予高劑量類固醇搶救視力」的邏輯完全一致：**器官功能岌岌可危時，經驗性治療優先於等待確診**。

**Pulmonary-renal syndrome 處置原則速記：**

| 步驟 | 內容 |
|---|---|
| 鑑別診斷 | AAV（GPA/MPA）、Anti-GBM disease、SLE、（少見：catastrophic APS） |
| 治療原則 | **不等檢驗結果**，立即經驗性高劑量類固醇 ± plasmapheresis |
| 目的 | 搶救肺臟/腎臟功能，降低死亡率與長期洗腎風險 |

### 27. 40 歲女性到門診, 他 20 歲有 asthma 病史有受到控制, 最近三個月開始惡化且血中 eosinophil counts 從 700 上升到 3000, 同時併有 nasal polyps 且抽血 IgE 12000。根據 2012 ACR/EULAR EGPA criteria, 下列何者正確?
- A 可以診斷 EGPA
- B 驗到 PR3 抗體可排除 EGPA
- C 要進一步問診、病例回顧和理學檢查
- D 有血尿可排除 EGPA

**建議答案：C（與使用者原答案 A 不同，已於下方說明判斷依據）**

**解析：**

**臨床情境重新檢視：** 40 歲女性，20 歲起氣喘（控制良好），近 3 個月惡化，**eosinophil 從 700 上升到 3000**，合併 **nasal polyps**，**IgE 12000**——這些表現符合 EGPA **前驅期（asthma、鼻息肉）+ 嗜酸性球浸潤期（eosinophilia）**的特徵，但**目前病歷中並未提供任何血管炎相關的證據**（例如：周邊神經病變、紫斑、肺部浸潤影像、心臟侵犯、腎絲球腎炎、切片證實血管炎，或 ANCA 陽性等）。

- **C（較佳答案）** 不論是 **1990 ACR 分類標準**或 **2022 ACR/EULAR EGPA 分類標準**，這些「分類標準」的設計前提都是**先確立病人已經是血管炎（vasculitis）診斷**，才能套用權重積分系統去分類是否為 EGPA——換句話說，分類標準是用來**在已知血管炎的病人中，區分是哪一種血管炎**，而不是用來從零開始「診斷」血管炎本身。這位病人目前只呈現 EGPA 病程中的**前驅期與嗜酸性球期**特徵，**尚未有任何血管炎期（vasculitic phase）的證據**。因此下一步應該是**更完整的問診（是否有周邊神經症狀、皮膚病灶、腹痛）、病歷回顧、理學檢查（是否有 purpura、mononeuritis 徵象），必要時安排影像與 ANCA 檢測**，先確認是否真的進入血管炎期，才能進一步套用分類標準做診斷。
- **A（可能過早）** 光憑「氣喘 + 嗜酸性球增多 + 鼻息肉 + IgE 高」就直接宣稱「可以診斷 EGPA」，忽略了 EGPA 分類標準中**必須先有血管炎證據**這個前提，屬於過早下診斷，也可能誤診其他嗜酸性疾病（如單純的嚴重氣喘合併鼻息肉，或需要排除的 hypereosinophilic syndrome）。
- **B（錯）** **PR3-ANCA（c-ANCA）陽性並不能排除 EGPA**——雖然 EGPA 較常見的是 **MPO-ANCA（p-ANCA）陽性（約 40%）**，但仍有少數 EGPA 病人可以是 PR3-ANCA 陽性，或兩者皆陰性（ANCA-negative EGPA，常以心臟侵犯為主的表型），ANCA 亞型不能用來排除診斷。
- **D（錯）** 血尿雖然在 EGPA 較少見（相對於 GPA/MPA），但**並非排除診斷的依據**——EGPA 仍有少數病人可以有腎臟侵犯（包括血尿、蛋白尿），不能因為出現血尿就排除 EGPA。

**觀念重點：** 血管炎分類標準（無論哪個疾病）都預設「已經確立血管炎診斷」為前提，臨床上遇到只有前驅期/嗜酸性期表現、尚無血管炎證據的病人，正確步驟是**先完整評估是否已進入血管炎期**，而非急著套公式診斷。

**⚠️ 提醒：** 此答案為根據分類標準邏輯的推論判斷，與作答者原選項（A）不同，若有確切公布答案可再核對確認。

### 28. 15 歲男性發燒 39.2 度腹痛去急診觀察 2 天, 抽血發現 CRP 和 SAA 高。病患兩年前曾因無菌性腹膜炎接受過手術。下列何者是最可能的診斷?
- A FMF
- B AOSD
- C CAPS
- D TRAPS

**答案：A（FMF）**

**解析：**

**Familial Mediterranean Fever (FMF)** 是體染色體隱性遺傳的自體發炎疾病（**MEFV 基因**突變），好發於**地中海沿岸族群**（Sephardic 猶太人、亞美尼亞人、土耳其人、阿拉伯人等），核心臨床特徵是**反覆發作、短暫（12–72 小時）的發燒合併漿膜炎（serositis）**：

- **腹膜炎（peritonitis）**：最常見的漿膜炎表現，發作時劇烈腹痛，臨床上常與急性闌尾炎或其他外科急腹症難以區分，因此**許多 FMF 病人在確診前，因反覆腹痛被誤診為外科急症而接受手術（如闌尾切除術）**，術中/術後病理常發現只是**無菌性腹膜炎（aseptic/sterile peritonitis）**，找不到感染源——這正是這題「兩年前曾因無菌性腹膜炎接受過手術」這個病史所指向的**經典 FMF 教學提示**。
- **胸膜炎（pleuritis）**：另一常見漿膜炎表現。
- 其他特徵：**關節炎**（常單一大關節）、**丹毒樣紅斑（erysipelas-like erythema，常見於小腿/足踝）**。
- **抽血 CRP、SAA（serum amyloid A）顯著上升**：SAA 特別重要，因為 FMF 病人若長期未控制，**SAA 持續慢性升高**可能導致 **AA amyloidosis（澱粉樣變性）**，是 FMF 最嚴重的長期併發症（尤其侵犯腎臟），這也是治療 FMF 需要規律使用 **colchicine** 的重要理由（除了預防急性發作，也預防澱粉樣變性）。

**逐一排除其他選項：**

- **B（錯）** AOSD（成人史迪爾氏症）典型表現是**每日一次的高燒（quotidian fever）、鮭魚色紅疹（salmon-colored rash）、關節炎、咽喉痛、淋巴結腫大**，並非以反覆短暫發作的腹膜炎為特徵，且好發年齡通常較 FMF 稍長，此病人 15 歲有反覆腹膜炎手術史，較不符合。
- **C（錯）** CAPS（cryopyrin-associated periodic syndrome，NLRP3 突變）典型表現是**遇冷誘發蕁麻疹、感音性聽力喪失、額頭突出（frontal bossing）**，較少以腹膜炎/腹痛為主要表現。
- **D（錯）** TRAPS（TNFRSF1A 突變）的發燒發作時間**較長（常持續 1–3 週）**，合併**遊走性肌痛、眼眶周圍水腫**，與 FMF 短暫發作（12–72小時）合併腹膜炎的型態不同。

**FMF 核心速記：**

| 項目 | 內容 |
|---|---|
| 基因 | MEFV（體染色體隱性遺傳） |
| 好發族群 | 地中海沿岸族群 |
| 典型發作 | 短暫（12–72h）發燒 + 漿膜炎（腹膜炎最常見） |
| 常見誤診 | 因反覆腹痛被誤認為外科急腹症而手術，術後常發現無菌性腹膜炎 |
| 長期併發症 | AA amyloidosis（SAA 持續上升是重要警訊） |
| 治療 | Colchicine（預防發作 + 預防澱粉樣變性） |

### 29. 6 個月大的嬰兒, 皮膚反覆蕁麻疹發作且會遇冷誘發, 且有看到 frontal bossing, 下列何者是最可能的基因變異?
- A MEFV
- B MVK
- C NLRP3
- D TNFRSF1A

**答案：C（NLRP3）**

**解析：**

**臨床情境：** 6 個月大嬰兒，**反覆蕁麻疹發作、遇冷誘發（cold-induced urticaria）**，合併 **frontal bossing（額頭突出）**——這是 **CAPS（Cryopyrin-Associated Periodic Syndrome）** 光譜疾病的經典表現，致病基因是 **NLRP3**（編碼 cryopyrin 蛋白，是 NLRP3 inflammasome 的核心組成分子）。

**CAPS 是一個光譜性疾病，由輕到重分為三個亞型：**

| 亞型 | 嚴重度 | 特徵 |
|---|---|---|
| FCAS（Familial cold autoinflammatory syndrome） | 最輕 | 遇冷誘發蕁麻疹、發燒、關節痛 |
| MWS（Muckle-Wells syndrome） | 中等 | 蕁麻疹 + 感音性聽力喪失 + 可能發展 amyloidosis |
| **NOMID / CINCA**（Neonatal-onset multisystem inflammatory disease） | **最重** | 蕁麻疹 + **frontal bossing（額頭突出）**、慢性腦膜炎、關節病變（如 patella overgrowth）、視力損傷 |

這位嬰兒同時有**遇冷誘發蕁麻疹**與**frontal bossing**，frontal bossing 是 CAPS 光譜中**較嚴重表型（NOMID/CINCA）**的特徵性顱顏骨骼變化，機轉與慢性發炎造成的顱骨過度骨化有關。無論是哪個亞型，**共同的致病基因都是 NLRP3**，這也是 NLRP3 inflammasome 活化後大量釋放 **IL-1β** 驅動全身發炎的機轉基礎，因此 CAPS 病人對 **anti-IL-1 治療（如 anakinra、canakinumab、rilonacept）反應極佳**，是治療首選。

**逐一排除其他選項：**

- **A（錯）** MEFV 是 **FMF** 的致病基因，典型表現是反覆短暫發燒合併漿膜炎（腹膜炎為主），與遇冷誘發蕁麻疹、frontal bossing 無關（呼應上一題）。
- **B（錯）** MVK 是 **MKD（Mevalonate kinase deficiency，又稱 Hyper-IgD syndrome, HIDS）** 的致病基因，典型表現是發燒合併淋巴結腫大、腹痛、皮疹，並非以遇冷誘發蕁麻疹為特徵。
- **D（錯）** TNFRSF1A 是 **TRAPS** 的致病基因，典型表現是**較長時間（1–3週）的發燒發作，合併遊走性肌痛、眼眶周圍水腫**，與此病人的表現型態不同。

**兒童自體發炎疾病基因速記：**

| 疾病 | 基因 | 招牌表現 |
|---|---|---|
| FMF | MEFV | 短暫發燒 + 腹膜炎 |
| MKD (HIDS) | MVK | 發燒 + 淋巴結腫大 + 腹痛 |
| CAPS (FCAS/MWS/NOMID) | **NLRP3** | **遇冷誘發蕁麻疹**、聽損（MWS）、**frontal bossing**（NOMID） |
| TRAPS | TNFRSF1A | 長時間發燒 + 遊走性肌痛 + 眼眶水腫 |

### 30. 下列機轉何者錯誤?
- A Mepolizumab 抑制 IL-4 receptor
- B Omalizumab 抑制 IgE
- C Tezepelumab 抑制 TSLP
- D Benralizumab 抑制 IL-5

**答案：A 與 D 皆錯誤（此題陷阱在於同時考驗 ligand vs. receptor 的精確區分）**

**解析：**

這題呼應風專考卷第 1 題同樣的核心觀念——**藥物到底是阻斷 ligand（細胞激素本身）還是 receptor（受體）**，是過敏免疫科最愛考的細節陷阱：

- **A（錯誤，本題答案之一）** **Mepolizumab** 抑制的是 **IL-5（ligand 本身）**，並非 IL-4 receptor。IL-4 receptor 是 dupilumab 的標靶，這裡完全張冠李戴——Mepolizumab 與 IL-4/IL-4R 訊息路徑無關。
- **D（錯誤，本題答案之一）** **Benralizumab** 抑制的是 **IL-5 receptor α (IL-5Rα)**，也就是**受體**，而非 IL-5 本身（ligand）。Benralizumab 結合 IL-5Rα 後，除了阻斷訊息傳遞，還會透過 **ADCC（antibody-dependent cell-mediated cytotoxicity）** 機轉，直接清除表現 IL-5Rα 的嗜酸性球與嗜鹼性球，這是它與純粹阻斷 ligand 的 mepolizumab 在藥理機轉上的關鍵差異（也是為什麼 benralizumab 降低周邊血 eosinophil 的效果特別顯著、幾乎能清空）。把它簡化寫成「抑制 IL-5」而不是「抑制 IL-5 receptor」，忽略了這個機轉上的本質差異，因此也是錯誤敘述。
- **B（正確，非本題答案）** **Omalizumab** 結合游離的 **IgE**，阻止 IgE 與肥大細胞/嗜鹼性球表面的 FcεRI 結合，是抗 IgE 治療的經典代表。
- **C（正確，非本題答案）** **Tezepelumab** 抑制 **TSLP（thymic stromal lymphopoietin）**，作用在 Th2 發炎路徑的**上游**（epithelial-derived alarmin），機轉與其他下游細胞激素（IL-4/5/13）阻斷劑不同。

**IL-5 pathway 藥物 ligand vs. receptor 速記（呼應風專第1題）：**

| 藥物 | 標靶 | Ligand 或 Receptor | 額外機轉 |
|---|---|---|---|
| Mepolizumab / Reslizumab | IL-5 | **Ligand** | — |
| Benralizumab | IL-5Rα | **Receptor** | 合併 ADCC，清除 eosinophil/basophil |
| Omalizumab | IgE | Ligand | — |
| Tezepelumab | TSLP | Ligand（上游 alarmin） | — |

### 31. 關於氣喘的敘述何者正確?
- A Anti-histamine 可抑制氣喘發作
- B Salbutamol 是 LABA, 一天使用一次
- C Tiotropium 為長效的 LAMA, 抑制氣管平滑肌的 M3 receptor
- D Formoterol 為短效 beta-2 agonist, 不可與 ICS 併用

**答案：C**

**解析：**

- **C（對，本題答案）** **Tiotropium** 是**長效型 muscarinic antagonist（LAMA）**，透過阻斷氣管平滑肌上的 **M3 muscarinic receptor**，抑制乙醯膽鹼（acetylcholine）誘發的支氣管收縮，達到支氣管擴張效果。在氣喘治療中，tiotropium 常作為**中重度氣喘、ICS+LABA 控制仍不理想時的 add-on therapy**（triple therapy: ICS + LABA + LAMA），也是 COPD 治療的重要骨幹藥物之一。
- **A（錯）** **Anti-histamine（抗組織胺）** 主要透過阻斷 H1 receptor，用於**過敏性鼻炎、蕁麻疹**等 IgE 媒介的過敏反應，但**對氣喘的支氣管收縮機轉幾乎沒有預防或治療效果**，並非氣喘控制用藥。
- **B（錯）** **Salbutamol（albuterol）** 是**短效型 beta-2 agonist（SABA）**，而非 LABA，作用快但**持續時間短（約 4–6 小時）**，臨床上是**需要時使用（PRN）作為急救緩解用藥**，並非每天固定使用一次的長效控制藥物。
- **D（錯）** **Formoterol** 恰好相反，是**長效型 beta-2 agonist（LABA）**，且其特點是**起效快（onset 快，類似 SABA）合併長效（持續 12 小時以上）**，因此可用於**急性緩解與長期控制雙重角色**。臨床上 formoterol **常與 ICS（如 budesonide）併用**，形成 ICS/formoterol 複方吸入劑（如 Symbicort），甚至發展出 **SMART/MART therapy（single maintenance and reliever therapy）**，允許病人用同一支吸入劑同時作為每日控制與急性發作時的緩解用藥，這是現行 GINA 氣喘治療指引的重要治療策略，與這句話「不可與 ICS 併用」完全相反。

**氣喘常用支氣管擴張劑分類速記：**

| 藥物類別 | 代表藥物 | 作用機轉 | 使用方式 |
|---|---|---|---|
| SABA | Salbutamol | β2 agonist | PRN 急救緩解 |
| LABA | Salmeterol、**Formoterol** | β2 agonist | 長期控制（Formoterol 起效快，可兼作緩解） |
| LAMA | **Tiotropium** | 阻斷 M3 receptor | 長期控制，add-on therapy |
| ICS/LABA 複方 | Budesonide/Formoterol | — | SMART/MART therapy 核心藥物 |

### 39. Recurrent angioedema 的診斷過程, 何者錯誤？
- A Recurrent angioedema 合併 urticaria, 最可能為肥大細胞相關機轉
- B C4、C1 功能皆正常, 家族中有血管性水腫病史, 故可能得診斷為 HAE with normal C1-INH
- C C4、C1 inhibitor 抗原皆低, C1q 正常, 故可能診斷為 type II HAE
- D 未服用 ACEi、C4 正常、沒有家族史, 可以根據抗組織胺反應性區分為 mast cell mediated or non-histaminergic type

**答案：C**

**解析：**

- **A（正確敘述，非本題答案）** **Angioedema 合併 urticaria（蕁麻疹）** 是重要的鑑別線索——**Bradykinin-mediated angioedema（如 HAE）本身不會合併蕁麻疹**，因為 bradykinin 造成的是單純血管通透性增加、組織水腫，不會誘發 mast cell 去顆粒化產生的風疹塊（wheal）。因此當 angioedema **合併** urticaria 時，最可能的機轉是**肥大細胞活化（mast cell-mediated / histaminergic）**，如過敏反應、慢性自發性蕁麻疹合併血管性水腫等。這是區分 histaminergic vs. bradykinin-mediated angioedema 最重要的臨床線索之一。
- **B（正確敘述，非本題答案）** **HAE with normal C1-INH（Type III HAE）** 的定義正是 **C4、C1-INH 抗原量與功能皆正常**，但仍有反覆家族性血管性水腫病史，致病機轉與 C1-INH 無關（如 Factor XII、plasminogen、angiopoietin-1、kininogen-1 等基因變異），好發於女性、與雌激素相關。此敘述完全符合 Type III HAE 的定義。
- **C（錯誤敘述，本題答案）** 這句話的分型**判斷錯誤**。回顧第 24 題討論過的分型邏輯：
  - **C1-INH 抗原量本身就偏低** → **Type I HAE**（quantitative deficiency，蛋白質量不足）
  - **C1-INH 抗原量正常/偏高，但功能檢測異常** → **Type II HAE**（qualitative defect，蛋白質功能異常但量正常）

  這句話描述的是「**C1 inhibitor 抗原皆低**」，也就是**抗原量不足**，這個組合對應的應該是 **Type I HAE**，而不是 Type II。此外，**C1q 正常**這個線索是用來區分 **hereditary（遺傳性，C1q 通常正常）vs. acquired（後天性，C1q 通常因被消耗而偏低，常見於淋巴增生疾病或自體抗體）angioedema**，並非用來區分 Type I 與 Type II。這句話把「抗原低」錯配到「Type II」，是這題的錯誤所在。
- **D（正確敘述，非本題答案）** 當病人**未使用 ACEi**（排除藥物誘發性 bradykinin-mediated angioedema）、**C4 正常**（較不支持典型遺傳性 HAE Type I/II，因為這兩型 C4 通常持續偏低）、**無家族史**（較不支持遺傳性病因），臨床上確實可以嘗試**以抗組織胺治療反應性**作為輔助鑑別工具——若對抗組織胺**有反應**，較支持 **mast cell-mediated（histaminergic）type**；若**無反應**，則需考慮 **non-histaminergic（如 bradykinin-mediated）type**，這是合理且常用的臨床鑑別思路。

**HAE 分型 vs. C1-INH 檢驗結果對照速記：**

| 分型 | C1-INH 抗原量 | C1-INH 功能 | C4 | C1q |
|---|---|---|---|---|
| Type I HAE | **低** | 低 | 低 | 正常 |
| Type II HAE | **正常/高** | 低 | 低 | 正常 |
| Type III HAE (normal C1-INH) | 正常 | 正常 | 正常 | 正常 |
| Acquired angioedema | 低（或正常但功能低） | 低 | 低 | **低（關鍵鑑別點）** |

### 40. 根據 IgG4RD 2019 criteria, 要先排除疾病下列何者為錯？
- A ANCA
- B Lymphoproliferative Castleman disease
- C Cryoglobulinemia
- D Non-necrotizing vasculitis

**答案：B（但 D 用詞也有疑慮，信心度中等，本地無原文可逐字核對）**

**解析：**

**2019 ACR/EULAR IgG4-RD classification criteria** 設計邏輯是先符合「entry criterion」，再檢查是否觸犯**排除條件（exclusion criteria）**——排除條件分為四大類：**臨床（clinical）、血清學（serologic）、影像（radiologic）、病理（pathology）**，目的是排除容易與 IgG4-RD 混淆的疾病。

- **A（是排除條件，非本題答案）** **ANCA 陽性**是血清學排除條件之一——因為 ANCA 陽性應優先考慮 AAV（GPA/MPA/EGPA），而非 IgG4-RD。
- **C（是排除條件，非本題答案）** **Cryoglobulinemia** 也是血清學排除條件之一，因為冷凝球蛋白血症本身可以造成類似 IgG4-RD 的臨床表現（如腺體腫大、血管炎），須優先排除。
- **B（較可能是本題答案）** **Castleman disease（尤其 multicentric Castleman disease）** 雖然是臨床上與 IgG4-RD **重要的鑑別診斷**（兩者都可能有淋巴結腫大、高丙種球蛋白血症等重疊表現），但它**並非 2019 ACR/EULAR 分類標準中正式列出的排除條件項目名稱**——排除條件是以**具體的臨床/血清學/影像/病理發現**（如發燒、ANCA陽性、cryoglobulinemia、necrotizing vasculitis 等）條列，而非直接列出「某個疾病名稱」作為排除項目。因此若題目問「下列何者為排除條件之一」，Castleman disease 本身**不是**分類標準中列出的正式排除條件項目，是這四個選項中最不符合的。
- **D（用詞可能有疑慮）** 病理排除條件中列出的是 **「necrotizing vasculitis（壞死性血管炎）」**，而不是「non-necrotizing vasculitis（非壞死性血管炎）」。事實上 IgG4-RD 本身的招牌病理特徵之一是 **obliterative phlebitis（阻塞性靜脈炎）**，這是一種**非壞死性**的血管病變，屬於**支持 IgG4-RD 診斷的正面病理特徵（inclusion criteria 的一部分），而非排除條件**。若題目原文確實寫的是「non-necrotizing vasculitis」，這個敘述本身也有問題——真正的排除條件應該是「necrotizing」而非「non-necrotizing」。

**⚠️ 提醒：** 這題因為兩個選項（B、D）都可能有問題，且無 2019 ACR/EULAR IgG4-RD 分類標準原文可逐字核對，若之後找到確切公布答案可再核對確認。

### 41. 根據下圖電腦斷層照片, 應診斷下列何者？
- A Tubulointerstitial nephritis
- B IgG4 related aortitis
- C Autoimmune pancreatitis
- D Retroperitoneal fibrosis

**影像描述：** 腎臟層次軸位 CT，箭頭指向腹主動脈兩側，可見主動脈周圍軟組織如披風狀包覆（periaortic soft tissue mantle/cuffing）。

**建議答案：D（Retroperitoneal fibrosis，與使用者原答案 B 不同，已於下方說明判讀依據）**

**解析：**

**IgG4-related disease** 侵犯主動脈周圍/腹膜後區域時，依病理侵犯範圍可分為兩種相關但影像表現不同的疾病：

| | IgG4-related aortitis | Retroperitoneal fibrosis (IgG4-related periaortitis/RPF) |
|---|---|---|
| 侵犯範圍 | 侷限在**主動脈血管壁本身**（尤其 adventitia）增厚 | 病灶**延伸超出血管壁**，形成包覆主動脈及周邊腹膜後結構的**軟組織腫塊/披風狀增厚（rind/mantle-like soft tissue）** |
| 典型影像 | 主動脈壁本身增厚，管腔周圍相對乾淨 | **主動脈周圍軟組織如披風狀包覆**，常見於腎動脈以下腹主動脈層次，可延伸包覆輸尿管，造成**hydronephrosis（腎水腫）** |
| 臨床影響 | 可能造成動脈瘤形成或破裂風險 | 因包覆輸尿管，常見表現為阻塞性腎病變 |

這張影像中，箭頭指向的軟組織位於**主動脈兩側、呈現包覆狀分布**，且切面位置在**腎臟層次**（接近輸尿管上段走行處），這正是 RPF 影像上最經典、最常被考的呈現方式——軟組織像披風一樣包住主動脈前外側（**通常不侵犯主動脈後壁**，這也是 RPF 影像上一個常被強調的特徵）。

- **D（判讀為正確答案）** 影像上主動脈周圍軟組織包覆的型態、加上切面位置在腎臟/輸尿管層次，是 **retroperitoneal fibrosis** 的典型呈現。
- **B（原答案，但判讀較不精確）** 若是 aortitis，影像上應該呈現**主動脈血管壁本身**的增厚/強化，而非這種延伸到血管壁外、形成周邊軟組織腫塊的型態。
- **A（錯）** Tubulointerstitial nephritis 是腎臟實質本身的病變，影像上會呈現腎臟皮質/髓質的異常，而非主動脈周圍軟組織病灶。
- **C（錯）** Autoimmune pancreatitis 病灶位置在胰臟，與此影像的解剖位置（主動脈周圍、腎臟層次）不符。

**⚠️ 提醒：** 此判讀根據影像上「主動脈周圍軟組織包覆、位置在腎臟/輸尿管層次」的典型 RPF 特徵，與原答案 B 不同，若有老師公布的正式答案可再核對確認。

### 42. 60 歲男性, 過去病史有痛風。6 週前因為急性痛風發作, 就醫接受 Diclofenac 25mg TID 10 days 和 colchicine 6 週, 四週前開始服用 allopurinol QD。三天前出現小便疼痛, 就醫接受 Baktar 治療, 就開始出現全身紅疹。下列敘述何者正確？
- A 造成過敏反應的藥物就是 Baktar
- B 為 IgE-mediated
- C 是 type IVc delayed reaction
- D 應該使用高劑量類固醇來治療

**臨床情境時間軸整理：**

- 6 週前：急性痛風發作，用 diclofenac 10 天（現已停用約 5 週）+ colchicine 用了 6 週（現在剛好療程結束）
- **4 週前**：開始服用 **allopurinol**（每天持續使用中）
- 3 天前：出現小便疼痛，開始服用 **Baktar（sulfamethoxazole/trimethoprim）**
- 現在：全身紅疹

**答案：C**

**解析：**

- **A（錯，本題最大陷阱）** 直覺上會認為「最近才開始吃的藥＝兇手」，也就是 3 天前才開始的 **Baktar**。但這題真正想考的是 **allopurinol hypersensitivity syndrome（AHS）**的經典時間軸特徵——**allopurinol 引起的嚴重皮膚藥物反應（SCAR，包括 DRESS、SJS/TEN）typical onset 是用藥後 2–6 週**，這位病人**恰好在 allopurinol 使用滿 4 週時**出現全身紅疹，時間點完全吻合 AHS 的典型延遲發作模式，遠比才吃 3 天的抗生素更符合致敏所需的時間。這也呼應台灣族群 **HLA-B*58:01 與 allopurinol SCAR 高度相關**的重要臨床議題（台灣是 allopurinol SCAR 的高風險地區之一），是過敏免疫科極愛考的經典陷阱——**「最近開始用的藥不一定是兇手，要看藥物本身典型的致敏潛伏期」**。
- **B（錯）** 這種**全身性、延遲發作（用藥後數週才出現）**的藥物疹型態，並非 **IgE-mediated（Type I，立即型）**反應——IgE-mediated 反應通常在**用藥後數分鐘到數小時內**發生（如蕁麻疹、anaphylaxis）。Allopurinol SCAR 的機轉是 **T cell-mediated（Type IV，delayed-type）**過敏反應，與 HLA-B*58:01 呈現藥物-胜肽複合物給細胞毒性 T 細胞辨識有關。
- **C（對，本題答案）** Allopurinol 引起的嚴重皮膚藥物反應（尤其進展為 SJS/TEN 這類角質細胞大量凋亡的型態）屬於 **Type IVc delayed reaction**——機轉核心是 **cytotoxic T lymphocyte (CTL) 與 NK cell** 透過釋放 **granulysin、perforin/granzyme B、FasL** 等細胞毒性分子，直接誘導角質細胞凋亡（keratinocyte apoptosis），與 Type IVa（monocyte 活化為主）、IVb（eosinophil 為主，如典型 DRESS）、IVd（neutrophil 為主，如 AGEP）的免疫機轉不同。
- **D（不夠精確，非本題最佳答案）** 是否使用**高劑量類固醇**取決於反應的**嚴重度分級**——若只是廣泛性 maculopapular rash，處置應先**立即停用可疑藥物（allopurinol）**、給予支持性治療與密切觀察是否進展；唯有進展成 **DRESS、SJS/TEN 等重症藥物疹（SCAR）**時，才會考慮全身性高劑量類固醇或其他免疫調節治療（如 IVIG、cyclosporine）。題目目前僅描述「全身紅疹」，尚未提供足夠嚴重度分級的資訊（如黏膜侵犯、發燒、器官侵犯、皮膚剝離），直接跳到「應該用高劑量類固醇」略嫌武斷，**停藥、找出致敏藥物才是第一步**。

**核心教學重點：** 遇到「最近開始用的藥」與「已使用一段時間、時間點恰好落在該藥物典型致敏潛伏期」的藥物同時存在時，**務必依照該藥物本身的典型 latency period（潛伏期）去判斷**，不要直覺認定「最新的藥」就是兇手——這是這題（以及許多藥物過敏考題）真正想測驗的臨床推理能力。

**Type IV 過敏反應細分速記：**

| 亞型 | 主要效應細胞 | 代表疾病 |
|---|---|---|
| IVa | Th1 / Monocyte | 接觸性皮膚炎 |
| IVb | Th2 / Eosinophil | Maculopapular exanthema、DRESS |
| **IVc** | **Cytotoxic T cell / NK cell** | **SJS/TEN、Fixed drug eruption** |
| IVd | Neutrophil | AGEP |

### 43. 患者因為 right knee infection, 接受 Oxacillin 2g Q6H + Ketorolac PRN, 十天後身上出現 itchy maculopapular rash without bullae formation, 打 Oxacillin 前有做 penicillin intradermal skin test (-), 下列敘述何者正確？
- A 是 IgE mediated
- B 是 type IV delay reaction
- C 因 penicillin intradermal skin test negative, 故非 oxacillin 過敏
- D Intradermal skin test 有 major antigenic determinant 和 minor antigenic determinant, 故所有因 β-lactam 過敏都可用此檢查來排除

**答案：B**

**解析：**

**先判斷過敏型態：** 病人使用 Oxacillin **十天後**才出現 **itchy maculopapular rash（無水泡）**——這種**延遲發作（用藥數天後才出現）、以斑丘疹為主、不合併蕁麻疹/血管性水腫**的型態，是典型的 **Type IV delayed hypersensitivity reaction（T cell-mediated）**，而非立即型的 IgE-mediated 反應。

- **B（對，本題答案）** 十天的延遲發作時間、加上 maculopapular rash（而非 urticaria/wheal）的皮疹型態，完全符合 **Type IV delayed reaction** 的臨床特徵——機轉是藥物（或其代謝產物）作為 hapten，與 T cell receptor 結合活化specific T cell，經過數天的致敏/活化過程後才在皮膚表現出來，與 IgE-mediated 立即型反應（通常在暴露後數分鐘至數小時內發生）在時間軸與皮疹型態上都截然不同。
- **A（錯）** IgE-mediated（Type I）反應應在**暴露後極短時間內（分鐘至數小時）**發生，典型表現是 **urticaria、angioedema、甚至 anaphylaxis**，與此病人「十天後才出現、無風疹塊」的臨床型態不符。
- **C（錯）** 這是這題的**核心陷阱**。**Penicillin intradermal skin test 只能偵測 IgE-mediated（Type I）立即型過敏反應**，對**Type IV delayed reaction 完全沒有偵測能力**——這種測試的設計原理就是評估肥大細胞/嗜鹼性球表面 IgE 是否會因抗原結合而活化，與 T cell 媒介的延遲型反應機轉無關。因此「skin test 陰性」只能說「病人對 penicillin 核心結構沒有 IgE-mediated 立即型過敏」，**不能**因此推論病人「對 oxacillin 沒有過敏」——尤其此病人的反應本來就是 Type IV，本來就不會被這種測試偵測到，這句話的邏輯推論是錯誤的。
- **D（錯）** Skin test 確實包含 **major antigenic determinant（penicilloyl-polylysine, PPL）** 與 **minor antigenic determinant mixture (MDM)**，但這個組合**只能評估對 penicillin 核心結構（core structure）的 IgE-mediated 過敏**，**無法涵蓋所有 β-lactam 過敏的情況**：
  1. 它只測 Type I（IgE-mediated），無法排除 Type II–IV 機轉的過敏反應。
  2. 不同 β-lactam（如 oxacillin、cephalosporin 等半合成青黴素）帶有**不同的側鏈（side chain）**，部分病人的過敏專一性是針對**側鏈結構**而非核心結構，這種 side-chain specific allergy **不會被標準 penicillin skin test 偵測到**。

  因此「所有 β-lactam 過敏都可用此檢查排除」是過度概化的錯誤敘述。

**Type I vs. Type IV 過敏反應與 Skin test 侷限性速記：**

| | Type I (IgE-mediated) | Type IV (T cell-mediated) |
|---|---|---|
| 發作時間 | 分鐘～數小時 | 數天 |
| 典型皮疹 | Urticaria、angioedema | Maculopapular rash |
| Penicillin skin test 可偵測？ | 可以 | **不行** |
| Skin test 侷限性 | 僅測 IgE-mediated、僅測 penicillin 核心結構，無法涵蓋 side-chain specific 或其他機轉的過敏 | — |

### 44. 55 歲女性, 因血尿而接受膀胱鏡檢查, 開始 30 分鐘左右發生休克狀態, 緊急處置與觀察後穩定出院, 隔天至風濕免疫科門診就醫, 請問下列敘述何者正確？
- A 抽 tryptase 來診斷
- B 是為 pseudoallergy
- C 可能是麻醉藥物過敏, 可藉 specific IgE 幫忙找過敏原
- D Basophil activation test 可以協助診斷

**建議答案：D（與使用者原答案 A 不同，已於下方說明判斷依據）**

**解析：**

**臨床情境的關鍵細節：** 病人是**術中（膀胱鏡檢查開始 30 分鐘）發生休克**，但**隔天**才到風濕免疫科門診——這個**時間差**是這題最重要的陷阱。

- **A（判斷不正確）** **Serum tryptase** 的診斷價值**高度依賴採檢時機**——tryptase 在肥大細胞活化後會快速上升，**約 1–2 小時達到高峰，並在 24 小時內逐漸回復到 baseline**。標準做法是在急性反應發生後**儘快（理想上 15 分鐘至 3 小時內）**抽血，並與**事後（症狀緩解至少 24 小時後）的 baseline tryptase** 做比較，兩者的差異（rise from baseline）才具有診斷意義。這位病人是**隔天**才來門診，此時再抽的 tryptase **已經錯過了急性期的診斷時間窗**，數值大概率已經回到 baseline，**無法用來回溯診斷「昨天那次休克是否為 mast cell activation/anaphylaxis」**——最多只能作為病人的「baseline tryptase」，供未來若再發作時比較之用，但**不能用來診斷已經發生過的這次事件**。
- **D（判斷為正確答案）** **Basophil activation test (BAT)** 相較於 tryptase，**對採檢時機的要求沒有那麼嚴苛**，可以在事件發生**數週後**才進行，仍能協助判斷病人的嗜鹼性球對可疑致敏原（如乳膠、特定藥物、局部麻醉劑等）是否有活化反應，是這位「隔天才來門診」病人身上**更適合、更具時效可行性**的診斷工具。
- **B（不夠精確，非本題答案）** 是否為 pseudoallergy（非 IgE-mediated 的直接肥大細胞活化，如某些藥物經由 MRGPRX2 途徑）目前資訊不足以判斷——膀胱鏡檢查過程中可能接觸的致敏原包括**乳膠手套/導管、局部麻醉劑、消毒液（如 chlorhexidine）**等，需要進一步病史與檢查釐清機轉，不能一開始就斷言是 pseudoallergy。
- **C（不夠精確）** 「麻醉藥物過敏」只是眾多可能致敏原之一（膀胱鏡檢查通常只需局部麻醉，不一定會用到全身麻醉藥物），且 **specific IgE 檢測對於術中過敏常見致敏原（如乳膠、chlorhexidine、部分 NMBA）的可及性與敏感度有限**，臨床上這類術中過敏的黃金標準通常是**皮膚測試（skin prick/intradermal test）**搭配病史還原，specific IgE 並非唯一或最佳工具。

**核心教學重點：** **Tryptase 是「時效性」檢驗，錯過急性期窗口就失去診斷意義；Basophil activation test 則沒有這麼嚴苛的時間限制**，這是鑑別這兩個檢查適用時機的關鍵。

**⚠️ 提醒：** 此答案與使用者原選項（A）不同，若有確切公布答案可再核對確認。

### 45. 下列敘述何者錯誤？
- A SCID 可用幹細胞移植治療
- B Selective IgA deficiency 可用 IVIG 治療
- C DiGeorge Syndrome 可用 MHC 相容腮腺移植治療
- D Wiskott-Aldrich syndrome 因會表現血小板低下, 故應切除脾臟治療

**答案：B**

**解析：**

- **A（正確敘述，非本題答案）** **SCID（Severe Combined Immunodeficiency）** 是免疫系統重症缺損疾病，**造血幹細胞移植（HSCT）**是目前唯一可治癒的標準治療，也是新生兒篩檢（NBS）納入 SCID 篩檢的重要理由——愈早診斷、愈早移植，預後愈好。
- **B（錯誤敘述，本題答案）** **Selective IgA deficiency（選擇性 IgA 缺乏症）病人應避免使用 IVIG（或任何含 IgA 的血液製劑）**，這是免疫學的重要安全守則：
  1. 大多數 IVIG 製劑中仍含有微量 IgA。
  2. Selective IgA deficiency 病人體內可能已產生 **anti-IgA 抗體**，一旦輸注含 IgA 的血液製劑（IVIG、血漿、全血），有誘發**嚴重過敏性休克（anaphylaxis）**的風險。
  3. 此外，單純的 selective IgA deficiency（若沒有合併其他抗體缺損，如 CVID）通常**不需要 IVIG 補充治療**——這類病人多數 IgG 濃度正常、能產生正常的特異性抗體反應，IVIG 主要保留給有明確抗體功能缺損（如 CVID、XLA）合併反覆感染的病人使用。

  因此「Selective IgA deficiency 可用 IVIG 治療」不僅**沒有必要**，甚至可能**造成危險**，是這題的錯誤敘述。
- **C（正確敘述，非本題答案）** **Complete DiGeorge syndrome**（胸腺完全發育不全）可考慮 **MHC 相容胸腺移植（thymus transplantation）**作為重建 T cell 功能的治療選項（此處若原文為「腮腺」應為謄寫/記憶時的筆誤，正確應為「胸腺」，因為胸腺才是 DiGeorge syndrome 的病灶所在，腮腺與此疾病無關）。
- **D（正確敘述，但臨床上需謹慎，非本題答案）** Wiskott-Aldrich syndrome (WAS) 的經典三聯症是**血小板低下（合併小顆血小板）、濕疹、反覆感染**。**脾臟切除**確實可以改善 WAS 病人的血小板低下與出血傾向（因為異常小顆血小板主要在脾臟被破壞清除），臨床上曾被用來處理嚴重出血問題；但因為 WAS 病人本身**已有免疫缺損**，脾臟切除後會**顯著增加莢膜菌感染（encapsulated bacteria）的風險**，臨床上需要合併終身抗生素預防，且**造血幹細胞移植才是根治性治療**，脾臟切除僅是特定情況下的症狀處置選項，並非常規建議。

**觀念小提醒：** IgA 缺乏症病人輸血/使用血液製劑前，務必詢問過敏史並考慮使用 **IgA-depleted 或 washed blood products**，避免誘發致命性過敏反應，這是臨床上非常重要的安全守則。

### 46. 下列免疫缺失跟病原體何者正確
- A Antibody: Strep. pneumoniae, Haemophilus influenzae
- B T cell: CMV, PJP
- C NK cell: HSV, EBV
- D Phagocyte: CMV, PJP

**答案：C（但 A、B 亦為正確配對，僅 D 為錯誤配對——此題性質較接近「找出錯誤選項」）**

**解析：**

先完整檢視這四個免疫缺損與病原體的配對，會發現 **A、B、C 三個都是教科書等級的正確配對**，只有 **D 是錯誤的**：

- **A（正確配對）** **Antibody deficiency（如 XLA、CVID）** 因為缺乏 opsonizing antibody，特別容易感染**有莢膜的細菌（encapsulated bacteria）**，經典代表就是 **Strep. pneumoniae、Haemophilus influenzae**（還有 Neisseria），這是免疫缺損最經典的配對之一。
- **B（正確配對）** **T cell deficiency（如 SCID、DiGeorge syndrome）** 因為喪失細胞媒介免疫，容易感染**病毒（如 CMV、EBV、HSV、VZV）、胞內病原體（如分枝桿菌）**，以及**伺機性黴菌感染，如 PJP（Pneumocystis jirovecii pneumonia）**，CMV 與 PJP 都是 T cell deficiency 的經典指標性感染。
- **C（正確配對，本題答案）** **NK cell deficiency** 因為喪失對抗病毒感染細胞的天然殺傷能力，特別容易出現**嚴重/瀰漫性的 herpesvirus 家族感染**，包括 **HSV、EBV**（以及 VZV、CMV），這也是教科書經典配對。
- **D（錯誤配對）** **Phagocyte deficiency（如 CGD、LAD）** 典型感染的病原體是 **catalase-positive 細菌（如 Staph. aureus、Serratia、Burkholderia）與黴菌（如 Aspergillus、Candida）**，而**不是 CMV 或 PJP**——CMV 是病毒，PJP 是機會性黴菌感染，兩者都屬於**T cell deficiency**典型的感染病原體，並非 phagocyte deficiency 的特徵性配對，這個選項把病原體張冠李戴到錯誤的免疫缺損類型。

**免疫缺損 vs. 典型病原體配對總表：**

| 免疫缺損類型 | 典型病原體 |
|---|---|
| Antibody deficiency | 有莢膜細菌（Strep. pneumoniae、H. influenzae、Neisseria） |
| T cell deficiency | 病毒（CMV、EBV、HSV）、胞內菌（分枝桿菌）、PJP |
| NK cell deficiency | Herpesvirus 家族（HSV、EBV、VZV、CMV） |
| Phagocyte deficiency | Catalase(+) 細菌（Staph. aureus、Serratia、Burkholderia）、黴菌（Aspergillus、Candida） |

### 47. Fever pattern 與疾病的配對（圖片題）

**圖片來源：** Kelly's Textbook of Rheumatology, Figure 97-1 — Characteristic patterns of body temperature during inflammatory attacks in the familial autoinflammatory syndromes。

**解析：**

這張圖的核心教學重點是——**不同自體發炎疾病，發燒發作的「持續時間」與「型態」各有特徵**，是鑑別診斷的重要線索之一：

| 疾病 | 發燒持續時間 | 曲線型態特徵 | 合併症狀 |

| **MWS** | 中等（約 5 天） | 類似 FMF 但略長 | Chills、arthralgia、erythematous rash、conjunctivitis |

**這張圖最重要的考試重點整理：**

1. **持續時間光譜**：FCAS（數小時）< FMF（1–3天）< MWS（約5天）< HIDS（4–10天，鋸齒狀）< TRAPS（最長，可達數週）
2. **誘發因子**：只有 **FCAS** 圖上特別標示「cold exposure」這個明確的誘發因子，是遇冷誘發蕁麻疹/發燒最直接的圖像佐證（呼應第 29 題）。
3. **CINCA/NOMID 是特例**：這是唯一一個發燒型態被註記為「不具鑑別意義」的疾病，因為它的**招牌臨床特徵是慢性、持續性的關節變形與腦膜炎**，而非特定的發燒曲線型態——這也是這張圖表最容易被考的「陷阱」重點：**不要只靠發燒曲線去診斷 CINCA/NOMID，要看關節與神經系統的慢性侵犯**。
4. **HIDS 的鋸齒狀曲線**：與其他疾病的單一尖峰不同，HIDS 常呈現**發作中還有多次小波動的鋸齒狀外觀**，是圖形辨識上的另一個特徵。

### 48. 下列何者非合適的治療選項
- A TRAPS: steroid, etanercept
- B CAPS: steroid, rilonacept
- C FMF: colchicine, anakinra
- D MKD: steroid, infliximab

**建議答案：D（與使用者原答案 C 不同，已於下方說明判斷依據）**

**解析：**

逐一檢視四個疾病的標準治療選項：

- **A（合適的治療，非本題答案）** **TRAPS**：急性期可用**類固醇**緩解症狀；**Etanercept**（TNF soluble receptor fusion protein）因其藥理結構與 TRAPS 的致病機轉（TNFR1 shedding 缺陷）有理論上的契合性，歷史上曾是 TRAPS 生物製劑治療的重要選項之一（雖然現在 anti-IL-1，如 canakinumab，已逐漸成為第一線生物製劑，但 etanercept 仍是被接受、有實證支持的治療選項）。
- **B（合適的治療，非本題答案）** **CAPS**：急性期可用類固醇；**Rilonacept**（IL-1 trap，同時中和 IL-1α 與 IL-1β）已獲**FDA 核准專門用於治療 CAPS（FCAS、MWS）**，是明確合適的治療選擇。
- **C（合適的治療，非本題答案）** **FMF**：**Colchicine 是標準第一線治療**；對 colchicine 反應不佳或無法耐受的病人，**Anakinra（anti-IL-1）** 是已建立實證的次線治療選項，這個配對是合適的。
- **D（不合適的治療，本題答案）** **MKD（HIDS）**：急性期可用類固醇緩解症狀沒問題，但 **infliximab（單株抗體類 anti-TNF）並非 MKD 的標準治療選項**。MKD 的生物製劑治療主流是 **anti-IL-1（anakinra、canakinumab）**，這與 IL-1β 過度分泌的核心機轉直接對應；anti-TNF 藥物（尤其單株抗體類，如 infliximab）在 MKD 的證據有限、效果不一致，並非被廣泛建議的治療選項，因此這個配對是四者中最不合適的。

**自體發炎疾病生物製劑治療速記：**

| 疾病 | 首選/標準治療 | 備註 |
|---|---|---|
| FMF | Colchicine（首選）→ Anakinra（次線） | — |
| CAPS | Anti-IL-1（Anakinra、Canakinumab、**Rilonacept**） | Rilonacept 專為 CAPS 核准 |
| TRAPS | Anti-IL-1（現多為首選）、Etanercept（歷史選項，仍可用） | 避免單株抗體類 anti-TNF |
| MKD (HIDS) | **Anti-IL-1（Anakinra、Canakinumab）** | Anti-TNF（尤其單株抗體如 infliximab）證據有限，非標準選項 |

**⚠️ 提醒：** 此答案與使用者原選項（C）不同，若有確切公布答案可再核對確認。

### 49. 2019 ACR/EULAR IgG4-RD criteria, 何者為 exclusion criteria?
- A Serum IgG4 count 大於正常值五倍
- B Eosinophilia
- C PR3-ANCA positive
- D IgG4+/IgG>40%

**答案：C（正確；B 同樣也是排除條件之一）**

**解析：**

延續第 40 題的討論，**2019 ACR/EULAR IgG4-RD classification criteria** 的排除條件分為四大類（臨床、血清學、影像、病理），目的是排除容易與 IgG4-RD 混淆的疾病：

- **C（對，本題答案）** **PR3-ANCA（c-ANCA）陽性**是**血清學排除條件**之一——ANCA 陽性應優先考慮 **AAV（GPA/MPA/EGPA）**，而非 IgG4-RD，這點呼應第 40 題已確認的觀念。
- **B（也是排除條件，補充提醒）** **Unexplained peripheral eosinophilia（不明原因的周邊嗜酸性球增多）** 同樣被列在**臨床/血清學排除條件**中——因為顯著嗜酸性球增多會讓人優先考慮 **EGPA、hypereosinophilic syndrome** 等其他疾病，而非典型 IgG4-RD（IgG4-RD 本身雖然偶爾也可以有輕度嗜酸性球增多，但顯著上升應優先排除其他診斷）。
- **A（錯，屬於納入計分項目而非排除條件）** **Serum IgG4 > 正常值 5 倍**其實是**血清學納入計分項目（inclusion/weighted scoring criteria）**中的高分項目——IgG4 濃度越高，對「符合 IgG4-RD」的支持力道越強，屬於**加分**而非排除。
- **D（錯，屬於病理計分項目而非排除條件）** **IgG4+/IgG 比值 > 40%**是**病理納入計分項目**之一（搭配 IgG4+ plasma cell 數量），是支持 IgG4-RD 診斷的正面病理特徵，同樣屬於**加分**而非排除條件。

**排除 vs. 納入計分項目的核心區別速記：**

| 類別 | 功能 | 舉例 |
|---|---|---|
| **排除條件（exclusion）** | 出現任一項就**不能**分類為 IgG4-RD | ANCA 陽性、Cryoglobulinemia、不明原因嗜酸性球增多、necrotizing vasculitis |
| **納入計分項目（inclusion/weighted scoring）** | 分數加總 ≥ threshold 才能分類為 IgG4-RD | 高 Serum IgG4、IgG4+/IgG > 40%、特定影像/病理特徵 |

### 50. 以下何種 IgG4 病理表現分數最高
- A IgG4/IgG 20%
- B Obliterative phlebitis
- C Uninformative
- D Storiform fibrosis + dense lymphocyte infiltration

**答案：D**

**解析：**

**2019 ACR/EULAR IgG4-RD classification criteria** 中的**病理計分項目（pathology domain）**，是依照三大經典病理特徵（**dense lymphoplasmacytic infiltrate、obliterative phlebitis、storiform fibrosis**）**同時出現的數量**來給分，出現越多項特徵，代表病理型態越典型，分數也越高：

| 病理發現 | 給分邏輯 |
|---|---|
| Uninformative（無法判讀） | 最低分（0分） |
| Indeterminate（有部分特徵但不完整） | 低分 |
| 三大特徵中**任一項**單獨出現 | 中等分 |
| 三大特徵中**兩項同時出現** | 較高分 |
| 三大特徵**全數出現** | 最高分 |

逐一分析選項：

- **D（對，本題答案）** **Storiform fibrosis（席紋狀纖維化）+ dense lymphocyte infiltration（緻密淋巴球浸潤）**是**兩項經典病理特徵同時出現**，比起單一特徵單獨出現，代表病理型態更典型、更符合 IgG4-RD 的特徵性組織學表現，**給分也更高**。
- **B（次高，非本題答案）** **Obliterative phlebitis** 雖然也是三大經典特徵之一，但題目只描述**單一項**特徵出現，給分低於「兩項特徵同時出現」的組合（如 D）。
- **A（低分，非本題答案）** **IgG4/IgG 比值 20%** 遠低於分類標準所需的**高分門檻（通常需 >40% 才能拿到較高分）**，20% 屬於偏低比例，給分很低甚至不給分。
- **C（最低分）** **Uninformative（切片無法判讀/資訊不足）** 定義上就是**0分**，是四個選項中分數最低的。

**核心觀念：** IgG4-RD 病理計分的精神是**「特徵性病理表現同時出現越多項，診斷確定性越高，給分越高」**，單一特徵出現的診斷意義遠不如多項經典特徵同時並存。

### 51. 吃了 amoxicillin 30 分鐘後起紅疹加上呼吸喘, 以下何者正確
- A Type III allergy
- B 給予 epinephrine IM
- C （忘記）
- D 給 anti-histamine

**答案：B**

**解析：**

**臨床情境：** 服用 amoxicillin **30 分鐘內**出現紅疹合併**呼吸喘**——這是典型 **Type I（IgE-mediated）立即型過敏反應**，且已有**呼吸道症狀（喘）**，代表病情已達到 **anaphylaxis（過敏性休克）** 的診斷等級，屬於需要立即處置的醫療急症。

- **B（對，本題答案）** **Epinephrine（腎上腺素）肌肉注射（IM，建議大腿外側 vastus lateralis）**是 anaphylaxis **唯一的第一線、最重要的急救藥物**，且**應立即給予、不應延遲**。Epinephrine 透過 α1、β1、β2 受體作用，同時達到**血管收縮（提升血壓）、支氣管擴張（緩解喘鳴）、減少肥大細胞持續釋放介質**等多重效果，是唯一能有效逆轉 anaphylaxis 致命併發症（呼吸道阻塞、休克）的藥物。
- **A（錯）** 這種**服藥後極短時間內（30分鐘）**發生的立即型反應，是 **Type I（IgE-mediated）過敏反應**，並非 **Type III（immune complex-mediated，如 serum sickness，通常在暴露後 1–2 週才發作）**，兩者機轉與時間軸完全不同。
- **D（錯）** **抗組織胺（anti-histamine）只能作為 anaphylaxis 的輔助治療（adjunct therapy）**，用來緩解皮膚症狀（如癢、蕁麻疹），**對呼吸道症狀與低血壓沒有立即、有效的治療效果**，且起效速度遠不及 epinephrine。臨床上絕對不能把抗組織胺當作 anaphylaxis 的第一線/替代治療，這是常見卻危險的錯誤處置。

**Anaphylaxis 急救速記：** 懷疑 anaphylaxis（尤其合併呼吸道或心血管症狀）時，**立即肌肉注射 epinephrine，不要猶豫、不要先用抗組織胺代替**，其他藥物（抗組織胺、類固醇）都只是輔助，無法取代 epinephrine 的角色。

**Epinephrine 劑量補充：** 濃度 1:1000（1mg/mL），成人 IM 劑量 **0.3–0.5 mg**（大腿外側），兒童 0.01 mg/kg（最大 0.3mg），5–15 分鐘未改善可重複注射；給藥途徑務必是 **IM**，不是 SC 或常規 IV。

### 52. AS 病人吃了 salazine 之後三天全身有水泡 + skin detach + oral mucosal break, 為以下何種情況
- A Acute urticaria
- B Fixed drug eruption
- C SJS/TEN
- D DRESS

**答案：C**

**解析：**

**臨床情境：** AS 病人服用 **sulfasalazine（含 sulfonamide 結構）** 後 **3 天**出現**全身水泡 + 皮膚剝離（skin detachment）+ 口腔黏膜破損**——這三個特徵組合起來，是 **SJS/TEN（Stevens-Johnson syndrome / Toxic epidermal necrolysis）** 教科書等級的典型表現。

- **C（對，本題答案）** SJS/TEN 的核心病理機轉是 **cytotoxic T cell 與 NK cell 透過 granulysin、perforin/granzyme B、Fas-FasL** 誘導**角質細胞大量凋亡（keratinocyte apoptosis）**，導致**表皮與真皮分離**，臨床上表現為**廣泛性水泡、皮膚剝離（類似燙傷）**，並且**幾乎必定合併至少一處黏膜侵犯**（口腔、眼睛、生殖器黏膜），此病人「全身水泡 + skin detach + oral mucosal break」三者兼具，完全符合 SJS/TEN 的診斷核心特徵。**Sulfasalazine（含 sulfonamide 結構）**也是文獻上明確記載會誘發 SJS/TEN 的高風險藥物之一，時序（用藥後 3 天，屬於 delayed-type Type IVc 反應的典型潛伏期）也吻合。
- **A（錯）** Acute urticaria（急性蕁麻疹）表現是**風疹塊（wheal）**，通常**數小時內**發作消退，**不會有水泡、皮膚剝離或黏膜破損**，與此病人的表現完全不同。
- **B（錯）** Fixed drug eruption（固定性藥物疹）典型表現是**局部、單一或少數幾處**圓形/橢圓形的紅斑或紫斑（偶爾中央可形成水泡），**再次暴露相同藥物時會在「同一部位」復發**，並非「**全身性**」廣泛分布，與此病人「全身水泡」的描述不符。
- **D（錯）** DRESS（Drug Reaction with Eosinophilia and Systemic Symptoms）典型表現是**廣泛性斑丘疹（morbilliform rash）、發燒、顏面水腫、淋巴結腫大、嗜酸性球增多、內臟器官侵犯（尤其肝臟）**，發作時間通常**較晚（用藥後 2–8 週）**，且**通常不會有顯著的皮膚剝離或大範圍水泡**，這是 DRESS 與 SJS/TEN 在皮膚表現上的重要鑑別點——DRESS 較少見到 SJS/TEN 這種表皮壞死剝離的病理型態。

**藥物疹嚴重型態鑑別速記：**

| 疾病 | 典型皮疹 | 黏膜侵犯 | 發作時間 |
|---|---|---|---|
| **SJS/TEN** | 廣泛水泡、**皮膚剝離** | **幾乎必有**（口腔/眼/生殖器） | 用藥後數天至數週（Type IVc） |
| DRESS | 廣泛斑丘疹，較少水泡剝離 | 少見 | 用藥後 2–8 週（較晚，Type IVb） |
| Fixed drug eruption | 局部固定部位紅斑/紫斑，可能水泡 | 少見（除非侵犯範圍大） | 再次暴露後數小時內復發同一部位 |
| Acute urticaria | 風疹塊（wheal） | 無 | 數分鐘至數小時（Type I） |

### 53. Sjögren syndrome 的病人如果出現持續性唾液腺腫大、體重減輕, 須懷疑以下診斷?

**答案：Lymphoma（B-cell non-Hodgkin lymphoma），最常見型態為 MALToma**

**解析：**

**Primary Sjögren syndrome (pSS)** 病人罹患 **B-cell non-Hodgkin lymphoma** 的風險比一般人高約 **5–20 倍**，是 pSS 最嚴重的長期併發症之一。

- **最常見的型態：MALT lymphoma（Mucosa-Associated Lymphoid Tissue lymphoma，又稱 marginal zone B-cell lymphoma of MALT type）**，好發於**唾液腺（尤其腮腺 parotid gland）**，這也是為什麼「持續性唾液腺腫大」是這題最重要的警訊——正常 pSS 的唾液腺腫大多半是反覆發作、對稱性、可能隨病情起伏消長；但若腫大變成**單側、持續不消退、質地變硬**，就要高度懷疑局部已經從良性淋巴增生（benign lymphoepithelial lesion）轉變為**惡性淋巴瘤**。
- **合併體重減輕**：屬於典型的 **B symptoms**（發燒、夜間盜汗、非刻意體重減輕），是懷疑淋巴瘤/惡性疾病的經典全身性警示症狀，與局部唾液腺腫大合併出現時，惡性轉化的臨床懷疑度會大幅提高。

**pSS 病人發展淋巴瘤的其他風險因子（也是臨床上該提高警覺的線索）：**

| 風險因子 | 說明 |
|---|---|
| **持續性唾液腺腫大** | 尤其單側、質地變硬、不隨病程消退 |
| Purpura（紫斑） | 皮膚小血管炎表現 |
| **低補體（C3、C4 down）** | 常與 cryoglobulinemia 並存 |
| Cryoglobulinemia | Mixed cryoglobulinemia 與淋巴瘤風險相關 |
| Lymphadenopathy | 全身性淋巴結腫大 |
| Monoclonal gammopathy | M-protein 出現 |
| 唾液腺切片 focus score 高、出現 germinal center-like structure | 病理層面的高風險指標 |

**臨床意義：** 上述任一項風險因子出現，都應該考慮安排進一步檢查（如影像學評估唾液腺、必要時切片病理），以early 偵測淋巴瘤轉化。

### 54. Sjögren syndrome 的病患如果出現口乾及吞嚥困難, 可能是以下哪個原因造成的

**答案：唾液分泌減少（salivary hypofunction）**

**解析：**

**Sjögren syndrome** 的核心病理機轉是**淋巴球（主要是 CD4+ T cell）浸潤破壞外分泌腺體（尤其唾液腺、淚腺）**，導致腺體結構被破壞、分泌功能下降：

- **口乾（xerostomia）**：唾液腺（parotid、submandibular、minor salivary glands）功能下降，唾液分泌量顯著減少，是 pSS 最核心的 sicca 症狀之一。
- **吞嚥困難（dysphagia）**：唾液在正常吞嚥過程中扮演**潤滑食團、幫助食物在口咽腔與食道順利下滑**的重要角色。當唾液分泌大幅減少，食物（尤其乾燥/固態食物）在口腔與食道推送過程會變得**乾澀、不順**，病人常主訴需要配水才能吞嚥，這種吞嚥困難本質上是**唾液分泌不足的續發症狀**，而非食道本身結構性病變或動力異常（與 SSc 的食道蠕動異常機轉不同）。

**臨床意義：** 這題點出一個重要觀念——pSS 病人的吞嚥困難，第一直覺應先考慮**唾液分泌不足導致的機械性吞嚥困難**，而非急著懷疑其他食道疾病（如 SSc 的食道蠕動異常、惡性腫瘤壓迫等），臨床評估上可用 **Schirmer's test、唾液流速測定（unstimulated whole saliva flow rate）** 等客觀檢查佐證唾液分泌不足的程度。

### 57. 50 歲男性喘三個月, 手指有 ulcer, 有輕微的肌肉無力, 抽血發現 CRP 高, 肺功能急速惡化, 應該驗什麼及懷疑什麼

**答案：驗 anti-MDA5 抗體，懷疑其相關的 rapidly progressive interstitial lung disease (RP-ILD)**

**解析：**

**臨床情境拆解：**

- **喘三個月，肺功能急速惡化**：提示**快速進展的間質性肺病（rapidly progressive ILD, RP-ILD）**。
- **手指潰瘍（digital ulcer）**：提示血管病變，也是 **anti-MDA5 associated dermatomyositis** 的特徵性皮膚表現之一（除了典型的 Gottron's papules、heliotrope rash，anti-MDA5 病人常合併**皮膚潰瘍性血管病變**，包括手指潰瘍、掌部丘疹潰瘍等）。
- **輕微肌肉無力**：這是這題的關鍵線索——**anti-MDA5 associated dermatomyositis 的招牌特徵正是「clinically amyopathic」或「hypomyopathic」**，也就是**肌肉侵犯往往很輕微，甚至完全沒有明顯肌無力**，與典型 DM/PM 的顯著近端肌肉無力形成強烈對比。
- **CRP 高**：反映全身性發炎活性上升。

這四個特徵組合——**輕微/幾乎沒有肌肉症狀 + 兇猛的肺部侵犯 + 皮膚血管病變（手指潰瘍）**——正是 **anti-MDA5 抗體陽性皮肌炎**最經典的臨床表現型，這型病人最令人畏懼的併發症就是 **rapidly progressive ILD（RP-ILD）**，病程可在數週內快速惡化導致呼吸衰竭，死亡率高，是皮肌炎家族中預後最差的亞型之一，需要**及早診斷、積極治療（通常需要合併多種免疫抑制劑，如高劑量類固醇 + calcineurin inhibitor + IVIG，甚至更積極的合併療法）**。

**anti-MDA5 DM 核心特徵速記：**

| 特徵 | 說明 |
|---|---|
| 肌肉侵犯 | 輕微或無（clinically amyopathic/hypomyopathic） |
| 肺部侵犯 | **RP-ILD**，兇猛、死亡率高 |
| 皮膚表現 | 手掌丘疹潰瘍、**手指潰瘍**、口腔潰瘍、皮膚血管病變 |
| 其他 | 常合併 arthralgia、發燒 |
| 預後不良因子 | 合併 anti-Ro52 陽性（呼應風專考卷第 27 題討論） |

### 58. 考 Dupilumab 的適應症

**答案：Dupilumab 適應症總覽（機轉：結合 IL-4Rα，同時阻斷 IL-4 與 IL-13 訊息路徑）**

**解析：**

Dupilumab 是目前**適應症擴展最廣**的生物製劑之一，核心機轉都圍繞在 **Type 2 inflammation（Th2-driven inflammation）** 這條軸線上，因為 IL-4/IL-13 是 Th2 發炎反應的核心細胞激素：

| 適應症 | 核准年齡下限（大致） | 備註 |
|---|---|---|
| **Atopic dermatitis (AD)** | **6 個月以上**（中重度） | 最早核准的適應症，也是核准年齡最低的族群 |
| **Asthma（氣喘）** | 6 歲以上 | 中重度、type 2/eosinophilic 表型，或口服類固醇依賴型氣喘 |
| **CRSwNP（Chronic rhinosinusitis with nasal polyps）** | 成人 | 慢性鼻竇炎合併鼻息肉 |
| **Eosinophilic esophagitis (EoE)** | 已擴展至 1 歲以上 | 嗜酸性食道炎，近年新增適應症 |
| **Prurigo nodularis（結節性癢疹）** | 成人 | 頑固性搔癢皮膚病 |
| **COPD（合併 type 2 inflammation / eosinophilic phenotype）** | 成人 | 近年新核准（BOREAS、NOTUS trial 證實降低急性惡化），呼應第 9 題討論 |
| **Chronic spontaneous urticaria (CSU)** | 成人 | 較新核准的適應症之一 |

**核心機轉統整：** 上述所有適應症的共同分子基礎都是 **Type 2 inflammation**——無論是皮膚（AD）、下呼吸道（asthma、COPD 特定表型）、上呼吸道（CRSwNP）、食道（EoE）、皮膚慢性搔癢（prurigo nodularis）或蕁麻疹（CSU），只要疾病機轉核心牽涉 **IL-4/IL-13 訊息路徑驅動的 Th2 型發炎反應**，阻斷 IL-4Rα 這個共同受體次單元就能同時涵蓋多種器官/疾病的治療效果，這也是 dupilumab 能夠跨這麼多適應症的分子邏輯基礎。

**觀念重點：** 適應症會隨新的臨床試驗結果持續擴展（如 COPD、CSU 都是相對晚近才新增的適應症），建議之後若遇到考題涉及「Dupilumab 是否適用於某疾病」，先回歸「這個疾病是否為 type 2/Th2-driven inflammation 主導」這個核心邏輯去判斷。

### 59. 關於 CRSwNP, 下列敘述何者正確
- A Dupilumab 是唯一可用的生物製劑?
- B Omalizumab 是第一線生物製劑?
- C Benralizumab 和 Mepolizumab 可以改善嗅覺和鼻塞?
- D 使用 Dupilumab 必須停用鼻用類固醇?

**解析（信心度中等，建議之後可再核對原始題目/公布答案）：**

- **A（錯）** **Dupilumab 並非 CRSwNP 唯一可用的生物製劑**——目前 CRSwNP 已核准的生物製劑至少有三種：**Dupilumab（anti-IL-4Rα）、Omalizumab（anti-IgE）、Mepolizumab（anti-IL-5）**，並非只有 dupilumab 一種選擇。
- **D（錯）** 使用 Dupilumab 治療 CRSwNP **不需要停用鼻用類固醇**。臨床試驗（如 SINUS-24/52 trial）的設計本身就是讓病人在**背景維持使用鼻用類固醇（如 mometasone furoate nasal spray）**的基礎上，加上 dupilumab 作為 add-on therapy，兩者是**併用**而非互斥。
- **B（使用者答案，信心度中等）** 目前並沒有明確的指引將 **omalizumab 定位為 CRSwNP 的「第一線」生物製劑**——三種已核准的生物製劑（dupilumab、omalizumab、mepolizumab）在 EPOS/EUFOREA 等指引中，**並未建立絕對的優先順序**，臨床選擇通常依照病人**共病情況**決定（例如合併過敏性氣喘且 IgE 高，可能優先考慮 omalizumab；合併 eosinophilic 氣喘，可能優先考慮 mepolizumab；若需要療效證據最強、對鼻息肉大小與嗅覺改善效果最顯著，dupilumab 常是實務上較優先選擇）。「omalizumab 是第一線」這個敘述信心度不高。
- **C（部分正確，需留意細節）** **Mepolizumab** 確實在 CRSwNP 臨床試驗（SYNAPSE trial）中證實可以改善鼻息肉評分、鼻塞與嗅覺；但 **Benralizumab** 的 CRSwNP 臨床試驗（OSTRO trial）雖然在鼻息肉評分上有改善，但**嗅覺改善的統計顯著性較不一致**，把 benralizumab 和 mepolizumab **一起**描述為「都能改善嗅覺和鼻塞」，精確度上可能有疑慮。

**⚠️ 提醒：** 這題四個選項都有各自的細節疑慮，信心度不高，建議之後若有老師公布的正式答案可再核對確認。

### 60. 關於 SSc (systemic sclerosis), 何時須評估 HSCT（造血幹細胞移植 Haematopoietic Stem Cell Transplantation）?
- A PAH（考的有 cardiac echo 的腔室壓力和右心導管的參數）
- B ILD（考的有肺功能及治療過程）
- C Renal crisis（目前的 eGFR 及目前洗腎）
- D 皮膚的硬化程度

**答案：D（但 B 同樣是重要適應症，A 與 C 描述的情境反而多半是禁忌症而非適應症）**

**解析：**

**Autologous HSCT** 治療 SSc 的關鍵試驗（**ASTIS、SCOT、ASSIST trial**）已確立其在**瀰漫型皮膚硬化症（diffuse cutaneous SSc, dcSSc）合併快速進展、高風險器官侵犯**病人身上的角色，但**必須嚴格篩選適合的病人**，因為移植前置化療（conditioning regimen）本身有相當風險，需要在「疾病嚴重度足以承擔移植風險」與「器官功能還沒差到承受不了移植」之間取得平衡。

- **D（對，本題答案）** **皮膚硬化程度（modified Rodnan skin score, mRSS）** 是 dcSSc 評估 HSCT 適應症的**核心指標之一**——主要臨床試驗（如 SCOT、ASTIS）都以**高 mRSS 分數、快速進展的瀰漫型皮膚硬化**作為主要的**收案適應症（indication）**，代表疾病處於活躍、進展快速的階段，移植帶來的長期效益可能大於風險。
- **B（也是重要適應症，補充提醒）** **早期、進展中的 ILD**（有肺功能持續下降、影像上有活動性發炎證據，但**尚未進入 end-stage fibrosis**）同樣是評估 HSCT 的**核心適應症之一**——這也是為什麼括號裡特別強調「肺功能及治療過程」，重點在於掌握 ILD 是否處於**還能受益於移植的早期進展階段**，而非已經纖維化到不可逆的末期。
- **A（較可能是禁忌症，而非適應症）** **顯著的 PAH（透過 echo 估計腔室壓力、右心導管確診的血流動力學參數）通常是 HSCT 的相對或絕對禁忌症**，而非適應症——因為移植前置化療對心肺功能負擔極大，合併中重度 PAH 的病人接受移植風險過高，通常會被排除在移植候選名單之外。
- **C（較可能是禁忌症，而非適應症）** **Renal crisis 合併顯著 eGFR 下降或已洗腎**，同樣通常是 HSCT 的**相對或絕對禁忌症**——移植前置化療藥物（如 cyclophosphamide）具有腎毒性，腎功能已嚴重受損或依賴透析的病人，接受移植的風險效益比不佳，一般也會被排除。

**觀念重點：** 這題的巧妙之處在於，**四個選項描述的器官侵犯，並非全部都是「適應症」**——皮膚硬化程度與早期進展性 ILD（B、D）是**促使考慮 HSCT 的正面適應症**；而顯著 PAH 與 renal crisis（A、C）則更多是**移植前需要排除、屬於禁忌症**的器官侵犯型態。

**SSc HSCT 適應症 vs. 禁忌症速記：**

| 器官侵犯 | 角色 |
|---|---|
| 高 mRSS、快速進展的瀰漫型皮膚硬化 | **適應症** |
| 早期、進展中的 ILD（尚未 end-stage） | **適應症** |
| 顯著 PAH（echo/RHC 證實） | **相對/絕對禁忌症** |
| Renal crisis、顯著腎功能不全或已洗腎 | **相對/絕對禁忌症** |

### 68. 一位接受腎臟移植, 三個月後的患者, 因肌酸酐逐漸上升接受切片, 病理顯示毛細血管內皮細胞腫脹, C4d 免疫染色陽性, 血清檢測發現 donor-specific antibodies（DSA）陽性。以下何者可能是最佳診斷？
- A Acute T cell-mediated rejection
- B Calcineurin inhibitor nephrotoxicity
- C Acute antibody-mediated rejection
- D 慢性移植腎病變（chronic allograft nephropathy）

**答案：C**

**解析：**

**臨床情境拆解：**

- **腎臟移植 3 個月後，肌酸酐逐漸上升**：提示移植腎功能惡化，需要鑑別排斥反應、藥物毒性、其他病因。
- **切片病理：微血管內皮細胞腫脹（microvascular endothelial swelling）**：反映**體液性（抗體媒介）攻擊血管內皮**的病理特徵。
- **C4d 免疫染色陽性**：**C4d 是補體活化後的裂解片段，會共價結合在微血管內皮上，是抗體媒介排斥（antibody-mediated rejection, AMR）的經典病理標記**，代表抗體結合後活化了 classical complement pathway。
- **Donor-specific antibodies (DSA) 陽性**：直接證實病人體內已產生**針對捐贈者 HLA 抗原的特異性抗體**，是 AMR 診斷的血清學核心證據。

這三項合在一起（**微血管內皮病變 + C4d 陽性 + DSA 陽性**）正是 **Banff classification** 對 **Acute Antibody-Mediated Rejection (AMR)** 的診斷三要素，缺一不可：

1. **組織學上的急性組織損傷證據**（如 microvascular inflammation、peritubular capillaritis、glomerulitis）
2. **C4d 染色陽性（或其他補體活化的證據）**
3. **血清中 DSA 陽性**

- **C（對，本題答案）** 三項證據完整吻合 Acute Antibody-Mediated Rejection 的診斷標準。
- **A（錯）** **Acute T cell-mediated rejection (TCMR)** 的病理特徵是**間質性淋巴球浸潤（interstitial lymphocytic infiltrate）與腎小管炎（tubulitis）**，並非以微血管內皮病變、C4d 陽性為主，且 TCMR 通常**DSA 陰性**（機轉是 T cell 直接攻擊移植組織，而非抗體媒介），與此病人的病理與血清學表現不符。
- **B（錯）** **Calcineurin inhibitor（CNI，如 tacrolimus、cyclosporine）腎毒性**的病理特徵通常是**小動脈玻璃樣變性（arteriolar hyalinosis）、腎小管空泡化（tubular vacuolization）**，屬於**藥物毒性造成的結構性損傷**，並不會出現 C4d 陽性或 DSA 陽性，這兩項是排除 CNI 腎毒性、指向免疫媒介排斥的關鍵鑑別點。
- **D（錯）** **慢性移植腎病變（chronic allograft nephropathy）** 是一個較籠統的病理描述，通常反映**長期、慢性、非特異性**的移植腎損傷（間質纖維化、腎小管萎縮），並非用來描述這種**急性、有明確免疫學證據（C4d、DSA 陽性）**的排斥反應，且此病人移植僅 3 個月，時間上也更符合**急性**排斥反應的病程，而非慢性病變。

**Banff AMR 診斷三要素速記：**

| 要素 | 內容 |
|---|---|
| 組織學證據 | 微血管發炎（microvascular inflammation）、內皮細胞腫脹 |
| 補體活化證據 | **C4d 染色陽性** |
| 血清學證據 | **Donor-specific antibody (DSA) 陽性** |

### 69. 在 SLE 致病機轉中,「Interferon (IFN) signature」是指什麼？
- A 血清的 IFN-α 濃度升高
- B PBMC（周邊血液單核細胞）IFN 受體表達升高
- C 受到 type I IFN 調控的相關基因過度表達模式
- D 由免疫螢光染色顯示腎小球沉積的 IFN 蛋白過度表達

**答案：C**

**解析：**

**「Interferon (IFN) signature」** 是 SLE 致病機轉研究中非常重要的概念，指的是**周邊血液單核細胞（PBMC）中，一群受 type I interferon 調控的下游基因（interferon-stimulated genes, ISGs）呈現過度表達的基因表現模式（gene expression pattern）**，通常透過 **microarray 或 RNA sequencing** 等基因表現分析技術偵測得到。

- **C（對，本題答案）** IFN signature 的本質是一個**基因表現層次（transcriptomic）的概念**，而非單純測量某個蛋白質的濃度——它反映的是**下游一整組 type I IFN 調控基因（如 MX1、IFI27、IFI44L、OAS1 等）同時被活化、表現量上升**的整體模式。約 60–80% 的 SLE 病人可以偵測到陽性 IFN signature，與疾病活性、特定臨床表現（如皮膚、腎臟侵犯）有一定相關性，也是近年 **anifrolumab（anti-IFNAR1）** 等 type I IFN 標靶藥物研發的重要理論基礎（呼應風專考卷第 1 題討論過的 anifrolumab 機轉）。
- **A（錯）** 血清 IFN-α 濃度本身**因為半衰期短、濃度極低，用傳統方法很難穩定準確測量**，臨床與研究上通常不是直接測「IFN-α 蛋白濃度」，而是改用下游基因表現量作為間接但更穩定的替代指標，這也是為什麼會發展出「IFN signature」這個概念，而非直接測 IFN-α 濃度。
- **B（錯）** IFN signature 測的是**下游被 IFN 訊息路徑活化的基因表現**，而非 IFN receptor（IFNAR）本身的表現量高低，這兩者是不同層次的概念。
- **D（錯）** 腎小球切片的免疫螢光染色主要用於**評估免疫複合物沉積**（如 IgG、IgA、IgM、C3、C1q 沉積型態，用於 lupus nephritis 分類），並非用來偵測 IFN signature——IFN signature 是**周邊血 PBMC 的基因表現分析**，與腎臟切片的免疫螢光染色是完全不同的檢測方法與應用層面。

**核心觀念：** IFN signature = **PBMC 中一組 interferon-stimulated genes 的過度表達模式**，是**基因表現層次**的概念，反映 type I IFN 訊息路徑在 SLE 病人體內處於**慢性活化**狀態，是 SLE 致病機轉「先天免疫系統過度活化」這條軸線的重要證據。

### 70. 下列哪一種狼瘡小鼠模型具複雜多因性, 製造最大量的 ANA 和 anti-dsDNA, 並模擬人類 SLE 漸進性腎炎過程, 可作為藥物長期療效觀察？
- A NZB/WF1
- B MRL/lpr
- C BXSB
- D Pristane-induced lupus model

**答案：A（NZB/W F1）**

**解析：**

**NZB/W F1（New Zealand Black x New Zealand White F1 混種小鼠）** 是 lupus 動物模型研究史上**最經典、最早建立、也是應用最廣泛**的自發性狼瘡小鼠模型，完全符合題目描述的四個特徵：

1. **複雜多因性（complex/multifactorial）**：NZB/W F1 是**兩個近親品系（NZB × NZW）雜交產生的 F1 混種鼠**，疾病表現型是由**多個基因座（polygenic）交互作用**共同決定，並非單一基因突變造成，機轉上更貼近人類 SLE 這種**多基因、多因子（遺傳+環境）**共同致病的複雜疾病模式。
2. **產生最大量的 ANA 與 anti-dsDNA**：NZB/W F1 是目前各種狼瘡小鼠模型中，**自體抗體（尤其 ANA、anti-dsDNA）力價最高、最穩定**的品系之一，是研究 SLE 自體抗體產生機轉的標準模型。
3. **模擬人類 SLE 漸進性腎炎過程**：NZB/W F1 的腎絲球腎炎進展是**緩慢、漸進式**的（通常在數月齡後才逐漸出現顯著 proteinuria、免疫複合物沉積、腎功能惡化），這個**緩慢漸進**的病程與人類 lupus nephritis 的自然病史高度相似，也因此特別適合用來**觀察疾病長期演變與藥物長期療效**。
4. **可作為藥物長期療效觀察**：正因為病程進展緩慢、可預測，NZB/W F1 是**lupus nephritis 藥物研發中最常用的臨床前試驗（preclinical trial）動物模型**，被廣泛用於評估新藥（如免疫抑制劑、生物製劑）對長期腎臟保護效果的療效觀察。

**逐一排除其他選項：**

- **B（錯）** **MRL/lpr** 是由**單一基因突變（Fas/lpr）**造成大量淋巴球增生（lymphoproliferation）與淋巴結腫大，疾病進展**快速、猛爆**（數週內即可發病死亡），較常用於研究**急性、快速進展**的自體免疫疾病機轉，並非以「複雜多因性、漸進式病程」見長，與此題描述不符。
- **C（錯）** **BXSB** 品系的特色是**雄性小鼠（因帶有 Y 染色體上的 Yaa 基因重複）疾病進展比雌性更快、更嚴重**，這與大多數狼瘡模型（雌性較嚴重）相反，是其研究上的獨特之處，但並非以「產生最大量自體抗體、複雜多因性」為主要特色。
- **D（錯）** **Pristane-induced lupus model** 是透過**化學物質（pristane）誘導**正常品系小鼠產生狼瘡樣疾病的**誘導性（inducible）模型**，主要用於研究**環境因子誘發自體免疫疾病**的機轉，並非自發性、多基因決定的複雜模型，與題目描述的「複雜多因性」性質不同。

**SLE 小鼠模型速記：**

| 模型 | 特色 | 主要用途 |
|---|---|---|
| **NZB/W F1** | **複雜多基因、自體抗體力價最高、漸進式腎炎** | **長期藥物療效觀察的標準模型** |
| MRL/lpr | 單一基因（Fas/lpr）突變，病程快速猛爆 | 研究急性淋巴增生、快速進展疾病機轉 |
| BXSB | 雄性病程比雌性更嚴重（Yaa 基因） | 研究性別相關致病機轉 |
| Pristane-induced | 化學誘導，非自發性 | 研究環境因子誘發自體免疫 |

### 71. 一位 8 歲女童因過去一年以來反覆發燒、細菌感染住院, 請問以下哪項敘述對本病患適合？
- A 若反覆細菌感染, 且正常血清 IgG, IgA, IgM, IgE 濃度, 正常白血球與淋巴球分類, 須考慮補體缺損, 可檢驗 C3, C4, CH50 與 AH50 (AP50)
- B 若補體檢驗為瀰漫型 Neisseria 奈瑟氏菌感染, 需考慮補體 C1q, C2 缺失
- C 不需要安排補打常規與非常規疫苗
- D CH50 檢測為 0, 代表補體調控分子有缺失

**答案：A**

**解析：**

- **A（對，本題答案）** 這是評估反覆感染病人的標準臨床邏輯：若病人**體液性免疫（Ig 濃度）正常**、**細胞數量與分類（WBC、淋巴球分類）也正常**，卻仍反覆出現細菌感染，下一步應該**考慮補體系統缺損**。標準補體篩檢組合是 **C3、C4（個別成分濃度）+ CH50（classical pathway 總功能篩檢）+ AH50/AP50（alternative pathway 總功能篩檢）**，這四項組合可以有效篩檢 classical、alternative 兩條補體活化路徑是否有功能性缺損，是臨床上評估補體缺損的標準第一步檢查。
- **B（錯）** 這句話把補體缺損的臨床關聯**配對錯誤**。**反覆/瀰漫性 Neisseria（腦膜炎雙球菌、淋病雙球菌）感染**經典上與**terminal complement pathway（C5–C9，membrane attack complex, MAC）缺損**相關，因為 MAC 是直接溶解革蘭氏陰性菌（尤其 Neisseria 這類對補體溶菌作用特別敏感的菌種）的關鍵機制。**C1q、C2 缺損**（classical pathway 早期成分）則經典上與 **SLE-like 自體免疫疾病**相關（因為免疫複合物清除功能受損），並非以 Neisseria 感染為主要臨床表現，這題把兩種完全不同的補體缺損表型搞混了。
- **C（錯）** 補體缺損病人**並非免疫缺損中「須避免施打疫苗」的族群**——因為補體缺損主要影響**先天免疫的效應機制（如溶菌、opsonization）**，並不影響 T/B cell 產生疫苗特異性免疫反應的能力，所以**常規疫苗（包括活性疫苗）通常仍可安全施打**。事實上，**terminal complement deficiency 病人因為對 Neisseria 感染風險特別高，臨床上反而建議額外補打腦膜炎雙球菌疫苗（meningococcal vaccine，包含更多血清型的疫苗）**，這與選項 C「不需要補打疫苗」完全相反。
- **D（錯）** **CH50 = 0（無法偵測）** 代表 **classical pathway 中某一個結構性補體成分（C1q、C1r、C1s、C4、C2、C3、C5–C9 任一項）完全缺乏**，導致整條古典路徑的溶血反應無法完成，並非specifically指向**補體調控分子（regulatory protein，如 C1-inhibitor、Factor H、Factor I）**缺損。調控分子缺損通常不會讓 CH50 直接降到 0，反而可能造成補體**過度活化、慢性消耗**的表現（如 C1-INH 缺損造成的 HAE，C4 會偏低但 CH50 不一定降到 0）。

**補體缺損與臨床表現配對速記：**

| 補體缺損 | 典型臨床關聯 |
|---|---|
| C1q、C2、C4（classical pathway 早期成分） | SLE-like 自體免疫疾病（免疫複合物清除障礙） |
| **C5–C9（terminal pathway, MAC）** | **反覆/瀰漫性 Neisseria 感染** |
| C3 | 反覆化膿性感染（有莢膜細菌），類似 antibody deficiency 的表現 |
| C1-INH（調控蛋白） | HAE（不是感染，是血管性水腫） |
| Factor H、Factor I（調控蛋白） | aHUS、反覆感染（alternative pathway 過度活化消耗） |

### 72. 下列那些 defective signal pathway 較容易引起 intra-cellular pathogens 的感染, 尤其是 mycobacterial infections
- A IL12/23-IFNγ signaling
- B CD40L-CD40 signaling
- C ICOS-ICOSL signaling
- D Toll-like receptor signaling

**答案：A**

**解析：**

**IL-12/23–IFN-γ signaling axis** 是先天免疫系統對抗**胞內病原體（尤其分枝桿菌 mycobacteria，包括結核菌與非結核分枝桿菌 NTM）**最核心的防禦迴路，機轉如下：

- **Macrophage** 吞噬分枝桿菌後，釋放 **IL-12（與 IL-23）**
- IL-12 刺激 **T cell 與 NK cell** 分泌 **IFN-γ**
- IFN-γ 回頭作用在 macrophage 上（IFN-γ receptor），**強化 macrophage 的殺菌能力（如活化吞噬體酸化、活性氧/氮自由基產生）**，形成一個**macrophage-T cell 之間的正回饋迴路**

這條路徑上任一環節的基因缺陷（IL-12Rβ1、IL-12p40、IFN-γR1/R2、STAT1、NEMO 等），都會造成一群稱為 **Mendelian Susceptibility to Mycobacterial Disease (MSMD)** 的疾病，病人對**分枝桿菌（包括 BCG 疫苗株、環境非結核分枝桿菌）與部分胞內菌（如沙門氏菌）** 異常易感染，是免疫學中「IL-12/23-IFN-γ 缺損 = 分枝桿菌感染」這個經典配對的分子基礎。

**逐一排除其他選項：**

- **B（錯）** **CD40L-CD40 signaling** 缺損會造成 **Hyper-IgM syndrome（X-linked，CD40L 突變）**，因為 class switch recombination 障礙，病人**IgG、IgA、IgE 顯著低下，IgM 正常或升高**，臨床上易感染的病原體以 **PJP（Pneumocystis jirovecii）、Cryptosporidium** 為主，並非以分枝桿菌感染為典型特徵。
- **C（錯）** **ICOS-ICOSL signaling** 缺損主要影響 **Tfh cell 功能與 germinal center 反應**，造成類似 **CVID** 的表現型（抗體功能缺損、反覆呼吸道感染），並非分枝桿菌感染的經典致病路徑。
- **D（錯）** **Toll-like receptor signaling** 缺損（如 IRAK4、MyD88 deficiency）主要造成對**化膿性細菌（pyogenic bacteria，如 Strep. pneumoniae、Staph. aureus）**的易感染性，並非以分枝桿菌感染為特徵性表現。

**胞內病原體/分枝桿菌易感染性核心配對速記：**

| 缺損路徑 | 疾病 | 典型感染病原體 |
|---|---|---|
| **IL-12/23–IFN-γ axis** | **MSMD** | **分枝桿菌（BCG、NTM）、沙門氏菌** |
| CD40L-CD40 | Hyper-IgM syndrome | PJP、Cryptosporidium |
| ICOS-ICOSL | CVID-like | 反覆呼吸道感染（抗體缺損型） |
| TLR/IRAK4/MyD88 | 先天免疫缺損 | 化膿性細菌 |

### 73. Hyper IgE syndrome (HIES) or Job syndrome 的 triads, 下列何者為非？
- A Recurrent skin infections (abscess)
- B Pneumonia, even pneumatocele formation
- C High IgE level (usually over 2000 IU/mL) or/and eosinophilia
- D Vasculitis

**答案：D**

**解析：**

**Hyper-IgE syndrome (HIES, Job syndrome)**，尤其是最經典的 **Autosomal Dominant HIES（STAT3 突變）**，典型的臨床三聯症（triad）是：

- **A（是招牌表現，非本題答案）** **反覆皮膚感染，尤其是冷膿瘍（cold abscess）**——之所以稱為「cold」，是因為 STAT3 訊息路徑異常導致**局部發炎反應不足**，膿瘍雖然仍會形成（通常是 Staph. aureus 感染），但**缺乏典型紅、腫、熱、痛的發炎表現**，這是 HIES 皮膚感染的招牌特色。
- **B（是招牌表現，非本題答案）** **肺炎，甚至進展成肺氣囊腫（pneumatocele）**——同樣因為發炎反應不足、組織修復異常，反覆肺炎後容易在肺部留下**氣囊腫**這種結構性後遺症，是 HIES 肺部侵犯的特徵性影像表現，也容易繼發黴菌（如 Aspergillus）二次感染。
- **C（是招牌表現，非本題答案）** **血清 IgE 顯著上升（通常 >2000 IU/mL）合併嗜酸性球增多**，是這個疾病命名的由來，也是實驗室診斷上最重要的線索。
- **D（非典型表現，本題答案）** **Vasculitis（血管炎）並非 HIES 的核心三聯症之一**。HIES 除了上述三大感染/免疫學特徵外，還常合併**其他系統性表現**，如：
  - 特徵性臉部外觀（course facial features）
  - 骨骼異常（如關節過度伸展、脊椎側彎、病理性骨折）
  - 牙齒異常（乳牙滯留）
  - 血管異常（如動脈瘤、血管扭曲）——但這是**血管結構異常（structural vascular abnormality）**，並非**發炎性血管炎（vasculitis）**，兩者機轉完全不同，不應混為一談。

**HIES 經典三聯症速記：**

| 三聯症 | 內容 |
|---|---|
| 皮膚 | 反覆冷膿瘍（cold abscess） |
| 肺臟 | 反覆肺炎，可進展成肺氣囊腫 |
| 實驗室 | IgE 顯著上升（>2000 IU/mL）+ 嗜酸性球增多 |

### 74. 下列有關 DiGeorge Syndrome 的敘述, 何者錯誤？
- A Complete DiGeorge Syndrome 會出現類似 B 細胞缺損的表現
- B 典型三聯症（triad）為先天性心臟病、免疫缺陷、甲狀旁腺功能低下
- C 最常見的基因缺陷為染色體 22q11.2 區段缺失
- D 兒童期應定期評估生長情形, 若有身材矮小, 應評估是否有餵食困難或生長激素缺乏問題

**答案：B（信心度中等）**

**解析：**

- **B（判斷為錯誤敘述）** 嚴格來說，DiGeorge syndrome 最初被描述的經典三聯症應該是 **「thymic aplasia/hypoplasia（胸腺發育不全）+ hypoparathyroidism（副甲狀腺功能低下）+ congenital heart disease（先天性心臟病）」**。題目把「thymic aplasia」直接替換成「免疫缺陷（immunodeficiency）」——雖然兩者有因果關係（胸腺發育不全 → T cell 發育缺陷 → 免疫缺陷），但**「免疫缺陷」是胸腺發育不全造成的下游結果，並非疾病原始定義三聯症中的字面描述**，這個替換可能是此題被設計為錯誤敘述的原因。此外，DiGeorge syndrome 目前更常以 **CATCH-22** 這個更完整的縮寫來描述（**C**ardiac abnormality、**A**bnormal facies、**T**hymic aplasia、**C**left palate、**H**ypocalcemia、**22**q11 deletion），涵蓋的特徵遠不只三項，把它簡化成單純「三聯症」也可能不夠完整/精確。
- **A（正確敘述，非本題答案）** **Complete DiGeorge syndrome**（胸腺完全發育不全）因為 T cell 完全無法正常發育成熟，臨床表現類似 **SCID（severe combined immunodeficiency）**——雖然 B cell 數量本身正常，但因為**缺乏 T cell help**，導致 B cell 無法有效進行 class switch recombination、產生有效的特異性抗體反應，臨床上呈現出**類似體液性（B cell）免疫缺損的表現**，這是重要且正確的教學觀念。
- **C（正確敘述，非本題答案）** **22q11.2 微小片段缺失（22q11.2 deletion）** 是 DiGeorge syndrome **最常見**的基因致病機轉，佔絕大多數病例。
- **D（正確敘述，非本題答案）** DiGeorge syndrome 病人常合併**餵食困難（因顎裂、肌肉張力低下等）**與**生長激素缺乏**，兒童期定期評估生長曲線、並在身材矮小時進一步評估這兩個常見病因，是合理且正確的臨床追蹤建議。

**⚠️ 提醒：** 這題 B 選項的「錯誤」判定，取決於考試是否嚴格要求「三聯症」的原始定義用詞（thymic aplasia 而非 immunodeficiency），細節上稍有主觀判斷空間，若有老師公布的正式答案可再核對確認。

### 75. 下列關於 dendritic cell, 何者錯誤
- A Immature DC 的 take up antigen 能力較 mature DC 強
- B Mature DC 的 antigen presenting 能力較 immature DC 強
- C Plasmacytoid DC 對 B cell 產生抗體沒有幫忙
- D TSLP 可藉由誘導 immature DC 表現 OX40L 來強化 Th2 的分化

**答案：C**

**解析：**

- **A（正確敘述，非本題答案）** **Immature DC** 專門負責**抗原捕獲（antigen capture）**，具有較強的**吞噬（phagocytosis）與胞飲（macropinocytosis）能力**，適合在周邊組織持續監控、攝取抗原。
- **B（正確敘述，非本題答案）** DC 成熟過程中，**抗原攝取相關的胞器活性下降，取而代之的是 MHC class II 表現量上升、共同刺激分子（CD80/CD86）表現量上升**，因此 **mature DC 的抗原呈現（antigen presentation）與活化 T cell 的能力遠強於 immature DC**，這是 DC 生物學的核心概念——「immature DC 負責抓抗原，mature DC 負責呈現抗原、活化 T cell」。
- **C（錯誤敘述，本題答案）** **Plasmacytoid DC (pDC)** **並非對 B cell 抗體產生沒有幫助**——pDC 除了以大量分泌 **type I interferon（IFN-α）**、在抗病毒先天免疫反應中扮演核心角色外，其分泌的 **IFN-α 也能直接作用在 B cell 上，促進 B cell 分化成漿細胞（plasmablast/plasma cell）、增強抗體產生**。這個機轉在**SLE 等自體免疫疾病**中特別重要——pDC 過度活化、持續分泌 IFN-α，被認為是驅動 B cell 過度活化、產生大量自體抗體的重要上游機轉之一（也呼應第 69 題討論的 IFN signature 概念）。因此「pDC 對 B cell 產生抗體沒有幫忙」是錯誤的敘述，pDC 其實透過 IFN-α 這條路徑，對 B cell 抗體反應有正面促進作用。
- **D（正確敘述，非本題答案）** **TSLP（thymic stromal lymphopoietin）** 是黏膜上皮細胞受到外界刺激（過敏原、機械性損傷、微生物）後釋放的 **alarmin**，作用在 **immature DC** 上，誘導其**表現 OX40L**，接著 DC 表面的 OX40L 與 naive T cell 表面的 OX40 結合，強力促進 **T cell 往 Th2 方向分化**，是氣喘、異位性皮膚炎等過敏性疾病上游機轉的重要一環（也呼應這份考卷第 9 題討論過 tezepelumab 標靶 TSLP 的機轉背景）。

**核心觀念：** pDC 不只是「抗病毒的 IFN-α 工廠」，在自體免疫疾病（尤其 SLE）機轉中，pDC 分泌的 IFN-α 也是**驅動 B cell 過度活化、產生自體抗體**的重要上游推手，這是免疫學考題常考的延伸連結。

### 76. 下列關於 dendritic cell 的敘述, 何者錯誤
- A Dendritic cell 是一種 mononuclear phagocyte
- B Human mature DC 可分為 myeloid DC 或 plasmacytoid DC
- C Langerhans cell 是屬於 conventional DC
- D Conventional DC 是抗病毒 cytokine interferon alpha 及 beta 的主要分泌者

**建議答案：D（與使用者原答案 C 不同，已於下方說明判斷依據）**

**解析：**

- **D（判斷為錯誤敘述，本題答案）** **分泌大量抗病毒 type I interferon（IFN-α、IFN-β）的主要細胞是 plasmacytoid DC (pDC)，而非 conventional DC**。pDC 甚至因為這個特性，過去被稱為「natural interferon-producing cell」，是先天免疫系統中對抗病毒感染最重要的 IFN 來源。**Conventional DC（myeloid DC）的核心專長是抗原呈現、活化 T cell（尤其 cross-presentation 給 CD8+ T cell）**，並非以大量分泌 type I IFN 為其功能特色，這句話把兩種 DC 亞群的招牌功能講反了。
- **C（正確敘述）** **Langerhans cell** 是位於**表皮層**的 DC，雖然在細胞起源上有其獨特性（部分可來自 yolk sac/胎肝前驅細胞，具有局部自我更新能力，與其他 cDC 主要依賴骨髓前驅細胞持續補充的模式不同），但依照**功能與細胞系譜分類，Langerhans cell 仍被歸類在 conventional DC（myeloid DC）家族**之下，與 plasmacytoid DC 是不同的分類，這句話是正確的。
- **A（正確敘述）** Dendritic cell 屬於**mononuclear phagocyte system** 的一員，與 monocyte、macrophage 同屬一個大家族（雖然 DC 的核心功能是抗原呈現，與 macrophage 以吞噬清除為主的功能有所區別，但在細胞系譜分類上仍歸屬於此系統）。
- **B（正確敘述）** 人類 DC 依細胞系譜可分為 **myeloid DC（conventional DC，又可再細分 cDC1、cDC2）與 plasmacytoid DC (pDC)** 兩大類，這是標準的 DC 分類方式。

**Conventional DC vs. Plasmacytoid DC 功能速記：**

| | Conventional DC (myeloid) | Plasmacytoid DC |
|---|---|---|
| 招牌功能 | **抗原呈現、活化 T cell**（含 cross-presentation） | **大量分泌 type I IFN（IFN-α/β）**，抗病毒 |
| 包含細胞 | Langerhans cell、cDC1、cDC2 | pDC |

**⚠️ 提醒：** 此答案與使用者原選項（C）不同，若有確切公布答案可再核對確認。

### 77. 一個 12 歲男孩因近一年內反覆發生鼻竇炎與支氣管炎就診, 檢查發現 IgG, IgA, IgM 均顯著降低, B cell 數量正常, T cell 功能及數量正常, 接種疫苗後特異性抗體效價未明顯上升。下列何者正確
- A 多數患者為 X-linked 遺傳
- B 多數患者的 B cell 可分化為 functional plasma cell
- C 接受 IVIG 替代治療, 其自體免疫與淋巴增生表現通常可完全控制
- D 病患可能有腸胃道症狀, 如慢性腹瀉或 IBD 表現

**建議答案：D（與使用者原答案 A 不同，已於下方說明判斷依據）**

**解析：**

**臨床情境判讀：** 12 歲男孩，反覆鼻竇炎/支氣管炎，**IgG、IgA、IgM 均顯著降低**，**B cell 數量正常**，**T cell 功能與數量正常**，**接種疫苗後特異性抗體效價無法有效上升**——這是 **Common Variable Immunodeficiency (CVID)** 的典型臨床與檢驗表現：B cell 數量本身沒有問題，但**終末分化（terminal differentiation）成能產生抗體的漿細胞（plasma cell）的功能出現障礙**，導致體液性免疫功能低下。

- **D（判斷為正確答案）** CVID 病人除了反覆感染外，**約 20–30% 會合併腸胃道侵犯**，表現包括**慢性腹瀉、吸收不良、類似 IBD 的腸病變（enteropathy）、nodular lymphoid hyperplasia** 等，是 CVID 重要的腸胃道併發症，此敘述正確。
- **A（判斷為錯誤）** **CVID 大多數病人是散發性（sporadic）**，並非以 X-linked 遺傳為主；已知的少數單基因致病機轉（如 TACI/TNFRSF13B、ICOS、CD19 等）也多為體染色體遺傳模式，並非 X-linked。這題容易與 **X-linked agammaglobulinemia (XLA, Bruton's disease)** 搞混——但 XLA 的關鍵檢驗特徵是 **B cell 數量顯著降低甚至幾乎缺乏**，與此病人「B cell 數量正常」的檢驗結果不符，這正是這題用來排除 XLA、指向 CVID 的關鍵線索。
- **B（錯）** CVID 的**核心缺陷正是「B cell 數量正常，但無法有效分化成 functional plasma cell」**，這句話與 CVID 的疾病定義本身矛盾——如果多數 B cell 都能正常分化成有功能的漿細胞，就不會出現 Ig 顯著降低、疫苗反應不佳的臨床表現了。
- **C（錯）** **IVIG 替代治療只能改善體液性免疫缺損（預防感染）**，但**無法完全控制 CVID 常合併的自體免疫（如 autoimmune cytopenia）與淋巴增生表現（如 granulomatous disease、lymphoproliferation、淋巴瘤風險上升）**，這些併發症通常需要額外的免疫調節治療，IVIG 並非萬能，這句話過度誇大了 IVIG 的療效範圍。

**CVID vs. XLA 鑑別速記：**

| | CVID | XLA (Bruton's) |
|---|---|---|
| 遺傳模式 | 多數散發性，少數體染色體遺傳 | X-linked（BTK 基因突變） |
| B cell 數量 | **正常** | **顯著降低/幾乎缺乏** |
| 缺陷本質 | B cell 終末分化障礙 | B cell 發育本身受阻（pre-B → B cell 階段） |
| 常見併發症 | 腸胃道症狀、自體免疫、淋巴增生 | 主要以反覆感染為主 |

**⚠️ 提醒：** 此答案與使用者原選項（A）不同，若有確切公布答案可再核對確認。
