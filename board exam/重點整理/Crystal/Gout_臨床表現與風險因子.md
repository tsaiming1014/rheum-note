# Gout 臨床表現與風險因子（Kelly Ch.97）

## 一、疾病分期概念

Gout 是 **MSU 晶體沉積的慢性疾病**，分幾個階段：

1. **無症狀高尿酸血症** → 晶體沉積但無症狀
2. **急性發作（Gout Flare）**
3. **間歇期（Intercritical period）**：完全無症狀，但晶體仍存在
4. **慢性痛風石性痛風（Tophaceous gout）/ 慢性痛風關節炎**

---

## 二、急性發作（Gout Flare）

**典型特徵**：

- 急性單關節炎為主（也可 oligo/polyarticular）
- **好發部位**：第一 MTP（podagra）最常見，其次 midfoot、ankle、knee
- 疼痛在 **24 小時內達到最高峰**
- 自然病程 **1-2 週**緩解
- 典型發作特徵（SUGAR study 最有鑑別力）：
  - Male sex
  - Dietary trigger
  - First MTP 受侵犯（當次或過去）
  - 臨床可見痛風石
  - 關節紅腫（erythema）
  - 關節極度壓痛
  - ≥ 2 次急性發作且在 2 週內完全緩解
  - 行走困難

- 嚴重發作可有發燒等全身症狀
- 緩解期皮膚可能脫屑
- 若無治療，發作越來越頻繁，侵犯範圍擴大至上肢（olecranon bursa、腕、手指小關節）

**誘發因素**：

- 高 SUA、急性疾病/手術、利尿劑、高普林飲食/酒精、脫水、關節外傷
- 開始 ULT 初期（晶體釋放）
- 高溫、極端濕度
- 低劑量 aspirin、住院

---

## 三、痛風石（Tophi）

**定義（G-CAN）**：MSU 晶體 + 宿主組織反應形成的有序結構

**外觀**：透明皮膚下白堊色或漿液性皮下結節，常有表面血管增生

**好發部位**：第一 MTP、耳廓（helix）、olecranon bursa、Achilles 肌腱、手指腹、腕

**非典型部位**：頭頸部、眼、耳鼻喉、心臟、乳房、腸道、脊椎、神經（壓迫性神經病變，如 carpal tunnel）

**臨床意義**：

- 通常**不痛**，但急性發炎時會痛
- 造成關節變形、活動受限
- **骨內痛風石 → 骨侵蝕（erosion）**
- 破潰 → 繼發感染風險
- Chronic gouty arthritis（持續性滑膜炎）

**高風險族群**：年老、女性、利尿劑使用者、CKD → 早期即出現痛風石

---

## 四、特殊臨床情境

- **Eruptive tophaceous gout**：腎移植患者、老年人合併 CKD + 利尿劑 → 快速進展
- **MSU + CPPD 共存**：同一關節可同時找到兩種晶體
- **Gout + RA**：難以區分 flare 來源，需仔細鑑別
- **Gout + OA**：痛風好發於 OA 已受損關節（尤其 DIP、第一 MTP）
- **Gout + PsA**：乾癬病人有高 SUA，第三趾痛風石 + 甲癬可擬似 sausage digit

## 五、鑑別診斷

| 最重要 | 其他 |
|--------|------|
| **Septic arthritis**（必須排除） | Fracture |
| CPPD（pseudogout） | Reactive arthritis |
| BCP arthritis/tendinitis | Psoriatic arthritis |
| | RA（多關節型） |

Tophus 的 DDx：RA nodule、tumoral calcinosis、lipid deposits、osteophyte

## 六、Factors Associated With Developing Gout（Table 97.3）

> 核心概念：**大多數風險因子都是透過升高 SUA 來增加 gout 風險**

### 遺傳／生理

| 因子 | 說明 |
|------|------|
| Male sex | 女性保護效應至停經後消失 |
| Ancestry | 台灣原住民、Māori、太平洋島民盛行率高出 2 倍以上 |
| **SLC2A9** | 腎小管 urate 轉運（最強遺傳訊號） |
| **ABCG2** | 腸道 urate 分泌；變異 → 早發型、tophus 多、allopurinol 反應差 |
| SLC17A1/SLC17A3、GCKR | 其他 urate 轉運相關基因 |
| 家族史 | 第一親等有 gout → 風險近 2 倍 |

### 藥物（升 SUA 機轉）

| 藥物 | 備註 |
|------|------|
| **利尿劑** | 最常見誘因，尤其 thiazide、loop diuretic |
| Cyclosporine / Tacrolimus | 腎移植常用，cyclosporine 影響更大 |
| ACE inhibitors | |
| 非 losartan 的 ARB | Losartan 例外，具輕度降尿酸效果 |
| Beta blockers | |
| Pyrazinamide（抗結核） | |
| Ritonavir（HIV 用） | |
| 低劑量 aspirin | 升 SUA（但心血管適應症時不應停） |

### 飲食

| 因子 | 方向 |
|------|------|
| 紅肉、海鮮 | ↑ SUA |
| 啤酒、烈酒 | ↑ SUA |
| 含糖飲料（高果糖） | ↑ SUA |
| 低脂乳製品、咖啡、維他命 C | ↓ SUA（但對已確診 gout 效果有限） |
| DASH diet | ↓ SUA、↓ 發病風險 |

> 注意：單一飲食成分對 SUA 的影響其實**很小**，無法取代藥物

### 共病

| 共病 | 備註 |
|------|------|
| 肥胖、體重增加 | Bariatric surgery 可顯著降 SUA |
| 高血壓 | |
| CKD | 腎臟排尿酸下降；也影響藥物選擇 |
| 高血脂（高三酸甘油酯） | |
| 心衰竭 | |
| 阻塞性睡眠呼吸中止（OSA） | |
| 貧血、鐮刀型細胞貧血 | 細胞轉換率高 |
| 乾癬、血液惡性腫瘤 | 高細胞轉換 → 高 urate 生成 |
| 鉛暴露 | |

> 特殊例外：**糖尿病控制差（高血糖）→ 尿糖 uricosuric 效果 → 反而降 SUA、降 gout 風險**
