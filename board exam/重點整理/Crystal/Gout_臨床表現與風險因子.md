# Gout 臨床表現與風險因子（Kelly Ch.97）

## 一、流行病學

**盛行率與趨勢：**

Gout 是最常見的發炎性關節炎，全球盛行率呈持續上升趨勢，與人口老化、肥胖盛行率增加、代謝症候群普及、飲食型態改變（高普林飲食、含糖飲料）有關。台灣的盛行率是全球數一數二高的地區之一，與遺傳因子（尤其 **ABCG2** 基因變異在漢人/亞裔族群中盛行率較高）及飲食習慣（海鮮、飲酒）都有關。

**性別與年齡分布：**

- 男性明顯多於女性，男女比約 **3-4:1**
- 機轉：雌激素具有促進尿酸排除（uricosuric）的作用，停經前女性因雌激素保護，痛風發生率明顯較低
- 男性好發年齡：**30-50 歲**；女性好發年齡明顯較晚，多在**停經後（60 歲以後）**
- 若**年輕女性（停經前）出現痛風**，臨床上要提高警覺，考慮是否有次發性原因（如腎功能不全、利尿劑使用、遺傳性尿酸代謝異常等）

**核心觀念：**

高尿酸血症是必要但非充分條件——**大多數高尿酸血症的人終其一生不會發生痛風**。共病症（高血壓、CKD、心血管疾病、糖尿病、代謝症候群）皆與 gout 有密切雙向關聯。

**重點速記：**

| 面向 | 重點 |
|---|---|
| 性別 | 男 >> 女（3-4:1），雌激素保護停經前女性 |
| 好發年齡 | 男性 30-50 歲；女性停經後（60+） |
| 種族 | 太平洋島民、Māori、台灣原住民/漢人族群盛行率高（與 ABCG2 相關） |
| 飲酒 | 啤酒、烈酒風險最高，紅酒相對較低 |
| 遺傳 | SLC2A9、ABCG2、SLC22A12（尿酸轉運蛋白基因） |

## 二、疾病分期概念

Gout 是 **MSU 晶體沉積的慢性疾病**，分幾個階段：

1. **無症狀高尿酸血症** → 晶體沉積但無症狀
2. **急性發作（Gout Flare）**
3. **間歇期（Intercritical period）**：完全無症狀，但晶體仍存在
4. **慢性痛風石性痛風（Tophaceous gout）/ 慢性痛風關節炎**

---

## 三、急性發作（Gout Flare）

**典型特徵**：

- 急性單關節炎為主（也可 oligo/polyarticular）
- **好發部位**：第一 MTP（podagra）最常見，其次 midfoot、ankle、knee
- 疼痛在 **24 小時內達到最高峰**
- 自然病程 **1-2 週**緩解
- 典型發作特徵（SUGAR study 最有鑑別力）：
  - Male sex
  - Dietary trigger
  - First MTP 受侵犯（當次或過去）
  - 臨床可見痛風石
  - 關節紅腫（erythema）
  - 關節極度壓痛
  - ≥ 2 次急性發作且在 2 週內完全緩解
  - 行走困難

- 嚴重發作可有發燒等全身症狀
- 緩解期皮膚可能脫屑
- 若無治療，發作越來越頻繁，侵犯範圍擴大至上肢（olecranon bursa、腕、手指小關節）

**誘發因素**：

- 高 SUA、急性疾病/手術、利尿劑、高普林飲食/酒精、脫水、關節外傷
- 開始 ULT 初期（晶體釋放）
- 高溫、極端濕度
- 低劑量 aspirin、住院

---

## 四、痛風石（Tophi）

**定義（G-CAN）**：MSU 晶體 + 宿主組織反應形成的有序結構

**外觀**：透明皮膚下白堊色或漿液性皮下結節，常有表面血管增生

**好發部位**：第一 MTP、耳廓（helix）、olecranon bursa、Achilles 肌腱、手指腹、腕

**非典型部位**：頭頸部、眼、耳鼻喉、心臟、乳房、腸道、脊椎、神經（壓迫性神經病變，如 carpal tunnel）

**臨床意義**：

- 通常**不痛**，但急性發炎時會痛
- 造成關節變形、活動受限
- **骨內痛風石 → 骨侵蝕（erosion）**
- 破潰 → 繼發感染風險
- Chronic gouty arthritis（持續性滑膜炎）

**高風險族群**：年老、女性、利尿劑使用者、CKD → 早期即出現痛風石

---

## 五、特殊臨床情境

- **Eruptive tophaceous gout**：腎移植患者、老年人合併 CKD + 利尿劑 → 快速進展
- **MSU + CPPD 共存**：同一關節可同時找到兩種晶體
- **Gout + RA**：難以區分 flare 來源，需仔細鑑別
- **Gout + OA**：痛風好發於 OA 已受損關節（尤其 DIP、第一 MTP）
- **Gout + PsA**：乾癬病人有高 SUA，第三趾痛風石 + 甲癬可擬似 sausage digit
