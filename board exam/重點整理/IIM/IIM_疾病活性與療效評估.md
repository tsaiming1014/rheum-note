# IIM 疾病活性與療效評估量表

> IMACS core set、MMT-8、MDAAT，以及 2016 ACR/EULAR 療效標準（TIS）

## 一、總覽：活性 vs 損傷 vs 功能

IIM 的評估分三個層次，量表不可混用：

| 層次 | 意義 | 代表工具 |
|---|---|---|
| **疾病活性（activity）** | 可逆、反映目前發炎 | IMACS CSM 六項、MDAAT |
| **損傷（damage）** | 不可逆累積 | Myositis Damage Index (MDI) |
| **功能/失能** | 病人層面影響 | HAQ、CMAS、FI-2/3 |

主導組織：**IMACS**（International Myositis Assessment and Clinical Studies group，成人）與 **PRINTO**（兒童）。

---

## 二、IMACS Core Set Measures（CSM 六項）

評估「疾病活性」的核心六項，臨床試驗的共同語言：

1. **Physician Global Activity（醫師整體活性）** — VAS 0–10 cm
2. **Patient/Parent Global Activity（病人整體活性）** — VAS 0–10 cm
3. **Muscle strength（肌力）** — **MMT-8**（見第三節）
4. **Physical function（身體功能）** — **HAQ**（成人）/ CHAQ（兒童）
5. **Muscle enzymes（肌肉酵素）** — ≥2 項：CK、aldolase、LDH、AST、ALT
6. **Global extramuscular activity（肌肉外整體活性）** — **MDAAT**（見第四節）

> 記憶：兩個 global（醫師、病人）＋ 肌力 ＋ 功能 ＋ 酵素 ＋ 肌肉外。

---

## 三、MMT-8（Manual Muscle Testing-8）★重點

由 IMACS 從 MMT-24 精簡驗證（Rider 2010）而來；涵蓋**近端、遠端、軸向**肌群，**單側（慣用側／右側優先，無法測才用左側）**施測。

**8 條肌群**

| 部位 | 肌群 |
|---|---|
| 軸向 | 1. 頸屈肌（neck flexors） |
| 近端上肢 | 2. 三角肌中段（deltoid，肩外展） |
| 近端上肢 | 3. 肱二頭肌（biceps，肘屈曲） |
| 近端下肢 | 4. 臀大肌（gluteus maximus，髖伸展） |
| 近端下肢 | 5. 臀中肌（gluteus medius，髖外展） |
| 近端下肢 | 6. 股四頭肌（quadriceps，膝伸展） |
| 遠端上肢 | 7. 腕伸肌（wrist extensors） |
| 遠端下肢 | 8. 踝背屈肌（ankle dorsiflexors） |

**重點**：IIM 最弱的 5 條肌肉中有 4 條（頸屈肌、三角肌、臀大肌、臀中肌）被納入 → 對近端無力敏感。

### 0–10 計分定義（Kendall 量表，IMACS 實際採用）

每條肌群 0–10 分。判斷三段式：**0 不動 → 1–3 看 ROM（去重力 vs 抗重力）→ 4–10 看能維持姿勢並抗多大阻力**。

| 分數 | 定義 | 區段 |
|---|---|---|
| **0** | 肌肉無任何動作（No movement） | 無動作 |
| **1** | 〔水平面/去重力〕**不完整** ROM | 看 ROM |
| **2** | 〔水平面/去重力〕**完整** ROM | 看 ROM |
| **3** | 〔抗重力姿勢〕**部分** ROM | 看 ROM |
| **4** | Gradual release（維持測試姿勢時逐漸被壓下） | 看姿勢 |
| **5** | 維持姿勢，**不加壓力** | 看姿勢 |
| **6** | 抗**輕微**壓力維持 | 抗阻力↑ |
| **7** | 抗**輕到中等**壓力維持 | 抗阻力↑ |
| **8** | 抗**中等**壓力維持 | 抗阻力↑ |
| **9** | 抗**中等到強**壓力維持 | 抗阻力↑ |
| **10** | 抗**強**壓力維持（正常） | 抗阻力↑ |

**總分**：8 條肌群單側合計 **0–80**；雙側版（軸向 1 條單側＋5 近端＋2 遠端雙側）**0–150**。

> 註：IMACS 實際計分系統**不使用獨立的「Trace」級**；MRC 對照上 MRC grade 5 ＝ Kendall 10。

## 四、肌肉外活性：MDAAT / MYOACT / MITAX

**MDAAT（Myositis Disease Activity Assessment Tool）** = 評估 7 大器官系統活性的工具，由兩部分組成：

- **MYOACT**（Myositis Disease Activity Assessment **Visual Analogue Scales**）：各器官系統用 VAS 評活性
- **MITAX**（Myositis **Intention-To-Treat** Activity Index）：仿 BILAG，依「醫師是否打算治療該系統」評分

7 大系統：全身（constitutional）、皮膚、骨骼（關節）、腸胃、肺、心臟、肌肉。

## 五、其他常用量表

| 量表 | 對象 | 內容 |
|---|---|---|
| **CMAS**（Childhood Myositis Assessment Scale） | JDM | 觀察型肌力/耐力，0–52 |
| **FI-2 / FI-3**（Functional Index in Myositis） | 成人 | 重複動作測耐力，物理治療常用 |
| **CDASI**（Cutaneous DM Disease Area & Severity Index） | DM 皮膚 | 分 activity 與 damage 兩分量 |
| **IBM-FRS**（IBM Functional Rating Scale） | IBM | 功能評分 0–40 |
| **HAQ / HAQ-DI** | 成人 | 失能指數 0–3 |
| **MDI**（Myositis Damage Index） | 全 IIM | 累積損傷（不可逆） |

## 六、療效標準演進：2004 DOI → 2016 ACR/EULAR TIS

### 2004 IMACS 初步「改善定義（DOI）」

≥3 項 CSM 改善 ≥20%，且至多 2 項惡化 ≥25%（且不得是 MMT）。

### 2016 ACR/EULAR Myositis Response Criteria（Aggarwal et al.）★現行標準

以 6 項 CSM 的**變化百分比**經 conjoint analysis 加權，算出 **Total Improvement Score（TIS, 0–100）**。

**成人 conjoint 權重（各項最高分）**

| CSM | 最高分 |
|---|---|
| 肌力 MMT-8 | 32.5 |
| 醫師整體活性 | 20 |
| 肌肉外活性（MDAAT） | 20 |
| 病人整體活性 | 10 |
| 身體功能 HAQ | 10 |
| 肌肉酵素 | 7.5 |
| **合計** | **100** |

> 重點：**肌力權重最高（32.5）、肌酶最低（7.5）**。兒童版（JDM）則 CMAS 與 MMT 權重最高。

**反應等級門檻（TIS）**

| 等級 | 成人 DM/PM | 幼年型 JDM |
|---|---|---|
| Minimal（最小改善） | ≥ 20 | ≥ 30 |
| Moderate（中度改善） | ≥ 40 | ≥ 45 |
| Major（重大改善） | ≥ 60 | ≥ 70 |

> 成人 20/40/60、兒童 30/45/70。（兒童數值依 Rider 2017 JDM 版本，引用前建議再核對。）

## 七、實務記憶串接

- **要評活性** → IMACS CSM 六項（含 MMT-8、MDAAT）
- **要評療效** → 把六項變化丟進 2016 TIS → 看落在 20/40/60 哪一級
- **要評累積損傷** → MDI（與活性分開）
- **DM 皮膚單獨追蹤** → CDASI；**IBM 功能** → IBM-FRS；**兒童** → CMAS
