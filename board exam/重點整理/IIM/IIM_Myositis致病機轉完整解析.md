# Myositis 致病機轉完整解析
## 從基因背景、自體抗體到肌肉外表現

---

## 一、前言：IIM 是什麼？

特發性炎性肌病（Idiopathic Inflammatory Myopathies, IIM），俗稱 myositis，是一群以**慢性近端肌無力、肌酶上升、肌肉炎症浸潤**為共同表現的全身性自體免疫疾病。主要亞型包括：Dermatomyositis（DM）、Polymyositis（PM）、Inclusion Body Myositis（IBM）、Immune-Mediated Necrotizing Myopathy（IMNM）、Anti-synthetase syndrome，以及 Clinically Amyopathic DM（CADM）。

IIM 之所以被歸為同一類疾病，是因為分類最初建立在**臨床症候群**上（Bohan & Peter, 1975），而非病因。不同亞型走著不同的免疫路徑，卻都「碰巧」傷及骨骼肌。隨著各亞型的 myositis-specific autoantibodies（MSAs）陸續被發現，現代觀點已將 IIM 視為一個**暫時性的分類框架**，未來可能如淋巴瘤一樣進一步細分為各自獨立的疾病。

---

## 二、為何不同抗體都造成「肌炎」？

### 靶抗原的共同特徵

MSAs 的靶抗原表面上各不相同，但有一個關鍵共通點：**幾乎都參與蛋白質合成途徑**（aminoacyl-tRNA synthetases、SRP、Mi-2 等）。這不是巧合。

這些分子在**受損、再生中的肌肉細胞**以及特定組織（肺泡細胞、皮膚角質細胞）中高度表現。組織損傷 → 靶抗原大量釋放 → 在特定 HLA 背景下被免疫系統辨識為「外來抗原」→ 啟動自體免疫攻擊。不同的 HLA 型加上不同的環境觸發因子，決定了針對哪個靶抗原產生抗體，但最終都匯聚到「傷害骨骼肌」這個共同終點。

### 下游路徑的共同匯聚點

- 骨骼肌損傷 → 近端肌無力
- MHC class I 過度表現 → ER stress → 肌纖維損傷
- 炎症細胞浸潤 → CK 上升

## 三、致病機轉全貌

### Stage 1｜遺傳背景

**HLA 基因**是最強的遺傳風險因子，其功能是將抗原呈現給 T cells，因此不同的 HLA 型決定了哪種分子更容易被辨識為自體抗原：

- HLA-DRB1\*0301 + DQA1\*0501 → 白人最主要風險；與 anti-synthetase 相關
- HLA-DRB1\*08 → 黑人風險因子
- HLA-DR3 → anti-Jo-1；DR5 → anti-SRP；DR7 → anti-Mi-2；DRB1\*1101 → anti-HMGCR

**非 HLA 基因**同樣重要：

- TNF、IL-1、補體基因（C4、C2）多型性
- Complement C4 低 copy number → IIM 風險增加
- VCP、SQSTM1、FYCO1 → IBM（autophagy 缺陷）

### Stage 2｜環境觸發

不同環境因子觸發不同亞型，這本身也是致病機轉的線索：

| 觸發因子 | 相關亞型 |
|---------|---------|
| UV 光照 | Anti-Mi-2 陽性 DM（latitude gradient 最強） |
| 吸入性物質（煙、粉塵）| Anti-Jo-1 陽性 anti-synthetase syndrome |
| Statins | Anti-HMGCR IMNM |
| 病毒感染（enteroviruses、HIV、COVID-19）| 多種亞型 |
| 免疫 checkpoint inhibitors | ICI-induced myositis |
| 惡性腫瘤 | Anti-TIF1-γ、anti-NXP2 陽性 DM |

Anti-synthetase syndrome 尤其值得注意：吸入性觸發因子、高達 70–80% 的 ILD 盛行率，都指向**肺部才是免疫反應的起始地**，肌肉損傷反而是次要現象。

### Stage 3｜打破免疫耐受，產生自體抗體

1. 組織損傷釋放 **DAMPs**（如 HMGB1）
2. DAMPs 活化 **TLR-3、TLR-7、TLR-9** → innate immunity 啟動
3. **Plasmacytoid DCs** 大量產生 **Type I IFN（IFNα/β）**
4. Type I IFN → 上調 MHC class I、促進 APC 活化
5. APCs 呈現自體抗原給 **CD4+ T cells**
6. CD4+ T cells 活化 → 協助 B cells → 產生 **MSAs**

MSAs 通常只有一種（mutually exclusive），有時在臨床症狀出現前就已存在。

### Stage 4｜三條平行的效應機制

#### 路徑 A：血管路徑（DM 為主）

補體活化 → MAC（C5b-9）沉積於微血管內皮 → 微血管壞死、capillary loss → 缺血（ischemia）+ 組織低氧（hypoxia） → HIF-1α/2α 上調、代償性血管新生但不足 → **Perifascicular atrophy**（距血管最遠的 perifascicular 區域最缺氧）

浸潤細胞以 CD4+ T cells、macrophages、B cells 為主，位於 perivascular/perimysial。

#### 路徑 B：CTL 路徑（PM、IBM 為主）

CD8+ CTL 辨識肌纖維表面 MHC class I 呈現的自體抗原 → Perforin-granzyme B 直接殺傷非壞死肌纖維 → Fas-FasL pathway 誘導凋亡

PM 和 IBM 有 CD8+ T cells 的 clonal proliferation（抗原驅動），浸潤細胞位於 endomysial，直接侵入非壞死肌纖維。

#### 路徑 C：抗體直接損傷（IMNM）

Anti-SRP / Anti-HMGCR 抗體 → MAC（C5b-9）沉積於肌纖維 **sarcolemma**（非血管）→ 肌纖維壞死（幾乎無淋巴球浸潤）

Anti-HMGCR 的特殊機轉：Statins 上調再生肌肉中的 HMGCR 表現 → 成為自體免疫靶抗原 → 即使停藥後仍持續。

### Stage 5｜MHC Class I 過度表現與 ER Stress（非免疫路徑）

正常骨骼肌不表現 MHC class I，但 IIM 患者的肌纖維廣泛且早期出現 MHC class I 過度表現，甚至出現在遠離炎症浸潤的區域。

炎症細胞因子（TNF、IFNγ）+ HMGB1 → 誘導 MHC class I 大量表現 → MHC class I 分子在 ER 內滯留堆積 → **ER stress** → **NFκB 活化** → 進一步上調 MHC I、ICAM-1、MCP-1 → Autophagy（TRAIL）、Apoptosis（TWEAK）、Necroptosis → 肌纖維損傷（即使沒有 CTL）

這條路徑解釋了為何**炎症消退後肌無力仍持續**，也說明為何單純免疫抑制治療往往不足夠。

### Stage 6｜細胞激素放大迴路

主要促炎細胞激素：IL-1α/β、TNF、IFNα、IL-6、IL-15、IL-18

- **TNF** → 直接誘導肌細胞死亡 + 抑制 myogenic miRNAs（miR-1、miR-133a/b、miR-206）→ 抑制肌肉再生
- NFκB 調控的 miRNAs（miR-146a/b、miR-31、miR-223）→ 降低 dystrophin 蛋白 → 加重肌肉損傷
- **HMGB1** → 上調 MHC I + 降低 sarcoplasmic reticulum 的 Ca²⁺ 釋放 → 肌肉疲勞
- 代謝損傷：AMPD1 缺乏 → ATP/phosphocreatine 下降 → 肌肉無力與疲勞

## 四、各 MSA 的致病機轉

| 抗體 | 靶抗原功能 | 主要路徑 | 關鍵機制 |
|------|----------|---------|---------|
| Anti-synthetase（Jo-1等）| Aminoacyl-tRNA synthetase（蛋白質合成）| 血管 + 肺 | 肺部為起始地；perimysial fragmentation；perifascicular MHC II 上調 |
| Anti-Mi-2 | 核內 helicase（chromatin remodeling）| 血管（DM）| UV → complement → MAC → perifascicular atrophy；Type I IFN |
| Anti-MDA5 | Innate immune receptor（偵測 dsRNA）| Type I IFN innate pathway | 病毒樣觸發；Type I IFN surge；rapid progressive ILD；DAD |
| Anti-TIF1-γ | Transcriptional regulator（腫瘤抑制）| Paraneoplastic | 腫瘤抗原與再生肌肉交叉反應；feed-forward loop |
| Anti-NXP2 | Nuclear matrix protein（轉錄調控）| Paraneoplastic + 缺血 | 慢性炎症 → calcinosis；鈣訊號調控失常 |
| Anti-SRP | Signal recognition particle（蛋白質轉位至ER）| 抗體直接損傷 | MAC 沉積在 sarcolemma；ER stress；無 CTL |
| Anti-HMGCR | HMG-CoA reductase（膽固醇合成）| 抗體直接損傷 | Statin 上調再生肌肉中靶抗原；即使停藥仍持續；MAC 沉積 |
| Anti-cN1A | Cytosolic 5′-nucleotidase（嘌呤代謝）| CTL + 退化 | CD8 endomysial 攻擊；amyloid-β 堆積；autophagy 缺陷（IBM）|

## 五、肌肉外表現的致病機轉

### Heliotrope Rash & Gottron's Papules/Sign

DM 的補體/血管路徑不只傷害肌肉，同樣作用於皮膚微血管：

MAC 沉積於皮膚微血管 → 血管通透性增加、血管擴張 → 紅斑（erythema）+ 水腫（edema）

**Heliotrope rash**：眼瞼皮下組織疏鬆，水腫特別明顯；紫紅色 = 血管擴張（紅）+ 輕度缺氧（紫）。

**Gottron's papules/sign**：指關節伸側反覆機械摩擦 → 微創傷 → DAMPs 放大 type I IFN 炎症 → Papules（表皮增生 + 真皮炎症浸潤）vs Sign（單純血管擴張性紅斑）。

值得注意的鑑別診斷：DM 皮疹打**關節面（knuckles）**，SLE 皮疹打**指骨間（phalanges）**，因為兩者血管炎症分布模式不同。

### Mechanic's Hands

Anti-synthetase syndrome 的免疫反應起於肺部，但 tRNA synthetase 也存在於皮膚角質細胞，尤其在反覆摩擦的部位表現更多：

T cell 攻擊角質細胞的 tRNA synthetase → 慢性炎症 → 角質層過度增生（hyperkeratosis）→ 手指橈側 / 指腹粗糙、脫屑、皸裂

Mechanic's hands **無典型血管炎性紅斑**，形態反映的是 T cell 慢性角化性炎症（而非 DM 的補體/血管路徑）。

### Arthritis

滑膜細胞也表現 tRNA synthetase / MDA5 → 系統循環中的自體抗體和活化 T cells 侵犯滑膜 → 對稱性小關節炎

通常 non-erosive，因為缺乏 RA 中的 pannus formation 和大量 osteoclast 活化；少數 anti-Jo-1 患者可出現侵蝕性關節炎，機轉未明。

### Calcinosis

機轉分三個層次：

1. **微血管損傷 → 局部缺血**：DM 的 MAC 沉積 → 毛細血管減少 → 局部組織缺氧 → 粒線體釋放 → 磷酸鈣濃度局部改變

2. **組織壞死 → 異位鈣化**：壞死的肌肉 / 脂肪細胞釋放磷脂 → 成為 hydroxyapatite 結晶的成核位置 → dystrophic calcification

3. **炎症持續是關鍵驅動力**：有效控制炎症可阻止進展；好發於摩擦部位（肘、膝）= 反覆微創傷 + 局部炎症的疊加

**JDM 最嚴重的原因**：兒童再生能力強 → 更多壞死再生循環 → 更多磷脂釋放；Anti-NXP2 干擾細胞內鈣訊號調控，可能直接促進鈣化。

## 六、為何治療困難？

IIM 至少有三條平行的損傷路徑同時運作：

1. **免疫細胞直接攻擊**（CTL perforin/granzyme、MAC 沉積）→ 免疫抑制可壓制
2. **MHC I / ER stress 的非免疫機轉** → 免疫抑制無法完全修復
3. **代謝損傷**（ATP/phosphocreatine 下降、AMPD1 缺乏）→ 免疫抑制無法改善

這就是為何臨床上常見「炎症指標改善、CK 下降，但肌無力依然存在」的現象。未來的治療方向需要針對非免疫路徑（ER stress、代謝修復）進行多靶點聯合治療。

## 七、總結

```
遺傳背景（HLA + 非HLA基因）
        ↓
環境觸發（UV、感染、statin、腫瘤、吸入物）
        ↓
組織損傷 → DAMPs → TLR 活化
        ↓
Innate immunity：plasmacytoid DC → Type I IFN ↑
        ↓
Adaptive immunity：APC → CD4+ T → B cells → MSAs
                                 → CD8+ CTL
        ↓
效應機制（依亞型）：
  DM     → 補體活化 → MAC → 微血管損傷 → 缺血性肌肉損傷
  PM/IBM → CD8+ CTL → perforin/granzyme → 肌纖維直接破壞
  IMNM   → 抗體 + MAC 沉積在 sarcolemma → 壞死性肌病
        ↓
MHC I 過度表現 → ER stress → NFκB → 非免疫性肌纖維損傷
        ↓
細胞激素（TNF、IL-1、IFN）→ 抑制肌肉再生 + 放大炎症
        ↓
肌纖維壞死、萎縮、功能喪失 → 近端肌無力
同時 → 皮膚（皮疹、mechanic's hands）
      關節（non-erosive arthritis）
      鈣化（calcinosis）
      肺（ILD）
```

IIM 的本質是：**同一套「基因 × 環境」啟動的免疫風暴，因 HLA 型和觸發因子的不同，走向各自的抗體亞型和效應路徑，但最終以骨骼肌和多器官損傷為共同代價。**
