# IBD-Associated Arthritis(腸病性關節炎)臨床表現與治療

> 資料來源:Kelly's Textbook of Rheumatology, Ch079 Inflammatory Bowel Disease–Associated Arthritis
> 相關筆記:[其他SpA 致病機轉(ReA、IBD-A)](其他SpA_致病機轉_ReA_IBD.md)

---

## 一、臨床表現(Clinical Manifestations)

關節表現與其他 SpA 相同:**周邊/軸性關節炎 + enthesitis + dactylitis**,可任意組合,並可合併 IBD 與 SpA 的關節外表現(uveitis、erythema nodosum、pyoderma gangrenosum)。

### 周邊關節炎 — 歷史上分 Type I / Type II(Orchard 1998)
> ⚠️ 此分類**風濕科已不再使用**,改用 ASAS peripheral/axial SpA 標準;但文獻常引用,須認得。

| | **Type I(寡關節)** | **Type II(多關節)** |
|---|---|---|
| 分布 | 寡關節、**不對稱**、游走性 | 多關節、**對稱**(似 RA) |
| 好發關節 | 膝、踝(下肢大關節) | **MCP**(掌指關節) |
| 侵蝕性 | 罕見 | 進展時可侵蝕 |
| 病程 | **自限性**、短暫(易復發) | 持續性,常需免疫抑制治療 |
| **與腸道活性關係** | **平行**(flare 與腸炎同步;**全大腸切除可治癒**) | **獨立**於腸道活性 |
| 關節外表現 | 常合併 EN、uveitis | 較少 |

> 兩型可隨時間互相轉變(categories are fluid)。Type I 關節炎可早於腸道疾病數年出現。

### 軸性(axial)
- 範圍:無症狀 sacroiliitis → 發炎性下背痛 → 真正 AS。
- 發炎性背痛特徵:隱匿起病、晨僵、休息加重活動改善、夜間後半段痛醒、> 3 個月、< 45 歲發病。
- **與腸道活性多半獨立**;**全大腸切除對軸性無效**(對比 Type I 周邊)。

### Enthesitis / Dactylitis
- 是 SpA 的標誌特徵,但在 IBD-arthritis **比 PsA 少見**;dactylitis 約 15%(CD 與周邊關節炎者較多)。

### 其他相關肌肉骨骼疾病
- **SAPHO syndrome**(與 HLA-B27 相關;IBD 見於 4.8% SAPHO 患者)。
- **Hypertrophic osteoarthropathy**:遠端皮膚/骨增生、杵狀指 → 腸道嚴重度的不良預後徵象。

## 二、治療(Management)

**核心原則:風濕科 + 腸胃科共同照護(multidisciplinary)** → 縮短診斷延遲、腸與關節結果更好。需依「關節型態 + IBD 類型(UC/CD)與嚴重度」決定;**同一藥對 IBD 與對 SpA 的劑量常不同**。

> 大方向:周邊關節炎可用小分子/csDMARD;**軸性疾病若 NSAID 失敗則需 TNF 抑制劑**;所有軸性患者都應做物理治療。

### NSAIDs — 有爭議
- 理論上抑制 COX 可能誘發 IBD 復發,但**證據薄弱且矛盾**(大型 meta-analysis 未見關聯)。
- 結論:**緩解期 IBD 短期使用 NSAID 是合理選項**;出現腸道惡化即停藥;使用前先與腸胃科討論。

### 類固醇
- 關節內注射 → 寡關節型周邊關節炎有效。
- 全身性 → 僅用於嚴重周邊關節炎 flare,最短療程最低劑量。
- **軸性疾病避免使用**(無效 + 副作用)。

### 口服(csDMARD)
- 主要用於**周邊關節炎**,對**軸性無證據**。
- **Sulfasalazine**:周邊關節炎有效;UC 比 CD 更受益。
- **MTX**、**Azathioprine**:對周邊關節炎有部分效果。

### 考試重點口訣
- **「兩個避用」**:**Etanercept**(對 IBD 無效會誘發)、**IL-17i(secukinumab/ixekizumab)**(誘發/惡化 IBD)。
- **TNFi 分流**:Golimumab → **UC**;Certolizumab → **CD**;Adalimumab/Infliximab → **CD + UC** 皆可。
- **唯一同時對「軸性 + IBD」皆有效**:**JAK 抑制劑**(tofacitinib/upadacitinib)。
- **Ustekinumab / IL-23i**:對 IBD 有效但**對軸性無效**(只適周邊)。
- **Vedolizumab**:腸道專一,**換藥可能誘發關節炎**;不常規用於 IBD-arthritis。

### Probiotics / 糞菌移植
- IBD-arthritis **無證據支持**;UC(尤其 pouchitis)有輕微證據,CD 無;PsA 糞菌移植試驗無效。
