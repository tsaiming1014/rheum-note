## 一、Reactive Arthritis(反應性關節炎,ReA)

**核心概念:SpA 中唯一「已知明確感染觸發」的關節炎** — 經典的 host–pathogen interaction(宿主─病原交互作用)。

### 觸發病原(arthritogenic organisms)

| 類型 | 病原 | Attack rate |
|---|---|---|
| 腸道型 (post-enteric) | *Salmonella, Shigella, Campylobacter, Yersinia* | 1–30% |
| 泌尿生殖型 (post-venereal) | ***Chlamydia trachomatis*** (美國最常見) | 4–8% |
| 新興/非典型 | *C. pneumoniae*, SARS-CoV-2, HIV, *C. difficile* 等 | — |

### 致病機轉的關鍵爭點:「無菌性」其實不無菌

ReA 傳統被稱為 **"sterile" arthritis**,因為常規培養關節液長不出細菌。但分子研究(PCR / RT-PCR)推翻了這個觀念,而 **腸道型與披衣菌型機轉不同**:

**1. Chlamydia 型 — 活菌持續存在 (persistence)**

- 披衣菌會感染 **單核球 (monocytes)**,藉此從生殖道 **播散 (dissemination)** 到滑膜。
- 在滑膜內進入一種異常的 **「persistence(持續態)」** — 菌仍 **活著且有代謝活性**,但調控細胞分裂的基因發生 **轉錄停滯 (transcriptional arrest)**,急性感染時高表現的基因被強烈下調。
- 有趣的觀察:滑膜內的披衣菌偏偏是 **眼部 (ocular/trachoma) serovar** 而非生殖道 serovar,可能解釋 attack rate 低與常合併眼部症狀。
- **臨床意義**:抗 TNF 會 up-regulate 披衣菌複製 → Chlamydia 型 ReA 使用抗 TNF 要小心;且 chronic 型可用 doxycycline+rifampin 等 **組合抗生素**。

**2. 腸道型 — 只剩細菌碎片,非活菌**

- 滑膜可測到 **細菌 DNA / 降解產物(如 LPS)**,但 **測不到活菌**(Yersinia 是唯一可能例外)。
- 究竟是這些殘留抗原驅動發炎,還是引發 autoimmune response,**仍未定論**。

### 宿主遺傳:HLA-B27 角色被「高估」
- B27 **不是發病必要條件**(超過半數患者 B27 陰性)。
- B27 的真正作用是 → **促成慢性化 (chronicity)**、**更猛烈的急性症狀**、更易出現經典三聯症(關節炎+結膜炎+尿道炎)。

## 二、IBD-Associated Arthritis(腸病性關節炎)

**核心概念:gut–joint axis(腸─關節軸)** — 腸道發炎/免疫活化如何引發滑膜炎與 enthesitis。

### Step 1:腸道屏障破壞 + Dysbiosis(菌叢失衡)
- IBD 與 SpA 都有 **gut diversity 下降**,促發炎菌增加(如 ***Prevotellaceae*** ↑;axSpA 患者 *Ruminococcus gnavus*、*Dialister* ↑ 且與疾病活性相關)。
- 屏障破壞 → 微生物抗原暴露給黏膜免疫細胞 → IEC 釋放 **IL-1、IL-6 危險訊號** → DC 釋放 **IL-12、IL-23** → 驅動 **Th1/Th17** → 分泌 **IL-17、IFN-γ** → 局部組織損傷(即 IBD 或 subclinical gut inflammation)。

### Step 2:從腸到關節(From Gut to Joint)— 四個假說
1. **共享遺傳 + 共同環境觸發**(shared genetics)
2. **腸道通透性增加** → 微生物抗原 **系統性累積** → 刺激關節發炎
3. 黏膜免疫細胞接觸抗原 → 引發 **全身性發炎反應** → 觸發關節炎
4. **細胞移行 (cell trafficking)**:發炎細胞由腸道移行至滑膜 — 證據是腸與關節 **共享 macrophage / T cell**,黏膜 T 細胞對滑膜蛋白有親和性,且 T 細胞受體可同時對細菌與組織胜肽反應(分子模擬)。

### 共享的遺傳機轉(SpA 與 IBD 重疊)

| 基因 | 機轉 |
|---|---|
| **HLA-B27** | IBD-spondylitis 中 50–60% 陽性(周邊關節炎關聯較弱)。機轉:**homodimer 形成 → unfolded protein response → 危險反應(IL-1/6/TNF/IL-23/IL-17)或 autophagy**;另可呈現細菌胜肽 → T 細胞 **交叉反應 (cross-reactivity)** |
| **ERAP1/ERAP2** | 修剪胜肽供 MHC-I 呈現,與 B27 交互作用(ERAP1 與 AS 關聯強於 IBD) |
| **IL-23R polymorphism** | → Th17 ↑、IL-17/IL-22/IL-21 ↑(IBD 與 SpA 共享) |
| **CARD9** | dectin-1/2 訊號、辨識 β-glucan;突變影響腸道恆定;β-glucan 暴露可在 SKG mouse 誘發 IL-23 依賴性 SpA 表型 |
| **NOD2/CARD15** (16q12) | IBD 最一致的感受性位點 → S100 蛋白 (DAMPs) → 發炎(**calprotectin = S100A8/A9**,即糞便鈣衛蛋白標記);但與 AS 無明確關聯 |

### 關鍵共通主軸
**IL-23/IL-17 軸 + ROR-γt⁺ T 細胞** 是腸與關節共享的核心通路;再加上 **biomechanical stress/trauma** 作用於 **enthesis(附著點)**,被視為發炎的起始部位(IL-22 在此促進骨形成)。動物模型(HLA-B27 大鼠、CARD9 null mice)一致顯示:**germ-free 環境下不發病 → 微生物是發病的必要條件**。

## 三、兩者機轉對照(重點記憶)

| | **ReA** | **IBD-arthritis** |
|---|---|---|
| 觸發 | **特定感染**(腸道/泌尿) | **慢性腸道發炎 + dysbiosis** |
| 滑膜內病原 | Chlamydia:**活菌持續態**;腸道菌:**僅碎片** | 無特定活菌;**抗原/細胞移行** |
| 核心假說 | 持續感染 + 分子模擬 | gut–joint axis(4 假說) |
| HLA-B27 | 主導 **慢性化**,非必要 | spondylitis 型 50–60%,周邊型弱 |
| 共通軸 | IL-23/IL-17、enthesitis | IL-23/IL-17、enthesitis |
