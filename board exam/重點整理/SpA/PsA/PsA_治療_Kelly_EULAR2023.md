# PsA 治療總整理（Kelly base + 2023 EULAR）

> 基礎架構取自 Kelly（Ch78），凡 EULAR 2023 與 Kelly 不同、或額外強調之處，以 **(2023 EULAR)** 標示。

## 〇、總原則

PsA 為多 domain 疾病（peripheral arthritis、skin/nail、axial、dactylitis、enthesitis），**不是所有藥對所有 domain 有效**，依當下最嚴重 domain 選藥。Kelly 主要引用 **GRAPPA 2021**（domain-based）；另有 ACR、EULAR。

![PsA GRAPPA 2021 treatment schema](../../images/PsA%20GRAPPA%202021.png)

**(2023 EULAR)** 採 **treat-to-target**：目標 **remission（替代為 LDA）**；3 個月活性下降 ≥50%、6 個月達標；csDMARD 約 **12 週**評估是否續用。

## 一、NSAIDs

第一線症狀治療，缺 controlled trial 證據。對 axial 症狀有效（AS 資料顯示或可延緩 radiographic progression），**對 skin psoriasis 無效**，須權衡 renal/CV 風險。

## 二、Glucocorticoids

全身性使用**非實證、無 RCT**，但高達 35% 患者曾用口服類固醇。⚠️ **停藥可能誘發 psoriasis rebound**，原則為最短療程、最低有效劑量。Intraarticular GC 對 oligoarticular 或局部 enthesitis（如 plantar fasciitis）相當有效。

## 三、csDMARDs

整體對 peripheral joints、enthesitis、dactylitis 有小到中度效益；**axial 與 nail 不反應**。

- **Methotrexate**：多數風濕科首選。MIPA study 陰性（但劑量不足、dropout 高）；SEAM-PsA 單用 ACR20 50.7%、皮膚與 enthesitis 也改善。
- **Leflunomide**：RCT PsARC 59% vs 30%。
- **Sulfasalazine**：6 個 RCT，placebo 反應率也偏高。
- **Cyclosporine**：對皮膚有效，無 vs placebo 的 RCT。
- ⚠️ **Hydroxychloroquine 不建議**（無效且可能惡化 psoriasis）。

**(2023 EULAR)** MTX 仍為**首選 csDMARD**；劑量要足 **20–25 mg/週（≈0.3 mg/kg）+ folate**；有臨床顯著皮膚侵犯時優先 MTX。**MTX + leflunomide 併用不建議**（efficacy/safety 差）。**Dactylitis 無獨立條文**，視同 oligoarthritis（GC 注射 + csDMARD）。

## 四、bDMARDs

### TNF inhibitors

adalimumab、etanercept、infliximab、golimumab、certolizumab。對關節療效相近，改善 enthesitis/dactylitis/axial，**延緩 radiographic progression**。皮膚以 adalimumab/certolizumab/infliximab 較佳（PASI-75 ~59–64%），**etanercept 皮膚療效最差**。Biosimilars 已證 non-inferiority。MTX 併用主要見於 infliximab（提升 drug survival、減少 anti-drug antibody）。

**(2023 EULAR)** bDMARD + MTX 併用主要支持 **TNFi**；**uveitis 共病 → 首選 anti-TNF 單株抗體（adalimumab/infliximab），etanercept 不算**。

### IL-17 blockers

secukinumab、ixekizumab（anti-IL-17A），對關節與皮膚均佳。⚠️ **IBD 患者避用**（IL-17 於 IBD 試驗失敗/惡化，有誘發 case report）。

**(2023 EULAR)** **Bimekizumab（IL-17A/F）已核准用於 PsA**（Kelly 仍列為 future therapy）；安全性上 IL-17A、尤其 **IL-17A/F 易黏膜皮膚 candidiasis**。（→ [IL-17A 與 IL-17F 機轉差異與 Bimekizumab](PsA_IL17A_F機轉差異.md)）

### IL-23 / IL-12-23 blockers

**Ustekinumab**（p40）改善關節、皮膚、dactylitis、enthesitis、延緩破壞，⚠️ **axial SpA 試驗失敗 → 活動性 axial 不選用**。**Guselkumab、risankizumab**（p19）ACR20 與 PASI 佳。

**(2023 EULAR)** **axial 不建議 IL-23 / IL-12-23 pathway**（axSpA 試驗陰性）；**Risankizumab 已核准用於 Crohn's disease** → IBD 共病可選 IL-23i。

### T cell inhibitor

**Abatacept (CTLA4)**：ACR20 39.4%，但**皮膚療效差（PASI-75 16.4%）→ 臨床少用**。

**(2023 EULAR)** csDMARD 失敗後的 bDMARD 線**不建議 abatacept**，僅留作換藥（Rec 10）選項。

## 五、tsDMARDs（口服）

- **Apremilast (PDE4i)**：ACR20 40% vs 19%，皮膚中等，安全性佳。
- **JAKi：tofacitinib、upadacitinib**（upadacitinib ACR20 70.6% vs 36.2%）。

**(2023 EULAR) Apremilast 定位**：僅 **mild disease**（oligoarticular/entheseal、無預後不良因子、皮膚有限）且 bDMARD 與 JAKi 皆不適合時才用；**療效低 + 無結構保護證據**，故排最後。

**(2023 EULAR) JAKi 定位與安全警語**：定位於 **bDMARD 失敗之後**（不與 bDMARD 同階）；援引 RA **ORAL-Surveillance** 的 CV/malignancy 警訊（FDA/EMA 全球警告）；**HZ 風險高於 bDMARD**；⚠️ 謹慎族群 = **≥65 歲、現/過去長期吸菸、ASCVD/CV 風險、惡性腫瘤風險、VTE 風險**。

## 六、藥物選擇排序（EULAR 與 Kelly 最大差異）

Kelly 以 GRAPPA domain schema 平行陳述；**(2023 EULAR) 明確給出 step-up 順序與 non-MSK 導向選藥**，為考試重點。

**Peripheral arthritis 路徑：**

1. NSAIDs ± 局部 GC（短期）
2. **csDMARD（首選 MTX）**，12 週評估
3. 失敗 → **bDMARD**（關節療效不分優劣，依 non-MSK 表現選 mode of action）
4. 失敗或不適合 → **JAKi**（安全警語）
5. Mild + 上述皆不適合 → **apremilast**

**依 non-MSK 表現選藥（Rec 9）：**

| 共病/表現 | 優先 |
|---|---|
| 臨床顯著皮膚（BSA>10% 或顏面/生殖器/重大影響 QoL） | IL-17A / IL-17A/F / IL-23 / IL-12-23 |
| Uveitis | **anti-TNF 單株抗體** |
| IBD | anti-TNF 單株抗體 / IL-23i / IL-12-23i / JAKi（**避 IL-17i**） |

**Enthesitis（Rec 7）**：NSAIDs/局部 GC 無效 → bDMARD（各藥療效相近）。

**Axial（Rec 8）**：NSAIDs 無效 → **IL-17Ai → TNFi → IL-17A/Fi → JAKi**（擇一）；**csDMARD 與 IL-23/IL-12-23 無效**；IL-17Ai 排第一（唯一 axial PsA 試驗 MAXIMISE/secukinumab）。

## 七、其他

- **(2023 EULAR) 換藥（Rec 10）**：bDMARD/JAKi 反應不佳或不耐 → 換另一 bDMARD 或 JAKi，**可含同類內換一次**。
- **(2023 EULAR) 減量（Rec 11）**：持續緩解 → **tapering（減量，非停藥）**；停藥常復發；off-label。
- **Treat-to-target（Kelly：TICOPA）**：tight control 優於 standard care（ACR20 61.7% vs 44.6%），但不良事件較多。
- **Lifestyle（Kelly）**：減重 + 運動；高 BMI 與 anti-TNF 反應差相關。

## 八、高頻考點速記

1. **HCQ 不建議**（惡化 psoriasis）。
2. **IBD → 避 IL-17i**；**active axial → 避 IL-23/ustekinumab**。
3. **Uveitis → anti-TNF 單株抗體（非 etanercept）**。
4. **Abatacept、etanercept 皮膚療效差**。
5. **(2023 EULAR)** JAKi 排 bDMARD 之後（ORAL-Surveillance；HZ 風險高）。
6. **(2023 EULAR)** apremilast 僅 mild disease、無結構保護。
7. **(2023 EULAR)** MTX 20–25 mg/週；MTX+LEF 不併用。
8. **(2023 EULAR)** csDMARD/IL-23 對 axial 無效。
9. 達 ACR20 ≠ 達標，<35% 患者達 MDA → PsA 仍有 unmet need。
