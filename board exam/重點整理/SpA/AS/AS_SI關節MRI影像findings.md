# Axial SpA — 薦腸關節（SI joint）MRI 影像 Findings 與定義

> 整理自 Kelly's Textbook of Rheumatology Ch076（Axial Spondyloarthritis）、Ch058（Imaging in Rheumatic Diseases），病灶精確定義整合自其引用的 ASAS/OMERACT MRI working group 定義。

![SI joint MRI 各類病灶總覽](../../images/SI%20joint%20MRI.png)

## 一、MRI 序列與掃描平面

判讀 SI joint 必須同時涵蓋「發炎」與「結構」兩類資訊，因此需要兩種互補序列：

| 序列 | 偵測重點 |
|---|---|
| **Water-sensitive（STIR 或 T2 fat-suppressed）** | 發炎變化 — 主要是 **bone marrow edema（BME）** |
| **T1-weighted** | 結構性病灶 — erosion、fat lesion、sclerosis、ankylosis |

幾個關鍵原則：

- **診斷用 MRI 必須同時包含上述兩種序列。**
- **不需打顯影劑**：注射 gadolinium 對 SI joint / 脊椎的診斷準確度沒有額外幫助（但若要清楚顯示 synovitis 則為例外，見下文）。
- **建議平面**：semicoronal / oblique coronal（平行於 S2 椎體後緣），可加 semiaxial（垂直於 semicoronal）輔助判讀 SI joint。

## 二、SI joint 解剖分區（理解發炎病灶的基礎）

SI joint 分成兩個 compartment，發炎病灶的命名取決於發炎落在哪個結構：

| 分區 | 位置 | 結構 |
|---|---|---|
| **Synovial / cartilaginous compartment** | 前下 2/3（anteroinferior） | 真正的滑膜關節：透明軟骨（薦骨側）+ 纖維軟骨（腸骨側）、滑膜、關節腔、關節囊 |
| **Ligamentous compartment** | 後上 1/3（posterosuperior） | 骨間薦腸韌帶（interosseous ligaments）= 附著點，無滑膜 |

## 三、Active inflammatory lesions（活動性發炎病灶）

| 病灶 | 定義 / 影像特徵 |
|---|---|
| **Bone marrow edema (BME) / Osteitis** | **最能代表 active sacroiliitis 的 MRI finding。** 位於 juxta-articular（關節旁軟骨下骨）；STIR/T2FS 呈高訊號，對應 T1 呈低訊號。為診斷核心病灶。 |
| **Capsulitis** | 關節囊的發炎；訊號沿關節囊（前方和／或後方）分布，位於關節間隙之外。 |
| **Synovitis** | 關節腔內滑膜的發炎；訊號落在關節間隙之內（前下軟骨部）。 |
| **Enthesitis** | 附著點發炎；可影響 SI joint 韌帶部（interosseous ligaments）、interspinal / supraspinal ligaments。BME 出現在關節外、附著點處即代表 enthesitis。 |

## 四、Structural lesions（結構性病灶）

| 病灶 | 定義 / 影像特徵 |
|---|---|
| **Erosion（骨侵蝕）** | 軟骨下骨缺損；常與 BME 區域緊鄰出現。CT 為偵測 erosion / sclerosis 的 gold standard，MRI（尤其 T1）亦能顯示。 |
| **Sclerosis（硬化）** | 關節旁骨硬化，T1/T2 皆呈低訊號。 |
| **Fat metaplasia / Fat lesion（脂肪化生）** | 骨髓內 homogenous、明亮、邊界銳利（sharply marginated）的訊號（T1 高訊號）。是發炎消退後出現的中間產物，為新骨形成前的中間階段。 |
| **Backfill** | 特指 erosion 腔內的 fat metaplasia（fat metaplasia in an erosion cavity）。代表 erosion 被新組織填補修復，常見於 TNF / IL-17 / JAK 抑制劑治療後。 |
| **Bone spurs（骨刺）** | 結構性增生病灶之一。 |
| **Ankylosis（骨性僵直）** | 跨越關節的骨橋接，為病程末期表現（end stage）。 |

> **病程演進**：發炎期 erosion → fat metaplasia（含 backfill）→ new bone formation → ankylosis。

## 五、Capsulitis vs Synovitis 的差別

兩者都屬 active inflammation，常被混淆，核心差別在於**發炎位置**：

| | **Synovitis（滑膜炎）** | **Capsulitis（關節囊炎）** |
|---|---|---|
| **發炎部位** | 關節腔內的滑膜（synovial / cartilaginous compartment，前下部） | 關節囊本身（關節腔腹側/背側邊緣外緣，可向內外側延伸至骨膜） |
| **MRI 位置** | 訊號在關節間隙（joint space）之內 | 訊號在關節間隙之外、沿關節囊（前方和／或後方）呈線狀／囊狀 |
| **訊號** | water-sensitive 關節腔內高訊號；打顯影劑後 enhancement | 沿關節囊高訊號 / enhancement |
| **判讀陷阱** | 不打顯影劑時很難與單純積液（effusion）區分 → 故 optimal 顯示 synovitis 需要 gadolinium | 因發炎在關節外緣、向周邊延伸，位置與單純積液不同，較易辨識 |

一句話總結：

- **Synovitis** = 發炎在關節腔內的滑膜（間隙之內），非顯影下易與積液混淆。
- **Capsulitis** = 發炎在關節囊（間隙之外、沿前／後緣，可向骨膜延伸）。

## 六、ASAS「positive MRI」定義（分類標準用）

要符合 ASAS axial SpA 分類標準中的 **"active sacroiliitis on MRI"（ASAS-positive MRI）**：

1. SI joint 的 **BME 必須「highly suggestive of axial SpA」**，**且**
2. BME 須出現於 **至少 2 個部位（≥2 sites）**，**和／或** **至少 2 個連續切面（≥2 consecutive slices）**。

更新與補充重點：

- 後續資料顯示，將 **erosion（結構性損傷）納入判讀可進一步提升診斷效力**；但 SI joint fat lesion 及脊椎病灶單獨並未證實能提升診斷效力。
- ASAS MRI working group 強調：**輕微 BME 不可過度判讀**，須結合臨床情境，並考量是否合併 erosion 等結構變化。
- **ASAS 標準是「分類標準」而非「診斷標準」**：診斷不能僅靠 MRI，須結合臨床（症狀）、實驗室（HLA-B27、CRP）與影像綜合判斷。

## 七、False positive 與重要鑑別

| 情境 | 特徵 |
|---|---|
| **運動員、產後婦女的偽陽性 BME** | 位於關節 anterior upper（前上）和／或 posterior lower（後下），中段（midportion）被保留（spared） |
| **真正 axial SpA** | 發炎雖也可見於上述區域，但通常會同時侵犯關節中段；且常合併 fat metaplasia、erosion（常緊鄰 BME），大幅提高為 axial SpA 的可能性 |
| **Osteitis condensans ilii（重要鑑別）** | 多見於產後婦女；特徵為 anterior iliac sclerosis（前側腸骨硬化）+ 常見 anterior BME |
| **兒童** | 正常變異常模擬 BME，過半健康兒童符合成人 erosion 定義，需兒童專用標準 |

## 八、評分系統（scoring systems）

- **SI joint 發炎**：SPARCC（Spondyloarthritis Research Consortium of Canada）、Berlin score — 兩者對變化高度敏感；SPARCC 另有 SSS（結構損傷進展評估），可在治療 3 個月即區分有效與安慰劑。
- **脊椎發炎**：Berlin、SPARCC、Canada–Denmark — 後者同時評估椎體與後外側結構（costovertebral、facet、棘突/橫突）的發炎與損傷，常用於臨床試驗。
