# AS 疾病活性與評估工具

> 資料來源：Firestein & Kelley's Textbook of Rheumatology, Ch76 Axial Spondyloarthritis；切點與組成依 ASAS 標準定義補充。

---

## 一、總覽：為什麼需要這麼多工具？

AS 的症狀（脊椎疼痛、僵硬、活動受限）可能來自**當下的發炎活性**，也可能來自**累積的結構性損傷**。Kelly 強調這兩者必須區分，因此 ASAS（Assessment of SpondyloArthritis international Society）發展出不同面向的標準化工具。

一句話記憶法：

> **活性看 ASDAS / BASDAI，功能看 BASFI，活動度看 BASMI，附著點炎看 MASES，結構看 mSASSS，療效看 ASAS response。**

| 面向 | 主要工具 | 性質 |
|---|---|---|
| 疾病活性 | ASDAS（首選）、BASDAI | 複合 / 純主觀 |
| 身體功能 | BASFI | 純主觀 |
| 脊椎活動度 | BASMI | 客觀理學測量 |
| 附著點炎 | MASES | 客觀理學測量 |
| 結構損傷 | mSASSS（首選）、BASRI、SASSS | X 光評分 |
| 治療反應 | ASAS20 / 40 / 5-6 / partial remission | 複合改善標準 |

## 二、疾病活性（Disease Activity）

### BASDAI（Bath AS Disease Activity Index）

**純病人主觀**問卷，6 題，回想過去一週，每題 0–10（VAS 或 NRS）。

| 題目 | 內容 |
|---|---|
| Q1 | 疲倦 / 疲勞程度 |
| Q2 | 脊椎（頸 / 背 / 髖）疼痛程度 |
| Q3 | 周邊關節疼痛 / 腫脹 |
| Q4 | 觸壓痛（附著點炎 enthesitis）程度 |
| Q5 | 晨僵**嚴重度** |
| Q6 | 晨僵**持續時間**（0 = 無；10 = ≥2 小時）|

計算公式：

$$BASDAI = \frac{Q1 + Q2 + Q3 + Q4 + \dfrac{Q5 + Q6}{2}}{5}$$

- 先把兩題晨僵（Q5、Q6）平均成一項，再與前 4 題一起取平均 → **0–10 分**。
- **BASDAI ≥ 4** = 至少中度活性，傳統上是世界多數國家**啟動生物製劑（TNFi）的門檻**。

侷限：完全主觀、不含發炎指標（CRP / ESR）、難以區分「活性」與「損傷或共病造成的症狀」。

### ASDAS（AS Disease Activity Score）

ASAS 開發的**複合指標**，結合主觀項目與**急性期反應物**，較能客觀反映發炎。Kelly 指出 **ASDAS 與 CRP 與預後（影像學進展）相關，尤其在疾病早期**，這是它優於 BASDAI 之處。

5 個組成（4 主觀 + 1 客觀）：

1. 背痛（BASDAI Q2）
2. 晨僵持續時間（BASDAI Q6）
3. 病人整體評估（Patient Global, PtGA）
4. 周邊關節疼痛 / 腫脹（BASDAI Q3）
5. **CRP（mg/L）或 ESR** —— 首選 **ASDAS-CRP**

> 各項有加權係數，臨床多用計算器 / App，不需手算。

疾病活性狀態切點（**需熟記**）：

| ASDAS 值 | 活性狀態 |
|---|---|
| **< 1.3** | Inactive disease（無活性）|
| **1.3 – < 2.1** | Low disease activity（低活性）|
| **2.1 – ≤ 3.5** | High disease activity（高活性）|
| **> 3.5** | Very high disease activity（極高活性）|

變化量（改善）切點：

- **ΔASDAS ≥ 1.1** = clinically important improvement（臨床上有意義的改善）
- **ΔASDAS ≥ 2.0** = major improvement（顯著改善）

### ASDAS 臨床決策門檻

**國際（2022 ASAS–EULAR）—— 以 ASDAS 為主：**

- **啟動生物製劑門檻**：ASDAS **> 2.1**（即 high disease activity 以上）。
- **續用 / 治療反應**：治療後 **ΔASDAS ≥ 1.1**（clinically important improvement）視為有意義反應；**ΔASDAS ≥ 2.0** 為 major improvement。
- 治療目標（T2T）：達 **inactive disease（ASDAS < 1.3）** 或至少 **低活性（ASDAS < 2.1）**。

**台灣健保（附表二十一之一，2026/06）—— 仍以 BASDAI 為準：**

- **啟動門檻**：連續兩次 **BASDAI ≥ 6**，且 **ESR > 28 mm/hr** 且 **CRP > 1 mg/dL**，兩次間隔經 ≥ 4 週充分治療。
- **續用條件**：初次使用治療 **12 週**評估 BASDAI，與治療前比較**進步 ≥ 50% 或減少 ≥ 2 分**方可續用；其後每 **12 週**評估、重新申請。

> 重點區分：**國際指引已轉向 ASDAS（>2.1 啟動、Δ≥1.1 續用），但台灣健保目前條文仍用 BASDAI≥6 與 BASDAI 改善幅度。考試 / 臨床申請要分清楚是問哪一套。**

### 三大工具組成成分比較（BASDAI vs ASDAS vs ASAS40）

> 圖例：✅ = 包含（Y）；🟧 = 不包含（N）

| 組成成分 | BASDAI | ASDAS | ASAS40 |
|---|:---:|:---:|:---:|
| 脊椎疼痛 Spinal pain | ✅ | ✅ | ✅ |
| 發炎 / 晨僵 Inflammation | ✅ | ✅ | ✅ |
| **CRP（mg/L）或 ESR** | 🟧 | ✅ | 🟧 |
| 疲倦 Fatigue | ✅ | 🟧 | 🟧 |
| 關節疼痛 / 腫脹 Joint pain/swelling | ✅ | ✅ | 🟧 |
| 局部壓痛 / 附著點炎 Enthesitis | ✅ | 🟧 | 🟧 |
| 功能 Function | 🟧 | 🟧 | ✅ |
| 病人整體評估 Patient global | 🟧 | ✅ | ✅ |

重點解讀：

- **只有 ASDAS 含客觀發炎指標（CRP/ESR）** → 故較能反映真實發炎、與影像進展相關。
- **只有 BASDAI 含疲倦與附著點炎** → 純主觀、面向最廣但無客觀指標。
- **功能（Function）只出現在 ASAS40** → BASDAI、ASDAS 都不含。
- **病人整體評估（Patient global）**：ASDAS 與 ASAS40 有、**BASDAI 沒有**。

## 三、身體功能（Function）

### BASFI（Bath AS Functional Index）

評估病人**日常活動能力 / 身體功能**，純主觀問卷，**10 題**，回想過去一週，每題 0–10（0 = 容易，10 = 不可能）。

| # | 內容 |
|---|---|
| 1 | 彎腰穿襪子 / 撿東西（不靠輔助）|
| 2 | 彎腰從地上撿筆 |
| 3 | 不靠輔助伸手拿高處架子的東西 |
| 4 | 不用手撐從無扶手椅子站起來 |
| 5 | 平躺後不靠協助坐起 |
| 6 | 無支撐站立 10 分鐘而不適 |
| 7 | 不用扶手 / 輔具爬 12–15 階樓梯 |
| 8 | 不轉身體、僅轉頸回頭看肩後 |
| 9 | 從事費體力活動（物理治療、運動、園藝）|
| 10 | 一整天活動（在家或工作）|

計算：10 題取平均 → **BASFI 0–10**，分數越高功能越差。

> BASFI 反映功能受損，可能來自當下發炎，也可能來自累積結構損傷，不能單用它判斷活性。

## 四、脊椎活動度（Spinal Mobility）

### BASMI（Bath AS Metrology Index）

評估**脊椎與髖部活動度**，反映的是**結構性損傷／僵硬累積**，而非當下發炎活性。詳細檢查手法與正常切點見 [AS_理學檢查_BASMI.md](AS_理學檢查_BASMI.md)。

5 項測量（每項 0–10 分，取平均 → BASMI 0–10，越高越僵）：

| 測量項目 | 反映部位 |
|---|---|
| Tragus-to-wall distance（耳屏到牆）| 頸 / 胸椎、整體姿勢 |
| Cervical rotation（頸椎旋轉角度）| 頸椎 |
| Lumbar flexion — modified Schober（改良 Schober）| 腰椎前彎 |
| Lumbar side flexion（腰椎側彎）| 腰椎側向 |
| Intermalleolar distance（兩踝間距）| 髖關節外展 |

相關但**不計入 BASMI** 的理學測量（Kelly 強調）：

- **改良 Schober test**：站立時於 L5 棘突（薦髂窩、Venus 凹）做記號，往上 10 cm 再做一記號；最大前彎後若兩記號間距 **< 15 cm** 表腰椎前彎受限。
- **Chest expansion（胸廓擴張）**：第 4 肋間，最大吐氣後最大吸氣量；年輕慢性發炎性下背痛者 **< 5 cm** 強烈提示 AS。
- **Occiput / tragus-to-wall**：偵測頸椎前傾，與椎體骨折相關。

## 五、附著點炎（Enthesitis）

### MASES（Maastricht AS Enthesitis Score）

AS 最常用的附著點炎量表。對 **13 個附著點**逐點評估壓痛，每點 0（無）或 1（有壓痛），**總分 0–13**，分數越高代表附著點炎越廣泛。

> 與 BASDAI Q4 不同：Q4 是病人主觀整體感受，MASES 是醫師逐點理學檢查的客觀計分。

---

## 六、結構損傷（Structural Damage, X 光評分）

脊椎 X 光**不列入分類標準**，但用來追蹤結構進展。Kelly 提到 3 套已驗證評分法：

| 評分 | 全名 | 評估範圍 | 特點 |
|---|---|---|---|
| BASRI | Bath AS Radiology Index | 頸椎 + 腰椎（可含 SI 關節）| 含 SI 關節分級 |
| SASSS | Stoke AS Spine Score | 僅腰椎 | 範圍小 |
| **mSASSS** | modified SASSS | 頸椎前緣 + 腰椎前緣 | **對變化最敏感**，試驗首選 |

### mSASSS 詳解（最重要）

評估部位：側位 X 光上，**頸椎（C2 下緣～T1 上緣）+ 腰椎（T12 下緣～S1 上緣）的前緣椎角（anterior corners）**。

每個椎角給 0–3 分：

| 分數 | 影像表現 |
|---|---|
| 0 | 正常 |
| 1 | 侵蝕 erosion、硬化 sclerosis 或方形化 squaring |
| 2 | 骨贅 syndesmophyte（非橋接）|
| 3 | 橋接 bridging syndesmophyte（椎體間融合）|

- 總分範圍 **0–72**。
- **Syndesmophyte 是 AS 影像進展最重要的獨立預測因子**（Kelly）。
- mSASSS 約需 **2 年**才能顯示有意義變化，敏感度仍受限；MRI 能更早偵測發炎（骨髓水腫 BME）。

## 七、治療反應標準（ASAS Response Criteria）

ASAS 改善標準的 4 個核心 domain：**BASFI、晨僵（morning stiffness）、病人整體評估（PtGA）、疼痛（pain）**。

| 標準 | 定義 |
|---|---|
| **ASAS20** | 4 domain 中 ≥3 項改善 ≥20% 且 ≥10 units，且剩餘 domain 無 ≥20% 且 ≥10 units 惡化 |
| **ASAS40** | 4 domain 中 ≥3 項改善 ≥40% 且 ≥20 units，且剩餘 domain 無任何惡化 |
| **ASAS 5/6** | 6 domain（上述 4 項 + 急性期反應物 + 脊椎活動度）中 ≥5 項改善 ≥20% |
| **ASAS partial remission** | ASAS20 的 4 個 domain 數值皆 **< 20 units** |

---

## 八、臨床應用門檻（2022 ASAS–EULAR）

- 啟動生物製劑：傳統用 **BASDAI ≥ 4**；2022 更新指引**轉向採用 ASDAS ≥ 2.1（高活性）**作為門檻。
- nr-axSpA（非放射性）：EMA / ASAS–EULAR 另要求**客觀發炎證據**（MRI 骨髓水腫 BME 和/或 CRP 升高）。
- 美國：僅需風濕科醫師判斷 + ICD 診斷碼，不強制客觀發炎。
- 治療目標（Treat-to-Target）：以 ASDAS 定義，例如達 **inactive disease（ASDAS < 1.3）** 或 **低活性（ASDAS < 2.1）** 後考慮減量 / 停藥研究。
