# axSpA 治療重點整理

**來源：** Kelly's Textbook of Rheumatology Ch076 + 2026 ACR/SAA/SPARTAN 治療指引
**更新日期：** 2026-06

---

## 一、治療目標與原則（ASAS–EULAR 2022 總則）

axSpA 是可能嚴重影響生活品質的疾病，需要**多專科整合管理**，由風濕科醫師協調。治療目標為：

- 最大化健康相關生活品質
- 控制症狀與發炎
- 預防進行性結構損傷
- 維持功能與社會參與

治療必須基於**醫病共識**，並根據個別患者的軸向/周邊/關節外表現及合併症調整。

---

## 二、疾病活動度評估

### 主要工具

| 工具 | 說明 |
|------|------|
| **BASDAI** | 0–10 分，≥4 為中重度活動（舊版啟動生物製劑門檻） |
| **ASDAS** | 整合 CRP，≥2.1 為活動性疾病；2022 ASAS–EULAR 建議以此取代 BASDAI |
| **ASAS-20/40** | 臨床試驗常用療效評估 |
| **ASAS partial remission** | 4 個域均 <20 分 |

### Treat-to-Target（T2T）

TICOSPA 試驗以 **ASDAS < 2.1** 為 target，每 4 週評估，雖主要終點未達統計顯著，但 2022 ASAS–EULAR 仍建議追蹤 ASDAS 至少有 **ΔASDAS ≥ 1.1**（clinically important improvement）再繼續治療。

**2026 ACR 建議反對嚴格 T2T 策略**，改以 **symptom-prompted treatment** 為主，目標為 **低疾病活動度**（low disease activity，即 ASDAS < 2.1）即可，不強制定期調藥。 (2026 ACR 新增)

## 三、非藥物治療

### 物理治療（強烈建議）

- **主動**物理治療（supervised exercise）優於被動（按摩、熱療、超音波）
- 水中運動優於陸地負重運動
- 家庭自主運動（back exercise、伸展）優於不運動
- 監督式團體物理治療優於個別治療
- **每日俯臥 15–30 分鐘**：對抗駝背趨勢
- 睡眠姿勢：建議完全仰臥、硬床墊、頸枕勿過高

### 各種運動方式的建議 (2026 ACR 新增)

- 有氧運動 > 非特定運動（條件建議）
- 伸展運動 > 非特定運動
- 瑜伽、太極均條件建議
- 水中運動 > 負重運動
- **建議重複療程，優於單一療程** (2026 ACR 新增)
- TENS：條件建議 (2026 ACR 新增)

### 不建議的治療

- **強烈不建議**：脊椎手法治療（尤其脊椎融合/骨質疏鬆患者）
- 條件不建議：牽引（traction）、radiofrequency ablation (2026 ACR 新增)
- 條件不建議：替代醫學（naturopathic、homeopathic、ayurvedic） (2026 ACR 新增)
- 條件不建議：大麻素（THC/CBD） (2026 ACR 新增)
- 條件建議：針灸與按摩 (2026 ACR 新增)

### 衛教與自我管理

- 患者教育、自助團體、戒菸
- 告知 AAU 可能在疾病任何時期發生（尤其 HLA-B27 陽性者）
- **脊椎融合者建議配戴醫療警示手環**（告知急救人員「脆性脊椎」） (2026 ACR 新增)
- 避免接觸性運動（脊椎融合/骨鬆時） (2026 ACR 新增)
- 諮詢職業調整（駕駛、重勞力工作） (2026 ACR 新增)
- **抑鬱症篩查與管理**（疾病活動及難治性患者） (2026 ACR 新增)
- 心血管疾病風險篩查 (2026 ACR 新增)
- **不建議**：常規心電圖篩查心傳導缺損；不建議心超篩查瓣膜疾病 (2026 ACR 新增)

## 四、藥物治療——第一線：NSAIDs

NSAIDs 為活動性 axSpA 的**標準第一線藥物**。

- 全劑量、**持續使用**優於按需使用（尤其需要控制症狀者）
- 無特定 NSAID 偏好；若需夜間濃度可選**長效製劑**
- 一種無效 → 換另一種 NSAID（不建議兩種同時使用）
- 2–4 週評估療效

### 特殊族群注意

- IBD：避免 continuous NSAIDs（條件不建議）；若需使用優先選 COX-2 選擇性 NSAIDs
- CV/腎臟疾病：謹慎評估

### 關節腔注射 glucocorticoid

- 局部注射（SIJ、周邊關節、附著點旁）：條件建議
- **全身性 glucocorticoid：強烈不建議**（無效且風險高）
- **ACTH：強烈不建議** (2026 ACR 新增)

---

## 五、藥物治療——何時升級至 b/tsDMARD

### 傳統啟動門檻

| 標準 | 說明 |
|------|------|
| **BASDAI ≥ 4** | 多數國家傳統要求 |
| **ASDAS ≥ 2.1** | 2022 ASAS–EULAR 新建議門檻 |
| 美國（FDA） | 只需風濕科診斷，無固定數字 |

**nr-axSpA 額外要求（EMA/ASAS–EULAR）**：需有客觀發炎證據（MRI BME 或 CRP 升高）

### 高風險患者可直接 bDMARD 作為初始治療 (2026 ACR 新增)

有以下情形的患者，**不需先試 NSAIDs**，可直接以 bDMARD 為初始治療：

- 已有 **syndesmophytes**
- **CRP 持續升高**

## 六、藥物治療——第二線：b/tsDMARD

### TNFi（5 種已核准）

| 藥物 | 劑型 | 用法 |
|------|------|------|
| Infliximab | 靜脈 | 5 mg/kg，第 0/2/6 週後每 6–8 週 |
| Etanercept | 皮下 | 50 mg qW |
| Adalimumab | 皮下 | 40 mg q2W |
| Golimumab | 皮下/靜脈 | 50 mg q4W |
| Certolizumab pegol | 皮下 | 200 mg q2W 或 400 mg q4W |

- 均可**單藥治療**，不需合併 MTX（IFX 因最高免疫原性，可考慮低劑量 MTX）
- 無特定 TNFi 優先

### IL-17i

| 藥物 | 機轉 | 核准疾病 |
|------|------|---------|
| Secukinumab | IL-17A | r-axSpA、nr-axSpA |
| Ixekizumab | IL-17A | r-axSpA、nr-axSpA |
| **Bimekizumab** | IL-17A + IL-17F（雙重） | r-axSpA、nr-axSpA、PsA |

- 2022 ASAS–EULAR：IL-17i 與 TNFi **側排同等**地位
- 2026 ACR：TNFi = IL-17i（同等建議，不分優劣）

### JAKi

| 藥物 | 靶點 | 資料 |
|------|------|------|
| Upadacitinib | JAK1 | r-axSpA + nr-axSpA Phase III，FDA 核准（需 TNFi 失敗） |
| Tofacitinib | JAK1/3 | r-axSpA Phase III，無 nr-axSpA FDA 適應症 |

- 2019 ACR：條件建議
- **2026 ACR：強烈建議**（升格，與 TNFi/IL-17i 同等強度） (2026 ACR 新增)
- 仍建議 TNFi/IL-17i 優先於 JAKi（條件建議 TNFi > JAKi、IL-17i > JAKi）
- **有 CV 風險或年齡 ≥65 歲：建議避免 JAKi**（2022 ASAS–EULAR）

### 藥物選擇受關節外表現影響

| EMM | 偏好 |
|-----|------|
| IBD（活動性或非活動性） | TNFi 單株抗體（IFX/ADA/CZP for Crohn；IFX/ADA/GOL for UC）；JAKi 也適合 (2026 ACR 新增) |
| 反覆/慢性葡萄膜炎 | TNFi 單株抗體（ADA/IFX 最佳）；若復發 → 繼續 TNFi + 局部治療 |
| 乾癬 | IL-17i 優先 |

### 明確無效/不建議的藥物

- **IL-23i（risankizumab 等）：強烈不建議**（Phase III 均失敗）(2026 ACR 新增強調)
- Abatacept、Rituximab、Tocilizumab、Sarilumab、Belimumab、Anakinra：不建議
- **Bisphosphonates**（用於治療 axSpA）：條件不建議
- **csDMARD**（MTX、SSZ、HCQ、AZA、LEF）：純軸向疾病不建議；周邊關節炎可考慮 SSZ

## 七、治療無效時的策略

### 換藥原則

| 情境 | 建議 |
|------|------|
| 第一個 TNFi **原發性**無效 | 換 IL-17i（或 JAKi） |
| 第一個 TNFi **繼發性**無效 | 換另一 TNFi 或 IL-17i |
| b/tsDMARD 失敗 | 換**不同作用機轉**的藥物 |
| 換藥前 | **不建議**加 csDMARD 而非換藥 |
| **不建議** | 換同一 TNFi 的 biosimilar |

### ≥2 個藥物類別失敗時 (2026 ACR 新增)

重新評估原因：

- 診斷是否正確？
- 治療依從性（adherence）？
- **Nociplastic pain**（中樞敏感化）？

### 雙重標靶療法 (2026 ACR 新增)

難治性患者可考慮：

- bDMARD + tsDMARD
- 兩種 bDMARD
- （乾癬合併 axSpA：bDMARD + tsDMARD 或 JAKi + apremilast/deucravacitinib）

### 劑量調整 (2026 ACR 新增)

難治性患者可考慮超過 FDA 核准最高劑量（條件建議）

---

## 八、Nociplastic Pain 的管理（全新章節）(2026 ACR 新增)

axSpA 合併 nociplastic pain（中樞敏感化）時，傳統抗發炎藥物效果有限，需要額外處理：

| 建議強度 | 治療 |
|---------|------|
| 條件建議 | Gabapentinoid、SSRI/SNRI、muscle relaxant |
| 條件建議 | 認知行為治療（CBT） |
| **強烈不建議** | 鴉片類止痛藥（over b/tsDMARD） |

---

## 九、緩解期（Inactive axSpA）管理

| 議題 | 建議 |
|------|------|
| NSAIDs | 改為**按需使用**（不建議持續使用） |
| b/tsDMARD | 可考慮**減量/延長間隔（tapering）** |
| **突然停藥** | **強烈不建議** |
| b/tsDMARD + NSAIDs | 可停 NSAIDs，只留 b/tsDMARD |
| b/tsDMARD + csDMARD | 可停 csDMARD |
| 停藥後復發 | 重啟**同一藥物**優於換藥 |
| 緩解期例行 MRI | **強烈不建議**（以確認緩解為由） |
| 不活動期例行脊椎 X 光 | **不建議**定期追蹤（每 2 年） |

---

## 十、骨質健康 (2026 ACR 新增)

- **強烈建議**跌倒風險評估與衛教
- **DXA 篩查**：條件建議；有 syndesmophytes/脊椎融合時，**同時掃脊椎 + 髖部**（避免只掃髖部）
- 注意：syndesmophytes 可造成腰椎 DXA 假性偏高
- 不建議每年一次 DXA（可每 2～5 年）
- 骨鬆確診後：可加 anabolic 或 antiresorptive 藥物（搭配 b/tsDMARD）

---

## 十一、手術與圍手術期

| 情境 | 建議 |
|------|------|
| 嚴重髖關節病變 | **全髖置換（THA）**（強烈建議） |
| THA 後預防 HO | **Celecoxib**（條件建議） (2026 ACR 新增) |
| THA 後預防 HO | **不建議**局部放射治療 (2026 ACR 新增) |
| 嚴重駝背 | 脊椎截骨手術（elective osteotomy，條件建議） |
| 頸椎受累 + 全身麻醉 | **強烈建議**術前告知麻醉科「脆性脊椎」，取得知情同意 (2026 ACR 新增) |

---

## 十二、共同照護原則 (2026 ACR 新增)

| 合併症 | 共管要求 |
|--------|---------|
| IBD | 必須與 **gastroenterologist** 共管 |
| 葡萄膜炎（活動性） | 必須與 **ophthalmologist** 共管 |
| 乾癬（活動性） | 必須與 **dermatologist** 共管 |
