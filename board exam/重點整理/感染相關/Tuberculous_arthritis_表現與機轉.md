# Tuberculous Arthritis — 結核性關節炎表現與機轉

## 背景

**Musculoskeletal TB（骨骼肌肉結核）** 約佔所有 TB 病例的 1-3%，佔肺外結核（extrapulmonary TB）的 10-15%。感染途徑主要是**由肺部原發病灶血行性擴散（hematogenous spread）**，也可經淋巴系統擴散，或由鄰近的骨髓炎直接蔓延。

## 骨骼肌肉結核的分型

| 型態 | 說明 |
|---|---|
| **Tuberculous spondylitis（Pott's disease）** | 最常見型態，侵犯脊椎（好發**胸腰椎交界**），從椎體前緣開始破壞，可造成椎體塌陷、**gibbus deformity（駝背畸形）**，並沿腰大肌形成 **paravertebral cold abscess（冷膿瘍）**，可延伸至腹股溝 |
| **Tuberculous arthritis（周邊關節結核）** | 通常單一關節，慢性病程，好發**承重大關節（髖、膝最常見）** |
| **Poncet's disease** | 屬於**反應性關節炎**，是身體其他部位有 TB 感染時誘發的關節炎，關節本身沒有結核菌感染，機轉類似其他反應性關節炎 |
| Tuberculous osteomyelitis | 骨髓炎型態 |
| Tuberculous tenosynovitis/bursitis | 可見經典的 **"rice bodies"（米粒體）**於肌腱鞘內 |

## 臨床表現

- **慢性、隱匿性（indolent）發作**，病程以數週到數月計算，與化膿性關節炎的急性劇烈表現截然不同
- **約 90% 為單關節侵犯**，好發承重關節：**髖、膝**最常見，其次踝、腕、肘
- 漸進性疼痛腫脹，可合併輕微全身結核症狀（低燒、夜間盜汗、體重減輕），但常常表現隱晦、容易被忽略
- **關節破壞速度緩慢**，與化膿性關節炎的快速軟骨破壞不同

## 經典影像學表現：Phemister Triad

1. **關節周圍骨質疏鬆（juxta-articular osteoporosis）**
2. **周邊性骨侵蝕（peripherally located bone erosion）**——從關節邊緣（bare area，缺乏軟骨保護處）開始侵蝕，而非像化膿性關節炎那樣瀰漫性破壞中央軟骨
3. **關節間隙逐漸、緩慢變窄**（相對於化膿性關節炎因蛋白酶快速分解軟骨，造成關節間隙早期就快速狹窄）

## 致病機轉

- *M. tuberculosis* 經血行性擴散，最初好發於**血流豐富的 subchondral bone / metaphysis（骨骺端）**，之後才侵犯到關節本身
- 與化膿性細菌不同，結核菌**不會分泌能快速分解軟骨的酵素**，關節破壞主要是透過**肉芽腫性發炎反應（granulomatous inflammation）**——乾酪樣壞死肉芽腫（caseating granuloma）、Langhans giant cell、類上皮細胞（epithelioid histiocyte）——這是一種 **delayed-type hypersensitivity、Th1 為主導**的慢性免疫反應，破壞速度遠比化膿性感染緩慢
- 滑膜先產生慢性肉芽腫性發炎，形成類似 pannus 的反應組織，從**關節邊緣（marginal）**開始侵蝕，逐漸向中心破壞，這也解釋了為何影像上侵蝕多從周邊開始

## 診斷

- **關節液分析**：白血球中度上升，**以淋巴球為主**（不同於化膿性關節炎的嗜中性球為主），低葡萄糖、高蛋白
- **AFB smear（抗酸染色）敏感度低（約 20%）**；培養較敏感但**耗時數週**；**PCR（如 Xpert MTB/RIF）**較快速且敏感度尚可
- **滑膜切片**（找乾酪樣肉芽腫）診斷率通常優於單純關節液檢查
- 需同時評估是否合併肺結核（但骨骼肌肉結核病人診斷當下，肺部不一定有活動性病灶，可能是潛伏結核的再活化）

## 與化膿性關節炎鑑別重點

| | 化膿性關節炎 | 結核性關節炎 |
|---|---|---|
| 發作速度 | 急性、劇烈 | 慢性、隱匿 |
| 關節液細胞 | 嗜中性球為主 | 淋巴球為主 |
| 軟骨破壞機轉 | 細菌酵素快速分解 | 肉芽腫性免疫反應，緩慢 |
| 關節間隙變化 | 早期快速狹窄 | 緩慢狹窄，周邊侵蝕為主 |
