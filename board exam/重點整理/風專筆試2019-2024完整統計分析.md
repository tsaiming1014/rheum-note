## 二、各主題年年必考重點精析

---

### 🔴 1. SLE / Lupus Nephritis / APS（最高頻，9%）

| 考點 | 出現年份 |
|------|----------|
| 2019 ACR/EULAR SLE 分類標準（算分） | 2022、2023、2024 |
| Lupus Nephritis 分類（Class I–VI）、活性指標 | 2021、2022、2023、2024 |
| LN induction/maintenance 治療（CYC vs MMF） | 2019、2021、2022、2023 |
| SLE 懷孕用藥（HCQ、SSZ、MMF禁忌） | 2019、2020、2021、2022 |
| Neonatal lupus、CHB 發生率（2%）、複發率 | 2019、2020、2022 |
| Anti-SSA/SSB 的臨床意義 | 2019、2021、2024 |
| SLEDAI-2k、BILAG、flare 定義 | 2021、2022、2024 |
| SLE photosensitivity → Anti-SSA/Ro | 2019、2021、2023 |
| SLE 心血管風險（脂質弔詭、疾病活性） | 2019、2023 |
| SLE with HLH 治療 | 2022、2023 |
| APS 診斷標準（Revised Sapporo vs 2023 new） | 2021、2022、2023 |
| Catastrophic APS 治療（抗凝+類固醇+IVIG） | 2023、2024 |
| SLE PAH（CXR 肺動脈擴大、RVSP） | 2022、2024 |
| Belimumab vs Anifrolumab 比較 | 2024 |

**高頻陷阱：**
- 2019 SLE criteria：fever 2分、autoimmune hemolysis 4分 → 總分 ≥10 才符合
- LN poor prognosis factor：**fibrous crescents（chronic）不算 active**；active 是 cellular/fibrocellular crescents
- SLE with MMF：**不可打 VZV live vaccine**
- SLE 懷孕：MMF 禁用（致畸胎）；HCQ 可用；SSZ 搭配 folic acid 可用

### 🔴 2. SpA / AS / PsA（8%，最多細節）

| 考點 | 出現年份 |
|------|----------|
| ASDAS 計算與臨床意義（改善 1.1 有意義） | 2019、2021、2022、2023、2024 |
| mSASS 評分（滿分72、C1 upper-C7 lower + L1-L5） | 2021、2022、2024 |
| ASAS axial SpA 分類標準 | 2019、2021、2023 |

| IL-17 在 SpA 致病機轉 | 2019、2021、2022、2023、2024 |
| IL-23 對 AS 無效（但對 PsA 有效） | 2021、2022、2023 |
| PsA：GRAPPA 2021（IBD→禁用IL-17；Uveitis→monoclonal TNFi） | 2021、2022、2023 |
| PsA biologics 選擇情境題 | 2022、2023、2024 |
| AS 復健（伸展、水中、漸進肌力訓練） | 2019、2021、2022、2023 |
| nr-axSpA 與 AS 的差異 | 2022、2023 |
| Axial SpA 首選治療 = **NSAID** | 2023、2024 |
| SpA 骨新生機轉（IL-22、BMP、Sclerostin） | 2019、2023、2024 |
| SpA 合併 IBD bDMARD 選擇 | 2021、2022、2023 |
| JAKi 在 AS（台灣：upadacitinib + tofacitinib） | 2023、2024 |

**高頻陷阱：**
- BASFI 評估**功能**（非疾病活性）；ASDAS/BASDAI 評估活性
- NSAID 不反應不是 SpA 的 feature（SpA 特徵是 NSAID 有反應）
- Etanercept：**不適用**於合併 IBD 或 Uveitis 的 SpA
- AS：**不建議**全身類固醇；局部注射可行

### 🔴 3. RA（8%，治療與影像為核心）

| 考點 | 出現年份 |
|------|----------|
| DAS28 計算（公式、疾病活性分層） | 2019、2021、2022、2023 |
| ACPA（anti-CCP）：filaggrin → citrullination；HLADRB1 + smoking | 2019、2021、2022、2023 |
| MTX 使用前評估、懷孕停藥3個月 | 2019、2020、2021 |
| RA sonography：OMERACT grading（GS/PD） | 2019、2020、2021、2022 |
| PsA vs RA 超音波/X-ray 差異 | 2019、2020、2021 |
| RA 頸部 C1-C2 subluxation（flexion >3mm） | 2019、2020、2021 |
| RA-ILD（停 MTX、可用 RTX/Abatacept） | 2021、2023 |
| Felty syndrome（RA + 白血球低 + 脾臟腫大） | 2019 |
| RA 共病（CV risk、ILD、感染） | 2021、2022、2023 |
| RA biologics（bDMARD failure 後選擇） | 2019、2021、2022 |
| RA secondary failure bDMARD 原因 | 2021 |
| IRAE 關節炎（ipilimumab/nivolumab → **可用類固醇**） | 2021、2022、2023 |

**高頻陷阱：**
- ACPA 的 antigen 是 citrullinated peptide，filaggrin 只是其中一個來源（非唯一 antigen）
- RA 臨床 remission 時超音波可發現 subclinical synovitis → 可輔助減藥決策
- IRAE（免疫檢查點抑制劑副作用）→ **可以**用類固醇

### 🔴 4. 藥物學（7.5%，幾乎每題都是重點）

| 藥物 | 重要考點 |
|------|----------|
| **MTX** | 懷孕停3個月；腎功能 Cr>1.6 需注意；AE 考慮停藥 |
| **Leflunomide** | 備孕禁用（cholestyramine washout）；eGFR<30 謹慎；ALT正常可用 |
| **AZA** | TPMT deficiency 禁用；與 allopurinol DDI（xanthine oxidase） |
| **HCQ** | 視網膜毒性（retinal maculopathy）；half-life 40 天；COVID 無效 |
| **Rituximab** | 消耗 pre-B 到 mature B（不含 plasma cell）；ADCC/complement/direct lysis |
| **Baricitinib** | JAK1/JAK2；JAK2 → 貧血；COVID 可用 |
| **Tofacitinib** | JAK1/JAK3；CV risk 警語（二線） |
| **Upadacitinib** | JAK1 selective；AS、PsA 已核准 |
| **Denosumab** | anti-RANKL；不是 anti-RANK；停藥要接 bisphosphonate |
| **Tocilizumab** | anti-IL-6R；COVID 可用；CHF 謹慎 |
| **Dupilumab** | anti-IL-4Rα；結膜炎副作用 |
| **Pegloticase** | recombinant uricase；注意 **G6PD deficiency** |
| **Febuxostat** | 與 warfarin 有 DDI；與 AZA 可以合用（不走 XO） |
| **Secukinumab** | anti-IL-17A；IBD 禁忌 |
| **Ustekinumab** | anti-IL-12/23；IBD 適用 |
| **CYC** | DNA alkylating（非 purine analogue）；LN 用 Euro-lupus vs NIH protocol |
| **類固醇** | Osteonecrosis（SLE 好發股骨頭）；cataract + 眼壓升高；不造成 PU（RR 僅1.1） |

### 🔴 5. 基礎免疫學（6%，每年必考）

| 考點 | 出現年份 |
|------|----------|
| **T cell subsets + cytokines** 配對（Th1/Th2/Th17/Treg/Tfh） | 2019、2021、2022、2023、2024 |
| **JAK-STAT pathway**（JAK1/2/3/TYK2 + STAT）；**TNF 不走 JAK** | 2019、2021、2022、2023、2024 |
| TCR 活化→calcineurin→NFAT（非 NFkB） | 2021、2022 |
| miRNA 抑制 mRNA translation | 2019、2021、2022 |
| CTLA-4 抑制（非促進）T cell activation | 2022 |
| PD-1/PD-L1 機轉 | 2022 |
| Inflammasome/Caspase-1 → IL-18（非IL-1α） | 2024 |
| Ferroptosis（鐵離子） | 2024 |
| NETosis 與 RA/SLE | 2021、2022、2023 |
| BAFF 受體（TACI/BCMA/BAFFR）；APRIL 差異 | 2024 |
| Complement 活化路徑 | 2022、2023 |
| IL-17A vs IL-17F；receptor | 2024 |

**高頻陷阱：**
- Th1 → IFN-γ, IL-2（非 TNF-α）；Tfh → IL-21（非IL-17）；Treg → TGF-β
- JAK pathway：IL-2、IL-4、IL-6、IL-12、IFN 都走 JAK；**TNF-α 不走**
- TCR activation → calcineurin → NFAT（Cyclosporine 抑制 calcineurin，不是 AP-1）

## 三、重要陷阱題整理（跨年反覆出現）

| 陷阱 | 正確答案 |
|------|---------|
| CTLA-4 作用 | **抑制** T cell activation（非促進） |
| Tfh 的 cytokine | **IL-21**（非 IL-17） |
| JAK + TNF-α | TNF 不走 JAK-STAT |
| ACPA antigen | citrullinated peptide（filaggrin 只是來源之一） |
| DM muscle biopsy 主要 cell | **CD4+ T cells**（非 CD8） |
| IIM dysphagia 原因 | 咽部肌肉無力（非 vocal cord palsy） |
| IRAE 關節炎用類固醇 | **可以用**（不影響腫瘤治療） |
| SpA NSAID 無效 | **不是** SpA 的 feature（SpA 對 NSAID 有反應） |
| mSASS 滿分 | **72 分**（非 48 分） |
| BASFI | 評估**功能**（非疾病活性，ASDAS 才是活性） |
| LN poor prognosis factor | **fibrous crescents**（chronic lesion）→ 差預後 |
| GCA + visual loss 類固醇 | 需**高劑量或 pulse**（非 20mg 起） |
| MAINRITSAN RTX vs AZA | RTX **優於** AZA |
| Pegloticase 禁忌 | **G6PD deficiency** |
| Neonatal lupus CHB 發生率 | **2%**（複發率 18-20%） |
| SERM + ONJ | **SERM 不易**造成 ONJ（bisphosphonate 和 denosumab 才會） |
| Osteonecrosis in SLE | 好發**股骨頭**（非踝關節） |
| CPPD birefringence | **Strong positive**（MSU 是 strong negative） |

## 四、備考策略建議

### 最優先（必拿分）
1. SLE 2019 criteria 算分 + LN 治療 + 懷孕用藥
2. JAK-STAT pathway（哪些 cytokine 走、哪些不走）
3. DAS28 計算（每年至少1題情境題）
4. SpA/PsA biologics 選擇（GRAPPA 2021）
5. 藥物副作用與 DDI 大全

### 高 CP 值主題
- IIM MSA 抗體臨床對應（固定3-4題）
- Gout 2015/2020 criteria + 降尿酸藥選擇
- SSc 抗體 + ILD/PAH 治療
- AAV 分類 + 治療（GPA/MPA/EGPA）

### 不可忽略的「小題大作」
- Fibromyalgia：診斷標準 + 藥物 IA 等級
- DEXA + FRAX + GIOP（每年穩定出現）
- 超音波 OMERACT grading + 各疾病特徵
