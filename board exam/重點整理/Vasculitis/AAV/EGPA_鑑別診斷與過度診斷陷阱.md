# EGPA 鑑別診斷與過度診斷陷阱

> 核心主題:嗜酸球疾病的鑑別。EGPA 最容易被「過度診斷」,因為它與單純過敏、HES 共用同一條 Th2/嗜酸球軸線。鑑別的靈魂在於找到**血管炎**這個獨有成分。
> 相關筆記:[EGPA 機轉深探](AAV_EGPA機轉深探.md)、[三大疾病完整比較](AAV_三大疾病完整比較.md)、[分類診斷與 2022 標準](AAV_分類診斷的來龍去脈與2022標準.md)、[歷屆考題](AAV_歷屆考題.md)

---

## 一、為什麼 EGPA 容易被過度診斷?

### 1. 分類標準 ≠ 診斷標準

2022 ACR/EULAR 是 **classification criteria(分類標準)**,不是 **diagnostic criteria(診斷標準)**。兩者目的完全不同:

| | 分類標準（Classification） | 診斷標準（Diagnostic） |
|--|--|--|
| 目的 | 把**已確診是血管炎**的病人,歸到 GPA/MPA/EGPA 哪一類 | 從未篩選的人群中診斷有沒有病 |
| 前提 | **必須先確立「這是血管炎」並排除 mimics** | 無此前提 |
| 誤用後果 | 拿來「診斷」會大量偽陽性 | — |

使用 2022 標準的官方前提:**必須先排除血管炎的 mimics(感染、惡性腫瘤、其他疾病),並確立有血管炎**,才能套用計分。

### 2. 用 EGPA 2022 分類標準實際算一次

EGPA 分類標準(總分 ≥ 6 分):

| 項目 | 分數 |
|------|------|
| 阻塞性氣道疾病（asthma） | +3 |
| 鼻息肉 | +3 |
| 血中嗜酸球 ≥ 1×10⁹/L | +5 |
| 血管外嗜酸球浸潤（切片） | +2 |
| Mononeuritis multiplex | +1 |
| cANCA 或 PR3-ANCA 陽性 | −3 |
| 血尿 | −1 |

陷阱:一個**單純過敏氣喘 + 鼻息肉 + 嗜酸球高**的人,光這三項就 3 + 3 + 5 = **11 分**,遠超過 6 分。

→ **完全沒有任何血管炎證據,就「被分類」成 EGPA。** 這就是為什麼「前提(必須先證明有血管炎)」比算分更重要。

### 3. pANCA weak(+) 的陷阱

- **pANCA(IIF 篩檢)特異性差**:IBD、自體免疫肝炎、感染、其他結締組織病、甚至健康人都可能弱陽。
- 真正有意義的是**抗原專一性的 MPO-ANCA(ELISA 確認)**,而非單純 IIF 的 pANCA pattern。
- weak(+) 且 **MPO-ANCA 陰性** → 臨床意義很低,不該當成 EGPA 證據。
- 補充:在 2022 計分裡,加分的是 PR3(而且是負分),MPO 根本不計分;但醫師看到「ANCA+」心理上會更傾向 EGPA,這是**認知偏誤**的來源。

> **一句話:單純 asthma + pANCA weak(+),若沒有血管炎或血管外嗜酸球浸潤的證據,不能診斷 EGPA。**

## 二、EGPA vs 單純過敏 / 氣喘

EGPA 比過敏多了**兩樣東西**:**(A) 系統性血管炎** 與 **(B) 嗜酸球組織浸潤造成的器官破壞**。

Th2 / 嗜酸球 / IgE 這條軸線是過敏和 EGPA **共用**的,所以**不能用來鑑別**。

```
單純過敏/氣喘  =  Th2 軸線(IL-4/5/13、IgE、嗜酸球) ← 到此為止
                              │
EGPA          =  Th2 軸線  +  血管炎  +  嗜酸球組織浸潤的器官損傷
                              ↑ 這才是 EGPA 獨有
```

| 特徵 | 單純過敏 / Asthma | EGPA |
|------|------------------|------|
| Th2 偏移、IgE、嗜酸球升高 | ✅ 有 | ✅ 有（共用,**不能鑑別**） |
| 鼻息肉、rhinosinusitis | 可有 | ✅ 常見 |
| 血中嗜酸球程度 | 通常 < 1.5×10⁹/L | 常 ≥ 1.5×10⁹/L,甚至更高 |
| 系統性症狀（發燒、體重減輕） | ❌ | ✅ |
| Mononeuritis multiplex | ❌ | ✅ 血管炎指紋 |
| 腎炎 / 肺泡出血 | ❌ | ✅（尤其 ANCA+） |
| 心肌炎（eosinophilic） | ❌ | ✅ 最重要死因 |
| 可觸性紫斑、皮膚血管炎 | ❌ | ✅ |
| 切片：血管炎 / 血管外嗜酸球浸潤 | ❌ | ✅ |
| MPO-ANCA | ❌ | ~40% 陽性 |

### 讓 asthma「升級」成 EGPA 的證據

出現**第三期(系統性血管炎期)**的證據:

1. **切片證實血管炎**(最直接);或
2. **Mononeuritis multiplex**——因為只有 40% EGPA 有 ANCA,不能靠 ANCA 確診;神經病變來自 vasa nervorum 缺血(血管炎),不是嗜酸球毒素;或
3. 其他血管炎器官表現:腎絲球腎炎、肺泡出血、可觸性紫斑。

## 三、EGPA vs HES（嗜酸球增多症候群）

### 1. 為什麼心臟分不出來?因為機轉相同

EGPA 和 HES 的心臟損傷,**本質上都是嗜酸球毒性顆粒蛋白**(MBP、ECP、EDN、eosinophil peroxidase)直接浸潤心肌造成的,走同一條路:

```
大量活化嗜酸球浸潤心內膜/心肌
        ↓
釋放毒性顆粒蛋白(MBP、ECP…)→ 內皮損傷、心肌壞死
        ↓
① Löffler 心內膜炎(急性壞死期)
        ↓
② 血栓期(心室內附壁血栓 → 栓塞)
        ↓
③ 纖維化期(endomyocardial fibrosis)
        ↓
Restrictive cardiomyopathy + 房室瓣逆流
```

→ 這套「**Löffler endocarditis → 心內膜心肌纖維化 → 限制型心肌病**」HES 和 EGPA **完全一樣**。光看心臟影像或心肌切片的嗜酸球浸潤,**無法區分**。兩者也都以心臟為主要死因。

### 2. 差別在「心臟以外」——血管炎這個成分

| 鑑別點 | EGPA | HES |
|--------|------|-----|
| 心臟病變本身 | 嗜酸球性心肌炎 / Löffler | 嗜酸球性心肌炎 / Löffler（**幾乎一樣,不能鑑別**） |
| 心臟切片有無血管炎 | 可見小血管 necrotizing vasculitis ± granuloma | **純嗜酸球浸潤,無血管炎、無 granuloma** |
| 氣喘 / 鼻息肉 / 過敏前驅期 | ✅ 典型三期演變 | 不一定 |
| Mononeuritis multiplex | ✅（vasa nervorum 缺血） | ❌（若有神經病變多為瀰漫性 / 毒性） |
| 腎炎、肺泡出血、可觸性紫斑 | ✅ 血管炎證據 | ❌ |
| ANCA（MPO） | ~40% 陽性 | 通常陰性 |
| FIP1L1-PDGFRA 融合基因 | ❌ | ✅ myeloid 型（**可用 imatinib**） |
| 異常 T cell clone（產 IL-5） | ❌ | ✅ lymphocytic 型 |
| 嗜酸球持續時間 / 程度 | 隨疾病波動 | ≥1.5×10⁹/L 持續 ≥6 個月（或合併器官損傷） |

### 3. 實務上怎麼判斷一個「嗜酸球性心肌病」是 EGPA 還是 HES?

不是問「這個心臟像誰」,而是問三個問題:

1. **身上其他地方有沒有血管炎證據?**（mononeuritis multiplex / 紫斑 / 腎炎 / 切片血管炎）→ 有則偏 EGPA
2. **有沒有 EGPA 的典型背景?**（氣喘 + 鼻息肉 + 過敏史 ± MPO-ANCA）→ 有則偏 EGPA
3. **有沒有 HES 的 clonal 證據?**（FIP1L1-PDGFRA 或異常 T cell clone）→ 有則為 HES（且 PDGFRA(+) 要用 imatinib,治療完全不同）

> Kelly 原文:「The presence of vasculitis, either proven by biopsy or strongly implied by **documented multifocal neuropathy**, is often essential to distinguish EGPA from HES because only 40% of patients with EGPA test positive for ANCA.」
>
> 核心:**用「血管炎」來鑑別,而不是用心臟或嗜酸球。**

## 四、總結（口試可直接講的版本）

1. **EGPA vs 單純過敏**:單純氣喘只有 Th2/嗜酸球軸線;EGPA 在此之上額外疊加「系統性血管炎」與「嗜酸球組織浸潤的器官破壞」。

2. **過度診斷陷阱**:2022 ACR/EULAR 是分類標準不是診斷標準,前提是「已確立有血管炎並排除 mimics」。單拿 asthma + 嗜酸球 + 鼻息肉去算分(11 分)會大量偽陽性。asthma + pANCA weak(+) 若 MPO 陰性且無任何血管炎證據,**還不符合 EGPA**。

3. **EGPA vs HES**:心臟病變機轉相同(嗜酸球毒性蛋白 → Löffler → 心內膜心肌纖維化),**心臟本身分不出來**。差別在 EGPA 有系統性血管炎(mononeuritis multiplex、紫斑、腎炎、切片血管炎、MPO-ANCA);HES 無血管炎,要找 FIP1L1-PDGFRA / T cell clone。**治療不同**:PDGFRA(+) HES 用 imatinib;EGPA 用類固醇 ± Mepolizumab / CYC / RTX。
