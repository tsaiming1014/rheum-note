## 🔹 2019

### 口試 7：AAV, Vascular Diseases and Immune System

**呂明錡醫師**

- **Vasculitis 目前有哪些藥物可以使用**
  - Steroid、CYC、RTX、AZA（維持）、MTX（維持）、MMF（維持）
  - EGPA：Mepolizumab（IL-5i）
  - GCA：Tocilizumab（IL-6i）
  - **Avacopan**（C5a receptor antagonist）：AAV 用

- **AAV 為什麼可以打 RTX？可能機轉是？**
  - AAV 疾病活性與活化 B cells 數量相關
  - ANCA 由 B cells/plasma cell precursors 產生
  - 鼻腔、肺、眼眶切片中有自體反應性 B cells 形成類淋巴濾泡結構
  - RTX（anti-CD20）→ 清除 premature + mature B cells →（ADCC、complement lysis、apoptosis）

- **ANCA 致病機轉詳述**

**蔡世滋醫師（臨床案例）**

> 情境：30 多歲女性，右眼嚴重突出，曾住院做過氣切，腳上有傷口結痂，Cre 2+

- 懷疑什麼疾病？DD 有哪些？（AAV > IgG4-RD）
- 比較像 GPA 還是 MPA？（右眼突出 = orbital pseudotumor → 更像 **GPA**）
- 應安排哪些檢查？（ANCA、brain MRI、腎臟超音波、鼻腔/腎臟切片）
- 如何治療？（CYC 或 RTX + 高劑量 steroid）
- 如果患者想懷孕？（RTX 停用 ≥12 個月後再懷孕；改用 AZA 維持）
- 比較 GPA、EGPA、MPA 的不同（強調 GPA 也會有 eosinophilia）

## 🔹 2020

### 口試 5：Vasculitis

**吳詹永嬌主任（AAV 臨床題）**

> 情境：18 歲，joint pain，nasal bloody discharge once，non-blanchable rash

1. **問診要點** — 鼻出血次數/血量、呼吸道症狀（喘、咳血）、腎臟症狀（血尿、泡沫尿、水腫）、皮膚病變範圍、全身症狀

2. **要做哪些檢查，解讀結果，如何治療**
   - 血液：CBC、腎功能、ANCA（c/pANCA）、anti-GBM Ab、補體
   - 尿液：UA + urine microscopy（RBC cast = 重要）
   - 影像：CXR、腎臟超音波
   - 切片：腎臟或鼻腔切片（granuloma？pauci immune？）

3. **後來喘、咳血，CXR 和 Chest CT 判讀，後續處置和治療**
   - 肺浸潤 + 咳血 → pulmonary hemorrhage
   - Induction：高劑量 steroid + RTX 或 CYC；Plasma exchange（DAH 合併嚴重腎炎時）；Avacopan

**另一位老師**

1. **ANCA 在 AAV 的致病機轉**（ANCA 結合中性球表面 PR3/MPO → 過度活化 → NETosis → 血管損傷 → vicious cycle）
2. **補體和 NET 有角色嗎？**（C5a priming）
3. **除了傳統治療外，有其他藥物選擇嗎？**（Avacopan、Mepolizumab）
4. **PR3 和 MPO 平常在哪裡？既然在細胞質中，為何會產生自體抗體？**（azurophilic granules → NETosis 釋放 → 打破耐受）

## 🔹 2021

### 筆試

**50. Vasculitis 和相關敘述，何者正確？**

- EGPA → nasal bloody discharge（GPA 較常見）
- **Cryoglobulinemic vasculitis → 會有 low level complement factor** ✓

**51. AAV 的 GN，何者錯誤？**

- A. 常見表現包括 skin purpura and renovascular hypertension
- B. No or few immune complex deposition ✓
- C. Anti-PR3 titer 跟嚴重程度相關
- D. CYC 和 RTX for 治療 ✓
- **答案 A**（renovascular hypertension 較少見）

**52. Vasculitis，何者正確？**

- A. IIF：HSP、Takayasu
- B. MPA、PAN、GCA 有 granuloma（誤，應為 GPA、EGPA）
- C. GPA、MPA、HSP 有 cellular crescent
- **D. HSP、SLE、cryoglobulinemic vasculitis 有 LCV** ✓

**53. Small vessel vasculitis，何者正確？**

- **AAV 中 EGPA 的 cardiomyopathy 比例較高** ✓（約 15–25%，EGPA 主要死因）

**54. EGPA 治療，何者錯誤？**（藥物：RTX、CYC、TCZ、MEPO）

- EGPA 治療以 CYC（severe）、Mepolizumab（eosinophilic）為主
- **TCZ（tocilizumab）並非 EGPA 標準治療** ✓ 答案

## 🔹 2022

### 筆試

**41. GPA 的治療**

| 嚴重度 | 誘導治療 | 維持治療 |
|--------|---------|---------|
| Severe | RTX 或 CYC + high-dose steroid | RTX 或 AZA/MTX |
| Non-severe | MTX 或 MMF + steroid | MTX 或 AZA |

**42. EGPA 的 clinical manifestations**

| 系統 | 表現 |
|------|------|
| 呼吸道 | Asthma（幾乎 100%）、rhinosinusitis |
| 心臟 | Eosinophilic myocarditis（最重要死因） |
| 神經 | Mononeuritis multiplex |
| 腎臟 | 較 GPA/MPA 輕微 |
| 皮膚 | Purpura、eosinophilic skin infiltration |

**44. MPA 的治療**

| 嚴重度 | 誘導 | 維持 |
|--------|------|------|
| Severe（腎/肺） | RTX 或 CYC + steroid | RTX 或 AZA |
| Non-severe | MTX 或 MMF + steroid | MTX 或 AZA |

**92. ANCA 細項、盛行率百分比**

| 疾病 | cANCA（PR3） | pANCA（MPO） |
|------|------------|------------|
| GPA | 60–90% | 5–20% |
| MPA | 30–40% | 50–60% |
| EGPA | 10–25% | 30–40% |

### 口試第 5 站：AAV / Vasculitis

**第一個老師**

- GPA、EGPA、MPA：cANCA/pANCA 百分比、侵犯器官各論（新 criteria）、guideline 治療
- 各種 ANCA 特色：誰侵犯心臟多、誰侵犯周邊神經多、比例

| 疾病 | 心臟侵犯 | 周邊神經侵犯 |
|------|---------|------------|
| GPA | 少見 | Mononeuritis multiplex（較少） |
| MPA | 少見 | Mononeuritis multiplex（中等） |

- B. Procalcitonin
- **C. ANCA** ✓
- D. dsDNA

**47. 53 歲女性，有 asthma 病史，因 autoimmune thyroiditis 使用 eltroxin。四周前兩眼眼窩腫脹，漸出現 diplopia、strabismus、proptosis。MRI：multicentral enlargement in extraocular muscles；TSH 正常、thyrotropin receptor antibody 陰性。以下哪些檢測合適？**

1. CXR + ACE（排除 sarcoidosis）
2. c-ANCA, p-ANCA（排除 GPA 或 Churg-Strauss）
3. Serum immunoelectrophoresis IgG/A/M/IgG4（排除 IgG4-RD）
4. Anti-AChR antibody + single fiber EMG（排除 ocular MG）

- **D.（1)+(2)+(3)+(4)** ✓

**50. 血管炎通論題，選錯的**

- A. Chapel Hill classification…
- **B. AAV 使用 complement inhibitor 無效 → 錯**（其實有效）

**51. AAV management，選錯的**

- **B. MAINRITSAN trial 中 RTX 維持治療未優於 AZA → 錯**（RTX 優於 AZA，sustained remission 較多）

**78. 下列組合哪幾項正確？**

1. GPA → c-ANCA(+) ✓
2. HSP → elevated IgE ✗
3. PAN → paranasal sinusitis ✗
4. Takayasu → pulseless disease ✓
5. Leukocytoclastic vasculitis → p-ANCA(+) ✗
6. Churg-Strauss → allergic disorder ✓
7. GCA → elevated IgA ✗

## 📊 高頻考點總結

| 考點 | 出現年度 |
|------|---------|
| GPA/MPA/EGPA 三者比較（抗體、器官、盛行率） | 2019, 2021, 2022 |
| ANCA 致病機轉（NETosis、PR3/MPO 暴露、vicious cycle） | 2019, 2020 |
| **Avacopan（C5a 抑制劑）** 機轉與應用 | 2019, 2020, 2021, 2024 |
| **EGPA 心臟侵犯 15–25%（最重要死因）** | 2021, 2022, 2023 |
| EGPA → Mepolizumab（IL-5i） | 2019, 2020, 2021, 2022 |
| Pauci-immune GN / 腎切片 IF | 2020, 2021 |
| 治療 guideline（CYC/RTX 誘導、維持、PLEX、IVIG） | 2019, 2021, 2022, 2024 |
| 2022 ACR/EULAR classification criteria | 2021, 2024 |
| RTX 維持優於 AZA（MAINRITSAN） | 2024 |

**結論**：AAV 幾乎每年必考，筆試與口試皆有。核心高頻：三疾病比較表、ANCA 機轉、Avacopan、EGPA 心臟侵犯、治療 guideline。
