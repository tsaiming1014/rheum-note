# AAV 三大疾病完整比較：GPA、MPA、EGPA

## 一、快速記憶：三個人設

| | 人設 | 核心邏輯 |
|--|--|--|
| **GPA** | 🔨 **破壞王** | 肉芽腫 → T cell + 巨噬細胞 → 上呼吸道全面破壞 |
| **MPA** | 🤫 **沉默殺手** | 純血管炎 → 沒有早期症狀 → 腎臟悄悄壞掉 |
| **EGPA** | 🤧 **過敏體質者** | Th2 + 嗜酸球 → 氣喘開場，嗜酸球到處浸潤 |

### 每個疾病只記三個招牌表現

**GPA：鼻 × 眼 × 肺結節**
- Saddle-nose deformity（鼻塌了）
- Orbital pseudotumor（眼球突出，50% 失明）
- 空洞性肺結節（Cavitary nodule）

**MPA：腎 × 纖維化肺 × 延遲診斷**
- RPGN（急速腎衰竭）
- Pulmonary fibrosis（ILD）← GPA 反而少見
- 診斷時腎已很爛（缺乏早期 ENT 症狀提醒）

**EGPA：氣喘 × 心肌炎 × 多發性單神經炎**
- Asthma（near 100%）
- Eosinophilic myocarditis（ANCA 陰性者多）
- Mononeuritis multiplex（區別 HES 的關鍵）

### 終極口訣

```
GPA → 破壞王：鼻塌、眼突、肺有洞（PR3）
MPA → 沉默殺手：腎最重，ILD，診斷晚（MPO）
EGPA → 過敏體質：喘必有，心臟怕，神經痛（40% ANCA）
```

## 二、致病機轉差異

### 共同基礎：ANCA 致病的核心流程

三種疾病都屬於 AAV（ANCA-associated vasculitis），共同的致病機制如下：

```
遺傳風險 + 環境暴露
        ↓
B cell 自體耐受破壞 → ANCA 產生
        ↓
炎症刺激（TNF, G-CSF, LPS, C5a）→ Neutrophil Priming
        ↓
MPO/PR3 轉位至細胞膜表面
        ↓
ANCA Fab（結合MPO/PR3）+ Fc（結合FcγRIIIb）
        ↓
中性球活化：Respiratory burst + Degranulation + Mac-1上調
        ↓
C5a 補體放大 + NETs 惡性循環
        ↓
Pauci-immune Necrotizing Vasculitis
```

**為什麼是 Pauci-immune？**  
ANCA 的靶抗原（MPO/PR3）平常藏在中性球顆粒裡，不是遊離在血液中的抗原，因此 ANCA 不會在血管壁形成抗原—抗體複合體沉積。傷害是中性球「親自動手」造成的，血管壁留下的是中性球暴走的痕跡，而非免疫複合體沉積痕跡。

---

### GPA 的特殊機轉：PR3 + T cell 軸 → 肉芽腫

#### PR3 的獨特性質

| 性質 | PR3 | MPO |
|------|-----|-----|
| 細胞表面表現 | **Constitutive（不需 priming 就有）** | 需要 priming 才外露 |
| 促炎功能 | **直接促進 TNF-α、IL-1β 釋放** | 主要靠氧化破壞 |
| 抑制劑 | α1-antitrypsin（SERPINA1） | 相對較少依賴單一抑制劑 |

#### 肉芽腫的形成機制

```
PR3 constitutive 表現於中性球表面
        ↓
CD4+ T cells 被持續活化（Th1 response）
        ↓
T cells 活化 macrophage → 上皮樣細胞 + 巨細胞
        ↓
Necrotizing Granuloma 形成
```

MPA 沒有這個過程：MPO 平常藏在顆粒裡，T cell 較少被持續刺激，所以**只有 vasculitis，沒有 granuloma**。

#### 為什麼 GPA 特別愛打上呼吸道？

- **α1-antitrypsin（PR3 的煞車）在血液濃度高，在呼吸道黏膜濃度低** → PR3 在呼吸道活性更強、煞車失靈
- *S. aureus* 定植於鼻腔 → 反覆 priming 中性球 → 上呼吸道持續的 ANCA-中性球反應 → 局部 granuloma + 破壞
- **SERPINA1 基因變異**（Z、S allele）→ α1-antitrypsin 缺乏 → PR3 更難被抑制 → 解釋 GPA 的遺傳易感性

---

### MPA 的機轉：MPO + 純 Neutrophil 軸

MPO-ANCA 的機轉比較「單純」：

- MPO 需要 priming 才外露，T cell 參與較少
- 以純粹的中性球活化 → 氧化破壞（HOCl）→ 血管壁 fibrinoid necrosis 為主
- 沒有 granuloma，沒有上呼吸道破壞
- 腎臟的 glomerular capillary 是最脆弱的靶（血流量大、表面積大、容易被活化中性球傷害）
- 遺傳上：MPO-ANCA 對應 **HLA-DQ**（GPA 對應 HLA-DP）→ 從基因層面就是不同的免疫方向

---

### EGPA 的機轉：Th2 + Eosinophil 軸（完全不同的驅動力）

```
Th2 免疫偏移
    ├── IL-4, IL-13 → IgE 升高 → 過敏體質 → 氣喘、鼻息肉
    └── IL-5 → 嗜酸球分化 + 活化 + 存活
                    ↓
            嗜酸球組織浸潤
                    ↓
       釋放 MBP、ECP、eosinophil peroxidase
                    ↓
           直接毒性損傷（≠ 血管炎）
```

#### ANCA 陽性 vs 陰性的分裂

| | **ANCA 陽性（~40%）** | **ANCA 陰性（~60%）** |
|--|--|--|
| 主要武器 | 嗜酸球 ＋ MPO-ANCA／中性球 | 純嗜酸球 |
| 腎炎（GN） | ✅ 較常見 | ❌ 少見 |
| 肺泡出血 | ✅ 可見 | ❌ 少見 |
| 心肌炎／心肌病變 | ❌ 少見 | ✅ **較常見** |

**遺傳證據**：EGPA 的 8 個相關基因中，有 7 個與正常人的嗜酸球計數相關，且 HLA 區域只與 MPO-ANCA 陽性 EGPA 相關，與 ANCA 陰性 EGPA 幾乎無關 → 說明嗜酸球成分是獨立的遺傳特質，不是 ANCA 的副產品。

**治療印證**：Mepolizumab（anti-IL-5）有效 → 確認 IL-5／嗜酸球軸是 EGPA 核心。

---

### 致病機轉差異總結

```
相同起點：AAV，都攻擊小血管
          │
          ├── PR3-ANCA → T cell 參與強 → Granuloma + 破壞性上呼吸道
          │   α1-antitrypsin 在呼吸道缺乏 → 呼吸道最嚴重         → GPA
          │
          ├── MPO-ANCA → 純中性球 → 乾淨的 Vasculitis
          │   腎臟毛細血管最脆弱，缺乏早期症狀                    → MPA
          │
          └── Th2 偏移 → IL-5 → 嗜酸球
              ├── ANCA (+)：嗜酸球 + 血管炎 → 腎炎、肺泡出血
              └── ANCA (-)：純嗜酸球毒性 → 心肌炎、心肌病變      → EGPA
```

### 耳鼻喉（ENT）

| 表現 | GPA | MPA | EGPA |
|------|-----|-----|------|
| 發生率 | **83–99%** | 1–20% | 48–77% |
| 性質 | **破壞性（Destructive）** | 罕見 | **過敏性（Allergic）** |
| 鼻部 | 鼻出血、膿性鼻涕、血痂、鼻中隔穿孔、**Saddle-nose deformity** | 偶見 | 鼻息肉（nasal polyps）、鼻塞 |
| 鼻竇 | 鼻竇炎、骨侵蝕 | 偶見 | 過敏性鼻竇炎 |
| 喉部 | **Subglottic stenosis（~15%）**，可危及生命 | 罕見 | 罕見 |
| 耳部 | 傳導性聽損、感音性聽損 | 罕見 | 偶見 |
| 其他 | 口腔潰瘍、牙齦炎、軟骨炎 | — | — |

> ⚠️ **GPA vs EGPA 的 ENT 鑑別**：GPA 是破壞性（骨侵蝕、穿孔）；EGPA 是過敏性（息肉），幾乎不具破壞性

## 四、口試回答框架

### 標準答題順序

> **① 定義一句話 → ② ANCA 類型 → ③ 招牌三表現 → ④ 器官依發生率帶過**

---

### GPA 示範答法

「GPA 是以**壞死性肉芽腫性發炎合併血管炎**為特徵的疾病，主要跟 **PR3-ANCA（c-ANCA）** 相關。

最具特色的三個表現是：**Saddle-nose deformity、Orbital pseudotumor、空洞性肺結節**。

依器官：發生率最高的是**上呼吸道（83–99%）**，有破壞性鼻竇炎、鼻中隔穿孔，嚴重可有 subglottic stenosis。**眼睛（34–61%）**是 GPA 特有的，可有 scleritis 和 orbital pseudotumor。**肺（66–85%）**以空洞性肺結節和肺泡出血為主。**腎臟（66–77%）**為 pauci-immune 壞死性新月形 GN。」

---

### MPA 示範答法

「MPA 是**純血管炎**，沒有肉芽腫，跟 **MPO-ANCA（p-ANCA）** 相關。

它的特色是**症狀隱匿，常到腎臟嚴重衰竭才被診斷**。最常見的是**腎臟（69–100%）**，以 RPGN 為主。肺部跟 GPA 不同，MPA 比較容易有**肺間質纖維化（ILD）**，而不是空洞結節。眼睛幾乎不受影響，這點和 GPA 完全相反。」

---

### EGPA 示範答法

「EGPA 是以**嗜酸球性肉芽腫性發炎合併血管炎**為特徵，特別的是**只有約 40% 的患者有 ANCA（通常 MPO-ANCA）**。

核心是 **Th2 免疫反應**驅動：**氣喘（near 100%）加上周邊嗜酸球增多**是必要條件。

最重要的三個表現：**氣喘、嗜酸球性心肌炎（約 15%，ANCA 陰性患者多）、Mononeuritis multiplex（51–76%，用來和 HES 鑑別）**。

ANCA 陽性患者比較會有腎炎和肺泡出血；ANCA 陰性患者心臟病變更明顯。」

---

### 鑑別診斷被問到時

- **GPA vs MPA**：有無肉芽腫、有無上呼吸道破壞、PR3 vs MPO
- **GPA vs EGPA 的 ENT**：破壞性 vs 過敏性
- **EGPA vs HES**：有無 mononeuritis multiplex、有無 vasculitis biopsy 證據
