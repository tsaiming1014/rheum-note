# 血管炎（Vasculitis）分類

血管炎是一群以血管壁發炎為特徵的異質性疾病，可導致缺血與器官損傷。Kelly 採用最廣泛使用的分類系統：先依「主要受侵犯血管口徑」分類，再依共同臨床特徵細分（依據 2012 修訂版 Chapel Hill 共識會議 CHCC 命名）。

> 重要觀念：分類標準（classification criteria）≠ 診斷標準（diagnostic criteria）。分類標準是為了在臨床研究中建立同質性病人族群，強調高特異度、犧牲敏感度，不可拿來診斷個別病人。

## 一、依血管口徑分類

| 血管口徑 | 定義 |
|---|---|
| 大血管 Large vessel | 主動脈及其主要分支與對應靜脈 |
| 中血管 Medium vessel | 主要內臟動/靜脈（腎、腸系膜、冠狀動脈） |
| 小血管 Small vessel | 實質內小動脈、小動脈、微血管、小靜脈、靜脈 |

---

## 二、各口徑下的主要疾病

### 🔴 小血管血管炎（Small-Vessel）— 再分兩大類

**(1) ANCA 相關血管炎（AAV）** — 壞死性、寡免疫複合體（pauci-immune）、與 ANCA 相關

- GPA（肉芽腫合併多血管炎，舊稱 Wegener's）→ 肉芽腫、上下呼吸道、C-ANCA / PR3
- MPA（顯微鏡下多血管炎）→ 腎絲球腎炎、P-ANCA / MPO
- EGPA（嗜伊紅性肉芽腫合併多血管炎，舊稱 Churg-Strauss）→ 氣喘、血液嗜伊紅球增多；ANCA 陽性率較低（38–73%），若陽性多為 MPO

> 趨勢：可改依 ANCA 抗原特異性（PR3 vs MPO）分類，因基因關聯與治療反應、復發更貼近 ANCA 型別而非 GPA/MPA 之臨床分型。

**(2) 免疫複合體媒介小血管血管炎** — 有免疫複合體沉積（與 AAV 相反）

- IgA 血管炎（舊稱 Henoch–Schönlein）— 唯一有驗證過分類標準者
- 冷凝球蛋白血管炎（CV）— Type I（單株）/ Type II、III（混合型，與 HCV 相關）
- 低補體性蕁麻疹性血管炎（HUV / anti-C1q）
- 抗腎絲球基底膜病（anti-GBM / Goodpasture's）

### 🟡 中血管血管炎（Medium-Vessel）

- 結節性多動脈炎（PAN）— 與 HBV 相關；定義上排除腎絲球腎炎與 ANCA（以與 MPA 區分）
- 川崎病（Kawasaki disease）— 多見 <5 歲兒童，冠狀動脈炎為重要併發症

### 🔵 大血管血管炎（Large-Vessel）

- 巨細胞動脈炎（GCA）— 僅 ≥50 歲；可侵犯顱外大血管（主動脈），約 50% 合併 PMR
- 高安氏動脈炎（Takayasu, TAK）— 多 <50–60 歲，女性 9:1

> GCA 與 TAK 有許多重疊（症狀、組織學、基因），是否為「同一疾病光譜」仍有爭議。

## 三、其他分類

- 可變口徑血管炎（Variable vessel）：貝賽特病（Behçet's）、Cogan's syndrome
- 單一器官血管炎（Single-organ）：原發性中樞神經系統血管炎（PACNS）、孤立性皮膚血管炎、血管炎性神經病變
- 次發性血管炎（Secondary）：續發於 SLE、Sjögren、RA、復發性多軟骨炎；或藥物（hydralazine、PTU → MPO-ANCA）、感染、免疫檢查點抑制劑（ICI）
- 單基因血管炎（Monogenic）：
    - DADA2（ADA2 缺乏）— 似 PAN，年輕人中風；TNF 抑制劑有效
    - VEXAS（UBA1 體細胞突變）— 男性，合併 RP、PAN、GCA 與血液異常
    - COPA syndrome（COPA 基因）— 肺出血、ILD、關節炎，可 ANCA 陽性
