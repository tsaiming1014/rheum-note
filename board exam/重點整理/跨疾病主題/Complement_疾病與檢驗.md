# 補體系統 — 臨床檢驗與各疾病變化

> 整合自 Kelly's Textbook of Rheumatology Ch.26（Complement System）與臨床判讀重點

## 一、基本框架

補體有三條活化途徑，最終匯流到 C3，再走向末端膜攻擊複合體（MAC, C5b-9）。

| 路徑 | 啟動物 | 特點 |
|------|--------|------|
| **古典途徑（classical）** | C1q 結合 IgM/IgG 免疫複合物 | 也可被 CRP、DNA、凋亡細胞啟動；切割 C4、C2 |
| **凝集素途徑（lectin）** | MBL/Ficolin 辨識病原醣類 | 啟動 MASP-1/2 → 切割 C4、C2（下游同古典途徑） |
| **替代途徑（alternative）** | C3 自發水解（tick-over） | 提供擴增迴路，**不經過 C4**；貢獻高達 80% 的 C3 沉積 |

臨床判讀主要看四個指標：

| 指標 | 反映什麼 |
|------|---------|
| **C4 ↓** | 古典途徑活化（免疫複合體驅動） |
| **C3 ↓** | 古典或替代途徑活化（C3 是兩條路徑匯流點） |
| **CH50 ↓** | 古典途徑整體功能；單一補體成分完全缺乏時會測不到（≈0） |
| **AH50（AP50）↓** | 替代途徑整體功能 |

判讀邏輯：

- **C3 與 C4 同時降** → 古典途徑活化（免疫複合體病）
- **C3 降、C4 正常** → 替代途徑活化
- **C4 不成比例極低** → cryoglobulinemia、IgG4-RD、血管性水腫

## 二、依補體變化型態分類疾病

### 型態 A：C3↓ + C4↓（古典途徑／免疫複合體病）

| 疾病 | 重點 |
|------|------|
| **Active SLE / Lupus nephritis** | C3、C4 同步下降，與活動度相關，收入 **SLEDAI**；lupus nephritis 高風險者常合併 anti-C1q↑ |
| **Sjögren's syndrome** | C3↓、C4↓ = **預後不良指標**（淋巴瘤風險↑、嚴重器官受累、死亡率↑） |
| **Mixed cryoglobulinemia** | 見型態 B |
| Subacute bacterial endocarditis | 慢性免疫複合體形成 |
| Serum sickness | 經典免疫複合體病 |
| Membranoproliferative GN（immune-complex type） | C3、C4 皆低 |

### 型態 B：C4↓ 不成比例（C4 ≫ C3 下降）

| 疾病 | 重點 |
|------|------|
| **Cryoglobulinemia（Type II/III）** | 免疫複合體（IgM-RF + IgG）強力活化古典途徑，C4 在 C3 上游被大量消耗；加上 C3 是急性期蛋白會被肝臟補回 → **C4 掉得比 C3 多**。RF↑、anti-CCP(−)，是強烈線索 |
| IgG4-related disease | 可見低補體，C4 下降為主 |

C4 不成比例低下的鑑別三大項：**Cryoglobulinemia、SLE、IgG4-RD**。

### 型態 C：C3↓ + C4 正常（替代途徑活化）

| 疾病 | 重點 |
|------|------|
| Post-streptococcal GN | C3 低、6–8 週內回復；C4 多正常 |
| C3 glomerulopathy（C3GN / Dense deposit disease） | C3 nephritic factor 穩定 C3 convertase → C3 持續低 |
| Atypical HUS | 替代途徑調控失常（factor H/I、MCP 突變） |

記憶點：**C4 正常但 C3 低 → 想替代途徑（以腎絲球疾病為主）**。

### 型態 D：C4 低、C3 正常 + 血管性水腫

| 疾病 | 重點 |
|------|------|
| Hereditary angioedema (HAE) | C1-INH 缺乏/失能 → C4 持續低（發作時更低），**C1q 正常**；C4 是最佳篩檢，確診測 C1-INH level/function |
| Acquired angioedema (AAE) | C4 低 + **C1q 也低**（與 HAE 的鑑別點） |

### 型態 E：補體「正常」的血管炎（重要鑑別）

| 疾病 | 重點 |
|------|------|
| ANCA-associated vasculitis（GPA/MPA/EGPA） | 典型 pauci-immune，補體多正常（C5a 參與致病 → avacopan 治療） |
| IgA vasculitis (HSP) | 補體通常正常 |
| Anti-GBM disease（Goodpasture） | 補體通常正常 |
| PAN（結節性多動脈炎） | 補體多正常（HBV 相關者可低） |
| 大血管炎（GCA / Takayasu） | 補體正常 |

### 型態 F：補體「正常或升高」（急性期反應）

補體是急性期蛋白，發炎時可上升：

| 疾病 | 重點 |
|------|------|
| **RA** | **血清補體正常或輕微↑**（急性期反應）；但**關節液 C3/C4↓、C5a↑、SC5b-9↑**（局部活化）。若血清出現低補體 → 想 rheumatoid vasculitis |
| **Psoriatic arthritis** | C3↑（急性期反應），**無補體活化訊號** |
| 感染、急性發炎、腫瘤 | 肝臟合成增加 |

> RA 動物模型：**替代途徑 + 凝集素途徑**是關節炎發展的必要條件，古典途徑非必要；ACPA 可同時活化古典與替代途徑。

## 三、遺傳性補體缺乏與疾病傾向

| 缺乏蛋白 | 頻率/特點 | 疾病傾向 |
|----------|-----------|----------|
| **C1q** | 最罕見（~70 例） | **SLE 風險最高（~90%）** |
| **C1r / C1s** | — | SLE >90% |
| **C4（尤其 C4A null）** | — | SLE ~70% |
| **C2** | **最常見（1:20,000）** | SLE ~15%（表現較輕） |
| **C3** | — | 反覆化膿性細菌感染（莢膜菌）、SLE |
| **末端 C5–C9（MAC）** | — | **僅 Neisseria 感染風險**（腦膜炎球菌） |
| **Properdin** | — | 腦膜炎球菌疾病 |
| **MBL** | 頻率高（~1:10） | 免疫缺損下感染風險↑、兒童反覆感染 |
| **C1-INH** | — | Hereditary angioedema |
| **Factor H / I、MCP** | — | aHUS、C3 glomerulopathy |

記憶口訣：

- **早期成分缺乏 → 自體免疫（SLE）**：C1q(90%) > C1r/s(90%) > C4(70%) > C2(15%)
- **末端成分缺乏 → 只有 Neisseria 感染風險**

### Lupus Paradox（補體的矛盾）

- **補體缺乏 → 容易得 SLE**（清除凋亡細胞/免疫複合體能力下降，自身抗原累積）
- **但多數 SLE 患者中，補體反而加重組織傷害**（免疫複合體 → 活化補體 → 腎炎）
- → 對多數 SLE 患者，**抑制補體**才是治療方向

## 四、臨床如何使用補體檢驗（口試答題版）

補體檢驗在臨床上有三大用途：**診斷鑑別、疾病活動度追蹤、評估補體缺乏**。

### 1. 要驗什麼

| 檢驗 | 用途 |
|------|------|
| **C3、C4（nephelometry）** | 最常用，反映途徑消耗；SLE 診斷與追蹤 |
| **CH50** | 評估整條古典途徑功能；篩檢遺傳性缺乏（=0 提示完全缺乏） |
| **AH50（AP50）** | 評估替代途徑功能 |
| **活化片段（C3a、C5a、C4d、C5b-9）** | 代表「正在進行」的補體活化 |
| **C1q、C1-INH（level + function）** | 評估血管性水腫、區分 HAE 與 AAE |
| **Anti-C1q 抗體** | 與 lupus nephritis 相關（見第五節） |
| 個別補體成分（C2, C5–C9…） | CH50/AH50 異常時進一步定位 |

實務起點：**以 C3、C4 搭配 CH50 作為篩檢組合**，再依需要加做。

### 2. 三大用途

- **診斷與鑑別**：用補體型態反推疾病（見第二節）；補體正常也可協助**排除**某些診斷（如 AAV）。
- **活動度監測**：SLE / lupus nephritis 的 C3、C4 與活動度相關，可追蹤病情、預測腎炎復發。但並非每種病都適用——例如 cryoglobulinemia 的 C4、cryocrit 即使臨床改善仍常持續異常。
- **評估補體缺乏**：反覆感染或早發/家族性自體免疫者，用 CH50/AH50 篩檢。

### 3. 判讀陷阱（加分項）

- 補體是**急性期蛋白**：發炎/感染時上升，可能遮蓋正在消耗的低補體 → 看趨勢而非單點。
- **檢體處理**：補體不穩定，採檢後需盡快分離、低溫保存，否則體外活化造成偽低。
- **CH50 = 0** 先想檢體處理不當或單一成分完全缺乏，而非全面消耗。
- 解讀務必**結合臨床與其他抗體**（anti-dsDNA、RF、cryoglobulin 等）。

## 五、Anti-C1q 抗體的臨床意義

- 可見於 SLE（與 lupus nephritis 相關），但也可見於健康人 → 單獨陽性不代表腎病。
- 病理機制：anti-C1q 需與腎絲球內的 C1q 免疫複合物共存，才會造成腎損傷。
- **Anti-C1q 陰性 → 幾乎可排除嚴重腎臟侵犯**（高陰性預測值）。

## 六、補體標靶治療藥物

| 藥物 | 標靶 | 適應症 |
|------|------|--------|
| **Eculizumab** | anti-C5 | PNH、aHUS、NMO、myasthenia gravis；難治性 TMA/scleroderma renal crisis 可考慮 |
| **Avacopan** | C5aR1 antagonist | ANCA-associated vasculitis |
| **Sutimlimab** | anti-C1s | Cold agglutinin disease |
| **Pegcetacoplan** | C3 inhibitor（compstatin 衍生物） | PNH |

## 七、一頁速記表

| 補體型態 | 首要聯想 |
|----------|---------|
| C3↓ C4↓ | SLE（flare/SLEDAI）、Sjögren's（壞預後）、免疫複合體病、SBE、serum sickness |
| C4 ≫ C3↓（C4 不成比例低） | Cryoglobulinemia、IgG4-RD |
| C3↓ C4 正常 | Post-strep GN、C3 glomerulopathy、aHUS |
| C4↓ C3 正常 + 血管性水腫 | HAE（C1q 正常）/ AAE（C1q 低） |
| 補體正常的血管炎 | AAV、IgA vasculitis、anti-GBM、PAN、GCA |
| 補體正常/↑ | 急性期：RA（血清正常↑、關節液↓）、PsA、感染、腫瘤 |
| CH50 = 0 | 某一補體成分完全缺乏 |
