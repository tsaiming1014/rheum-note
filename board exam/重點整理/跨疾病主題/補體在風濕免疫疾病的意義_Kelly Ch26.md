# 補體（Complement）在風濕免疫疾病的臨床意義
> 來源：Kelly's Textbook of Rheumatology, Ch. 26 — Complement System

---

## 一、補體的三條活化路徑

| 路徑 | 啟動物 | 特點 |
|------|--------|------|
| **Classical pathway（古典路徑）** | C1q 與 IgM/IgG 免疫複合物結合 | 也可被 CRP、DNA、凋亡細胞啟動 |
| **Lectin pathway（凝集素路徑）** | MBL、Ficolin 辨識病原糖類 | 啟動 MASP-1/2 → 裂解 C4、C2 |
| **Alternative pathway（替代路徑）** | C3 自發水解（tick-over） | 提供擴增迴路，貢獻高達 80% 的 C3 沉積 |

三條路徑均匯聚於 **C3** → 裂解 C5 → 形成 **MAC（C5b-9）**

---

## 二、補體檢測方法

| 檢測 | 意義 |
|------|------|
| **CH50** | 評估古典路徑整體活性（sheep RBC 溶血試驗） |
| **AP50** | 評估替代路徑活性（rabbit RBC 溶血試驗） |
| **C3、C4（nephelometry）** | 最常用，用於 SLE 診斷與追蹤 |
| **C3a、C5a、C4d、C5b-9** | 活化片段，代表正在進行的補體活化 |
| **Anti-C1q 抗體** | 與 lupus nephritis 相關 |

---

## 三、補體數值的臨床解讀

### 低補體（Hypocomplementemia）= C3↓、C4↓、CH50↓

**機制：消耗 > 合成**（secondary/acquired deficiency）
- 免疫複合物大量活化補體 → 補體被消耗

**臨床意義：低補體 = 疾病活躍的訊號**

| 疾病 | 補體變化 | 臨床意義 |
|------|----------|----------|
| **SLE** | C3↓、C4↓、CH50↓ | 疾病活化（flare）指標；包含於 SLEDAI 評分中 |
| SLE with lupus nephritis | C3↓、C4↓ + anti-C1q ↑ | 腎臟受侵高風險；anti-C1q 陰性 → 幾乎無嚴重腎病 |
| **Sjögren's Syndrome** | C3↓、C4↓ | 預後不良指標：淋巴瘤、嚴重病症、死亡率↑ |
| **Mixed Cryoglobulinemia** | C3↓、C4↓ | 消耗活化，與血管炎相關 |
| **RA** | 血清補體正常或輕微↑；**關節液 C3/C4↓** | 局部（關節腔）補體活化，非全身性 |

### 高補體（Hypercomplementemia）= C3↑

| 疾病 | 補體變化 | 機制 |
|------|----------|------|
| **Psoriatic Arthritis** | C3↑ | 急性期蛋白（acute-phase reactant）反應，非活化 |
| 感染、急性發炎 | C3↑ | 肝臟合成增加（急性期反應） |

## 四、補體缺乏症（Primary Deficiency）與疾病關聯

| 缺乏蛋白 | 疾病關聯 |
|----------|----------|
| **C1q** | 最強 SLE 風險（90% 患者發展成 SLE）；最稀少（~70 例已知） |
| **C1r / C1s** | SLE > 90% |
| **C4（尤其 C4A null）** | SLE 70% |
| **C2** | 最常見（1:20,000）；SLE 15%（表現較輕） |
| **C3** | 反覆細菌感染；SLE |
| **MBL** | 免疫缺損下感染風險↑（頻率高：1:10） |
| **C5–C9（MAC 成分）** | 僅 **Neisseria** 感染風險（腦膜炎球菌） |
| **Properdin** | 腦膜炎球菌疾病 |
| **C1-INH** | 遺傳性血管水腫（Hereditary Angioedema, HAE）|

### "Lupus Paradox"（矛盾）
- **補體缺乏 → 容易得 SLE**（清除凋亡細胞能力下降，自身抗原累積）
- **但在多數 SLE 患者中，補體反而加重組織傷害**（免疫複合物 → 活化補體 → 腎炎）
- → 對多數 SLE 患者，**抑制補體**才是正確治療方向

---

## 五、各疾病補體的角色摘要

### SLE
- C1q 沉積於腎絲球 + 血清 C1q↓（flare 時）
- 免疫複合物 → 古典路徑活化 → MAC 形成 → 腎炎
- C3↓、C4↓ 收入 **SLEDAI 評分**
- Anti-C1q 抗體：absent → 強烈預測無嚴重腎病

### Rheumatoid Arthritis (RA)
- **血清**補體：正常或輕微升高（急性期蛋白反應）
- **關節液**：C3↓、C4↓、C5a↑、SC5b-9↑（局部活化）
- ACPAs 可同時活化古典路徑和替代路徑
- 動物模型：替代路徑 + 凝集素路徑 是關節炎發展的必要條件（古典路徑非必要）

### Sjögren's Syndrome
- C3↓、C4↓ = **預後不良**（淋巴瘤風險↑、嚴重器官受累、早死）

### Mixed Cryoglobulinemia
- C3↓、C4↓ 常見（免疫複合物消耗）

### Psoriatic Arthritis
- C3↑（acute-phase reactant），**無活化訊號**

### Vasculitis（ANCA-associated）
- C5aR 抑制劑（avacopan）已批准治療 ANCA 相關血管炎

### Scleroderma Renal Crisis / TMA in SLE
- Eculizumab（anti-C5）可用於難治性病例

## 六、補體相關治療藥物

| 藥物 | 標靶 | 適應症 |
|------|------|--------|
| **Eculizumab** | anti-C5 | PNH、aHUS、NMO、myasthenia gravis |
| **Avacopan** | C5aR1 antagonist | ANCA-associated vasculitis |
| **Sutimlimab** | anti-C1s | Cold agglutinin disease |
| **Pegcetoplan（Pegcetacoplan）** | C3 inhibitor（compstatin derivative） | PNH |

---

## 七、Anti-C1q 抗體的臨床意義

- 可見於 SLE（與 lupus nephritis 相關）
- 亦可見於健康人 → 單獨陽性不代表腎病
- 病理機制：anti-C1q 需與腎絲球內的 C1q 免疫複合物共存，才會造成腎損傷
- **Anti-C1q 陰性 → 幾乎可排除嚴重腎臟侵犯**（高陰性預測值）

---

## 重點口訣整理

```
低補體（C3↓C4↓）：
  SLE flare > Sjögren's（壞預後）> Mixed cryo > RA關節液

高補體（C3↑）：
  PsA（急性期反應）、感染、急性發炎

補體缺乏 → SLE 風險：
  C1q(90%) > C1r/s(90%) > C4(70%) > C2(15%)
  C5-C9缺乏 → 只有 Neisseria 感染風險
```
