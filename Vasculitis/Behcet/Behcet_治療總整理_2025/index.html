    Behçet's Disease — 治療總整理

    來源： 2025 EULAR (Ann Rheum Dis 2026;85:1010–1025) + Kelly's Textbook of Rheumatology Ch94

    治療總則（Overarching Principles）

    內容
    A BD 呈復發緩解病程，可能威脅器官或生命；隨病程演進，表現可能趨緩
    B 治療目標：預防不可逆器官損傷、最大化健康相關生活品質
    C 整個病程中應持續評估器官侵犯，並排除 mimicker
    D 多專科團隊（皮膚、眼科、神經、腸胃、血管外科）、衛教、共同決策、治療依從性、生活型態改善（口腔衛生、戒菸、飲食、運動）
    E 需認識 BD 病程可變性以成功管理；男性、年輕患者器官侵犯較嚴重

    一、皮膚黏膜侵犯（Mucocutaneous）

    Rec 1（LoE 2）

    • 第一線：Colchicine 0.6–2.4 mg/day → 復發性皮膚黏膜病變
    • 對 colchicine 無效或無法耐受 → Apremilast 或 TNFαi（monoclonal）

    Rec 2（LoE 2/5）

    • 口腔/生殖器潰瘍可使用局部類固醇（topical GC）
    • 避免長期全身性類固醇（急性發作可短期使用，嚴重生殖器潰瘍可低劑量 GC 短程加速癒合）

    局部輔助治療（Kelly 補充）

    藥物 說明
    Topical GC 減少疼痛與潰瘍持續時間
    Pimecrolimus（局部 CNI） 可減少疼痛或潰瘍持續時間
    Sucralfate（局部） 保護屏障、止痛、可能縮短病程（RCT 支持）
    Benzydamine HCl 漱口水 減痛
    抗菌漱口水（Chlorhexidine、Tetracycline、Minocycline） 降低菌量 + 抗炎
    Topical PGE₂ gel 0.3 mg bid 減痛 + 預防新潰瘍
    局部麻醉藥（Lidocaine 2–5%、Tetracaine 0.5–1%） 症狀緩解
    Silver nitrate、CO₂/Nd:YAG 雷射 局部燒灼止痛

    難治性皮膚黏膜（Kelly 補充）

    若 colchicine + 局部治療仍不足，可考慮：

    • Dapsone 50–150 mg/day（可單用或與 colchicine 合用）
    • Apremilast 30 mg bid（PDE-4 抑制劑）：RCT 顯示減少口腔潰瘍數目及疼痛；2019 年 FDA 核准；副作用以腸胃道症狀為主
    • Thalidomide 100 mg/day（口腔/生殖器潰瘍有效，但需注意神經毒性與致畸性）
    • Monoclonal anti-TNFα（IFX / ADA）：有效，ADA 起效較快（中位數 42 天 vs. IFX 152 天）
    • Etanercept 25 mg biw：唯一有 RCT 支持的 TNF 製劑；但為 fusion protein 非 monoclonal antibody，EULAR 2025 未列為優先
    • Secukinumab（anti-IL-17）300 mg/month：小型試驗有效（300 mg 完全緩解率優於 150 mg）
    • Anakinra / Canakinumab（anti-IL-1）：部分有效，口腔/生殖器潰瘍可能仍難治
    • 其他備選（資源有限時）：AZA、MMF、MTX、Pentoxifylline 400 mg tid、低劑量 Isotretinoin 20 mg/day

    二、關節侵犯（Joint）

    Rec 3（LoE 2/3）

    • 急性關節炎第一線：Colchicine
    • 復發性或慢性關節炎 → 考慮免疫抑制劑（csDMARD）
    • 關節炎通常非侵蝕性；與丘疹膿皰性皮損共存時發生率更高

    三、眼部侵犯（Uveitis）

    Rec 4（LoE 2/5）

    • 所有 Behçet 葡萄膜炎患者必須給予免疫抑制治療
    • 目標：誘導並維持臨床及血管攝影緩解（clinical + angiographic remission）
    • 後節（posterior segment）威脅視力的炎症 → Infliximab（優先）+ 其他免疫抑制劑
    • GC 不可單獨使用（not as monotherapy）

    治療細節

    步驟 藥物 說明
    急性控制 GC 0.5–1 mg/kg/d（或 IV methylprednisolone pulse） 提供快速抗炎
    維持 / steroid-sparing AZA（傳統第一線） 特別用於年輕男性前葡萄膜炎（高後節進展風險）
    後節 / 難治 Monoclonal anti-TNFα（IFX 優先，ADA 替代） 大型非對照試驗顯示視力顯著改善
    前葡萄膜炎（輕症） 局部 GC 可考慮 Hypopyon uveitis 必須全身免疫抑制

    監測： Fundus fluorescein angiography — 血管滲漏（leakage）= 仍有活性，與長期復發相關。需與眼科醫師共同追蹤。

    四、動脈侵犯（Arterial）

    Rec 5（LoE 2/5/4）

    • 肺動脈 / 周邊動脈瘤：高劑量 GC + Infliximab（優先）
    • Cyclophosphamide 可作替代（不良反應較多，近年已逐漸被取代）
    • GC 應緩慢減量；免疫抑制劑（優先 monoclonal anti-TNFα）持續維持

    Rec 6（LoE 4/5）

    • 血管手術（若需要）不應延誤，應在迅速啟動藥物治療後盡快進行
    • 肺動脈瘤高出血風險者：Embolization 優於開放手術

    2025 年重要改變： IFX 正式取代 CYC 成為動脈侵犯第一線。RCT 顯示 IFX 在第 24 週緩解率高於 CYC，且輕中度不良事件較少。

    五、靜脈血栓（Venous Thrombosis）

    Rec 7（LoE 2/3）

    • 深靜脈血栓（含腦靜脈竇血栓 CVST）：高劑量 GC + 免疫抑制劑（優先 monoclonal anti-TNFα）
    • 大靜脈血栓 / 心內血栓 / 視力威脅性 CVST → 考慮 IV methylprednisolone pulse
    • 免疫抑制劑應持續維持（AZA 為 csDMARD 首選）

    Rec 8（LoE 5）

    • 抗凝劑（anticoagulant）可考慮加用，前提：

    • 出血風險低

    • 已排除共存的肺動脈瘤（動脈瘤 + 抗凝 = 致命出血風險）

    注意： 目前無 RCT 支持抗凝加 IS 優於單用 IS；動脈瘤常與靜脈血栓並存；最佳抗凝療程未定。

    Rec 9（LoE 5）

    • CVST 合併威脅視力的顱內高壓 → 立即考慮外科手術介入

    靜脈後遺症處置（Kelly 補充）

    • 下肢靜脈潰瘍（post-thrombotic syndrome）：臥床休息、彈性襪/繃帶、清創、抗生素
    • Maggot therapy（Lucilia sericata）：清創效果佳，合併免疫抑制劑使用
    • TNFαi 對腿部潰瘍癒合效果不確定（回顧性研究未顯示改善）

    六、胃腸道侵犯（Gastrointestinal）

    Rec 10（LoE 5）

    • 診斷、嚴重度評估、管理：均應以內視鏡（endoscopy）為基礎

    Rec 11（LoE 3）

    • 給予 5-ASA 或 Azathioprine ± GC
    • 嚴重或難治性 → Monoclonal anti-TNFα（ADA 結果良好）
    • JAK inhibitor 有小型 case series，需謹慎詮釋
    • 手術適應症：嚴重出血、穿孔、腸阻塞

    七、腦實質神經侵犯（Parenchymal Neuro-Behçet）

    Rec 12（LoE 2/5/4）

    • 活動性腦實質侵犯：高劑量 GC + 免疫抑制劑（優先 Infliximab）
    • GC 應緩慢減量（6–12 個月）；免疫抑制劑持續維持

    治療細節（Kelly 補充）

    • 嚴重急性發作：IV methylprednisolone 1 g pulse（可重複 7–10 次若恢復不佳）
    • 後續：口服 prednisolone 1 mg/kg/d 逐漸減量
    • 一線無效 → Anti-TNFα（IFX/ADA） 或 IFNα
    • 其他生物製劑報告：Tocilizumab、Anakinra、Canakinumab（個案系列，謹慎詮釋）

    皮膚黏膜 / 關節

    ↓ 無效或無法耐受
Apremilast（或 monoclonal anti-TNFα）
    ↓ 仍難治
Thalidomide / Dapsone / Biologics（etanercept、anti-IL-1、anti-IL-17）

    主要器官侵犯（眼 / 動脈 / 靜脈 / CNS）

    高劑量 GC
（大靜脈血栓、動脈、CNS → 優先 IV methylprednisolone pulse）
    +
csDMARD* 或直接 Monoclonal anti-TNFα（IFX 優先）
    ↓ 維持期

    csDMARD 選擇（依器官）：

    • 葡萄膜炎 → AZA ± CsA
    • 下肢靜脈血栓 / CNS → AZA
    • GI → AZA 和/或 5-ASA

    2025 年關鍵更新（vs 2018 EULAR）

    面向 舊觀念 2025 更新
    動脈侵犯第一線 CYC 為主 IFX + 高劑量 GC（CYC 降為替代）
    靜脈血栓 GC + IS 明確強調 Monoclonal anti-TNFα 優先
    CNS 侵犯 GC + IS（生物製劑次選） 明確推薦 IFX 優先
    緩解定義 臨床緩解 擴展至 Angiographic remission（葡萄膜炎）、內視鏡緩解（GI）
    Interferon-alpha 有效選項 歐洲已無法取得，不再列入主流
    Apremilast 有 RCT 支持 明確列入 mucocutaneous 第二線

    預後

    • 病程復發緩解，診斷常延遲
    • 主要致病/致殘原因：眼部侵犯（失明）、神經侵犯
    • 死亡率：土耳其隊列最高達 9.8%（血管併發症 44%、CNS 12%）
    • 男性、年輕患者病情較重，預後較差
    • 近年因生物製劑廣泛應用，整體預後已顯著改善
