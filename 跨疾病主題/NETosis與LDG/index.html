    NETosis 與 LDG 重點整理

    參考：Kelly's Textbook of Rheumatology｜專科考試重點級別

    一、什麼是 NET？

    Neutrophil Extracellular Trap（嗜中性球胞外陷阱）

    嗜中性球死亡時，將自身的 DNA + 組蛋白 + 顆粒蛋白 噴射到細胞外，形成網狀結構。原本目的是捕捉並殺死病原體，但在自體免疫疾病中因清除不良而持續釋放 DAMP，放大 IFN-α 迴路。

    二、NETosis 的兩種類型

    類型 機制 速度 細胞命運
    Vital NETosis 細胞核噴出，細胞仍活著 快（分鐘） 細胞存活，仍可吞噬
    Suicidal NETosis ROS 依賴（NADPH oxidase）→ 核膜崩解 → 細胞死亡 慢（數小時） 細胞死亡

    SLE 主要和 Suicidal NETosis 有關

    三、NET 的成分（記這三樣）

    1. dsDNA / chromatin → 自體抗原，TLR9 的配體
    2. Citrullinated histone → RA 的自體抗原（anti-CCP 的來源）；由 PAD4 酶執行 citrullination
    3. LL37（cathelicidin）/ MPO / elastase → 放大發炎、活化 pDC

    四、NETosis 在各疾病的角色

    疾病 NETosis 的角色
    SLE NET DNA → TLR9/TLR7 → IFN-α → 自體免疫放大；anti-NET 抗體陽性
    RA Citrullinated histone → anti-CCP 抗體的來源
    ANCA 血管炎 NET 釋放 MPO/PR3 → ANCA 自體抗原暴露；內皮損傷
    血栓（COVID/APS） NET 活化血小板與凝血路徑 → 血栓形成（immunothrombosis）
    痛風 MSU crystals 誘發 NETosis → 關節炎放大

    五、LDG（Low Density Granulocytes）

    定義

    正常嗜中性球分離時沉在高密度層。LDG 密度異常低，分離時混在 PBMC 層，在 SLE 患者周邊血中數量顯著增加。

    LDG vs 正常嗜中性球

    特性 正常 Neutrophil LDG
    分離位置 高密度層 PBMC 層（低密度）
    成熟度 成熟 較不成熟（partially immature）
    NETosis 傾向 低 極高（spontaneous）
    IFN-α 產生 低 高
    細胞毒性 低 高（直接損傷內皮）
    活化狀態 靜止 持續活化

    LDG 為什麼在 SLE 大量出現？

    IFN-α 作用於骨髓 → 促進不成熟嗜中性球提早釋放入血 → 這些不成熟細胞即為 LDG。

    六、LDG 的三大病理作用

    1. 自發性 NETosis → DAMP 工廠

    • 不需要外部刺激，spontaneous NETosis
    • 持續釋放 dsDNA、citrullinated histone、LL37
    • LL37 與 DNA 結合 → 抵抗 DNase I 降解 → 持續進入 pDC endosome → TLR9 活化 → IFN-α

    2. 直接損傷血管內皮

    • 釋放 ROS + MPO + elastase → 直接造成內皮細胞死亡
    • 解釋 SLE 患者早發性動脈粥樣硬化與心血管風險升高

    3. 促進血栓形成（Immunothrombosis）

    • NET 提供磷脂支架 → 活化凝血路徑（Factor XII 接觸活化）
    • 活化血小板 → 釋放更多促發炎物質
    • 與 SLE 合併 APS（抗磷脂抗體症候群） 的血栓風險有關

    七、SLE 的 LDG / NETosis 惡性循環（核心）

    遺傳易感性 + 環境觸發（感染/UV）
            ↓
初期 Type I IFN 產生（pDC 被 DAMP 活化）
            ↓
IFN-α 作用於骨髓
            ↓
不成熟嗜中性球提早入血 → LDG 增加
            ↓
        ┌──────────────────────────────┐
        │  LDG 自發 NETosis            │
        │  ↓                           │
        │  NET: DNA + LL37 複合體      │
        │  ↓                           │
        │  抵抗 DNase I 降解            │
        │  ↓                           │
        │  pDC TLR9/TLR7 活化          │
        │  ↓                           │
        │  IFN-α 大量分泌 ──────────────┘（正回饋）
        └──────────────────────────────
            ↓（同時）
        內皮損傷 + 血栓傾向 + B細胞活化 → 自體抗體

    八、考試常見考點整理

    考點 答案
    NETosis 的關鍵酶 PAD4（citrullination of histone）、NADPH oxidase（ROS）
    LDG 增加的原因 IFN-α 驅動骨髓提早釋放不成熟嗜中性球
    LDG 最重要的特性 自發性 NETosis（spontaneous）
    SLE 中清除 NET 的障礙 DNase I 活性下降、C1q 缺乏、LL37 保護複合體
    LL37 的角色 與 DNA 結合 → 保護 NET 不被 DNase I 降解 → 持續活化 TLR9
    NET 在 RA 的意義 Citrullinated histone → anti-CCP 抗原來源
    NET 促血栓的機制 提供磷脂支架 + 活化 Factor XII（接觸活化）
    ANCA 血管炎的連結 NET 釋放 MPO/PR3 → 成為 ANCA 自體抗原
    LDG 與心血管風險 直接損傷血管內皮 → 早發性動脈粥樣硬化
    治療切入點 阻斷 IFN-α（Anifrolumab）可間接減少 LDG 產生

    九、一句話總結

    NETosis 是嗜中性球的「自爆式武器」；LDG 是 IFN-α 製造出來的「缺陷嗜中性球」，天生就容易自發 NETosis，又反過來產生更多 IFN-α，是 SLE 自體免疫惡性循環的核心驅動者。
