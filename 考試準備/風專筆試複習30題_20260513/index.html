    ✅ 解答 & 重點解析

    題號 答案 核心考點
    Q1 C Tfh → IL-21（非IL-17）；IL-17是Th17分泌
    Q2 C TNF-α 走TRAF/NFκB，不走JAK-STAT
    Q3 D Fever 2分 + Autoimmune hemolysis 4分 = 6分，需≥10分才符合標準
    Q4 B LN尿沉渣可以正常，不一定有血尿或管型
    Q5 A RVSP 60 mmHg = 重度PAH，是突發猝死最可能原因
    Q6 C BASFI評估功能，不是疾病活性；ASDAS/BASDAI才是活性評估
    Q7 B PsA + 抽菸（CV risk）→ 避免JAKi；IL-17i為適合選擇
    Q8 D SpA特徵是對NSAID有反應；無反應不是典型表現
    Q9 B Carbamylation：Lysine → Homocitrulline；ACPA：Arginine → Citrulline
    Q10 B MTX經腎排泄，Cr > 1.6 mg/dL需避免使用
    Q11 A Atlanto-axial subluxation：flexion時前間距 > 3 mm
    Q12 A Denosumab靶點是RANKL，不是RANK本身
    Q13 B Adalimumab（TNFi）實際上是治療uveitis，不是造成
    Q14 C Opioid 不是 FM的Level IA用藥；TCA/SNRI/Pregabalin才是
    Q15 A Anti-MDA5 → amyopathic DM，肌肉無力往往輕微或缺乏
    Q16 D DM主要浸潤細胞是CD4+（非CD8+），合併perifascicular atrophy
    Q17 A 2017 EULAR/ACR IIM標準中，age ≥40 years不是單獨計分項目
    Q18 B 雙側對稱發作 = primary RP的特徵；secondary RP常為不對稱
    Q19 D SSc-PAH篩查切點：DLCO < 55%（非<80%）
    Q20 A 正確：(1)GPA→cANCA ✓ (4)Takayasu→pulseless ✓ (6)EGPA→allergic ✓
    Q21 C MAINRITSAN：RTX維持治療優於AZA，C的說法相反故錯誤
    Q22 C CPPD：rhomboid shape，strong positive birefringence；與鈣代謝異常相關
    Q23 C Pegloticase使用前須篩查G6PD缺乏（可引發溶血）
    Q24 C 探頭與皮膚間需耦合凝膠；需雙側dorsal + palmar view
    Q25 C 超音波可偵測subclinical synovitis，指導減藥決策
    Q26 A GIOP第一線 = Bisphosphonate；高骨折風險→Teriparatide；Denosumab停藥需銜接BP；SERM不造成ONJ
    Q27 C FM有allodynia，輕觸也會引發疼痛（非「輕觸不會痛」）
    Q28 C 肌力訓練是OA唯一有強力evidence支持的非藥物治療
    Q29 B SjS唇腺切片主要是CD4+ T cells和B cells，不是CD8+
    Q30 B Neonatal lupus CHB發生率約2%（母親SSA/SSB(+)）；複發率18–20%

    陷阱 正確答案

    BASFI評估什麼？ 功能（非疾病活性）
    Denosumab靶點？ RANKL（非RANK）
    Adalimumab與uveitis關係？ 治療uveitis（非造成）
    DM肌肉切片主要細胞？ CD4+（非CD8+）
    Bilateral symmetric RP → ? Primary RP（非secondary）
    SSc-PAH篩查DLCO切點？ < 55%（非<80%）
    CPPD birefringence？ Strong positive（MSU為strong negative）
    Neonatal lupus CHB發生率？ 2%，複發率18–20%
