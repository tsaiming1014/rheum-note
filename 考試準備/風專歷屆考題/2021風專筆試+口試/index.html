    2021 風專筆試＋口試 考題整理

    筆試

    1. T cell subset cytokines 配對，選錯誤

    • Th1 → TNF-α（部分正確，但主要分泌 IFN-γ）
    • Treg → IL-21（誤，IL-21 主要由 Tfh 分泌；Treg 主要分泌 TGF-β、IL-10）✓ 答案

    2. miRNA 可抑制 mRNA translation

    正確 ✓ — miRNA 與 RISC 複合體結合，抑制 target mRNA 的翻譯或促進其降解

    3. T cell activation，何者錯誤？

    • TCR 活化後透過 calmodulin 活化 NF-κB（誤，TCR 活化後透過 calcineurin 活化 NFAT，NF-κB 由其他路徑活化）✓ 答案

    4. RA animal model 驗證新藥效果的是哪個？

    • CIA（Collagen-Induced Arthritis） ✓ — 最常用於驗證新藥的 RA 動物模型

    5. 關節滑膜 hyaluronan 的主要分泌細胞？

    • Type B synovial lining cell（fibroblast-like synoviocyte, FLS） ✓

    6. 參與軟骨 matrix 分解的主要 cytokine？

    • IL-1β 和 TNF-α 是最主要的（促進 MMP 分泌，降解 type II collagen 和 proteoglycans）
    • IL-17 也有參與

    7. 何者可以抑制 osteoclasts 產生？

    • Osteoprotegerin（OPG） ✓ — OPG 是 RANKL 的 decoy receptor，競爭性抑制 RANKL 與 RANK 結合，從而抑制 osteoclast 分化

    8. SjS 流行病學，何者錯誤？

    • 原發性乾燥症盛行率高達 10–15%（誤，盛行率約 0.01–0.1%，並非 10–15%）✓ 答案

    9. IIM 與 cancer 關聯性，何者錯誤？

    • 男性 SLE 患者得到攝護腺癌的風險較高（誤，SLE 與 IIM 為不同疾病；IIM 相關 cancer 主要為卵巢癌、肺癌、大腸癌）✓ 答案
    • 某選項：IIM 得到 cancer 的危險因子為 old age、ILD、periungual erythema ✓（正確）

    10. IgG4-RD，何者錯誤？

    • 可以依賴 IgG4 level 追蹤疾病活動度與復發（誤，血清 IgG4 level 與疾病活動度相關性不佳，不應作為唯一追蹤指標）✓ 答案

    11. SLE 及抗病毒為下列哪個 cytokines？

    • Type I IFN（IFN-α/β） ✓ — SLE 患者 IFN-α 持續升高，參與自體免疫致病機轉，同時具抗病毒功能
    • 選項：IL-1、IL-6、IFN-α、IL-33

    12. COVID-19 thrombosis 可測何種抗體？

    • Anti-PF4（platelet factor 4）抗體 ✓
    • VITT（Vaccine-Induced Immune Thrombocytopenia and Thrombosis）相關
    • COVID-19 本身也可誘發 aPL，但通常一過性

    13. 下列何者與 thrombosis 最相關？

    • D-dimer ✓（急性血栓的敏感性標記，陰性預測值高）

    14. SLE 何種抗體有光敏感性（photosensitivity）？

    • Anti-Ro（SSA） ✓ — Anti-Ro 陽性與 photosensitivity、SCLE、neonatal lupus 相關

    15. Autoinflammatory syndrome

    疾病 基因 主要表現
    FMF MEFV（Pyrin） 週期性發燒、漿膜炎、關節炎

    16. PM 病人有 ILD，與何種抗體有關？

    • Anti-synthetase（最常見 anti-Jo-1） ✓
    • Anti-MDA5 ✓（常與 rapid progressive ILD 相關）
    • Anti-dsDNA（非 IIM 相關，是 SLE 的抗體）✗
