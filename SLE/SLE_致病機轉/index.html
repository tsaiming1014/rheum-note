    SLE 致病機轉整理（Kelly Ch80-81；補體部分另參 Ch26）

    總覽：一句話看懂 SLE 的故事

    帶有易感基因的人，在環境誘因下，凋亡細胞碎片與循環免疫複合體「做得多、清得少」，核酸廢棄物持續堆積在細胞外 → 這些廢棄物本身就是強力的 DAMP + 自體抗原，驅動先天免疫（TLR/cGAS-STING → IFN-I）大量活化 → 打破 T/B 細胞的周邊耐受性 → 分化出自體抗體分泌的漿細胞 → 抗體與抗原形成免疫複合體，沉積在組織中並活化補體 → 造成腎、皮膚、CNS、血管等器官的發炎與破壞。

    這條故事線有一個容易忽略的樞紐：第四節。 前面（遺傳、性別、環境）都只是「上游條件」，後面（先天/適應性免疫、自體抗體、器官損傷）都是「下游放大與執行」；真正把兩端接起來的，是「凋亡碎片與免疫複合體的產生 vs. 清除失衡」這一步——這也是本次重寫最主要補強的部分。

    一、遺傳易感性

    1-1 遺傳負荷概況

    比較對象 相對風險 (RR)
    同卵雙胞胎 316×
    手足 24×
    一般人 1×

    1-2 重要致病基因分類（依機轉分類，對應下文章節）

    機轉類別 代表基因 說明
    凋亡碎片清除障礙 C1q, C2, C4 Classical pathway 補體缺乏 → opsonization / 凋亡碎片清除↓（詳見四-2）
    循環免疫複合體清除 / FcR 功能 FCGR2A, FCGR2B, FCGR3A, ITGAM FCGR2A/3A 為活化型受器；FCGR2B 為抑制型，SLE 患者其在 marginal zone B cell 表現↓，B 細胞活化閾值降低；ITGAM(CD11b) 為 CR3 成分，介導補體調理後的吞噬（詳見四-3）
    自噬 / DNA 修復 ATG5, RAD51B, DRAM1 影響自噬相關抗原處理與自體核酸清除
    TLR 路徑 / IFN-I 誘導 TLR7, IRF5, IRF7, IRAK1, TNIP1 核酸感測 → IFN-α 過量
    細胞質核酸感測 IFIH1 (MDA5), DDX58 (RIG-I), IKBKE, IRF8 胞質 dsRNA/DNA 感測 → MAVS/STING 活化
    IFN-I 反應 STAT4, TYK2, SOCS1 IFN 訊號下游
    T/B 細胞活化閾值 PTPN22, BLK, BANK1, LYN, TNFAIP3 降低淋巴球活化門檻
    MHC HLA-DRB1*03:01（歐洲），HLA-DRB1*15:03（非裔） 抗原呈現
    器官損傷易感 APOL1（腎臟，非裔）, HAS2, PDGFRA 目標器官脆弱性

    1-3 單基因（Monogenic）SLE 關鍵概念

    • C1q 缺乏 → 單一最強的 SLE 風險因子：同時造成（a）凋亡碎片清除障礙（waste disposal hypothesis）、（b）減少對 pDC 的 IFN-α 抑制（C1q 正常會把刺激性免疫複合體導向 monocyte 而非 pDC）→ 雙重放大發炎
    • Interferonopathy 家族：TREX1, SAMHD1, RNASEH2A/B/C, ADAR1, STING 突變 → 胞質核酸持續堆積 → 持續 IFN-I 產生（Aicardi–Goutières 症候群為原型疾病，好發於幼兒，機轉與 SLE 有重疊）
    • DNASE1、DNASE1L3、DNASE2 突變：分別影響胞外 DNA/NETs、微粒子（microparticle）DNA、凋亡細胞內殘留 DNA 的降解，家族型突變可見於侵襲性 lupus 合併 anti-DNA 抗體或低補體性蕁麻疹性血管炎症候群
    • FAS 突變：影響淋巴球凋亡調控（詳見四-1）

    1-4 表觀遺傳學（Epigenetics）

    • T cell DNA 低甲基化：Lupus T cell 的 type I IFN-stimulated genes 調控區域甲基化程度下降 → 這些基因被「解除封印」而過度表現；同卵雙胞胎研究中，發病一方甲基化程度明顯較低，支持表觀遺傳（而非單純基因序列）的角色
    • 藥物誘發性 lupus 的機轉即是表觀遺傳：Procainamide、Hydralazine 抑制 ERK 路徑訊號 → DNMT1、DNMT3a（DNA 甲基轉移酶）表現下降 → 誘發性低甲基化，機轉與自發性 SLE 的 T cell 低甲基化相同
    • 低甲基化的下游效應：除了活化 IFN-stimulated genes，也會增加內生性 retroelement 的表現，這些核酸本身可再活化核酸感測路徑（與四、五節相連）
    • X 染色體劑量效應：詳見第二節

    二、女性好發原因（9:1）

    女性好發 SLE ← 多重機轉共同作用

1. 荷爾蒙：estrogen 促進 pDC 活化；prolactin 上升
2. X 染色體劑量效應
   - TLR7 位於 X 染色體，部分逃脫 X 染色體不活化（XCI escape）
   - 女性可雙對偶基因表現 TLR7 → 更強的 IFN-I 反應
   - Klinefelter (47,XXY) 男性：SLE 風險升高 14 倍
3. X 染色體相關基因：TLR7、TASL、IRAK1 均位於 X 染色體
4. 排卵相關的 DNA 去甲基化 → retroelement 表現 → 核酸免疫刺激（與一-4 呼應）

    三、環境觸發因子

    觸發因子 代表 致病機轉
    紫外線（UV-B） 日曬 DNA 斷裂、角質細胞凋亡↑、自體抗原（Ro/La/Sm）移位到凋亡小體表面；詳見 SLE 光敏感性筆記
    病毒 EBV EBERs 誘導 IFN-I；EBNA2 活化 SLE 相關基因位點；EBNA-1 與 Sm 抗原序列相似（分子擬態）

    四、核心樞紐——廢棄物「做得多、清得少」

    這一節回答的問題是：為什麼會有這麼多自體核酸/細胞碎片，可以持續活化 TLR7/9、cGAS-STING？ 答案不是單一事件，而是「產生增加」與「清除下降」同時發生、互相加乘，而且清除障礙又分成兩條平行但不同的路徑——凋亡碎片清除、與循環免疫複合體清除——臨床上很容易被當成同一件事，但機轉並不相同。

    遺傳易感性（C1q/C2/C4、DNASE家族、FCGR、FAS…）＋ 環境觸發（UV、EBV、氧化壓力、藥物）

    4-1 廢棄物產生增加（Increased generation）

    機轉 說明
    UV-B 誘發凋亡 角質細胞、皮下淋巴球 DNA 斷裂 → 凋亡增加；UV 同時使 Ro/La/Sm 等抗原由細胞核移位到凋亡小體表面，讓免疫系統直接暴露到平常藏起來的自體抗原（詳見 SLE 光敏感性筆記）
    氧化壓力（oxidative stress） 粒線體受損 → 釋出 oxidized mitochondrial DNA，本身即具高度免疫刺激性，可活化 cGAS-STING 與 RIG-I/MDA5-MAVS 路徑
    NETosis 中性球（尤其低密度顆粒球 LDG）主動將核酸＋組蛋白＋HMGB1＋LL37＋elastase 擠出細胞外；非典型凋亡，但同樣造成廢棄物大量增加（詳見第五節）
    FAS/FasL 路徑異常 部分單基因 SLE 患者帶有 FAS 突變 → 淋巴球凋亡調控失常
    EBV 感染細胞凋亡 病毒誘發宿主細胞凋亡，釋出 EBER 等免疫刺激性 RNA

    4-2 凋亡碎片清除障礙（Impaired clearance of apoptotic debris）

    機轉 說明
    補體缺乏（C1q > C4 > C2） C1q 為 classical pathway 辨識蛋白，可直接結合凋亡/壞死細胞並促進吞噬（C1q receptor），也活化 classical pathway 進一步 opsonize；缺乏早期補體成分時，凋亡細胞無法被有效清除（waste disposal hypothesis）
    DNase 活性不足 DNase1、DNase1L3 分解胞外 DNA（含 NETs、循環微粒子 DNA）；DNase2 分解凋亡細胞被吞噬後、胞內殘留的 DNA；基因缺陷或活性不足 → DNA 持續存在並刺激免疫
    CRP（pentraxin 家族）功能異常 正常協助凋亡碎片清除；CRP 多型性與 SLE 相關
    巨噬細胞吞噬（efferocytosis）功能異常 吞噬凋亡細胞的能力下降 → 凋亡小體（apoptotic blebs）、microparticles 持續存在於細胞外環境

    4-3 循環免疫複合體清除障礙（Impaired clearance of circulating immune complexes）

    這是與 4-2 平行、但機轉不同的另一條清除路徑——針對已經形成的免疫複合體（而非單純凋亡碎片）：

    • 紅血球 CR1（CD35）介導的 immune adherence clearance：C3b/C4b 調理過的免疫複合體會經由 CR1 黏附在紅血球表面（immune adherence），由紅血球運送至肝、脾的巨噬細胞樣細胞清除；這些巨噬細胞不會分解免疫複合體本身，而是切斷 CR1，使複合體脫離紅血球後被清除——因此活動性 lupus 病人的紅血球表面 CR1 表現量會下降，是這條清除路徑被大量消耗、失能的指標
    • 補體的兩種 IC 清除機制：(1) Classical pathway＋CR1 運送，預防免疫複合體沉積於組織；(2) Alternative pathway 主導的溶解（solubilization），把已經形成的複合體再打散，避免持續致病
    • FcγR 失衡：活化型 FcγR（FCGR2A、FCGR3A）促進吞噬細胞內化並清除免疫複合體，但同時也會活化發炎細胞；抑制型 FCGR2B 在 SLE 患者的 marginal zone B cell 表現量下降，削弱了對 B 細胞活化的煞車，兩者失衡使 IC 清除與發炎反應的平衡被打破

    4-4 補體的雙重角色——一體兩面，容易考混

    階段 角色 對應機轉
    上游（清除功能） 早期 classical pathway（C1q、C4、C2）缺乏 = SLE 最強風險因子之一 凋亡碎片清除↓（4-2）＋ IC 清除/溶解↓（4-3）
    下游（效應器功能） 免疫複合體沉積組織後，活化補體 C3a/C5a 過敏毒素、招募發炎細胞、MAC（C5b-9）造成細胞膜破壞（詳見第八、九節）

    記憶重點：同一個補體系統，早期成分「缺乏」是風險因子；一旦免疫複合體形成並沉積，補體「活化」反而是傷害組織的效應器。 這也是為什麼補體低下（消耗性低補體血症）在 SLE 活動期反而常見——不是先天缺乏，而是被免疫複合體大量消耗。

    4-5 小結：故事線的轉折點

    • 4-1～4-3 的最終結果，都是讓自體核酸、凋亡小體、免疫複合體持續存在於細胞外環境，而不是被及時清除
    • 這些堆積物同時具備兩種身分：DAMP（活化先天免疫）＋ 自體抗原（Ro、Sm、DNA、histone，將驅動第六、七節的適應性免疫與自體抗體）
    • 遺傳因素（第一節）與環境因素（第三節）在此匯流成同一個下游結果——這正是連接「遺傳＋環境」與「先天免疫活化」之間的關鍵橋梁

    核心產生細胞

    細胞 角色
    pDC（漿細胞樣 DC） 最主要的 IFN-α 產生者；TLR7/9 活化；FcγRIIa 表現使其能攝入免疫複合體並活化

    • 提供自體抗原（Sm、DNA、histones）
    • 促進血管損傷 + 血栓形成
    • SLE 患者的 DNase I 無法有效清除 NETs（呼應第四節），形成惡性循環

    六、適應性免疫失調——周邊耐受性崩潰

    6-0 耐受性框架（先建立概念再看細節）

    正常免疫系統靠兩層機制避免攻擊自身：

    • 中央耐受性（central tolerance）：胸腺（T cell）、骨髓（B cell）中，對自身抗原有高親和力的淋巴球被 clonal deletion 或誘導 anergy
    • 周邊耐受性（peripheral tolerance）：逃過中央篩選的自體反應性淋巴球，靠 anergy、Treg 主動抑制等機制被壓制

    SLE 的核心問題主要發生在周邊耐受性：第四、五節持續存在的自體抗原＋IFN-I 訊號，提供了打破 anergy、活化 Treg 抑制之外的自體反應性 T/B 細胞的條件，讓原本該被壓制的 clone 被活化並擴增。

    T cell 異常

    異常 說明
    IL-2 分泌↓ → Treg 生成↓ → 周邊耐受性維持能力下降
    CD40L (CD154) 持續高表現 → 提供 B cell 過度 help
    TCR ζ chain → Fc receptor γ chain → TCR 訊號改變，活化閾值降低
    mTOR 活化、粒線體超極化 → T 細胞代謝異常
    Tfh / Tph 細胞↑ → 驅動 B cell 分化（Tph 表現高 PD-1，在 SLE 早期出現）
    Th17↑ / Treg↓ → 促發炎，破壞周邊耐受性平衡
    DNA 低甲基化 → IFN-stimulated gene 的 epigenetic 去抑制（見一-4）

    B cell 異常

    類別 說明
    Age-associated B cells (ABCs) T-bet+/IRF5+ B cell，TLR7 訊號依賴；可分化為 plasmablast 或 plasma cell
    Plasmablast（短效） 分泌 anti-dsDNA Ab；隨疾病活性波動；對 belimumab 有反應
    Long-lived plasma cells（長效） 分泌 anti-Sm、anti-Ro；骨髓利基保護；對一般免疫抑制劑抵抗；CAR-T 可能有效
    驅動因子 BLyS↑（belimumab 目標）、IL-10↑、IL-21↑
    活化異常 LYN↓、FCGR2B（抑制性 FcR）表現↓，尤其 marginal zone B cell（見四-3）→ 活化閾值降低
    耐受機制缺損 PTPN22 變異 → 自體反應 B cell 無法被反選擇
    雙受器活化模型 含核酸的自體抗原可同時engage BCR（辨識）與 endosomal TLR7/9（第二訊號）→ B 細胞不需 T cell help 即可完全活化 → 分化為漿細胞

    自體抗體的致病角色

  ├── 內皮下（Subendothelial）→  Class III / IV  ← 發炎最強
  └── 上皮下（Subepithelial） →  Class V（膜性）
                    ↓
【Step 3】IC 啟動補體系統（Classical Pathway）

  C1q → C4 → C2 → C3 convertase
              → C3a（過敏毒素、肥大細胞活化）
              → C5a（趨化因子：招募中性球 + 單核球）
              → MAC（C5b-9）：細胞膜穿孔
                    ↓
【Step 4】發炎細胞浸潤腎臟

  ├── 巡邏型單核球（Patrolling monocytes）
  │     ← TLR 活化後遊走至腎臟
  │     → 釋放 BLyS、IL-12、TNF
  │     → 加重發炎、促進硬化（sclerosis）
  ├── T cell（間質浸潤）→ 直接傷害腎小管
  ├── B cell（間質浸潤）→ 局部 IC 形成
  └── 中性球 → 釋放酶 + ROS → 組織破壞
                    ↓
【Step 5】Podocyte 損傷 → 蛋白尿
  Anti-dsDNA、Anti-Ro/La 等 Ab 直接作用
  IC 沉積 → 足細胞 slit diaphragm 破壞
                    ↓
【Step 6】慢性化：纖維化 + 硬化
  急性炎症 → 腎小管轉錄組的彈性決定是否進展
  TGF-β、PDGF → 間質纖維化
  →→→ Class VI（>90% 球性硬化）

    9-4. ISN/RPS 組織分類（重要！）

    Class 名稱 組織特徵 IC 位置 臨床表現 預後
    I Minimal Mesangial LM 正常 系膜（IF/EM） 無症狀 極佳
    II Mesangial Proliferative 系膜細胞↑ 系膜 輕微血尿/蛋白尿 佳
    III Focal（<50% 腎絲球） 內皮下增生；局灶 內皮下 血尿 + 蛋白尿 中
    IV Diffuse（≥50% 腎絲球） 彌漫增生；wire loop；新月體 內皮下（廣泛） 腎炎症候群 + 腎功能惡化 最差
    V Membranous 上皮下 IC；毛細管壁增厚 上皮下 腎病症候群（大量蛋白尿） 中
    VI Advanced Sclerotic >90% 全球性硬化 — 腎功能衰竭 最差（ESKD）

    Class IV-S vs IV-G：

    • IV-S = 各腎絲球中 <50% 小葉受累（segmental）
    • IV-G = 各腎絲球中 ≥50% 小葉受累（global）

    在 lupus nephritis 的免疫螢光中同時看到：

  IgG ✓
  IgM ✓   ← 全部陽性 = "Full House"
  IgA ✓
  C3  ✓
  C1q ✓

沿腎絲球基底膜（GBM）、系膜、腎小管基底膜呈顆粒狀
→ 高度提示 lupus nephritis（vs 其他 GN）

    電子顯微鏡可見：Tubuloreticular inclusions（內皮細胞內，與 IFN-α 高度相關）

    9-6. 與 Anti-DNA 抗體的特殊關係

    • anti-dsDNA 抗體從 lupus 腎臟中被洗脫（eluted）→ 直接致病證據
    • 部分 anti-dsDNA Ab 也與 NMDA receptor 交叉反應（→ 亦可作用於 CNS）
    • IC 中含核酸 → 進入系膜細胞 endosome → 活化 TLR9 → 腎局部 IFN-I 產生 → 放大腎臟炎症

    9-7. 腎小管間質病變（Tubulointerstitial Disease）

    • 見於 66% 的 LN 切片
    • 特徵：炎性浸潤 + 腎小管損傷 + 間質纖維化
    • 腎小管轉錄組（renal tubule transcriptome）健全度 → 決定急性 vs 慢性 nephritis 走向
    • 是預測長期腎臟預後最強的組織學指標之一

    9-8. 腎臟預後不良的因子

    因素 說明
    種族 非裔、拉丁裔 > 亞裔、白人
    APOL1 基因型 非裔相關，ESKD 風險↑
    TMA 合併存在 蛋白尿更高、腎功能更差
    間質纖維化 強烈預測腎功能喪失
    Class 轉換 15~50% 患者發生；自發或治療後
    持續低度蛋白尿 仍可能有活動性炎症（切片往往不一致）

    其他器官

    器官 機轉 特色
    皮膚 UV → DNA 斷裂 + 角質細胞凋亡 → 自體抗原暴露；角質細胞產生 IFN-κ → 招募炎症細胞；IC 沉積於真皮表皮交界（lupus band test） 皮膚型 lupus 中 anti-Ro 最相關
